# 检验过程质量控制专项内审检查表（条款顺序展开版）

**文件编号**：QUA-REC-2026-002A
**对应标准**：ISO 15189:2022 第 7.3 条；CNAS-CL02:2023 第 7.3 条
**版本**：第 A 版第 0 次修订
**生效日期**：2026-03-15
**编制人**：质量负责人
**审核人**：技术负责人
**批准人**：实验室主任

---

## 1 目的

1.1 依据 ISO 15189:2022 第 7.3 条（检验过程）及 CNAS-CL02:2023 第 7.3 条，对检验过程质量控制模块实施专项内部审核。
1.2 本表按条款编号顺序逐条展开 7.3 条全部子条款（7.3.1—7.3.8），为内审员提供可执行审核轨迹。

## 2 适用范围

2.1 适用于生化、免疫、血液、体液、微生物、分子、形态、输血相容性 8 个专业组的全部检测项目。
2.2 覆盖方法选择 → 方法验证 → 方法确认 → 测量不确定度 → 生物参考区间 → 文件化程序 → 质量保证（IQC、EQA、比对）全流程。

## 3 使用说明

3.1 抽样比例：每专业组检测项目抽样 ≥30%；定量项目测量不确定度、生物参考区间评审 100% 查验。
3.2 审核方法按"查阅文件 + 记录追溯 + 现场观察 + 访谈 + 质控数据抽取"5 种方式组合。
3.3 判定等级：符合 / 一般不符合 / 严重不符合 / 观察项 / 不适用。

## 12 符合性判定说明

12.1 符合：条款已文件化、已实施、有客观证据。
12.2 一般不符合：单点失效；30 日内整改。
12.3 严重不符合：多点失效或影响结果准确性；60 日内整改并验证。
12.4 观察项：改进建议；下次管理评审前处置。
12.5 不适用：需附书面说明。
