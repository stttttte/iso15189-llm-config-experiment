## 1 总体概述

### 1.1 不符合汇总

| 编号 | 等级 | 条款 | 核心现象 |

## 2 NC-2026-001 整改方案（文件控制失效）

### 2.1 基本信息

2.1.1 条款：ISO 15189:2022 第 4.2.4 条；CNAS-CL02:2023 对应条款。
2.1.2 等级：严重不符合。

### 2.2 根本原因

| 维度 | 具体原因 |
|------|----------|
| 机制 | 纸质文件分发无回收流程；DOC-002 对纸质版本管理规定不明确 |
| 监督 | 6 个月未开展文件现场核查；无版本审计清单 |
| 人员 | 对文件版本控制重要性意识不足 |

### 2.3 纠正措施

#### 2.3.1 措施 1.1：即时清理工作岗位

| 项目 | 内容 |
|------|------|
| 动作 | 逐岗检查拍照；收回过期/作废文件；销毁 |
| 负责人 | 科室主任 + 质量负责人 |
| 完成日 | 2026-02-05 |
| 输出 | 现场清理报告 + 照片证据 |

#### 2.3.2 措施 1.2：修订 DOC-002

| 项目 | 内容 |
|------|------|
| 新增内容 | 纸质分发记录、文件回收流程、月度版本确认表 |
| 负责人 | 质量负责人 |
| 完成日 | 2026-02-28 |
| 输出 | DOC-002 v1.6 |

#### 2.3.3 措施 1.3：月度文件现场核查

| 项目 | 内容 |
|------|------|
| 频率 | 每月 1 次；抽样 ≥50% |
| 核查内容 | 版本号、生效日期、过期文件移除 |
| 启动日 | 2026-02-01（持续） |
| 负责人 | 质量管理部门 |
| 输出 | 月度核查记录 |

#### 2.3.4 措施 1.4：全员培训

| 项目 | 内容 |
|------|------|
| 内容 | 版本重要性、文件控制程序、旧版本识别、风险 |
| 形式 | 集中 1.5 小时 + 部门讨论 1 小时 |
| 考核 | 笔试或口头提问；正确率 ≥80% |
| 完成日 | 2026-02-28 |
| 负责人 | 质量负责人 + 科室主任 |

### 2.4 验证方案

#### 2.4.1 验证标准

| 项目 | 标准 |
|------|------|
| 版本一致 | 100% 一致；0 过期 |
| 分发回收 | 记录完整可追溯 |
| 人员理解 | 5 人 3 问正确率 ≥80% |
| 月度核查 | 每月有记录 |
| 核查问题数 | ≤2 个/月 |

#### 2.4.2 验证节点

| 节点 | 日期 |

#### 2.4.3 证据清单

## 3 NC-2026-002 整改方案（设备维护记录不完整）

### 3.1 基本信息

3.1.1 条款：ISO 15189:2022 第 6.4.4 条；CNAS-CL02:2023 对应条款。
3.1.2 等级：一般不符合。

### 3.2 根本原因

| 维度 | 具体原因 |
|------|----------|
| 程序 | EQP-004 未要求维护后验证；表单无验证栏 |
| 培训 | 外包维护人员对记录完整性理解不足 |
| 管理 | 复核签名形式化 |

### 3.3 纠正措施

#### 3.3.1 措施 2.1：修订 EQP-004 与表单

| 项目 | 内容 |
|------|------|
| 新增要求 | 维护后必须进行功能检验或性能验证；故障维修须诊断测试；预防性维护须 IQC |
| 表单新增 | "维护后验证"栏（方法、结果、签名） |
| 完成日 | 2026-02-28 |
| 负责人 | 设备管理部门 + 质量部 |

#### 3.3.2 措施 2.2：二级审核制度

| 项目 | 内容 |
|------|------|
| 流程 | 维护人员填写→技术主管或质量人员复核 |
| 追踪 | 维护记录复核清单（月度） |
| 启动日 | 2026-02-01 |
| 负责人 | 设备管理部门主任 |

#### 3.3.3 措施 2.3：补充 2025 年缺陷记录

| 项目 | 内容 |
|------|------|
| 补充对象 | 8 条缺验证结果 + 3 条缺复核签名 |
| 补充依据 | 原维护单据、设备运行日志 |
| 签名要求 | 由当时维护人员或技术主管补签（注明补记日期） |
| 完成日 | 2026-03-15 |
| 负责人 | 设备管理部门 + 技术团队 |

#### 3.3.4 措施 2.4：外包维护管控

| 项目 | 内容 |
|------|------|
| 动作 | 书面通知外包单位；建立外包验收清单；下次维护现场培训 |
| 完成日 | 2026-02-15 |
| 负责人 | 设备管理部门 |

### 3.4 验证方案

#### 3.4.1 验证标准

## 4 NC-2026-003 整改方案（人员能力评估缺失）

### 4.1 基本信息

4.1.1 条款：ISO 15189:2022 第 6.2 条；CNAS-CL02:2023 对应条款。
4.1.2 等级：一般不符合。

### 4.2 根本原因

| 维度 | 具体原因 |
|------|----------|
| 程序 | PER-002 对评估时限与质量监督不足 |
| 意识 | 对高学历员工放松要求 |
| 机制 | 缺乏"培训-评估-监控"三阶段跟踪 |

### 4.3 纠正措施

#### 4.3.1 措施 3.1：立即评估员工 D

| 项目 | 内容 |
|------|------|
| 内容 | 理论 + 实践 + 问题处理 |
| 评估表 | 分项评分；总体结论；不合格需给出后续培养计划 |
| 完成日 | 2026-02-10 |
| 负责人 | 科室主任 + 技术负责人 |

#### 4.3.2 措施 3.2：修订 PER-002 与表单

| 项目 | 内容 |
|------|------|
| 时限 | 培训完成后 ≤2 周进行评估 |
| 标准 | 理论 ≥80%；实践独立完成并质量合格 |
| 表单 | 分项评分（样品准备、仪器操作、质量控制、报告审核）；总体结论 |
| 监控 | 人事部建立能力评估监控清单 |
| 完成日 | 2026-02-28 |
| 负责人 | 人事部 + 质量部门 |

#### 4.3.3 措施 3.3：补充员工 E 评估

| 项目 | 内容 |
|------|------|
| 动作 | 深入评估形成详细记录 |
| 完成日 | 2026-02-15 |
| 负责人 | 科室主任 |

#### 4.3.4 措施 3.4：新员工成长档案

| 项目 | 内容 |
|------|------|
| 建档 | 每一新员工独立档案 |
| 记录 | 培训、进度、评估、授权等关键节点 |
| 规则 | 未通过评估禁止独立操作 |
| 完成日 | 2026-02-28 |
| 负责人 | 科室主任 + 人事部 |

#### 4.3.5 措施 3.5：杜绝程序简化

| 项目 | 内容 |
|------|------|
| 原则 | 所有新员工无论学历完整执行流程 |
| 落实 | 管理会议强调；内审重点检查 |
| 启动日 | 2026-02-01 |
| 负责人 | 科室主任 + 质量部门 |

### 4.4 验证方案

#### 4.4.1 验证标准

| 项目 | 标准 |
|------|------|
| 员工 D 评估 | 评估表完整；有评分与结论 |
