# ISO 15189:2022 检验科文件体系生成指南

## 本套模板的用途

本套 MD 模板文件用于指导大语言模型 (LLM) 为医学实验室（检验科）生成符合 **ISO 15189:2022** 要求的质量管理体系文件。

**定位**：将"从零编写"变为"审核修改"，节省 70-80% 编写时间。

**输出物**：质量手册 + 程序文件(SOP) + 记录表单/作业指导书（三层文件结构）。

---

## 推荐使用的 LLM 模型

由于单个模板文件约 8K-12K tokens，请使用以下级别的模型：

| 推荐级别 | 模型 | 说明 |
|---------|------|------|
| 最佳 | Claude Opus / Claude Sonnet | 长上下文处理能力强，中文质量高 |
| 推荐 | GPT-4 / GPT-4o | 指令遵循度好 |
| 可用 | 其他 128K+ 上下文模型 | 需测试中文输出质量 |
| 不推荐 | GPT-3.5 / 4K 上下文模型 | 上下文不足，生成质量差 |

---

## 使用流程

```
步骤1：填写下方「检验科基本信息」
  ↓
步骤2：将本文件 + 01-质量手册-框架.md 一起发给 LLM
  ↓  LLM 生成质量手册初稿
步骤3：逐个发送 02~08 程序文件模板给 LLM
  ↓  每次附上本文件的「基本信息」部分作为上下文
  ↓  LLM 逐模块生成程序文件
步骤4：发送 09-记录表单清单.md 给 LLM
  ↓  LLM 生成配套表单
步骤5：发送 10-POCT专项.md（如有 POCT）
  ↓
步骤6：人工审核 → 签批发布 → 实施运行
```

**重要提示**：
- 每次发给 LLM 时，请把下方「基本信息」一并粘贴，确保 LLM 知道你的实验室具体情况
- 生成结果中所有 `【】` 包裹的内容为必须替换的占位符

| 批准 | 【姓名/岗位】 |

## 1 目的

建立并维护检验科人员管理制度，规范人员档案管理、入职/离职/调岗流程、岗位说明书管理、关键岗位代理人指定，确保所有人员的任职资格、能力、授权等信息完整、准确、可追溯，满足 ISO 15189:2022 第 6.2.1 条和第 6.2.5 条要求。

## 2 适用范围

适用于检验科所有在岗人员（含正式员工、合同制人员、进修生、实习生、外聘专家），涵盖人员全生命周期管理。

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求摘录

> **§6.2.1 总则**
> "实验室应有文件化程序，确保管理所有对检验前、检验和检验后过程有影响的人员。"

> **§6.2.5 人事记录**
> "实验室应保存每位人员的记录，包括但不限于：
> a) 能力要求；
> b) 相关的教育和专业资格（如适用，包括副本）；
> c) 以前工作经历的副本和/或参考文献；
> d) 职位描述；
> e) 培训和再培训记录；
> f) 对能力的评估；
> g) 继续教育和成就的记录；
> h) 不良事件/事故报告；
> i) 工作表现评价；
> j) 授权/许可文件；
> k) 与健康状况有关的信息（如适用）。"

### 3.2 其他适用标准/法规

- 《医疗机构临床实验室管理办法》
- 《临床实验室生物安全管理规范》
- 【其他适用的地方法规或行业标准】

| 人事专员/档案管理员 | 建立并维护人员档案，执行入职/离职/调岗手续，更新人员信息 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 人员档案 | 记录个人基本信息、学历、职称、执业资格、培训、评估、授权等全部人事信息的集合 |
| 岗位说明书 | 明确某一岗位的职责、权限、资质要求、工作范围的正式文件 |
| 关键岗位 | 对检验结果质量具有直接重大影响的岗位，如技术负责人、质量负责人、授权签字人等 |
| 代理人 | 经授权在关键岗位人员缺席时代行其职责的人员 |
| 人员360度视图 | 将人员的培训记录、能力评估、授权状态、协议签署、健康记录等信息汇聚展示的综合视图 |
| 颜色视觉检查 | 对需要进行颜色辨别工作的人员（如形态学检查）进行的色觉能力测试 |

## 6 程序内容

### 6.1 人员档案管理

#### 6.1.1 档案内容

每位人员的档案应至少包含以下内容：

| 序号 | 档案内容 | 要求 | 对应条款 |
|------|----------|------|----------|
| 1 | 基本信息 | 姓名、性别、出生日期、身份证号、联系方式、照片 | §6.2.5 |
| 2 | 学历信息 | 最高学历、学位、毕业院校、专业、毕业时间（附学历证书扫描件） | §6.2.5 b) |
| 3 | 职称信息 | 当前职称、取得时间、证书编号（附职称证书扫描件） | §6.2.5 b) |
| 4 | 执业资格 | 执业证书类型、编号、有效期、注册范围 | §6.2.5 b), j) |
| 5 | 工作经历 | 以前工作单位、岗位、起止时间 | §6.2.5 c) |
| 6 | 签名与印章 | 手写签名留样、电子签名授权 | §6.2.5 |
| 7 | 颜色视觉检查 | 色觉检查结果，适用于形态学检验岗位人员 | §6.2.1 |
| 8 | 健康状况 | 年度体检报告、传染病筛查结果、职业暴露记录 | §6.2.5 k) |
| 9 | 培训记录 | 见 JYK-SOP-06-003 | §6.2.5 e) |
| 10 | 能力评估记录 | 见 JYK-SOP-06-002 | §6.2.5 f) |
| 11 | 授权记录 | 见 JYK-SOP-06-004 | §6.2.5 j) |
| 12 | 不良事件记录 | 差错/事故报告及处理记录 | §6.2.5 h) |
| 13 | 工作表现评价 | 年度绩效考核结果 | §6.2.5 i) |
| 14 | 继续教育记录 | 学分、进修、学术活动记录 | §6.2.5 g) |

#### 6.1.2 档案建立与更新

1. 新员工入职 **3 个工作日内** 完成基本档案建立。
2. 人员资质信息变更（职称晋升、证书续注册等）应在 **变更后 5 个工作日内** 更新档案。
3. 每年 **【12月】** 进行一次档案完整性核查，确保所有人员档案齐全。

#### 6.1.3 档案保管与保密

- 纸质档案存放于 **【档案室/带锁文件柜】**，专人管理。
- 电子档案存储于 **【LIMS系统/人事管理系统】**，设置分级访问权限。
- 人员档案保存期限：在职期间 + 离职后 **【10】年**。

### 6.2 入职/离职/调岗管理

#### 6.2.1 入职流程

```
需求提出 → 资格审核 → 入职手续 → 档案建立 → 岗前培训 → 上岗考核 → 正式上岗
```

1. **需求提出**：专业组组长填写《人员需求申请表》，经技术负责人审核、科主任批准。
2. **资格审核**：人事专员核验应聘者学历、职称、执业资格原件，确认符合岗位任职条件。
3. **入职手续**：签署劳动合同/协议，收集个人信息表、证件复印件、体检报告、照片、签名留样。
4. **档案建立**：3个工作日内在人事系统中建立完整档案。
5. **岗前培训**：按 JYK-SOP-06-003 执行入职培训计划。
6. **上岗考核**：按 JYK-SOP-06-002 执行新员工能力评估（入岗后6个月内至少2次）。
7. **正式上岗**：考核合格后，由技术负责人签署上岗确认，进入正常工作状态。

#### 6.2.2 离职流程

```
离职申请 → 工作交接 → 授权终止 → 档案封存 → 系统权限注销
```

1. 离职人员提前 **【30天】** 提交书面离职申请。
2. 完成工作交接清单，由组长及技术负责人确认签字。
3. 终止所有检验授权（见 JYK-SOP-06-004）。
4. 注销 LIMS 系统账号及其他信息系统访问权限。
5. 人事专员封存人员档案，标记离职日期及原因。

#### 6.2.3 调岗流程

```
调岗申请 → 资格校验 → 培训计划 → 能力评估 → 授权变更 → 正式调岗
```

1. 调岗需经科主任批准。
2. 校验人员是否满足目标岗位任职条件。
3. 制定针对性培训计划（见 JYK-SOP-06-003）。
4. 完成目标岗位能力评估合格后方可独立工作（见 JYK-SOP-06-002）。
5. 原岗位授权终止，新岗位授权启用（见 JYK-SOP-06-004）。
6. 更新人员档案中的岗位信息。

### 6.3 岗位说明书管理

#### 6.3.1 岗位说明书内容

每个岗位应有正式的岗位说明书，至少包含：

| 内容项 | 说明 |
|--------|------|
| 岗位名称 | 如：临床化学检验技师 |
| 所属部门/小组 | 如：临床化学组 |
| 汇报关系 | 直接上级/下属 |
| 岗位职责 | 逐条列出核心职责 |
| 权限范围 | 可执行的检验项目、可签发的报告类型 |
| 任职资格要求 | 学历、职称、执业资格、工作年限、特殊技能要求 |
| 工作范围 | 负责的检验项目/仪器/区域 |
| 能力要求 | 与 JYK-SOP-06-002 中定义的能力要求对应 |

#### 6.3.2 岗位说明书编写与审批

1. 技术负责人组织编写各岗位说明书。
2. 质量负责人审核岗位说明书的合规性。
3. 科主任批准后生效。
4. 每 **【2年】** 或岗位职责发生重大变化时进行修订。

### 6.4 任职条件自动校验

1. 系统应根据岗位说明书中定义的任职条件，在以下场景自动校验人员资质：
   - 新员工入职分配岗位时
   - 人员调岗时
   - 证书到期时自动预警
   - 年度档案核查时

2. 校验项包括：

| 校验项 | 校验规则示例 |
|--------|-------------|
| 学历 | ≥ 岗位要求最低学历 |
| 职称 | ≥ 岗位要求最低职称 |
| 执业资格 | 持有效执业证书且在注册有效期内 |
| 工作年限 | ≥ 岗位要求最低年限 |
| 特殊资质 | 如 HIV 初筛上岗证、PCR 上岗证等在有效期内 |

3. 校验不通过时，系统应阻止分配岗位并生成告警记录。

### 6.5 关键岗位代理人指定与授权

#### 6.5.1 需指定代理人的关键岗位

| 关键岗位 | 最少代理人数量 |
|----------|---------------|
| 科主任 | 1 |
| 技术负责人 | 1 |
| 质量负责人 | 1 |
| 各专业组组长 | 各 1 |
| 授权签字人 | 各授权领域至少 1 |

#### 6.5.2 代理人要求

1. 代理人应满足被代理岗位的任职条件。
2. 代理人须经过被代理岗位的相关培训。
3. 代理人指定须经科主任批准，形成书面授权文件。
4. 代理人授权应明确代理范围、代理条件（如正式岗位人员请假/出差/培训期间）、代理期限。
5. 每年至少复核一次代理人名单。

# SOP 2：JYK-SOP-06-002《能力评估管理程序》

## 文件信息

| 项目 | 内容 |

| 文件编号 | JYK-SOP-06-002 |
| 文件名称 | 能力评估管理程序 |
| 版本 | 第【A】版 第【0】次修订 |
| 生效日期 | 【YYYY-MM-DD】 |
| 编制 | 【姓名/岗位】 |
| 审核 | 【姓名/岗位】 |
| 批准 | 【姓名/岗位】 |
| 发放范围 | 检验科全体人员 |
| 页码 | 第 1 页 共 【X】 页 |

---

## 1 目的

建立系统化的人员能力评估制度，明确各岗位能力要求及评估方法，确保所有从事检验活动的人员持续具备胜任其岗位的能力，满足 ISO 15189:2022 第 6.2.2 条要求。

## 2 适用范围

适用于检验科所有从事检验前、检验和检验后过程活动的人员（含正式员工、合同制人员、进修生等），涵盖新员工入岗评估、常规评估、复岗评估等全部评估场景。

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求摘录

> **§6.2.2 能力要求**
> "实验室应确定各项职能所需的能力，包括教育、资格、培训、技术知识、技能和经验的要求。"
>
> "应按照文件化程序，在规定的时间间隔内评估每个人员执行分配任务的能力。"
>
> "评估方法应至少包括以下适用方法的组合：
> a) 直接观察常规工作过程和程序，包括所有适用的安全实践；
> b) 直接观察对设备维护和功能检查的执行；
> c) 监控检验和检验后过程的记录和结果；
> d) 评审工作记录；
> e) 评估解决问题的能力；
> f) 检查特别提供的样品，如先前检验过的样品、实验室间比对材料、分割样品。"
>
> "在新员工入职后的规定期间内（如最初六个月）应至少进行两次能力评估，之后每年进行评估。"

### 3.2 其他适用标准/法规

- 《医学实验室质量和能力认可准则》（CNAS-CL02）
- 【其他适用的行业标准】

## 4 职责

| 角色 | 职责 |
|------|------|
| 科主任 | 批准年度能力评估计划，审核评估结果汇总报告 |
| 技术负责人 | 制定各岗位能力要求，组织年度评估计划编排，审核评估结果 |
| 质量负责人 | 监督评估流程的执行合规性，协调评估与质量监督的联动 |
| 各专业组组长 | 执行本组人员评估，记录评估结果，提出培训建议 |
| 质量监督员 | 参与直接观察评估，提供日常工作质量监控数据 |
| 被评估人员 | 配合评估安排，按要求完成各项评估任务 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 能力评估 | 通过系统化方法评价人员执行指定任务所需知识、技能和行为的过程 |
| 直接观察 | 评估人员在工作现场直接观看被评估人执行操作的评估方法 |
| 盲样测试 | 将已知结果的样品以常规标本形式提供给被评估人检测，评估其检验能力 |
| 入岗评估 | 新员工入职后在规定期间内进行的初始能力评估 |
| 复岗评估 | 人员离岗后重新返回岗位时进行的能力评估 |
| 加密评估 | 对复杂/高风险检验项目增加评估频次的安排 |

## 6 程序内容

### 6.1 各项职能的能力要求定义

#### 6.1.1 能力要求框架

每个岗位/检验项目的能力要求应从以下六个维度定义：

| 维度 | 说明 | 示例 |
|------|------|------|

#### 6.1.2 能力要求文件

1. 技术负责人组织各专业组制定《岗位/项目能力要求表》（JYK-FM-06-002-01）。
2. 能力要求应与岗位说明书（JYK-SOP-06-001）中的任职条件保持一致。
3. 新项目开展或岗位职责重大变化时，应及时修订能力要求。

### 6.2 评估计划编排

#### 6.2.1 年度评估计划

1. 每年 **【1月】** 由技术负责人组织编制年度能力评估计划（JYK-FM-06-002-02）。
2. 评估计划应覆盖所有在岗人员和所有授权项目。
3. **常规评估间隔不超过 12 个月**（即每年至少评估一次）。
4. 计划应明确：被评估人、评估项目、评估方法、评估人、计划评估时间。

#### 6.2.2 评估频次要求

| 评估类型 | 适用场景 | 频次要求 |
|----------|----------|----------|
| 新员工入岗评估 | 新入职人员 | 前6个月内至少2次 |
| 常规年度评估 | 在岗人员 | 每12个月至少1次 |
| 复岗评估 | 离岗超过2年 | 返岗前完成评估 |
| 加密评估 | 复杂/高风险项目 | 每6个月1次（见6.5） |
| 专项评估 | 新项目/新方法上线 | 开展前完成评估 |
| 不合格后复评 | 评估不合格人员 | 完成培训后1个月内 |

### 6.3 评估方法

#### 6.3.1 评估方法组合

每次评估应至少采用以下方法中的 **两种及以上组合**：

| 编号 | 评估方法 | 说明 | 适用场景 |
|------|----------|------|----------|
| M1 | 直接观察 | 评估人在工作现场观察被评估人执行标准操作流程（含安全操作） | 所有岗位 |
| M2 | 盲样测试 | 将已知结果样品作为常规样品送检，评估检验准确性 | 有定量/定性结果的检验项目 |
| M3 | 理论考核 | 笔试/口试，考核专业知识、SOP理解、质量控制知识 | 所有岗位 |
| M4 | 审查工作记录 | 回顾性审查被评估人的检验记录、质控记录、异常处理记录 | 所有岗位 |
| M5 | 检查特别提供的样品 | 使用先前检验过的样品、EQA材料、分割样品进行评估 | 实验室间比对项目 |
| M6 | 解决问题能力评估 | 提供异常情景（如仪器故障、质控失控、危急值），评估判断和处理能力 | 中级及以上人员 |

#### 6.3.2 各岗位评估方法矩阵

| 岗位/项目类别 | M1 直接观察 | M2 盲样 | M3 理论 | M4 记录审查 | M5 特殊样品 | M6 问题解决 |
|---------------|:-----------:|:-------:|:-------:|:-----------:|:-----------:|:-----------:|
| 样品采集/接收 | ● | — | ● | ● | — | ○ |
| 常规生化 | ● | ● | ● | ● | ○ | ○ |
| 常规免疫 | ● | ● | ● | ● | ○ | ○ |
| 血液体液 | ● | ● | ● | ● | ● | ○ |
| 形态学检查 | ● | ● | ● | ● | ● | ● |
| 微生物检验 | ● | ● | ● | ● | ● | ● |
| 分子检测 | ● | ● | ● | ● | ● | ● |
| 质谱检验 | ● | ● | ● | ● | ● | ● |
| 流式细胞术 | ● | ● | ● | ● | ● | ● |
| 结果审核/报告签发 | ● | — | ● | ● | — | ● |

说明：● = 必选，○ = 推荐，— = 不适用

### 6.4 新员工入岗评估

#### 6.4.1 评估安排

1. 新员工入岗后 **6个月内** 应完成至少 **2次** 能力评估。
2. 建议安排：
   - **第1次评估**：入岗后第 **【2-3】个月**，重点评估基本操作能力和安全规范。
   - **第2次评估**：入岗后第 **【5-6】个月**，全面评估独立工作能力。

#### 6.4.2 评估内容

| 评估次数 | 评估重点 | 评估方法 | 合格标准 |
|----------|----------|----------|----------|
| 第1次 | 基本操作规范、安全操作、SOP理解 | M1 + M3 | 理论≥【80】分，操作无重大偏差 |
| 第2次 | 独立工作能力、结果判读、异常处理 | M1 + M2 + M4 + M6 | 盲样结果偏差在允许范围内，操作规范 |

#### 6.4.3 评估结果处理

- **两次评估均合格**：由技术负责人签署独立上岗确认。
- **任一次评估不合格**：制定针对性培训计划（见 JYK-SOP-06-003），培训后 1 个月内复评。
- **经培训和复评仍不合格**：评估是否适合继续在该岗位工作，必要时调整岗位或终止聘用。

### 6.5 复杂项目加密评估安排

对于以下复杂/高风险检验项目，在常规年度评估基础上增加评估频次：

| 项目类别 | 加密评估间隔 | 加密评估方法重点 | 说明 |
|----------|-------------|-----------------|------|
| 形态学检查（血涂片/骨髓/尿沉渣） | 每6个月 | M2盲样（已知形态标本）+ M5（分割样品比对） | 形态辨认高度依赖主观判断 |
| 微生物检验（培养/鉴定/药敏） | 每6个月 | M2盲样（已知菌株）+ M1直接观察（无菌操作） | 涉及培养鉴定的主观判断 |
| 质谱检验 | 每6个月 | M2盲样 + M4审查记录 + M6问题解决 | 新技术，需强化能力监控 |
| 流式细胞术 | 每6个月 | M2盲样 + M1直接观察（设门策略） | 设门分析高度依赖操作者经验 |
| 分子检测（PCR等） | 每6个月 | M2盲样 + M1直接观察（防污染操作） | 高灵敏度项目，污染风险高 |

### 6.6 离岗超过2年的复岗评估

#### 6.6.1 适用条件

人员因以下原因离岗超过连续 **2年**，返岗前须重新进行能力评估：
- 长期病假
- 产假/育儿假
- 外派/借调
- 停职
- 其他原因导致的长期脱岗

#### 6.6.2 复岗评估流程

```
复岗申请 → 岗位条件复核 → 培训安排 → 能力评估（等同新员工标准） → 评估合格 → 重新授权 → 恢复上岗
```

1. 复岗人员需重新参加岗前培训（生物安全、SOP更新、仪器操作等）。
2. 能力评估标准 **等同于新员工入岗评估**（6个月内至少2次）。
3. 所有原有授权在复岗评估合格前处于 **暂停状态**。
4. 评估合格后由技术负责人重新签署授权（见 JYK-SOP-06-004）。

### 6.7 评估结果统计和不合格处理

#### 6.7.1 评估结果记录

每次评估应填写《能力评估记录表》（JYK-FM-06-002-03），内容包括：
- 被评估人信息
- 评估项目、评估方法
- 评估内容及评分标准
- 评估得分/结论（合格/不合格）
- 评估人签名及日期
- 不合格项说明及改进建议

#### 6.7.2 评估结果判定标准

| 评估方法 | 合格标准 |
|----------|----------|
| 理论考核 | 得分≥【80】分（满分100分） |
| 直接观察 | 无关键操作偏差，总评分≥【85】分 |
| 盲样测试 | 所有项目结果在允许偏差范围内 |
| 工作记录审查 | 无重大差错，记录完整规范 |
| 问题解决评估 | 正确识别问题并采取适当措施 |

#### 6.7.3 不合格处理流程

```
评估不合格 → 通知被评估人及其组长 → 分析不合格原因 → 制定培训计划 → 执行培训 → 复评（1个月内） → 合格/再次不合格
```

1. 评估不合格时，应书面通知被评估人并分析原因。
2. 由组长与技术负责人共同制定针对性培训计划。
3. 培训完成后 **1个月内** 进行复评。
4. 复评仍不合格的：
   - 限制/暂停其相关检验项目的授权。
   - 评估是否调整岗位。
   - 记录于人员档案并纳入下次管理评审议题。

#### 6.7.4 评估结果统计分析

1. 每年编制《年度能力评估汇总报告》（JYK-FM-06-002-04），内容包括：
   - 评估完成率（实际评估人次/计划评估人次×100%）
   - 一次评估合格率
   - 不合格项分布分析
   - 复评合格率
   - 趋势分析（与上年比较）
2. 报告提交管理评审。

### 6.8 能力评估与质量监督的联动

1. 日常质量监督中发现的人员能力问题应及时触发专项评估。
2. 能力评估结果应反馈给质量监督员，作为确定监督重点的依据。
3. 以下情况应启动联动机制：

| 触发条件 | 联动动作 |
|----------|----------|
| 质量监督发现某人员多次同类操作偏差 | 启动该人员该项目专项能力评估 |
| 室间质评结果不合格 | 评估相关操作人员能力 |
| 患者投诉涉及检验质量 | 评估相关操作人员能力 |
| 能力评估不合格 | 通知质量监督员加强对该人员的日常监督 |
| 新员工入岗 | 质量监督员纳入重点监督对象 |

# SOP 3：JYK-SOP-06-003《培训管理程序》

## 文件信息

| 项目 | 内容 |
|------|------|
| 文件编号 | JYK-SOP-06-003 |
| 文件名称 | 培训管理程序 |
| 版本 | 第【A】版 第【0】次修订 |
| 生效日期 | 【YYYY-MM-DD】 |
| 编制 | 【姓名/岗位】 |
| 审核 | 【姓名/岗位】 |
| 批准 | 【姓名/岗位】 |
| 发放范围 | 检验科全体人员 |
| 页码 | 第 1 页 共 【X】 页 |

## 1 目的

建立系统化的培训管理制度，规范培训计划制定、培训实施、效果评估及继续教育学分管理，确保检验科人员持续获得必要的知识和技能更新，满足 ISO 15189:2022 第 6.2.4 条要求。

## 2 适用范围

适用于检验科所有人员的培训活动，包括入职培训、在职培训、专业培训、安全培训、合规培训、POCT培训，以及继续教育和职业发展活动。

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求摘录

> **§6.2.4 继续教育和职业发展**
> "实验室应为所有人员提供继续教育方案。参加继续教育方案的人员应能证明其继续教育活动的效果。"
>
> "当使用电子系统或手工系统发布检验结果时，应对有权限进入检验信息系统的所有人员提供培训，以确保患者数据的保密性。"

> **§6.2.2**（与培训相关）
> "实验室应确保人员的能力通过适当的培训来获得和保持。"

### 3.2 其他适用标准/法规

- 《医疗机构临床实验室管理办法》
- 《实验室生物安全通用要求》（GB 19489）
- 【其他适用的继续教育管理规定】

## 4 职责

| 角色 | 职责 |
|------|------|
| 科主任 | 批准年度培训计划和培训预算 |
| 技术负责人 | 审核技术培训内容，确认培训师资 |
| 质量负责人 | 组织合规培训（公正性/保密性/安全），监督培训执行 |
| 培训管理员 | 制定培训计划、组织培训实施、管理培训档案、统计继续教育学分 |
| 各专业组组长 | 提出本组培训需求，协助组织本组专业培训，参与带教 |
| 全体人员 | 按计划参加培训，完成培训考核，保存继续教育证明 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 入职培训 | 新员工入职后的初始培训，包括通用培训和岗位专业培训 |
| 在职培训 | 在岗人员日常接受的持续性培训活动 |
| 继续教育 | 获取和更新专业知识、技能的持续性教育活动，通常以学分计量 |
| 合规培训 | 涉及法规遵从、公正性、保密性、患者权益等方面的培训 |
| POCT培训 | 对参与床旁检验（Point-of-Care Testing）人员的专项培训 |
| 培训效果评估 | 通过考核、实操、满意度调查等方式衡量培训达到预期目标的程度 |

## 6 程序内容

### 6.1 年度培训计划制定与执行追踪

#### 6.1.1 培训需求分析

每年 **【12月】** 进行下年度培训需求分析，信息来源包括：

| 来源 | 说明 |
|------|------|
| 能力评估结果 | 评估中发现的知识/技能薄弱环节（见 JYK-SOP-06-002） |
| 质量监督反馈 | 日常监督中发现的操作问题 |
| 不符合/纠正措施 | 内审/外审/投诉中发现的人员能力相关问题 |
| 新项目/新设备 | 即将开展的新项目或新采购的设备 |
| 法规/标准更新 | 新发布或修订的法规、标准 |
| 人员变动 | 新入职人员、调岗人员 |
| 管理评审建议 | 管理评审中提出的培训改进建议 |
| 人员自身需求 | 人员提出的专业发展需求 |

#### 6.1.2 年度培训计划编制

1. 培训管理员汇总培训需求，编制《年度培训计划》（JYK-FM-06-003-01）。
2. 计划应包含：培训课程名称、培训类型、培训对象、培训师资、培训时间、培训学时、考核方式、预算。
3. 质量负责人审核，科主任批准。
4. 年度培训计划应确保覆盖所有必修培训主题（见 6.2）。

#### 6.1.3 计划执行追踪

1. 培训管理员每 **【季度】** 汇总培训执行情况，编制《培训执行追踪表》。
2. 追踪内容：已完成/未完成课程、参训人次、考核通过率、延期原因。
3. 年底编制《年度培训总结报告》（JYK-FM-06-003-02），提交管理评审。

### 6.2 培训类型

#### 6.2.1 入职培训

新员工入职后 **【2周】** 内完成以下培训：

| 培训模块 | 培训内容 | 学时 | 考核方式 |
|----------|----------|------|----------|
| 科室概况 | 科室组织架构、工作流程、科室文化 | 【2】h | 无需考核 |
| 质量管理 | 质量方针/目标、质量管理体系概述 | 【2】h | 笔试 |
| 生物安全 | 实验室生物安全等级、防护措施、应急处理 | 【4】h | 笔试+实操 |
| 信息安全与保密 | 患者数据保密要求、信息系统使用规范 | 【2】h | 笔试 |
| 公正性与伦理 | 公正性承诺、利益冲突、患者权益 | 【2】h | 签署承诺书 |
| SOP体系 | 文件管理制度、SOP查阅和执行要求 | 【2】h | 笔试 |
| 岗位专业培训 | 本岗位SOP、仪器操作、质控操作 | 【视岗位】 | 实操考核 |

#### 6.2.2 在职培训

| 培训内容 | 频率 | 培训方式 |
|----------|------|----------|
| SOP更新培训 | SOP修订后即时 | 集中讲解+签字确认 |
| 新项目开展培训 | 项目上线前 | 理论+实操 |
| 新设备操作培训 | 设备启用前 | 厂家培训+科内转训 |
| 质量改进培训 | 按需 | 案例分析 |
| 仪器故障排除培训 | 每【半年】 | 模拟演练 |

#### 6.2.3 专业培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 专业学术讲座 | 每【月】1次 | 全科人员 |
| 疑难病例讨论 | 每【月】1-2次 | 相关专业组 |
| 新技术/新方法学习 | 按需 | 相关人员 |
| 学术会议/培训班 | 按需 | 外派人员 |

#### 6.2.4 安全培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 生物安全年度培训 | 每年至少1次 | 全科人员 |
| 消防安全培训 | 每年至少1次 | 全科人员 |
| 化学品安全培训 | 每年至少1次 | 接触化学品人员 |
| 职业暴露处置演练 | 每年至少1次 | 全科人员 |
| 应急预案演练 | 每年至少1次 | 全科人员 |

#### 6.2.5 合规培训

| 培训内容 | 频率 | 对应要求 |
|----------|------|----------|
| 公正性培训 | 每年至少1次 | ISO 15189:2022 §4.1.1.3 |
| 保密性培训 | 每年至少1次 | ISO 15189:2022 §4.1.1.4 |
| 患者权益培训 | 每年至少1次 | 相关法律法规 |
| 伦理培训 | 每年至少1次 | 科研伦理、知情同意 |
| 法律法规更新培训 | 法规更新后即时 | 新发布的法规 |

#### 6.2.6 POCT 培训

对参与 POCT（床旁检验）管理和操作的人员进行专项培训：

| 培训内容 | 适用人员 | 频率 |
|----------|----------|------|
| POCT设备操作培训 | POCT操作人员（含临床科室人员） | 上岗前+每年复训 |
| POCT质量控制培训 | POCT操作人员 | 上岗前+每年复训 |
| POCT结果判读培训 | POCT操作人员 | 上岗前 |
| POCT管理制度培训 | 检验科POCT管理人员 | 按需 |

### 6.3 培训课程/课件/签到管理

#### 6.3.1 培训课程管理

1. 每次培训应有正式的培训课程信息记录，包括：
   - 课程名称、课程编号
   - 培训类型（入职/在职/专业/安全/合规/POCT）
   - 培训日期、时间、地点
   - 培训师资（内部/外部）
   - 培训方式（面授/线上/实操/自学/混合）
   - 培训学时
   - 考核方式及合格标准

#### 6.3.2 培训课件管理

1. 培训课件应由培训师编制，技术负责人或质量负责人审核。
2. 课件存储于 **【培训课件文件夹/LIMS系统/共享盘】**。
3. 课件应注明版本和编制日期。
4. 涉及SOP内容的课件应与当前有效版本一致。

#### 6.3.3 培训签到管理

1. 每次培训应有《培训签到表》（JYK-FM-06-003-03），记录参加人员签名、培训时间。
2. 线上培训应有系统登录记录或在线签到截图。
3. 培训签到表与培训课件一同归档保存。

### 6.4 培训效果评估

#### 6.4.1 评估方式

| 评估方式 | 说明 | 适用场景 |
|----------|------|----------|
| 笔试/在线考核 | 理论知识测试，满分100分，合格线【80】分 | 理论培训 |
| 实操考核 | 现场操作演示，由培训师评分 | 技能培训 |
| 满意度调查 | 培训满意度问卷，评价课程内容/师资/组织 | 所有培训 |
| 行为观察 | 培训后一段时间观察工作行为是否改变 | 重点培训 |
| 结果分析 | 培训后检验质量指标是否改善 | 针对性培训 |

#### 6.4.2 考核不合格处理

1. 笔试/实操考核不合格者，安排 **【2周内】** 补考。
2. 补考仍不合格者，需重新参加培训。
3. 关键培训（如安全培训、SOP培训）未通过者，不得独立上岗。
4. 考核结果记录于个人培训档案。

### 6.5 继续教育学分管理

#### 6.5.1 学分要求

1. 根据【省/市卫生健康委员会】要求，各类人员年度继续教育学分要求：

| 人员类别 | 年度学分要求 | 其中I类学分 | 其中II类学分 |
|----------|-------------|------------|-------------|
| 高级职称 | ≥【25】分 | ≥【5】分 | ≥【20】分 |
| 中级职称 | ≥【25】分 | ≥【5】分 | ≥【20】分 |
| 初级职称 | ≥【25】分 | ≥【5】分 | ≥【20】分 |

#### 6.5.2 学分登记与跟踪

1. 培训管理员维护《继续教育学分登记表》（JYK-FM-06-003-04）。
2. 每季度汇总学分完成情况，对学分不足人员发出提醒。
3. 年底核查全员学分达标情况，纳入年度培训总结报告。

### 6.6 外部培训/会议参加记录

1. 人员参加外部培训/学术会议须事先填写《外部培训/会议申请表》（JYK-FM-06-003-05），经组长同意、科主任批准。
2. 培训/会议结束后 **【2周内】** 须提交：
   - 培训/会议总结报告
   - 证书/学分证明复印件
   - 科室内转训安排（如适用）
3. 外出培训学习到的新知识/新技术，应安排向本组或全科进行转训分享。

### 6.7 与合规培训的整合

#### 6.7.1 合规培训与其他培训的整合安排

为提高培训效率，可将合规培训与其他培训活动整合：

| 合规培训主题 | 可整合的培训场景 | 整合方式 |
|-------------|-----------------|----------|
| 公正性 | 入职培训、年度质量管理培训 | 设置独立模块 |
| 保密性 | 入职培训、信息系统培训 | 设置独立模块 |
| 患者权益 | 入职培训、伦理培训 | 合并课程 |
| 安全 | 入职培训、年度安全培训 | 可合并或分开 |

#### 6.7.2 合规培训记录要求

1. 合规培训须有 **独立的签到记录和考核记录**，即使与其他培训整合。
2. 公正性培训须签署《公正性承诺书》。
3. 保密性培训须签署《保密协议》。
4. 以上记录纳入人员档案（见 JYK-SOP-06-001）。

## 1 目的

建立系统化的人员授权管理制度，规范授权范围定义、授权签字人管理、授权到期管理及特殊岗位资质管理，确保所有检验活动由经授权的合格人员执行，满足 ISO 15189:2022 第 6.2.3 条要求。

## 2 适用范围

适用于检验科所有需要授权方可执行的活动，包括但不限于检验方法的选择/开发/修改/确认和验证、检验结果的审查/发布/报告、信息系统使用、设备操作等。

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求摘录

> **§6.2.3 批准/授权**
> "实验室管理层应批准/授权人员执行以下活动：
> a) 方法的选择、开发、修改、确认和验证；
> b) 检验结果的审查、发布和报告；
> c) 使用实验室信息系统。"
>
> "应保存所有授权的记录。"

> **§5.6.2.3**（与授权签字人相关）
> "实验室应规定谁有权发布检验结果和报告。"

### 3.2 其他适用标准/法规

- 《医疗机构临床实验室管理办法》
- CNAS-CL02 §5.1.6（授权签字人要求）
- 【其他适用的法规或认可要求】

| 角色 | 职责 |
|------|------|
| 科主任 | 批准授权签字人资格，签发授权书 |
| 技术负责人 | 审核授权申请，确认人员具备授权所需的能力和资质 |
| 质量负责人 | 监督授权管理制度执行，审查授权合规性 |
| 授权管理员 | 维护授权清单、跟踪授权到期、管理授权档案 |
| 各专业组组长 | 提出本组人员授权申请，监督授权执行 |
| 被授权人员 | 在授权范围内执行工作，不得超范围操作 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 授权 | 由实验室管理层正式批准人员执行特定活动的行为 |
| 授权签字人 | 经授权可在检验报告上签字并对结果负责的人员 |
| 授权范围 | 被授权人员可执行的具体活动或检验项目的范围 |
| 授权有效期 | 授权的起止时间，到期需重新评估和续期 |
| 设备操作授权 | 经培训和考核合格后获得的特定设备操作权限 |

## 6 程序内容

### 6.1 授权范围定义

#### 6.1.1 需要授权的活动类别

根据 ISO 15189:2022 §6.2.3 要求，以下活动需要正式授权：

| 类别 | 授权活动 | 授权级别 | 授权审批人 |
|------|----------|----------|------------|
| A | 方法的选择、开发、修改 | 高级授权 | 科主任 |
| B | 方法的确认和验证 | 高级授权 | 科主任 |
| C | 检验结果的审查 | 中级授权 | 技术负责人 |
| D | 检验结果的发布和报告签发 | 高级授权 | 科主任 |
| E | 实验室信息系统使用（不同权限级别） | 基础/中级授权 | 技术负责人 |
| F | 设备操作 | 基础授权 | 专业组组长 |
| G | 特殊检验项目操作 | 特殊授权 | 科主任 |

#### 6.1.2 授权级别说明

| 授权级别 | 要求 | 有效期 | 审批流程 |
|----------|------|--------|----------|
| 基础授权 | 完成培训+考核合格 | 【2】年 | 组长审核→技术负责人批准 |
| 中级授权 | 基础授权+中级及以上职称 | 【2】年 | 组长推荐→技术负责人审核→科主任批准 |
| 高级授权 | 中级授权+3年及以上经验 | 【2】年 | 技术负责人推荐→科主任批准 |
| 特殊授权 | 高级授权+特殊资质证书 | 同证书有效期 | 技术负责人推荐→科主任批准 |

### 6.2 授权签字人管理

#### 6.2.1 授权签字人资质要求

| 条件项 | 要求 | 校验方式 |
|--------|------|----------|
| 职称 | 中级及以上技术职称（主管技师/主管检验师及以上） | 系统自动校验 |
| 工作经验 | ≥3年本专业检验工作经验 | 系统自动校验 |
| 学历 | 本科及以上（医学检验相关专业） | 系统自动校验 |
| 培训 | 完成授权签字人培训（质量管理/法规/报告签发规范） | 培训记录确认 |
| 能力评估 | 最近一次能力评估合格 | 系统自动校验 |
| 执业资格 | 持有效执业证书 | 系统自动校验 |

#### 6.2.2 授权签字人申请与审批流程

```
申请 → 资质校验 → 技术负责人审核 → 科主任批准 → 签发授权书 → 系统授权配置 → 签名留样
```

1. 申请人填写《授权签字人申请表》（JYK-FM-06-004-01）。
2. 授权管理员核验资质条件（系统自动校验+人工核实）。
3. 技术负责人审核申请人的专业能力和工作表现。
4. 科主任批准后签发《授权签字人授权书》（JYK-FM-06-004-02）。
5. 在 LIMS 系统中配置相应的报告签发权限。
6. 留存签名样本（手写签名+电子签名）。

### 6.3 授权签字领域与范围配置

#### 6.3.1 授权领域划分

| 领域编号 | 授权领域 | 包含项目示例 |
|----------|----------|-------------|
| F08 | 【其他领域】 | 【根据实际情况补充】 |

#### 6.3.2 授权范围配置

1. 每位授权签字人的授权范围应精确到领域+项目级别。
2. 授权范围应与其专业能力和培训经历匹配。
3. 不得授权超出其能力范围的签字领域。
4. 授权范围变更需重新审批。

### 6.4 授权到期提醒

#### 6.4.1 到期提醒机制

| 提醒时间节点 | 提醒方式 | 提醒对象 |
|-------------|----------|----------|
| 到期前 **90天** | 系统自动通知 | 被授权人、组长、授权管理员 |
| 到期前 **30天** | 系统自动通知+邮件提醒 | 被授权人、组长、技术负责人 |
| 到期前 **7天** | 系统告警+邮件+短信 | 被授权人、技术负责人、科主任 |
| **到期当天** | 系统自动暂停授权 | 自动执行 |

#### 6.4.2 授权续期流程

```
到期提醒 → 能力评估确认 → 资质复核 → 续期申请 → 技术负责人审核 → 科主任批准 → 系统更新
```

1. 授权到期前完成以下条件确认：
   - 最近一次能力评估合格（见 JYK-SOP-06-002）
   - 执业证书/资质证书在有效期内
   - 无重大差错/事故记录
   - 完成必要的继续教育学分
2. 条件满足则提交续期申请，简化审批。
3. 条件不满足则暂停授权，待补齐后重新申请。

### 6.5 特殊岗位资质管理

#### 6.5.1 需要特殊资质的岗位/项目

| 特殊岗位/项目 | 所需特殊资质 | 资质发证机构 | 有效期管理 |
|---------------|-------------|-------------|------------|
| HIV 抗体初筛 | HIV初筛实验室上岗证 | 【省/市疾控中心】 | 【3】年，到期续证 |
| 产前筛查 | 产前筛查技术培训合格证 | 【省卫生健康委】 | 【3】年，到期续证 |
| 分子检测（PCR） | PCR上岗证/临床基因扩增检验技术培训合格证 | 【省临检中心】 | 长期有效/【3】年 |
| 微生物检验 | 【如适用】生物安全培训合格证 | 【】 | 【】 |
| 血型鉴定/输血相容性检测 | 【如适用】输血技术培训合格证 | 【】 | 【】 |
| 【其他特殊项目】 | 【所需资质】 | 【发证机构】 | 【有效期】 |

#### 6.5.2 特殊资质管理要求

1. 特殊资质证书原件/扫描件纳入人员档案（见 JYK-SOP-06-001）。
2. 系统应记录证书编号、发证日期、有效期、发证机构。
3. 证书到期前 **【90天】** 系统自动提醒续证。
4. 未持有效特殊资质的人员，**不得** 从事对应的检验项目。
5. 特殊资质到期未续证的，系统自动暂停其对应项目的操作授权。

### 6.6 设备操作授权清单

#### 6.6.1 设备授权管理原则

1. 所有检验设备的操作均需经过培训和考核后方可授权。
2. 设备操作授权以 **单台设备** 为单位管理。
3. 同类型设备（同品牌同型号）可合并授权，但更换型号需重新培训和授权。

#### 6.6.2 设备操作授权流程

```
设备培训（厂家/科内） → 操作考核 → 组长确认 → 授权登记 → 系统配置
```

#### 6.6.3 设备操作授权清单

应维护《设备操作授权清单》（JYK-FM-06-004-03），内容包括：

| 字段 | 说明 |
|------|------|
| 设备名称 | 如：全自动生化分析仪 |
| 设备型号 | 如：【品牌型号】 |
| 设备编号 | 实验室内部编号 |
| 授权操作人员 | 姓名列表 |
| 培训日期 | 完成培训的日期 |
| 考核日期 | 通过考核的日期 |
| 授权日期 | 正式授权的日期 |
| 授权人 | 批准授权的人员 |

### 6.7 授权暂停与终止

#### 6.7.1 触发条件

| 情况 | 处理方式 |
|------|----------|
| 能力评估不合格 | 暂停相关项目授权，直至复评合格 |
| 授权到期未续期 | 系统自动暂停 |
| 特殊资质证书过期 | 暂停对应项目授权 |
| 发生重大差错/事故 | 暂停相关授权，调查后决定恢复或终止 |
| 人员离职 | 终止全部授权 |
| 人员调岗 | 终止原岗位授权，新岗位重新授权 |
| 人员违反操作规程 | 视严重程度暂停或终止相关授权 |
| 离岗超过2年 | 暂停全部授权，复岗评估合格后恢复 |

#### 6.7.2 授权暂停/终止流程

1. 授权管理员在系统中标记暂停/终止状态及原因。
2. 书面通知被授权人。
3. LIMS系统同步更新权限配置。
4. 暂停授权的恢复需重新评估后审批。
5. 终止授权的记录永久保留。

# 附加生成规则

# 生成规则（强制）

你是实验医学科 ISO 15189:2022 质量管理体系文件专家。生成任何文件时，必须遵循以下全部规则。

## 2. 编号与层级

- 标题层级：一级"1"，二级"1.1"，三级"1.1.1"，最深不超过四级
- 文件编号体系：QM（质量手册）、PER（人员）、DOC（文件记录）、QUA（质量）、SAF（安全）、ANA（分析过程）、INS（设备物资）、MET（方法性能）
- 记录表格编号：[前缀]-REC-[序号]

## 3. 标准引用

- 必须引用 ISO 15189:2022 对应条款，格式：ISO 15189:2022 第 X.X.X 条
- 必须同时引用 CNAS-CL02:2023 对应条款
- 每个程序要求须追溯到至少一个标准条款

## 4. 语言风格

- 中文为主，专业术语保留英文原文并附中文
- 操作步骤使用祈使句（"应""必须""shall"）
- 禁止模糊表述。生成内容中出现以下任何词汇，必须立即替换为右侧的精确表述：

| 禁止用法 | 必须替换为 |
|----------|-----------|
| 大约/约/左右/差不多 | 具体数值或范围（如"2-8℃""30±2分钟"） |
| 一般来说/通常/往往 | 删除修饰词直接陈述；有例外用"除…外"列出 |
| 及时/尽快/尽早 | 明确时限（如"2小时内""当日完成""发现后30分钟内"） |
| 相关人员/有关部门 | 具体岗位（如"质量负责人""检验科主任""专业组长"） |
| 适当/合理/必要时 | 触发条件（如"当温度偏差>2℃时""质控违反Westgard规则时"） |
| 定期/经常/不定期 | 频率（如"每月1次""每季度至少1次""每6个月"） |
| 做好记录/注意保存 | 指向表格（如"填写PER-REC-003""记录于INS-REC-005"） |

## 5. 术语规范

必须使用 ISO 15189:2022 标准术语，禁止使用旧术语：

| 禁止使用 | 必须改为 |
|----------|---------|
| 分析前过程/分析前阶段 | 检验前过程 |
| 分析过程/分析阶段 | 检验过程 |
| 分析后过程/分析后阶段 | 检验后过程 |
| 不确定度（单独使用） | 测量不确定度 |
| 正常值范围/正常参考值 | 生物参考区间 |
| 标本 | 样品 |
| 检验项目 | 检测项目 |
| 准确度 | 正确度 |
| 质控品 | 质控物 |
| 线性范围 | 可报告范围 |

## 6. 版本与变更

- 新建文件版本号：第 A 版第 0 次修订
- 变更历史必须包含：修订日期、修订内容摘要、修订人、审批人

## 7. 记录表格

- 程序文件必须配套记录表格清单
- 表格须有表头（科室名称、文件编号、版本号）、填写说明、签名栏

## 8. 安全与合规

- 涉及患者信息的流程必须包含隐私保护措施
- 涉及生物样本的操作必须包含生物安全要求
- 涉及数据传输的流程必须说明加密和权限控制
