# ISO 15189:2022 检验科文件体系生成指南

## 本套模板的用途

本套 MD 模板文件用于指导大语言模型 (LLM) 为医学实验室（检验科）生成符合 **ISO 15189:2022** 要求的质量管理体系文件。

**定位**：将"从零编写"变为"审核修改"，节省 70-80% 编写时间。

**输出物**：质量手册 + 程序文件(SOP) + 记录表单/作业指导书（三层文件结构）。

---

## 推荐使用的 LLM 模型

由于单个模板文件约 8K-12K tokens，请使用以下级别的模型：

| 推荐级别 | 模型 | 说明 |
|---------|------|------|
| 最佳 | Claude Opus / Claude Sonnet | 长上下文处理能力强，中文质量高 |
| 推荐 | GPT-4 / GPT-4o | 指令遵循度好 |
| 可用 | 其他 128K+ 上下文模型 | 需测试中文输出质量 |
| 不推荐 | GPT-3.5 / 4K 上下文模型 | 上下文不足，生成质量差 |

---

## 使用流程

```
步骤1：填写下方「检验科基本信息」
  ↓
步骤2：将本文件 + 01-质量手册-框架.md 一起发给 LLM
  ↓  LLM 生成质量手册初稿
步骤3：逐个发送 02~08 程序文件模板给 LLM
  ↓  每次附上本文件的「基本信息」部分作为上下文
  ↓  LLM 逐模块生成程序文件
步骤4：发送 09-记录表单清单.md 给 LLM
  ↓  LLM 生成配套表单
步骤5：发送 10-POCT专项.md（如有 POCT）
  ↓
步骤6：人工审核 → 签批发布 → 实施运行
```

**重要提示**：
- 每次发给 LLM 时，请把下方「基本信息」一并粘贴，确保 LLM 知道你的实验室具体情况
- 生成结果中所有 `【】` 包裹的内容为必须替换的占位符
- 所有标注 `⚠️ 示例，请替换` 的文本为示例内容，必须替换为真实信息

# SOP 1：JYK-SOP-06-005《设施和环境管理程序》

## 对应条款：ISO 15189:2022 §6.3 (6.3.1~6.3.5)

---

### 标准原文要求摘录

> **6.3.1 总则**
> 实验室应具备开展实验室活动所需的设施和环境条件，并进行管理。实验室应记录开展实验室活动所需的设施和环境条件要求，以及对适当执行实验室活动可能产生不良影响的偏差。
>
> **6.3.2 实验室和办公设施**
> 实验室设施应充分用于实验室活动的正确执行。应考虑：(a) 能源、照明、暖通和通风要适合实验室活动；(b) 设施不应对结果的有效性和所需测量的准确性造成不良影响；(c) 设施应防止对人员健康和安全造成不良影响。
>
> **6.3.3 储存设施**
> 实验室应为样品、载玻片和组织块、留存样品、文件和记录、设备、试剂、实验室耗材、参考材料以及其他可能影响实验室活动结果的物品提供适当的储存条件。储存设施应确保储存物品的安全、功能和完整性。
>
> **6.3.4 人事设施和样品收集设施**
> 实验室应能使用合适的人事设施。在需要时，应能使用急救材料。应为样品收集提供设施，这些设施应确保患者隐私、舒适和无障碍通道。
>
> **6.3.5 设施维护和环境条件**
> 实验室应监控、控制和记录环境条件。当规定的环境条件可能影响测量结果的有效性或对人员健康和安全造成不良影响时，实验室应采取措施。

PCR实验室按照卫生部《医疗机构临床基因扩增检验实验室管理办法》要求设置，
分为以下四个独立分区：

  试剂准备区 → 标本制备区 → 扩增区 → 产物分析区
  （正压 +10Pa）  （负压 -5Pa）  （负压 -10Pa）  （负压 -15Pa）

气流方向：从试剂准备区→产物分析区递减压力梯度，防止扩增产物反向污染。
各区之间设缓冲间，配备传递窗和紫外灭菌装置。
人员和物品严格执行单向流动，不得逆向。
```

1. 发现异常 → 立即记录发现时间、偏离参数值
       ↓
2. 初步评估 → 判断是否影响当前检验活动
       ↓ 是                     ↓ 否

       ↓

       ↓

       ↓

       ↓

```

# SOP 2：JYK-SOP-06-006《设备管理程序》

## 对应条款：ISO 15189:2022 §6.4 (6.4.1~6.4.7)

---

### 标准原文要求摘录

> **6.4.1** 实验室应具备正确执行实验室活动所需的设备（包括但不限于测量仪器、软件、测量标准、标准物质、参考数据、试剂和耗材、辅助设备，以及采集和检查样品所需的设备）并对其进行管理。
>
> **6.4.2** 实验室应记录每个设备项目的唯一标识。实验室应保持设备登记册。
>
> **6.4.3** 设备在投入使用或重新投入使用前，应验证其符合规定的验收标准。
>
> **6.4.4** 设备应由经授权的人员操作。设备的使用和维护说明（包括安全操作）应易于被相关人员获取。
>
> **6.4.5** 实验室应有预防性维护计划。当发现设备有缺陷或偏离规定要求时，应将其停用、隔离或做出明确标记，直至验证其正确运行。实验室应检查这些缺陷或偏离对之前检验结果的影响。
>
> **6.4.6** 实验室应调查设备不良事件或发送给实验室的设备事件通知，并在必要时向有关部门报告。
>
> **6.4.7** 实验室应保存对实验室活动有影响的设备记录。

### 1. 目的

**必须包含的内容**：
- 建立设备全生命周期管理制度（选择→采购→安装→验收→使用→维护→退役）
- 确保所有设备处于良好运行状态，满足检验活动要求
- 满足 ISO 15189:2022 §6.4 的全部要求

---

### 2. 适用范围

**必须包含的内容**：
- 适用于本实验室所有用于检验活动的设备，包括：
  - 分析仪器（全自动/半自动）
  - 辅助设备（离心机、水浴箱、培养箱、冰箱等）
  - 测量器具（移液器、天平、温度计等）
  - 信息系统设备（LIS 服务器、电脑、条码打印机等）
  - 样品采集设备

---

### 6. 程序内容

#### 6.1 设备选择与采购 [§6.4.1]

> LLM 须描述设备选型和采购流程：

```
设备采购流程：

1. 需求论证 → 使用科室/专业组提出需求，填写《设备需求论证报告》
       ↓
2. 选型评估 → 技术负责人组织选型评估（性能指标、品牌对比、售后服务、
              与LIS接口兼容性、试剂开放/封闭性）
       ↓
3. 审批 → 科室负责人审批 → 医院设备科/采购中心
       ↓
4. 招标/采购 → 按医院采购流程执行
       ↓
5. 合同签订 → 明确安装、培训、保修、售后等条款
       ↓
6. 到货验收 → 转入 6.3 验收程序
```

#### 6.2 设备标识与登记册 [§6.4.2]

**6.2.1 设备唯一标识**

⚠️ 示例，请替换：

#### 6.3 设备验收 [§6.4.3]

> LLM 须详细描述设备验收程序（新购设备和维修/搬迁后重新投入使用的设备）：

**6.3.1 新设备验收**

```
新设备验收流程：

1. 开箱检查 → 核对装箱单、外观完整性、配件齐全性
       ↓
2. 安装确认（IQ）→ 安装环境符合要求、供电/供水/排水正常、
                    厂家工程师完成安装
       ↓
3. 运行确认（OQ）→ 各功能模块正常运行、软件版本正确、
                    与 LIS 接口连接正常
       ↓
4. 性能确认（PQ）→ 用标准品/质控品验证性能指标：
   ├─ 精密度（批内/批间）
   ├─ 正确度（与参考方法/实验室间比对）
   ├─ 线性范围
   ├─ 携带污染率
   ├─ 检出限/定量限（如适用）
   └─ 与现有设备的相关性/可比性
       ↓
5. 验收结论 → 技术负责人签署验收报告
       ↓
6. 合格 → 粘贴"在用"标识 → 建立设备档案 → 投入使用
   不合格 → 联系供应商处理 → 重新验收
```

**6.3.2 重新投入使用验收**

> 以下情况需要重新验收：
- 设备维修后
- 设备搬迁后
- 长时间停用后重新启用
- 更换关键部件后

验收项目根据具体情况由技术负责人确定，至少包含性能确认。

#### 6.4 设备使用 [§6.4.4]

**6.4.1 授权操作**

- 设备操作人员须经培训、考核合格后方可独立操作
- 培训记录归入个人培训档案（参见 JYK-SOP-06-002《人员培训管理程序》）
- 授权名单记录在《设备操作授权名单》JYK-FR-06-032

**6.4.2 设备使用说明**

- 每台设备操作区域应放置操作 SOP（作业指导书）
- 包含：开/关机程序、日常操作流程、质控流程、安全注意事项
- 制造商原版说明书保存在设备档案中

**6.4.3 设备安全保障**

- 电气安全（接地、漏电保护）
- 化学安全（废液收集、排放）
- 机械安全（防护罩、安全联锁）
- 辐射安全（如适用）

**6.5.2 故障设备管理**

```
设备故障处理流程：

1. 发现故障 → 操作人员立即停止使用该设备
       ↓
2. 粘贴"停用/故障"标识标签（红色），注明停用日期和原因
       ↓
3. 报告 → 通知组长和设备管理员
       ↓
4. 影响评估 → 技术负责人评估故障对已发出结果的影响
       ↓  有影响 → 启动结果召回程序
5. 维修 → 联系厂家或第三方维修
       ↓
6. 维修完成 → 执行重新投入使用验收（见 6.3.2）
       ↓
7. 合格 → 撕除"停用"标识 → 恢复使用 → 记录归档
```

# SOP 3：JYK-SOP-06-007《校准和计量溯源管理程序》

## 对应条款：ISO 15189:2022 §6.5 (6.5.1~6.5.3)

---

### 标准原文要求摘录

> **6.5.1** 实验室应建立并维护测量设备的校准程序。
>
> **6.5.2** 实验室应建立和维护检查结果的计量可追溯性，并以文件化信息加以记录。
>
> **6.5.3** 在计量可追溯性无法在技术上实现的情况下，实验室应记录所用的替代方法以证明检查结果的可靠性，例如：(a) 使用认证参考材料；(b) 参加适当的 EQA/PT 方案；(c) 使用具有已知特性的适当标准进行检查；(d) 与另一程序达成一致意见/协议；(e) 基质样品的相互比较。

### 6. 程序内容

#### 6.1 校准程序要求 [§6.5.1]

**6.1.1 需校准的设备清单**

> LLM 须列出需校准设备的分类：

| 设备类别 | 典型设备 | 校准方式 | 校准周期 |
|---------|---------|---------|---------|
| 分析仪器 | 生化分析仪、免疫分析仪、血球仪等 | 使用厂家配套校准品校准 | 按厂家要求/批号更换时 |
| 计量器具（强检） | 天平、温度计、移液器 | 外部法定计量检定机构 | 每年 1 次 |
| 辅助测量设备 | 温湿度计、压差计、定时器 | 外部校准或比对验证 | 每年 1 次 |
| 离心机 | 转速校准 | 外部计量检定 | 每年 1 次 |

**6.1.2 校准条件**

- 校准应在制造商说明书规定的环境条件下进行
- 校准前设备应预热至稳定状态
- 使用的校准品应在有效期内且储存条件符合要求
- 校准品的复溶/使用严格按说明书操作

**6.1.3 仪器校准流程**

```
仪器校准流程：

1. 确认校准时机（新批号试剂、定期校准、维修后、质控失控后等）
       ↓
2. 准备校准品（检查效期、复溶/平衡至室温）
       ↓
3. 按制造商程序执行校准操作
       ↓
4. 审核校准曲线/校准结果（斜率、截距、相关系数等）
       ↓
5. 校准通过 → 运行质控品验证校准有效性
       ↓ 质控通过
6. 记录校准日期、校准品批号和效期、校准结果 → 投入检测
       ↓ 质控不通过
7. 分析原因 → 重新校准或联系厂家
```

#### 6.3 校准状态记录和重新校准 [§6.5.1]

**6.3.1 校准状态标识**

- 分析仪器：在 LIS 或仪器系统中记录校准状态和下次校准日期

**6.3.2 重新校准触发条件**

> LLM 须列出需重新校准的条件：

- 试剂批号更换
- 制造商建议的定期校准周期到期

- 环境条件发生重大变化后
- EQA/PT 结果不满意后

校正系数更新流程：

#### 6.5 校准失控处理 [§6.5.1]

> LLM 须描述校准未通过时的处理流程：

```
校准失控处理流程：

1. 校准未通过（校准曲线不合格或校准后质控失控）
       ↓
2. 排查原因：
   ├─ 校准品问题（效期、储存、复溶）→ 更换校准品重新校准
   ├─ 试剂问题 → 更换试剂
   ├─ 仪器问题 → 联系厂家维修
   └─ 环境问题 → 调整环境条件
       ↓
3. 纠正后重新校准
       ↓
4. 运行质控验证
       ↓
5. 评估影响 → 追溯上次合格校准以来的检验结果是否受影响
       ↓
6. 如有影响 → 通知临床科室、必要时召回结果
       ↓
7. 记录到《校准失控处理记录表》JYK-FR-06-042
```

**标准要求的替代方法**：

| 替代方法 | 适用场景 | 实施要求 |
|---------|---------|---------|
| (a) 使用认证参考材料（CRM） | 有 CRM 可用的检测项目 | 采购并使用有证标准物质进行验证 |
| (b) 参加 EQA/PT 方案 | 大多数常规检测项目 | 按计划参加，评估偏倚 |
| (c) 使用已知特性的标准 | 特殊检测方法 | 使用明确赋值的标准品 |
| (d) 与另一程序达成一致 | 方法学比对 | 两种独立方法的结果一致性验证 |
| (e) 基质样品互比 | 同类仪器间比对 | 用患者样品进行仪器间比对 |

⚠️ 示例，请替换：
```
对于以下项目，实验室通过替代方法确保结果可靠性：
- 血细胞分析：使用全血质控品 + 参加卫生部/省临检中心 EQA
- 微生物鉴定和药敏：参加 EQA + 使用 ATCC 标准菌株
- 尿液有形成分：使用人工质控物 + 参加 EQA + 人员间比对
```

# SOP 4：JYK-SOP-06-008《试剂和耗材管理程序》

## 对应条款：ISO 15189:2022 §6.6 (6.6.1~6.6.7)

---

### 标准原文要求摘录

> **6.6.1** 实验室应有一个程序来管理试剂和耗材的接收、储存和验收测试。
>
> **6.6.2** 试剂和耗材应按照制造商的规定进行储存，且在规定条件下保管。

> **6.6.3** 每个新配方的试剂或每个新批号的试剂和耗材，在投入使用前，应验证其符合规定的验收标准。

> **6.6.4** 试剂和耗材的库存应以适当的方式进行管理。
>
> **6.6.5** 应有试剂和耗材的使用说明，且这些说明应易于被相关人员获取。
>
> **6.6.6** 实验室应调查试剂和耗材的不良事件，并在必要时向有关部门报告。
>
> **6.6.7** 实验室应保存试剂和耗材的记录。

### 6. 程序内容

#### 6.1 试剂选择与采购 [§6.6.1]

**6.1.1 试剂选择原则**

- 优先选择已获国家药品监督管理局（NMPA）注册/备案的体外诊断试剂
- 封闭系统设备使用厂家配套试剂
- 开放系统设备须经方法学验证后选择试剂
- 综合考虑：分析性能、临床适用性、成本效益、供应稳定性

**6.1.2 采购流程**

```
试剂采购流程：

1. 各专业组提出试剂需求计划（品种、预估用量、期望到货时间）
       ↓
2. 试剂管理员汇总需求 → 编制采购计划
       ↓
3. 技术负责人审核品种合理性 → 科室负责人审批
       ↓
4. 提交医院采购部门/药品耗材管理中心
       ↓
5. 采购部门按规定流程采购（招标/议价/直采）
       ↓
6. 到货 → 转入 6.2 接收程序
```

#### 6.2 接收和储存 [§6.6.1, §6.6.2]

**6.2.1 试剂接收**

> LLM 须详细描述试剂接收流程：

```
试剂接收流程：

1. 核对送货单与采购订单是否一致
       ↓
2. 检查外包装完整性（有无破损、泄漏、温度标签变色）
       ↓
3. 核对关键信息：
   ├─ 品名、规格、批号
   ├─ 数量
   ├─ 生产日期和有效期（剩余有效期应 ≥ 总有效期的 2/3）
   ├─ 储运条件是否符合要求（冷链运输的检查温度记录）
   └─ 注册证号/备案号
       ↓
4. 合格 → 签收 → 填写《试剂接收登记表》JYK-FR-06-046
   不合格 → 拒收 → 通知采购部门处理
       ↓
5. 立即按储存要求入库（冷藏品 ≤ 30 min 内放入冰箱）
       ↓
6. 标记为"待验收" → 转入 6.3 验收测试
```

**6.2.2 储存管理**

> LLM 须描述试剂储存要求：

- 按制造商说明书规定的温度条件分区储存
- 储存区域的温度监控（参见 JYK-SOP-06-005 环境监控）
- 危险化学品按 SDS 要求专柜储存
- 易燃易爆品按消防安全要求储存
- 生物制品与化学试剂分区
- 试剂标签朝外，便于识别

#### 6.3 验收测试 [§6.6.3]

> LLM 须详细描述新批号/新配方试剂的验收测试程序：

**6.3.1 验收测试触发条件**

- 新批号试剂首次使用前
- 试剂配方/方法学变更
- 更换供应商
- 同一批号不同运输批次（抽检）

**6.3.2 验收测试方法**

⚠️ 示例，请替换：
```
验收测试方案（按试剂类型）：

1. 定量检测试剂：
   - 用至少2个浓度水平的质控品，新旧批号平行检测
   - 判定标准：新批号结果与旧批号结果的偏差 ≤ 允许总误差的 1/2
   - 或与靶值的偏差在厂家声明的允许范围内

2. 定性/半定量检测试剂：
   - 用已知阳性和阴性样品检测
   - 判定标准：定性结果一致

3. 培养基：
   - 用 ATCC 标准菌株接种
   - 判定标准：菌落生长符合预期（形态、数量、生长时间）

4. 耗材（采血管等）：
   - 检查外观、密封性、添加剂量
   - 抽样检测是否影响检测结果
```

**6.3.3 验收记录**

- 验收合格：标记为"已验收"，可投入使用
- 验收不合格：标记"不合格"，隔离存放，退回供应商或销毁
- 记录到《试剂/耗材验收测试记录表》JYK-FR-06-047

#### 6.4 库存管理 [§6.6.4]

**必须包含的内容**：
- 已验收和未验收试剂分区存放并明确标识
- 先进先出（FIFO）原则
- 效期管理：定期（至少每月）检查效期，近效期试剂提前预警
- 安全库存量设定
- 库存盘点（至少每季度一次）
- 记录到《试剂/耗材库存台账》JYK-FR-06-048

⚠️ 示例，请替换：
```
效期预警机制：
- 剩余有效期 ≤ 3 个月：黄色标签标记，优先使用
- 剩余有效期 ≤ 1 个月：红色标签标记，评估是否可在效期内用完
- 过期试剂：立即撤出使用区域，标记"过期"，按报废流程处理
```

#### 6.5 使用说明管理 [§6.6.5]

**必须包含的内容**：
- 试剂说明书（含中文标签/说明书）存放在使用区域或电子化归档
- 操作人员须阅读并理解试剂说明书
- 制造商更新说明书时及时替换
- 内部制备试剂须有自编使用说明

#### 6.6 试剂不良事件报告 [§6.6.6]

> LLM 须描述试剂不良事件的处理和报告程序：

```
试剂不良事件处理流程：

1. 发现异常（试剂外观异常、检测结果批量偏移、质控失控等）
       ↓
2. 立即停止使用该批次试剂 → 隔离保存
       ↓
3. 报告组长和技术负责人
       ↓
4. 初步调查：排除仪器、操作、样品等其他原因
       ↓
5. 确认为试剂问题 → 填写《试剂不良事件报告表》JYK-FR-06-049
       ↓
6. 联系供应商/制造商 → 提供样品和数据
       ↓
7. 评估影响 → 追溯该批次试剂已检测样品的结果
       ↓
8. 必要时向 NMPA 不良事件监测系统报告
       ↓
9. 采取纠正措施（更换试剂、调整方法、通知临床等）
```

### 6. 程序内容

#### 6.1 外部产品和服务的评审和批准 [§6.8.1]

**6.1.1 需评审的外部产品和服务范围**

| 类别 | 举例 |
|------|------|
| 产品类 | 试剂、耗材、设备、校准品、质控品、参考物质 |
| 服务类 | 设备维修/保养、校准/检定、EQA/PT 方案、IT 系统维护、物流运输、废物处理 |
| 检验类 | 转诊实验室检测服务 |

**6.1.2 评审标准**

> LLM 须列出对外部供应商的基本评审标准：

- 法定资质（营业执照、医疗器械经营许可证、生产许可证等）
- 产品注册/备案证明
- 质量管理体系证书（ISO 认证，如有）
- 技术能力（产品性能、服务水平）
- 售后服务能力（响应速度、培训支持）
- 价格合理性
- 历史供货/服务质量记录
- 信誉和市场口碑

#### 6.2 供应商资格确定、选择和绩效评估 [§6.8.1]

**6.2.1 供应商准入评审**

```
供应商准入流程：

1. 收集供应商资质材料（证照复印件、产品注册证、质量体系证书等）
       ↓
2. 质量负责人组织资料审查
       ↓
3. 必要时现场考察或要求提供样品试用
       ↓
4. 技术负责人审核技术能力
       ↓
5. 科室负责人审批
       ↓
6. 合格 → 纳入《合格供应商清单》JYK-FR-06-058
   不合格 → 不予准入，通知供应商
```

**6.2.2 供应商绩效评估**

> LLM 须描述供应商定期绩效评估机制：

**评估频率**：至少每年一次

**评估维度**：

⚠️ 示例，请替换：

| 评估维度 | 权重 | 评分标准 |
|---------|------|---------|
| 产品质量 | 30% | 不合格率、质量投诉次数 |
| 供货及时性 | 20% | 交货准时率 |
| 售后服务 | 20% | 响应速度、问题解决率 |
| 价格合理性 | 15% | 价格竞争力、价格稳定性 |
| 资质合规性 | 15% | 证照有效、产品注册合规 |

**评估结果处理**：
- 优秀（≥ 90分）：保持合作，优先考虑
- 合格（70~89分）：保持合作，要求改进
- 不合格（< 70分）：发出整改通知，限期整改；连续两次不合格移出合格名录

记录到《供应商绩效评估表》JYK-FR-06-059

**6.2.3 对绩效评估采取的行动**

- 评估结果反馈给供应商
- 不合格供应商的整改跟踪
- 严重不合格的供应商终止合作
- 评估结果作为续签合同和采购决策的依据

#### 6.3 转诊实验室管理 [§6.8.2]

**6.3.1 转诊实验室清单维护**

> LLM 须说明转诊实验室清单的建立和维护要求：

**必须包含的内容**：
- 转诊实验室选择标准（优先选择已获 CNAS/CAP 认可的实验室）
- 转诊实验室资质审查（实验室资质、认可范围覆盖外送项目）
- 清单内容：实验室名称、联系方式、认可状态、可委托项目、TAT

⚠️ 示例，请替换：
```
转诊实验室清单（示例）：

| 序号 | 实验室名称 | 认可状态 | 委托项目 | 参考TAT |
|------|-----------|---------|---------|---------|
| 1 | XX省临床检验中心 | CNAS 认可 | 基因检测、罕见病筛查 | 5~10 工作日 |
| 2 | XX医学检验所 | CNAS 认可 | 特殊免疫、过敏原全套 | 3~5 工作日 |
| 3 | 国家参考实验室 | CNAS 认可 | 疑难微生物鉴定 | 10~15 工作日 |
```

**6.3.2 转诊实验室能力评估**

**评估频率**：准入评估 + 至少每年一次定期评估

**评估内容**：
- 资质有效性（认可证书有效期）
- 检测能力覆盖范围
- EQA/PT 参加情况和结果
- TAT 达成率
- 结果准确性（与本实验室或其他实验室比对）
- 客户服务满意度

记录到《转诊实验室评估表》JYK-FR-06-060

**6.3.3 结果反馈和质量监控**

- 接收转诊实验室结果后，由技术人员审核结果的合理性
- 发现异常结果及时与转诊实验室沟通核实
- 定期统计外送项目的结果不一致率
- 监控 TAT 达成情况

#### 6.4 样品转介管理 [§6.8.2, §6.8.3]

**6.4.1 外送样品流程**

```
外送样品流程：

1. 临床医生开具外送检验申请
       ↓
2. 采集样品 → 标识（患者信息、项目、采集时间）
       ↓
3. 实验室接收 → 登记外送申请
       ↓
4. 样品准备：
   ├─ 核对样品与申请单一致性
   ├─ 按转诊实验室要求处理样品（离心、分装等）
   ├─ 包装（三层包装：一级容器+吸附材料+二级容器+外包装）
   └─ 填写《外送样品交接单》JYK-FR-06-061
       ↓
5. 运送 → 符合运输温度和安全要求
       ↓
6. 交接 → 转诊实验室签收确认
       ↓
7. 等待结果
       ↓
8. 接收结果 → 审核 → 录入 LIS → 发放报告
```

**6.4.2 患者信息保护 [§6.8.3(a)]**

- 外送申请单不包含不必要的患者个人信息
- 与转诊实验室签署保密协议
- 电子数据传输采用加密方式

**6.4.3 结果传达 [§6.8.3(b)]**

- 转诊实验室的检验结果由本实验室传达给临床请求者
- 报告中注明委托实验室名称和检测方法
- 本实验室对向临床发出的所有报告负责

**6.4.4 结果完整性 [§6.8.3(c)]**

- 接收转诊结果后进行合理性审核
- 对可疑结果与转诊实验室确认
- 报告格式统一，使用本实验室报告模板

# 生成规则（强制）

你是实验医学科 ISO 15189:2022 质量管理体系文件专家。生成任何文件时，必须遵循以下全部规则。

## 2. 编号与层级

- 标题层级：一级"1"，二级"1.1"，三级"1.1.1"，最深不超过四级
- 文件编号体系：QM（质量手册）、PER（人员）、DOC（文件记录）、QUA（质量）、SAF（安全）、ANA（分析过程）、INS（设备物资）、MET（方法性能）
- 记录表格编号：[前缀]-REC-[序号]

## 3. 标准引用

- 必须引用 ISO 15189:2022 对应条款，格式：ISO 15189:2022 第 X.X.X 条
- 必须同时引用 CNAS-CL02:2023 对应条款
- 每个程序要求须追溯到至少一个标准条款

## 4. 语言风格

- 中文为主，专业术语保留英文原文并附中文
- 操作步骤使用祈使句（"应""必须""shall"）
- 禁止模糊表述。生成内容中出现以下任何词汇，必须立即替换为右侧的精确表述：

| 禁止用法 | 必须替换为 |
|----------|-----------|
| 大约/约/左右/差不多 | 具体数值或范围（如"2-8℃""30±2分钟"） |
| 一般来说/通常/往往 | 删除修饰词直接陈述；有例外用"除…外"列出 |
| 及时/尽快/尽早 | 明确时限（如"2小时内""当日完成""发现后30分钟内"） |
| 相关人员/有关部门 | 具体岗位（如"质量负责人""检验科主任""专业组长"） |
| 适当/合理/必要时 | 触发条件（如"当温度偏差>2℃时""质控违反Westgard规则时"） |
| 定期/经常/不定期 | 频率（如"每月1次""每季度至少1次""每6个月"） |
| 做好记录/注意保存 | 指向表格（如"填写PER-REC-003""记录于INS-REC-005"） |
