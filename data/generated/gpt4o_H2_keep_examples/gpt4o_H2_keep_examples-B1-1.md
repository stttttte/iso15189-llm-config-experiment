# 1. 目的

用于指导实验医学科依据 **ISO 15189:2022 第 5-8 章** 和 **CNAS-CL02:2023** 开展内部审核，评价管理体系的符合性、有效性及持续改进情况。

# 2. 适用范围

适用于实验医学科 2026 年度内部审核活动，覆盖管理要求、资源要求、过程要求和管理体系要求。

# 3. 引用文件

1. ISO 15189:2022《医学实验室 质量和能力的要求》
2. CNAS-CL02:2023《医学实验室质量和能力认可准则》
3. ISO 19011《管理体系审核指南》
4. 实验医学科现行质量手册、程序文件、作业指导书及记录表单

# 4. 填写说明

1. 内审员应依据检查条款逐项实施审核。
2. 审核方法可采用：  
   - **文件查阅**  
   - **记录抽查**  
   - **现场观察**  
   - **人员访谈**  
   - **系统追溯**  
   - **样品/报告追踪**
3. 符合性判定栏填写方式：  
   - **符合**  
   - **基本符合**  
   - **不符合**  
   - **不适用**
4. 发现问题时，应在“不符合描述/观察”栏记录事实证据。
5. 涉及电子记录时，应记录系统名称、查询路径、记录编号或截图编号。
