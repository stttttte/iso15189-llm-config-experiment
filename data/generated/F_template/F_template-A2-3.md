## 1. 目的

规范实验医学科检验设备的全生命周期管理，建立从设备选型采购、安装验收、日常使用、校准维护到故障处置和报废退役的系统化管理制度。通过有效的设备管理确保所有用于检验活动的设备持续满足性能规格要求，保障检验结果的准确性、可重复性和可追溯性，符合 ISO 15189:2022 第 6.4 条全部要求。

---

## 2. 适用范围

本程序适用于实验医学科拥有、租赁或借用的全部与检验活动有关的设备，包括：

- **检测分析系统**：全自动生化分析仪、化学发光免疫分析仪、全自动血液分析仪、全自动凝血分析仪、全自动尿液分析工作站、全自动微生物培养鉴定系统、荧光定量 PCR 仪、基因测序仪、流式细胞仪、血气分析仪、特定蛋白分析仪、电泳分析系统等
- **样品前处理设备**：自动开盖/去盖机、自动分杯系统、离心机（低速、高速、微量）等
- **辅助仪器设备**：水浴箱、恒温培养箱、CO2 培养箱、厌氧培养箱、高压蒸汽灭菌器、干燥箱、生物安全柜、超净工作台、纯水制备系统等
- **冷链储存设备**：医用冰箱（2~8℃）、医用低温冰箱（-20℃）、超低温冰箱（-80℃）、液氮罐等
- **计量器具**：各规格微量移液器、电子分析天平、标准温度计/温度探头、电子秒表/计时器等
- **信息与自动化设备**：LIS 服务器及终端、条码系统（扫描器/打印机）、实验室自动化轨道系统、数据接口设备等
- **POCT 设备**：部署于临床科室的便携式或小型即时检测分析仪
- **采样相关设备**：真空采血器具、采血椅等

## 4. 术语和定义

| 术语 | 定义 |
|------|------|
| 设备 | 实验室正确执行检验活动所需的全部仪器、软件、测量标准、标准物质、参考数据、辅助器具及采集样品所需的器材（ISO 15189:2022 第 6.4.1 条定义范畴） |
| 设备全生命周期管理 | 对设备从选型论证、采购、安装验收、投入使用、持续维护、校准、故障管理到报废处置全过程的系统化管理 |
| 安装确认（IQ） | Installation Qualification，验证设备已按制造商规范正确安装，安装环境满足规定要求 |
| 运行确认（OQ） | Operational Qualification，验证设备在安装完成后各功能模块可正常运行 |
| 性能确认（PQ） | Performance Qualification，使用标准物质或已知特性的样品验证设备能达到预期性能指标 |
| 预防性维护（PM） | Preventive Maintenance，按计划在设备正常运行期间对其进行的保养活动，旨在降低故障率和延长使用寿命 |
| 设备登记册 | 集中记载实验室所有设备唯一标识和核心信息的受控文件 |
| 设备不良事件 | 设备在正常使用中发生的导致或可能导致患者伤害、人员伤害或检验结果严重偏差的事件 |

## 6. 程序内容

### 6.1 设备规划与采购

#### 6.1.1 年度设备需求规划

每年 10 月，设备管理员汇总各专业组提交的下年度设备需求申请，编制《年度设备购置计划》，经技术负责人审核、科室负责人审批后报医院设备管理部门。

需求申请应包含以下内容：
- 申请类别：新增、更新换代、扩充备份
- 设备名称与预期技术参数
- 购置理由及临床效益分析
- 预算估算（含设备、试剂、耗材、维保的三年总成本）
- 场地与配套设施评估

#### 6.1.2 选型评估与论证

技术负责人组织选型评估小组（含相关专业组组长和设备管理员），对拟采购设备进行综合评估。评估维度包括：

| 评估维度 | 评估要点 |
|----------|----------|
| 技术性能 | 检测速度、精密度、正确度、分析灵敏度、检出限、测量范围是否满足临床需求 |
| 试剂体系 | 开放式/封闭式、试剂品种是否齐全、单测试成本 |
| LIS 兼容性 | 数据接口协议（HL7/ASTM）、双向通信能力、条码识别能力 |
| 售后服务 | 响应时间承诺、本地工程师配置、备品备件供应能力 |
| 运营成本 | 试剂耗材年费用、维保合同费用、人力成本 |
| 同类设备口碑 | 同级医院使用反馈 |

评估结论记录于《设备选型评估报告》。

#### 6.1.3 采购实施

- 按照医院统一采购管理规定进行招标或议价采购
- 采购合同须明确约定：设备配置清单与技术参数、安装调试时间节点、人员培训安排（培训内容、课时、人数）、保修期限及保修范围、售后响应时间（故障报修后到达现场的小时数）、备件供应年限承诺、软件免费升级期限

#### 6.1.4 到货验收

设备到货后 3 个工作日内完成开箱验收：

1. 核对送货单与合同配置清单，逐一确认主机、附件和配件
2. 检查外包装和设备外观有无运输损伤
3. 核对设备铭牌（型号和序列号）与合同一致
4. 拍照留存开箱全过程
5. 如发现短缺或损伤，当日通知供应商并形成书面记录

完成后填写《设备到货验收记录》。

### 6.2 设备标识与登记

#### 6.2.1 唯一标识体系

所有设备实行唯一编号管理，编号规则：

```
EQ-[类别码][专业组码]-[四位序号]

类别码：
  A = 分析仪器
  B = 辅助设备
  C = 计量器具
  D = 信息设备
  P = POCT 设备

专业组码：
  01 = 临床化学
  02 = 临床免疫
  03 = 临床血液
  04 = 临床微生物
  05 = 临床体液
  06 = 临床分子
  00 = 通用/公共

示例：EQ-A01-0001 = 临床化学组第 1 台分析仪器
      EQ-B00-0012 = 通用辅助设备第 12 台
      EQ-C03-0003 = 临床血液组第 3 台计量器具
```

每台设备在机身醒目位置粘贴标准化的标识卡。标识卡包含：设备唯一编号、设备名称、当前管理状态。管理状态使用三色标签：

- 绿色：正常在用
- 黄色：停用/维修中
- 红色：已报废/待报废

#### 6.2.2 设备登记册管理

设备管理员维护电子版和纸质版《设备登记册》，每台设备登记以下信息：

- 设备唯一标识编号
- 设备名称、型号、规格
- 制造商名称及联系信息
- 设备序列号/出厂编号
- 供应商名称与合同编号
- 购置日期、到货日期、启用日期
- 安装位置（楼层-房间号-工位号）
- 设备责任人
- 当前管理状态
- 校准/检定到期日
- 年度维保合同到期日

登记册在以下事件发生后 2 个工作日内更新：新设备入册、设备位置变更、设备状态变更、设备报废注销。设备管理员每季度盘点一次，确保登记册信息与实物一致。

### 6.3 设备安装与验收

#### 6.3.1 安装管理

设备安装前，设备管理员牵头进行安装前准备检查：

| 检查项目 | 具体内容 |
|----------|----------|
| 环境条件 | 温度、湿度、通风条件符合制造商要求 |
| 电源 | 电压稳定性、接地电阻、是否需要 UPS |
| 供水/排水 | 纯水水质、供水压力、排水设施 |
| 网络 | LIS 网络端口、IP 地址分配 |
| 空间 | 设备摆放空间和维修操作空间充足 |

安装由制造商工程师完成，实验室指派操作人员全程参与和学习。

#### 6.3.2 验收测试方案

新设备安装完成后，技术负责人组织制定《设备验收测试方案》，明确验收项目、方法、判定标准和责任人。验收分三个阶段：

**阶段一：安装确认（IQ）**

- 核查设备安装位置和连接符合制造商规范
- 验证电源、供水、排水、排气系统功能正常
- 确认软件已安装、版本号正确
- 确认 LIS 接口联通并能双向传输

**阶段二：运行确认（OQ）**

- 逐项测试设备各功能模块
- 验证自动进样、试剂管理、结果计算、数据传输等环节
- 测试报警系统和安全联锁功能
- 验证开机自检和关机程序

**阶段三：性能确认（PQ）**

根据设备类型，执行以下全部或部分性能验证：

| 性能指标 | 验证方法 | 最低接受标准 |
|----------|----------|--------------|
| 批内精密度 | 单一水平质控品连续测定 20 次 | CV% ≤ 1/4 TEa（总允许误差） |
| 日间精密度 | 2 个水平质控品每日 1 次，连续 20 个工作日 | CV% ≤ 1/3 TEa |
| 正确度验证 | 有证参考物质或与参考实验室比对 | 偏倚 ≤ 1/2 TEa |
| 可报告范围验证 | 6 个以上浓度水平标准品 | R² ≥ 0.990，回收率 95%~105% |
| 携带污染率 | 3H3L 法或交替法 | ≤ 制造商声称值 |
| 分析灵敏度/检出限 | 空白/低值样品重复测定 | ≤ 制造商声称值 |
| 方法比对 | 40 份覆盖检测范围的患者样品 | 满足偏倚目标，r ≥ 0.975 |

全部验收数据整理为《设备验收测试报告》，技术负责人签署验收结论。

#### 6.3.3 设备重新投入使用验收

以下情形必须重新验收后方可使用：

| 情形 | 最低验收要求 |
|------|-------------|
| 更换核心部件（光源、检测器、流路板等） | IQ + OQ + 完整 PQ |
| 一般维修（更换非核心部件） | 至少 2 个水平质控品检测 + 10 份患者样品比对 |
| 设备搬迁 | IQ + OQ + 简化 PQ（精密度 + 质控品检测） |
| 停用超过 30 天重新启用 | OQ + 2 个水平质控品检测 |
| 软件版本升级 | OQ + 验证升级功能的正确性 + 质控品检测 |

### 6.4 设备操作管理

#### 6.4.1 操作授权制度

每台设备均须建立操作授权名单：

- 新员工须完成理论培训（设备原理、操作流程、安全须知）和实操培训（在带教老师指导下实际操作至少 5 个工作日）
- 经考核合格后由技术负责人签字授权，记入《设备操作授权名单》
- 授权名单每年 1 月复核一次，离岗超过 6 个月的人员须重新授权
- 未经授权人员不得独立操作检验设备

#### 6.4.2 操作指导文件管理

每台设备应配置以下操作指导文件并置于操作区域随手可及处：

- **快速操作卡**：A4 单页塑封，涵盖开关机步骤、日常操作要点、常见报警代码和处理方法
- **作业指导书**：详细操作 SOP，涵盖开机准备、校准操作、样品检测流程、质控执行、关机维护、安全注意事项
- **制造商使用说明书**：原版说明书保存在设备档案中，电子版存储在实验室共享文件夹供查阅

文件更新后，旧版收回、新版替换，操作人员签名确认已阅读。

#### 6.4.3 设备运行安全管理

各类安全风险的管控措施：

| 安全类别 | 管控措施 |
|----------|----------|
| 电气安全 | 所有设备可靠接地；核心设备接入 UPS 供电；禁止多孔排插串联；每年检测一次接地电阻 |
| 化学安全 | 废液桶定时倾倒（液面达 2/3 时更换）；酸碱废液分别收集；溢出时按化学品泄漏应急预案处理 |
| 机械安全 | 离心机须平衡后方可启动，运行中禁止开盖；自动化轨道运行区域张贴"请勿伸手"警示标识 |
| 生物安全 | 处理开放性样品的设备操作人员佩戴手套、口罩和护目镜；设备送修或报废前须彻底消毒 |
| 信息安全 | 设备操作系统设置密码登录；LIS 传输采用内网，禁止外网连接；软件参数修改须授权 |

### 6.5 设备校准

#### 6.5.1 校准管理要求

所有影响检验结果的设备和器具均须纳入校准管理，设备管理员维护《校准计划台账》，内容包括：设备编号、校准方式、校准频率、校准到期日、校准实施者。

| 设备/器具 | 校准方式 | 校准频率 | 触发校准的额外条件 |
|-----------|----------|----------|---------------------|
| 全自动分析仪（生化、免疫等） | 配套校准品校准 | 更换新批号试剂时、维修后、质控持续偏移时；至少每 6 个月 1 次 | 质控违反 Westgard 规则且排除非校准因素后 |
| 血液分析仪 | 配套校准品校准 | 更换新批号试剂时、维修后；至少每 6 个月 1 次 | 同上 |
| 移液器 | 称量法（纯水称重） | 每季度自校 1 次；每年送第三方计量机构校准 1 次 | 跌落、碰撞或发现吸液异常后 |
| 电子天平 | 外部标准砝码校准 | 每个工作日使用前 | 搬动后 |
| 温度计/温度探头 | 冰点法或与有证标准温度计比对 | 每年 1 次 | — |
| 离心机转速 | 光电转速计验证 | 每年 1 次 | 异常振动或噪音后 |
| 水浴箱/培养箱温度 | 有证温度计多点验证 | 每半年 1 次 | 维修后 |
| 电子计时器 | 与授时中心标准时间比对 | 每年 1 次 | — |

#### 6.5.2 校准结果处理

- 校准合格：记录校准数据，粘贴或更新校准标签（含校准日期和下次到期日），设备继续使用
- 校准不合格：立即停用设备，粘贴"停用"标识；分析原因（设备故障、校准品问题、操作失误等）；故障修复后重新校准；评估上次校准合格至本次不合格期间出具结果的可靠性

校准记录统一填入《设备校准记录表》。

### 6.6 预防性维护

#### 6.6.1 维护计划编制

设备管理员于每年 12 月编制下年度《设备预防性维护计划表》，涵盖全部在用设备，维护内容和周期参照制造商说明书和设备实际使用强度确定。计划经技术负责人审核后执行。

#### 6.6.2 日常维护和保养执行

**日保养**（操作人员每日完成）：

- 开机自检记录
- 检查试剂余量和废液容量
- 清洁样品加样区域
- 打印机色带/碳粉检查
- 记录冰箱温度

**周保养**（操作人员/指定维护人每周完成）：

- 管路冲洗和清洗液更换
- 加样针/吸液针清洁
- 检查密封件和管路接头
- UPS 状态指示灯检查
- 纯水系统电阻率检测

**月保养**（设备管理员/指定维护人每月完成）：

- 深度管路清洗和去蛋白
- 光学系统清洁（光源、滤光片、检测器窗口）
- 机械传动系统润滑
- 系统日志检查
- 维护执行情况月度汇总

**季度/半年保养**（设备管理员组织或联系厂家）：

- 关键耗材更换（比色杯、反应杯、灯泡、管路等，按制造商周期）
- 管路密封性压力测试
- 计量器具自校
- 冰箱冷凝器清洁
- 安全设施功能测试

**年度保养**（制造商工程师或授权第三方）：

- 全面维护保养，含全部易损件检查和更换
- 整机性能测试和校准
- 软件系统检查和更新
- 出具维护保养报告

#### 6.6.3 维护记录管理

- 日常保养记录填写于《设备日常维护保养记录表》，每台设备一本
- 月度以上维护活动填写《设备维护/维修执行记录表》，存入设备档案
- 设备管理员每月统计维护计划执行率，目标值 ≥ 95%

### 6.7 设备故障管理

#### 6.7.1 故障发现与应急处置

设备出现以下情形之一时，操作人员须立即停止使用：

- 设备运行异常（异常噪音、振动、温度过高、漏液等）
- 自检不通过或持续报警
- 质控结果超出控制范围且无法通过简单排障恢复
- 检验结果明显异常（与临床情况严重不符）

停用后操作人员须：
1. 在设备上放置黄色"停用/故障"标识牌
2. 在 15 分钟内通知专业组组长和设备管理员
3. 在《设备故障/停用记录表》上记录故障现象、发现时间和初步判断

#### 6.7.2 故障对检验结果的影响评估

技术负责人在接到设备故障通知后 4 小时内完成影响评估：

- 确定故障发生的可能时间节点（以最近一次质控合格时间为起点）
- 回溯该时段内所有在该设备上检测的样品清单
- 判断故障类型是否可能影响检验结果的准确性
- 如确认结果受影响，立即执行以下措施：
  - 通知临床科室暂缓使用受影响的结果
  - 调取留样在备用设备或替代方法上复检
  - 发出更正报告
- 全部评估和处置过程记录于《设备故障影响评估记录》，存入设备档案

#### 6.7.3 维修管理

- 简单故障：设备管理员或厂家远程指导排障
- 复杂故障：联系制造商售后或签约第三方维修机构，约定到场时间
- 维修过程记录：故障描述、维修日期、维修内容、更换部件明细、维修人员签名
- 维修结束后，执行本程序 6.3.3 的重新投入使用验收
- 验收通过后，移除"停用/故障"标识，更新设备登记册中的维修记录

#### 6.7.4 故障期间的业务持续性保障

| 故障情形 | 应对措施 |
|----------|----------|
| 有同类备用设备 | 启用备用设备（须在验收合格状态），告知操作人员切换 |
| 无备用设备但非急诊项目 | 样品暂存于规定温度条件下，设备恢复后优先检测 |
| 急诊项目且无备用设备 | 立即联系合作实验室转送样品检测；同时通知急诊科预计恢复时间 |
| 批量设备故障（如停电） | 启动应急预案，启用 UPS 维持核心设备运行；联系院方后勤保障 |

### 6.8 设备不良事件管理

#### 6.8.1 事件分类与识别

| 事件类别 | 描述 | 示例 |
|----------|------|------|
| I 类（严重） | 导致或可能导致患者死亡、严重伤害 | 设备漏电致操作人员触电；试管破碎致样品喷溅伤眼 |
| II 类（中度） | 导致检验结果严重偏差并已影响临床决策 | 系统性校准漂移导致一批结果异常偏高 |
| III 类（轻度） | 设备异常但未造成实际伤害或结果偏差 | 进样器卡管致样品未检测到 |

#### 6.8.2 调查与纠正

1. 发现不良事件后立即停用设备并保全现场和证据
2. 设备管理员在 24 小时内填写《设备不良事件报告表》
3. 质量负责人牵头成立调查小组，在 5 个工作日内完成调查
4. 查明根本原因，制定纠正措施和预防措施
5. 跟踪措施执行效果，在 30 天内关闭事件处理

#### 6.8.3 外部报告

- I 类事件：在 24 小时内通过"国家医疗器械不良事件监测信息系统"上报，同时报告医院设备科和医务部
- II 类事件：7 个工作日内在线上报
- III 类事件：15 个工作日内在线上报
- 所有事件同步报告医院医疗器械临床使用安全管理委员会

#### 6.8.4 制造商安全通知与召回处理

- 设备管理员负责关注制造商发布的安全通知、现场安全整改通知和产品召回信息
- 收到通知后 24 小时内清查受影响设备并评估风险
- 涉及安全性的召回：立即停用受影响设备，按制造商指引执行整改
- 涉及性能的召回：评估是否继续使用，如影响结果准确性则停用
- 整改完成后执行重新验收
- 召回处理全过程记录存入设备档案

### 6.9 设备报废与退役

#### 6.9.1 报废标准

符合下列条件之一的设备可申请报废：

- 达到制造商声明的使用寿命终期
- 主要性能指标不能满足检验要求且维修后仍无法恢复
- 关键零部件已停产，无法获得替代品
- 单次维修费用超过设备原值的 30% 或累计维修费用超过原值 50%
- 存在不可消除的安全隐患
- 因技术进步已被更先进的设备替代且无继续使用价值

#### 6.9.2 报废审批流程

1. 使用部门提交《设备报废申请表》，附设备运行状况、故障维修记录和报废理由
2. 设备管理员核查设备档案，出具设备状况说明
3. 技术负责人出具技术鉴定意见，确认设备确已无法满足检验需求
4. 科室负责人签字审批
5. 报送医院设备科/国有资产管理部门审批
6. 财务部门办理资产注销手续

#### 6.9.3 报废执行

- 对报废设备进行彻底消毒处理（按生物安全要求执行）
- 清除设备内部残留的生物样品、化学试剂和废液
- 拆除设备唯一标识卡
- 删除设备中的患者数据和检验数据（信息安全要求）
- 在设备登记册中将状态变更为"已报废"，注明报废日期和批准文号
- 设备档案完整保存至报废后至少 3 年
