## 第二部分：填写说明

### 一、表格用途

本表用于记录医学检验科所有检验设备和计量器具的校准活动。每次校准事件（无论是内部校准还是外部送检）均应填写本表，作为设备档案的组成部分。

### 二、填写时机

| 场景 | 时机 | 填表要求 |
|------|------|---------|
| 定期校准 | 按校准计划到期执行 | 校准当日完成主表填写，审核签名应在 5 个工作日内完成 |
| 新设备验收校准 | 设备到货安装后、投入使用前 | 作为设备验收资料的组成部分 |
| 维修后校准 | 维修完成后、重新投入使用前 | 在"校准触发原因"栏勾选"维修后" |
| 质控失控后校准 | 质控失控排查确认需重新校准时 | 在"校准触发原因"栏勾选"质控失控后" |
| 试剂批号更换后校准 | 分析仪器更换校准品或试剂后 | 在"校准触发原因"栏勾选"试剂批号更换" |

### 三、关键字段详解

#### 3.1 校准环境条件

- **为什么要记录**：精密设备（天平、分光光度计、移液器等）的校准结果受环境温度、湿度、气压影响。记录校准时的环境条件是校准结果可溯源的前提条件。
- **允许范围的确定**：
  - 首先查阅设备制造商说明书中规定的操作环境要求。
  - 其次查阅校准方法（JJG/JJF）中规定的校准条件。
  - 如无明确规定，按实验室一般环境标准执行：温度 18～28℃，相对湿度 30%～80%。
- **测量仪器编号**：填写用于测量环境参数的温湿度计、气压计的设备编号，该仪器本身应已完成有效校准。

#### 3.2 测量不确定度

- **外部校准**：测量不确定度由校准机构在校准证书中给出，直接抄录至本表对应栏位。
- **内部校准**：实验室应依据 JJF 1059.1 评定测量不确定度。不确定度的评定过程和计算过程可单独形成技术报告，本表仅记录最终结果。
- **包含因子 k**：通常取 k=2，对应约 95% 的置信水平。如校准证书中 k 取其他值，如实填写。
- **满足预期用途的判定**：将测量不确定度与检验项目的允许总误差进行比较。当不确定度小于允许总误差的 1/3 时，通常认为满足要求。

#### 3.3 校准数据记录

- **测量次数**：一般应至少测量 3 次，取平均值。如校准方法另有规定，从其规定。
- **偏差计算**：偏差 = 实测均值 - 标准值。保留有效位数应与测量值一致。
- **判定标准**：偏差的绝对值不超过最大允许误差时判定为合格。最大允许误差依据校准规程或设备技术指标确定。

#### 3.4 签名要求

- **填表人**：必须是实际参与校准操作或亲自接收外部校准数据的人员。不得由他人代签。
- **一级审核**：质量负责人从记录完整性、数据逻辑性角度审核，确认无遗漏、无明显错误。
- **二级复核**：科室负责人或技术负责人从技术角度确认校准结论的正确性，以及设备是否可继续用于检验活动。

### 四、续页说明

校准项目超过 5 项时，可使用续页。续页应在右上角注明：

- 设备名称和编号
- 校准日期
- 页码（如"第 2 页 / 共 3 页"）

续页末尾由填表人签名确认。

## 第三部分：记录更正规则

### 一、基本原则

校准记录是设备管理和检验质量的关键文件证据。任何已完成的记录均应保持原始状态，不得擅自篡改。确需更正时，应确保更正过程完全可追溯。

### 二、纸质记录更正规范

1. **发现错误后**：在错误内容上画一条水平线（单划线），确保原始内容仍清晰可辨。
2. **填写正确内容**：在划线旁边或上方书写正确信息。
3. **更正人确认**：在更正处旁签署更正人姓名（或姓名缩写）和更正日期。
4. **必要时说明原因**：如更正涉及数值数据的变更，应在备注栏注明更正原因。

### 三、绝对禁止事项

| 禁止行为 | 原因 |
|---------|------|
| 使用涂改液、修正带或修正贴纸 | 覆盖原始信息，破坏可追溯性 |
| 使用橡皮擦除 | 完全消除原始信息，无法审计 |
| 撕毁记录页或重新誊写 | 破坏原始文件完整性 |
| 代替他人签名 | 违反签名真实性原则 |
| 使用铅笔填写（初始记录时） | 易被擦除修改，不具备证据效力 |

### 四、电子记录更正

在 LIS/LIMS 系统中，记录更正须满足审计追踪要求：系统自动保留原始数据、更正后数据、更正人身份、更正时间戳和更正原因。电子记录的更正权限应受到用户角色控制。

### 五、保存期限

| 文件类型 | 保存期限 | 保存方式 |
|---------|---------|---------|
| 校准记录表纸质原件 | 设备使用寿命期 + 3 年（最少 6 年） | 存入设备档案，专人管理 |
| 校准证书原件/复印件 | 同校准记录表 | 附于校准记录表之后 |
| 电子扫描备份 | 同纸质原件 | LIMS 系统存储，定期备份 |
| 测量不确定度评定报告 | 同校准记录表 | 存入设备技术档案 |
