## 1. 各字段定义与要求

| 字段 | 定义 | 取值/格式要求 |
|------|------|---------------|
| 设备名称（A1） | 设备完整的中文名称 | 与设备铭牌及设备登记册保持完全一致，禁止缩写 |
| 制造商（A2） | 设备生产企业的注册名称 | 填写全称 |
| 设备型号（A3） | 制造商在设备上标注的产品型号代码 | 与铭牌一致 |
| 出厂序列号（A4） | 制造商在出厂时分配的唯一识别编号 | 与铭牌一致 |
| 科室管理编号（A5） | 实验室内部赋予该设备的唯一管理编号 | 按科室编号规则填写 |
| 所属专业组（A6） | 使用该设备的专业组别 | 在预设选项中勾选一项 |
| 温度/湿度/气压（B1-B3） | 校准操作开始前，在设备所在位置读取的环境参数 | 使用在校准有效期内的温湿度计读取，数值精确到小数点后一位 |
| 校准原因（C3） | 触发本次校准的具体事件或条件 | 在预设选项中勾选，选择"其他"时须在说明栏填写具体事件描述 |
| 校准周期（C4） | 两次校准之间的最长间隔时间 | 依据制造商说明书、国家计量检定规程或行业标准确定，单位为月 |
| 校准品名称（D6） | 校准时使用的参考标准品的完整产品名称 | 填写完整商品名称和规格，禁止仅写缩写 |
| 校准品批号（D9） | 校准品外包装上印刷的批号标识 | 直接照抄包装标签上的批号 |
| 计量溯源性声明（D13） | 校准品量值可追溯至的上级参考标准的描述 | 查阅校准品说明书中的"溯源性"或"Traceability"段落，摘录上级参考物质或参考方法名称及编号 |
| 校准参数（E列） | 本次校准覆盖的具体测量物理量或化学量 | 每行一个参数，如"波长正确度""吸光度重复性""转速""温度示值误差" |
| 实测值（E列） | 被校设备对标准值的实际测量读数 | 至少测量 2 次，分别记录，填写有效数字位数与标准值一致 |
| 偏差（E列） | 实测均值与标准值之间的差异 | 偏差 = 实测均值 - 标准值 |
| 最大允许偏差（E列） | 判定校准合格与否的偏差限值 | 依据制造商说明书、计量检定规程或行业标准确定 |
| 合成标准不确定度 uc（F3） | 各不确定度分量合成后的标准不确定度 | 按 JJF 1059.1 方法计算 |
| 扩展不确定度 U（F5） | uc 乘以包含因子 k 后的结果，用于限定被测量真值的区间 | U = k x uc |
| 质控物验证（G部分） | 校准后用质控物确认校准有效性的检测 | 在校准完成后 2 小时内执行，至少使用 2 个浓度水平 |

## 2. 填写通用规定

1. **日期格式**：统一使用"YYYY年MM月DD日"
2. **时间格式**：统一使用 24 小时制"HH时MM分"
3. **数值要求**：所有测量数值必须标注单位；有效数字位数与对应标准值/说明书要求一致
4. **签名要求**：使用黑色或蓝黑色签字笔手写签名；禁止使用铅笔、红色笔、印章或电子签名代替手签（电子系统另有规定的除外）
5. **必填逻辑**：当第 E 部分判定存在不合格项，或第 G 部分出现失控水平时，第 I 部分"校准不合格处理"为强制填写项
6. **审核时限**：填表人在校准完成当日将记录提交一级审核人；一级审核人在 3 个工作日内完成审核签字；二级复核人在 5 个工作日内完成复核签字
7. **证书归档时限**：外部校准证书原件在收到后 5 个工作日内归入设备档案

# 附录二：记录更正规则

> 依据 ISO 15189:2022 第 7.7 条、CNAS-CL02:2023 第 7.7 条

## 1. 纸质记录更正

1. **更正方法**：在需要更正的错误文字或数字上画一条单横线，使原始内容清晰可辨认
2. **写入正确内容**：在原始内容的旁边（上方空白处或右侧）书写正确内容
3. **更正标注**：在更正处旁注明更正日期（YYYY.MM.DD 格式）并由更正人本人签全名
4. **禁止的行为**：
   - 禁止使用涂改液（修正液）覆盖
   - 禁止使用修正带覆盖
   - 禁止使用橡皮擦擦除
   - 禁止使用胶带粘贴覆盖
   - 禁止刮擦纸面使原字迹消失
   - 禁止撕毁原始记录页
   - 禁止以"重新抄写一份"替代更正
   - 禁止以任何方式使原始记录内容无法辨认

## 2. 电子系统记录更正

在 LIS、LIMS 或其他电子信息系统中修改校准记录时：
- 系统须自动保留原始数据不被覆盖
- 系统须自动记录修改时间（精确到秒）
- 系统须自动记录修改操作人身份
- 系统须自动记录修改原因（由操作人填写）
- 上述信息构成完整的审计追踪 (Audit Trail)，不可被删除或篡改

## 3. 记录保存

- 纸质记录原件保存期限：6 年（自校准日期起计算）
- 保存环境：干燥、通风、防潮、防虫蛀的专用档案柜中
- 电子记录保存：LIS/LIMS 系统服务器，每日自动备份，异地备份每周 1 次
- 保存期满后按科室《记录控制程序》执行销毁，销毁前须经质量负责人审批，并保留销毁清单
