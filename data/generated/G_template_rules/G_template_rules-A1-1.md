## 1 目的

建立实验医学科人员培训与能力评估的系统化管理制度，规范培训需求分析、培训计划制定、培训实施、培训效果评估、人员能力评估、继续教育管理的全过程，确保所有从事检验活动的人员持续具备胜任其岗位职责的能力，满足 ISO 15189:2022 第 6.2 条及 CNAS-CL02:2023 第 6.2 条的全部要求。

## 2 适用范围

2.1 本程序适用于实验医学科所有从事检验前过程、检验过程和检验后过程活动的人员，包括正式在编人员、合同制人员、进修人员、实习人员及参与床旁检验（Point-of-Care Testing, POCT）的临床科室人员。

2.2 本程序覆盖以下管理活动：

- 年度培训计划的制定与执行
- 入职培训、在职培训、专业技术培训、安全培训、合规培训、POCT 专项培训
- 培训课件与签到管理
- 培训效果评估与考核
- 新员工入岗能力评估、年度能力评估、复岗能力评估、专项能力评估
- 评估不合格的处理与复评
- 继续教育学分管理
- 能力评估与质量监督的联动机制

## 3 标准依据

### 3.1 ISO 15189:2022 条款要求

| 条款编号 | 条款内容要点 |
|----------|-------------|
| ISO 15189:2022 第 6.2.1 条 | 实验室应配备具有所需能力的人员执行其活动 |
| ISO 15189:2022 第 6.2.2 条 | 确定各项职能所需的能力（教育、资格、培训、技术知识、技能和经验），按文件化程序在规定时间间隔内评估人员能力；评估方法应包括直接观察、设备维护观察、监控检验记录和结果、评审工作记录、评估解决问题的能力、检查特别提供的样本；新员工入职后 6 个月内至少进行 2 次能力评估，之后每 12 个月评估 1 次 |
| ISO 15189:2022 第 6.2.3 条 | 授权人员执行特定活动 |
| ISO 15189:2022 第 6.2.4 条 | 为所有人员提供继续教育方案并证明其效果；对有权限进入检验信息系统的人员提供保密性培训 |
| ISO 15189:2022 第 6.2.5 条 | 保存人员记录，包括教育和专业资格、培训和能力评估记录 |

## 4 职责

| 角色 | 职责内容 |
|------|----------|
| 科主任 | 批准年度培训计划和年度能力评估计划；批准培训预算；审核年度培训总结报告和年度能力评估汇总报告 |
| 技术负责人 | 制定各岗位能力要求；组织编排年度能力评估计划；审核技术培训内容和培训师资；审核能力评估结果；签署独立上岗确认和复岗确认 |
| 质量负责人 | 组织合规培训（公正性、保密性、安全）；监督培训和评估流程的执行合规性；协调能力评估与质量监督的联动 |
| 培训管理员 | 制定年度培训计划；组织培训实施；管理培训课件与签到记录；维护培训档案；统计继续教育学分；编制培训执行追踪报告 |
| 各专业组组长 | 提出本组培训需求；协助组织本组专业培训和带教；执行本组人员能力评估；记录评估结果并提出培训建议 |
| 质量监督员 | 参与直接观察评估；提供日常工作质量监控数据；发现人员能力问题后触发专项评估 |
| 全体人员 | 按计划参加培训并完成考核；配合能力评估安排并完成各项评估任务；保存继续教育证明 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 能力评估（Competence Assessment） | 通过系统化方法评价人员执行指定任务所需知识、技能和行为的过程 |
| 直接观察（Direct Observation） | 评估人员在工作现场直接观看被评估人执行操作的评估方法 |
| 盲样测试（Blind Sample Testing） | 将已知结果的样本以常规标本形式提供给被评估人检测，评估其检验能力 |
| 入岗评估（Initial Competence Assessment） | 新员工入职后在规定期间内进行的初始能力评估 |
| 复岗评估（Return-to-Work Assessment） | 人员离岗后重新返回岗位时进行的能力评估 |
| 加密评估（Enhanced Frequency Assessment） | 对复杂或高风险检验程序增加评估频次的安排 |
| 入职培训（Orientation Training） | 新员工入职后接受的初始培训，包括通用培训和岗位专业培训 |
| 在职培训（On-the-Job Training） | 在岗人员日常接受的持续性培训活动 |
| 继续教育（Continuing Education） | 获取和更新专业知识、技能的持续性教育活动，以学分计量 |
| 合规培训（Compliance Training） | 涉及法规遵从、公正性、保密性、患者权益等方面的培训 |
| 培训效果评估（Training Effectiveness Evaluation） | 通过考核、实操、满意度调查等方式衡量培训达到预期目标的程度 |
| 检验前过程（Pre-examination Process） | 从临床医生开具检验申请单开始，到样本准备就绪可供检验为止的全部过程 |
| 检验过程（Examination Process） | 确定样本特性值的一组操作 |
| 检验后过程（Post-examination Process） | 检验完成后的全部过程，包括结果审核、报告发送、样本保存与处置 |
| 能力验证（Proficiency Testing） | 利用实验室间比对评价参加者的能力 |

## 6 程序内容

### 6.1 培训管理

#### 6.1.1 年度培训需求分析

每年 12 月由培训管理员组织开展下年度培训需求分析，信息来源包括：

| 信息来源 | 分析内容 |
|----------|----------|
| 能力评估结果 | 本年度能力评估中发现的知识或技能薄弱环节（依据 PER-SOP-002 第 6.2 节评估记录） |
| 质量监督反馈 | 日常质量监督中发现的操作偏差和规范性问题 |
| 不符合与纠正措施 | 内审、外审、投诉中发现的与人员能力相关的问题项 |
| 新项目与新设备 | 下年度计划开展的新检验程序或新采购的设备 |
| 法规与标准更新 | 新发布或修订的法律法规、行业标准 |
| 人员变动 | 新入职人员、岗位调整人员 |
| 管理评审建议 | 管理评审中提出的培训改进建议 |
| 人员自身发展需求 | 人员提出的专业发展和深造需求 |

培训管理员将需求分析结果汇总后形成《培训需求分析报告》，作为年度培训计划编制的依据。

#### 6.1.2 年度培训计划编制

1. 培训管理员依据需求分析结果，于每年 1 月 15 日前编制完成《年度培训计划》（PER-REC-001）。
2. 计划内容应包含：培训课程名称、培训类型（入职/在职/专业/安全/合规/POCT）、培训对象、培训师资、计划培训时间、培训学时、考核方式、预算。
3. 年度培训计划应确保覆盖本程序第 6.1.4 节规定的所有必修培训主题。
4. 质量负责人审核后，由科主任批准发布。

#### 6.1.3 计划执行追踪

1. 培训管理员每季度编制 1 次《培训执行追踪表》，追踪内容包括：已完成课程数/计划课程数、实际参训人次、考核通过率、未完成课程及延期原因。
2. 年底编制《年度培训总结报告》（PER-REC-002），内容包括：培训计划完成率、各类培训参训率、考核通过率、继续教育学分达标率、培训满意度统计、改进建议。
3. 年度培训总结报告提交科主任审批后，纳入管理评审输入。

#### 6.1.4 培训类型与内容要求

##### 6.1.4.1 入职培训

新员工入职后 2 周内完成以下全部培训模块：

| 培训模块 | 培训内容 | 学时要求 | 考核方式 |
|----------|----------|----------|----------|
| 科室概况 | 组织架构、工作流程、科室文化、岗位职责 | 2 学时 | 签字确认 |
| 质量管理体系 | 质量方针与质量目标、质量管理体系框架、文件管理制度 | 2 学时 | 笔试（合格线 80 分） |
| 生物安全 | 实验室生物安全等级、个人防护装备使用、职业暴露应急处置、废弃物处理 | 4 学时 | 笔试（合格线 80 分）+ 实操考核 |
| 信息安全与保密 | 患者数据保密要求、检验信息系统使用规范、密码管理、数据传输加密要求 | 2 学时 | 笔试（合格线 80 分） |
| 公正性与伦理 | 公正性承诺、利益冲突识别与报告、患者权益保护 | 2 学时 | 签署《公正性承诺书》 |
| 标准操作规程（SOP）体系 | 文件分类与编号体系、SOP 查阅与执行规范、变更管理 | 2 学时 | 笔试（合格线 80 分） |
| 岗位专业培训 | 本岗位 SOP 学习、设备操作、质控物操作、样本处理 | 依岗位确定，不少于 20 学时 | 实操考核 |

入职培训全部模块考核合格后，方可进入带教阶段。考核不合格者在 2 周内安排补考，补考仍不合格者须重新参加该模块培训。

##### 6.1.4.2 在职培训

| 培训内容 | 频率要求 | 培训方式 |
|----------|----------|----------|
| SOP 更新培训 | SOP 修订发布后 7 个工作日内完成 | 集中讲解 + 签字确认 |
| 新检验程序开展培训 | 新程序上线前完成 | 理论授课 + 实操考核 |
| 新设备操作培训 | 设备启用前完成 | 厂家培训 + 科内转训 + 实操考核 |
| 质量改进培训 | 每季度至少 1 次 | 案例分析、专题研讨 |
| 设备故障排除培训 | 每 6 个月至少 1 次 | 模拟演练 |

##### 6.1.4.3 专业技术培训

| 培训内容 | 频率要求 | 适用人员 |
|----------|----------|----------|
| 专业学术讲座 | 每月至少 1 次 | 全科人员 |
| 疑难病例讨论 | 每月 1~2 次 | 对应专业组人员 |
| 新技术与新方法学习 | 新技术引进前完成 | 对应专业组人员 |
| 外部学术会议与培训班 | 依审批安排 | 外派人员 |

##### 6.1.4.4 安全培训

| 培训内容 | 频率要求 | 适用人员 |
|----------|----------|----------|
| 生物安全年度培训 | 每年至少 1 次 | 全科人员 |
| 消防安全培训 | 每年至少 1 次 | 全科人员 |
| 化学品安全培训 | 每年至少 1 次 | 接触化学危险品的人员 |
| 职业暴露处置演练 | 每年至少 1 次 | 全科人员 |
| 应急预案演练 | 每年至少 1 次 | 全科人员 |

##### 6.1.4.5 合规培训

| 培训内容 | 频率要求 | 对应标准要求 |
|----------|----------|-------------|
| 公正性培训 | 每年至少 1 次 | ISO 15189:2022 第 4.1.1.3 条 |
| 保密性培训 | 每年至少 1 次 | ISO 15189:2022 第 4.1.1.4 条 |
| 患者权益保护培训 | 每年至少 1 次 | 相关法律法规 |
| 伦理培训 | 每年至少 1 次 | 科研伦理、知情同意 |
| 法律法规更新培训 | 新法规发布后 30 日内完成 | 新发布的法规、标准 |

合规培训须有独立的签到记录和考核记录，即使与其他培训整合进行。公正性培训须签署《公正性承诺书》（PER-REC-012），保密性培训须签署《保密协议》（PER-REC-013），签署记录纳入个人人事档案。

##### 6.1.4.6 POCT 专项培训

| 培训内容 | 适用人员 | 频率要求 |
|----------|----------|----------|
| POCT 设备操作培训 | POCT 操作人员（含临床科室操作人员） | 上岗前完成 + 每年复训 1 次 |
| POCT 质量控制培训 | POCT 操作人员 | 上岗前完成 + 每年复训 1 次 |
| POCT 结果判读培训 | POCT 操作人员 | 上岗前完成 |
| POCT 管理制度培训 | 实验医学科 POCT 管理人员 | 制度更新后 7 个工作日内完成 |

#### 6.1.5 培训课件与签到管理

##### 6.1.5.1 培训课件管理

1. 培训课件由培训师编制，技术负责人或质量负责人审核后使用。
2. 课件应注明版本号和编制日期。涉及 SOP 内容的课件应与当前有效版本保持一致，SOP 修订后课件应在 14 日内同步更新。
3. 培训课件按类别归档于实验室信息管理系统或指定共享文件夹，由培训管理员统一维护。

##### 6.1.5.2 培训签到管理

1. 每次培训应使用《培训签到表》（PER-REC-003），记录参加人员亲笔签名、培训日期与时间。
2. 线上培训应保存系统登录记录或在线签到截图作为参训凭证。
3. 培训签到表与培训课件、考核材料一并归档保存。

#### 6.1.6 培训效果评估

##### 6.1.6.1 评估方式

| 评估方式 | 具体说明 | 适用场景 |
|----------|----------|----------|
| 笔试或在线考核 | 理论知识测试，满分 100 分，合格线 80 分 | 理论培训 |
| 实操考核 | 由培训师在现场评分，依据操作评分表逐项评分 | 技能培训 |
| 满意度调查 | 使用《培训满意度调查表》（PER-REC-009）评价课程内容、师资水平和培训组织 | 所有培训 |
| 行为观察 | 培训后 1 个月内，由专业组组长或质量监督员观察工作行为是否改善 | 重点培训 |
| 结果分析 | 培训后 3 个月内，对比培训前后检验质量指标变化 | 针对性培训 |

##### 6.1.6.2 考核不合格处理

1. 笔试或实操考核不合格者，在 2 周内安排 1 次补考。
2. 补考仍不合格者，须重新参加该课程培训后再次补考。
3. 关键培训（生物安全培训、SOP 培训、设备操作培训）未通过考核者，不得独立上岗操作。
4. 所有考核结果记录于《培训考核记录表》（PER-REC-008）并归入个人培训档案。

#### 6.1.7 外部培训与会议管理

1. 人员参加外部培训或学术会议，须事先填写《外部培训/会议申请表》（PER-REC-005），经专业组组长同意、科主任批准后方可参加。
2. 培训或会议结束后 14 日内，参加人员须向培训管理员提交以下材料：
   - 培训或会议总结报告（使用 PER-REC-011 模板）
   - 证书或学分证明复印件
   - 科室内转训安排（由培训管理员协调，在返回后 30 日内完成转训分享）
3. 转训分享应以专题讲座形式向本专业组或全科人员进行。

#### 6.1.8 继续教育学分管理

##### 6.1.8.1 学分要求

依据省/市卫生健康委员会规定，各类人员年度继续教育学分要求如下：

| 人员类别 | 年度学分要求 | 其中 I 类学分 | 其中 II 类学分 |
|----------|-------------|--------------|---------------|
| 高级职称 | 不少于 25 分 | 不少于 5 分 | 不少于 20 分 |
| 中级职称 | 不少于 25 分 | 不少于 5 分 | 不少于 20 分 |
| 初级职称 | 不少于 25 分 | 不少于 5 分 | 不少于 20 分 |

##### 6.1.8.2 学分登记与追踪

1. 培训管理员维护《继续教育学分登记表》（PER-REC-004），按人员逐条登记每次继续教育活动的名称、日期、学时、学分类别和学分数。
2. 每季度末汇总 1 次学分完成进度，对学分完成率低于当季度目标（即累计完成年度学分数的 25%、50%、75%）的人员，由培训管理员以书面形式发出提醒。
3. 年底核查全员学分达标情况，将达标率和不达标人员名单纳入年度培训总结报告。

### 6.2 能力评估管理

#### 6.2.1 各项职能的能力要求定义

##### 6.2.1.1 能力要求框架

每个岗位和检验程序的能力要求应从以下六个维度进行定义：

| 维度 | 说明 | 示例 |
|------|------|------|
| 教育背景 | 学历和专业要求 | 医学检验技术本科及以上 |
| 专业资格 | 执业资质和职称要求 | 持有检验技师资格证书 |
| 培训经历 | 所需完成的培训课程 | 完成生物安全培训、本岗位 SOP 培训 |
| 技术知识 | 应掌握的理论知识范围 | 临床化学检验原理、质控规则 |
| 操作技能 | 应具备的实际操作能力 | 独立完成设备校准、样本处理、结果审核 |
| 工作经验 | 相关工作经历要求 | 从事临床检验工作 2 年以上 |

##### 6.2.1.2 能力要求文件管理

1. 技术负责人组织各专业组组长制定《岗位/项目能力要求表》（PER-REC-006），明确各岗位和各检验程序对应的六维能力要求。
2. 能力要求应与岗位说明书中的任职条件保持一致。
3. 开展新检验程序或岗位职责发生重大变更时，技术负责人在变更生效前完成能力要求的修订。

#### 6.2.2 评估计划编排

##### 6.2.2.1 年度能力评估计划

1. 每年 1 月由技术负责人组织编制《年度能力评估计划》（PER-REC-007）。
2. 评估计划应覆盖所有在岗人员和所有已授权的检验程序。
3. 计划应明确：被评估人姓名、评估的检验程序或岗位、评估方法组合、评估人、计划评估月份。
4. 质量负责人审核、科主任批准后发布执行。

##### 6.2.2.2 评估频次要求

| 评估类型 | 适用场景 | 频次要求 |
|----------|----------|----------|
| 新员工入岗评估 | 新入职人员 | 入职后 6 个月内至少完成 2 次（ISO 15189:2022 第 6.2.2 条） |
| 常规年度评估 | 在岗人员 | 每 12 个月至少 1 次（ISO 15189:2022 第 6.2.2 条） |
| 复岗评估 | 离岗超过连续 2 年的人员 | 返岗前完成，标准等同新员工入岗评估 |
| 加密评估 | 复杂或高风险检验程序 | 每 6 个月 1 次（见 6.2.5） |
| 专项评估 | 新检验程序或新方法上线 | 程序上线前完成 |
| 不合格后复评 | 评估不合格人员 | 完成针对性培训后 1 个月内复评 |

#### 6.2.3 评估方法

##### 6.2.3.1 评估方法组合

每次能力评估应至少采用以下方法中的 2 种及以上组合（ISO 15189:2022 第 6.2.2 条 a)~f)）：

| 编号 | 评估方法 | 具体说明 | 适用场景 |
|------|----------|----------|----------|
| M1 | 直接观察 | 评估人在工作现场观察被评估人执行标准操作规程，包括安全操作实践和设备维护操作（覆盖 ISO 15189:2022 第 6.2.2 条 a) 和 b)） | 所有岗位 |
| M2 | 盲样测试 | 将已知结果的样本作为常规样本送检，评估检验正确度和精密度 | 有定量或定性结果的检验程序 |
| M3 | 理论考核 | 笔试或口试，考核专业知识、SOP 理解程度、质量控制知识 | 所有岗位 |
| M4 | 工作记录审查 | 回顾性审查被评估人的检验记录、质控记录、异常处理记录（覆盖 ISO 15189:2022 第 6.2.2 条 c) 和 d)） | 所有岗位 |
| M5 | 特殊样本检查 | 使用先前检验过的样本、能力验证材料或分割样本进行评估（覆盖 ISO 15189:2022 第 6.2.2 条 f)） | 有实验室间比对的检验程序 |
| M6 | 问题解决能力评估 | 提供异常情景（如设备故障、质控失控、危急值处置），评估判断和处理能力（覆盖 ISO 15189:2022 第 6.2.2 条 e)） | 中级及以上人员 |

##### 6.2.3.2 各岗位评估方法矩阵

| 岗位或程序类别 | M1 直接观察 | M2 盲样测试 | M3 理论考核 | M4 记录审查 | M5 特殊样本 | M6 问题解决 |
|---------------|:-----------:|:----------:|:----------:|:----------:|:----------:|:----------:|
| 样本采集与接收 | 必选 | -- | 必选 | 必选 | -- | 推荐 |
| 临床化学（生化） | 必选 | 必选 | 必选 | 必选 | 推荐 | 推荐 |
| 临床免疫 | 必选 | 必选 | 必选 | 必选 | 推荐 | 推荐 |
| 临床血液与体液 | 必选 | 必选 | 必选 | 必选 | 必选 | 推荐 |
| 形态学检查 | 必选 | 必选 | 必选 | 必选 | 必选 | 必选 |
| 临床微生物 | 必选 | 必选 | 必选 | 必选 | 必选 | 必选 |
| 分子检测（PCR） | 必选 | 必选 | 必选 | 必选 | 必选 | 必选 |
| 结果审核与报告签发 | 必选 | -- | 必选 | 必选 | -- | 必选 |

#### 6.2.4 新员工入岗能力评估

##### 6.2.4.1 评估安排

依据 ISO 15189:2022 第 6.2.2 条，新员工入岗后 6 个月内应完成至少 2 次能力评估：

- **第 1 次评估**：入岗后第 2~3 个月，重点评估基本操作能力和安全规范执行情况。
- **第 2 次评估**：入岗后第 5~6 个月，全面评估独立工作能力。

##### 6.2.4.2 评估内容与合格标准

| 评估次数 | 评估重点 | 评估方法组合 | 合格标准 |
|----------|----------|-------------|----------|
| 第 1 次 | 基本操作规范、安全操作执行、SOP 理解 | M1 + M3 | 理论考核得分不低于 80 分；直接观察无关键操作偏差 |
| 第 2 次 | 独立工作能力、结果判读、异常处理 | M1 + M2 + M4 + M6 | 盲样测试结果偏差在允许范围内；操作规范无关键偏差；能正确识别异常情况并采取措施 |

##### 6.2.4.3 评估结果处理

1. 两次评估均合格：由技术负责人签署独立上岗确认，该人员正式获得独立操作授权。
2. 任一次评估不合格：由专业组组长与技术负责人共同制定针对性培训计划（依据本程序第 6.1 节），培训完成后 1 个月内安排复评。
3. 经培训和复评仍不合格：评估该人员是否适合继续在该岗位工作，必要时调整岗位或终止聘用。处理结果记录于个人人事档案。

#### 6.2.5 复杂项目加密评估

对以下复杂或高风险检验程序，在常规年度评估基础上增加评估频次至每 6 个月 1 次：

| 程序类别 | 加密评估方法重点 | 加密原因 |
|----------|-----------------|----------|
| 形态学检查（血涂片、骨髓、尿沉渣） | M2 盲样测试（已知形态标本）+ M5 分割样本比对 | 形态辨认高度依赖主观判断 |
| 临床微生物检验（培养、鉴定、药敏） | M2 盲样测试（已知菌株）+ M1 直接观察（无菌操作） | 培养鉴定涉及主观判断 |
| 分子检测（PCR 等） | M2 盲样测试 + M1 直接观察（防污染操作） | 高灵敏度程序，交叉污染风险高 |
| 流式细胞术 | M2 盲样测试 + M1 直接观察（设门策略） | 设门分析高度依赖操作者经验 |
| 质谱检验 | M2 盲样测试 + M4 记录审查 + M6 问题解决 | 新技术程序，须强化能力监控 |

#### 6.2.6 复岗评估

##### 6.2.6.1 适用条件

人员因以下原因离岗超过连续 2 年，返岗前须重新进行能力评估：

- 长期病假
- 产假或育儿假
- 外派或借调
- 停职
- 其他原因导致的长期脱岗

##### 6.2.6.2 复岗评估流程

```
复岗申请 → 岗位条件复核 → 复岗培训 → 能力评估（标准等同新员工入岗评估） → 评估合格 → 技术负责人重新签署授权 → 恢复上岗
```

1. 复岗人员须重新参加岗前培训，内容包括：生物安全更新培训、离岗期间修订的 SOP 培训、设备操作培训。
2. 能力评估标准等同新员工入岗评估（6 个月内至少 2 次评估）。
3. 所有原有授权在复岗评估合格前处于暂停状态。
4. 评估合格后由技术负责人重新签署授权。

#### 6.2.7 评估结果记录与不合格处理

##### 6.2.7.1 评估结果记录

每次能力评估应填写《能力评估记录表》（PER-REC-010），内容包括：

- 被评估人姓名、岗位、工号
- 评估的检验程序或岗位名称
- 评估方法及评估工具
- 各评估项目的评分标准和实际得分
- 评估结论（合格/不合格）
- 评估人签名及评估日期
- 不合格项的具体说明及改进建议

##### 6.2.7.2 评估结果判定标准

| 评估方法 | 合格标准 |
|----------|----------|
| 理论考核（M3） | 得分不低于 80 分（满分 100 分） |
| 直接观察（M1） | 无关键操作偏差，总评分不低于 85 分（满分 100 分） |
| 盲样测试（M2） | 所有检验程序结果在允许偏差范围内 |
| 工作记录审查（M4） | 无重大差错，记录完整、规范 |
| 问题解决评估（M6） | 正确识别问题性质并采取符合 SOP 要求的处理措施 |

##### 6.2.7.3 不合格处理流程

```
评估不合格 → 书面通知被评估人及其专业组组长 → 分析不合格原因 → 专业组组长与技术负责人制定针对性培训计划 → 执行培训 → 1 个月内复评 → 合格/再次不合格
```

1. 评估不合格时，应在评估完成后 3 个工作日内以书面形式通知被评估人，明确不合格项和改进要求。
2. 专业组组长与技术负责人共同分析不合格原因，制定针对性培训计划并记录于《不合格评估处理记录》（PER-REC-014）。
3. 培训完成后 1 个月内安排复评。
4. 复评仍不合格的处理措施：
   - 限制或暂停该人员对应检验程序的操作授权。
   - 评估是否调整该人员的岗位安排。
   - 处理结果记录于人员档案，并将该事项纳入下次管理评审议题。

##### 6.2.7.4 评估结果统计分析

1. 每年编制《年度能力评估汇总报告》（PER-REC-015），内容包括：
   - 评估完成率 = 实际评估人次 / 计划评估人次 x 100%
   - 首次评估合格率
   - 不合格项分布分析（按评估方法、按专业组、按检验程序分类统计）
   - 复评合格率
   - 与上年度的趋势对比分析
2. 汇总报告经技术负责人审核、科主任批准后，纳入管理评审输入。

#### 6.2.8 能力评估与质量监督的联动机制

1. 日常质量监督中发现的人员能力问题，质量监督员应在发现后 3 个工作日内书面报告质量负责人，由质量负责人决定是否启动专项能力评估。
2. 能力评估结果应在评估完成后 7 个工作日内反馈给质量监督员，作为确定日常监督重点的依据。
3. 以下情况应启动联动机制：

| 触发条件 | 联动动作 | 启动时限 |
|----------|----------|----------|
| 质量监督发现某人员 3 个月内出现 2 次及以上同类操作偏差 | 启动该人员该检验程序的专项能力评估 | 发现后 7 个工作日内 |
| 能力验证结果不合格 | 评估该检验程序所有操作人员的能力 | 收到结果后 14 个工作日内 |
| 患者投诉涉及检验质量 | 评估涉事操作人员的能力 | 投诉受理后 14 个工作日内 |
| 能力评估不合格 | 通知质量监督员将该人员纳入重点监督对象 | 评估结论确认后 3 个工作日内 |
| 新员工入岗 | 质量监督员将新员工纳入重点监督对象，持续至独立上岗确认后 3 个月 | 入岗当日 |

### 6.3 培训与能力评估档案管理

#### 6.3.1 个人培训与能力评估档案

依据 ISO 15189:2022 第 6.2.5 条，每位人员应建立个人培训与能力评估档案，内容包括：

| 档案类别 | 具体内容 |
|----------|----------|
| 教育与资格 | 学历证书复印件、学位证书复印件、执业资格证书复印件、职称证书复印件 |
| 培训记录 | 入职培训完成记录、在职培训签到与考核记录、外部培训证书与总结报告 |
| 能力评估记录 | 入岗评估记录、年度评估记录、专项评估记录、复岗评估记录 |
| 授权记录 | 独立上岗确认书、检验程序授权书 |
| 继续教育记录 | 继续教育学分登记表、学分证明 |
| 合规文件 | 公正性承诺书、保密协议 |

#### 6.3.2 档案保存要求

1. 培训管理员负责档案的日常维护和更新。
2. 纸质档案存放于实验医学科资料室专用档案柜，按人员姓名索引排列。
3. 电子档案保存于实验室信息管理系统，设置访问权限，仅科主任、技术负责人、质量负责人和培训管理员有权查阅全部档案。
4. 在职人员档案保存至离职后 5 年。

### 6.4 隐私保护要求

1. 人员培训与能力评估过程中涉及的个人信息（学历、资格、考核成绩等）属于内部保密信息，仅限上述授权人员查阅。
2. 能力评估结果的反馈应以一对一方式进行，不得在公开场合公布个人不合格信息。
3. 电子档案系统应具备操作日志记录功能，记录每次查阅和修改操作。
