## 附件一：表格填写说明

### 1. 总则

本表用于记录医学检验科所有设备的校准活动，包括内部校准和外部校准（法定计量检定、第三方校准、厂家校准）。每次校准活动应填写一份独立的校准记录表。

### 2. 各字段填写要求

#### 2.1 设备基本信息

| 字段 | 填写要求 |
|------|---------|
| 设备名称 | 填写设备的完整名称，如"全自动生化分析仪"，应与设备登记册一致 |
| 设备编号 | 填写实验室内部统一编号，格式参照设备编号规则 |
| 设备型号 | 填写制造商标注的设备型号，如"AU5800" |
| 出厂序列号 | 填写制造商赋予的唯一序列号，从设备铭牌获取 |
| 制造商 | 填写设备制造商的完整名称 |
| 所属专业组 | 填写设备当前使用的专业组名称，如"临床化学组" |
| 安装位置 | 填写设备当前放置的具体位置，如"三楼生化室 B 区" |

#### 2.2 校准信息

| 字段 | 填写要求 |
|------|---------|
| 校准日期 | 填写实际校准操作的日期，精确到日 |
| 校准开始/结束时间 | 填写校准操作的起止时间，精确到分钟 |
| 校准周期 | 填写设备规定的校准间隔月数，依据设备说明书、计量法规或实验室校准计划确定 |
| 下次校准日期 | 根据校准周期自动推算的下次校准截止日期 |

#### 2.3 校准环境条件

| 字段 | 填写要求 |
|------|---------|
| 温度 | 用经校准的温度计测量校准现场的环境温度，精确到 0.1℃ |
| 相对湿度 | 用经校准的湿度计测量校准现场的相对湿度，精确到 1% |
| 大气压力 | 对天平、气体分析等对气压敏感的设备，用气压计测量现场大气压，精确到 1 hPa；不适用的设备填写"不适用" |
| 供电电压 | 对电压敏感的精密设备，用万用表测量实际供电电压；不适用的设备填写"不适用" |
| 允许范围 | 依据设备说明书或校准方法规定的环境条件限值填写 |

#### 2.4 校准执行信息

| 字段 | 填写要求 |
|------|---------|
| 校准类别 | 勾选对应的校准方式，只能选择一项 |
| 校准单位名称 | 内部校准填写"本实验室"，外部校准填写校准机构全称 |
| 校准人员姓名 | 填写实际执行校准操作的人员姓名 |
| 校准资质/证书编号 | 外部校准填写校准机构的 CNAS/CMA 认可证书编号，内部校准填写校准人员授权编号 |
| 校准依据/方法 | 填写所使用的校准规程或方法编号，如"JJG 1036-2008"或制造商校准程序编号 |
| 标准器/校准品 | 填写所用的标准器或校准品名称、编号、有效期和溯源信息 |

#### 2.5 校准项目及结果

| 字段 | 填写要求 |
|------|---------|
| 校准项目 | 填写具体校准参数名称，如"重复性""示值误差""线性""灵敏度"等 |
| 标准值 | 填写标准器或校准品的参考值，含单位 |
| 实测值 | 填写被校设备的实际测量值，含单位；多次测量取平均值时注明"n=X" |
| 允许误差 | 填写该项目的最大允许误差范围，含单位 |
| 偏差 | 填写实测值与标准值之差，含单位和正负号 |
| 判定结果 | 偏差在允许误差范围内判定为"符合"，超出范围判定为"不符合" |

#### 2.6 测量不确定度

| 字段 | 填写要求 |
|------|---------|
| 测量不确定度 | 填写扩展不确定度 U 值，格式为"± 数值（单位）"，如"± 0.05 mg" |
| 不确定度来源 | 勾选不确定度的来源：校准证书中给出的或实验室自行评定的 |
| 包含因子 k | 填写扩展不确定度使用的包含因子，通常 k=2（对应约 95% 置信水平） |
| 满足预期用途 | 将测量不确定度与检验项目允许的总误差进行比较，判断是否满足临床用途要求 |

#### 2.7 签名栏

| 字段 | 填写要求 |
|------|---------|
| 填表人 | 由执行校准并记录数据的人员亲笔签名并注明日期，应在校准完成当日签署 |
| 一级审核人 | 由质量负责人审核校准记录的完整性、数据合理性后签名，应在校准完成后 3 个工作日内完成 |
| 二级复核人 | 由科室负责人或技术负责人对校准结论的准确性进行复核并签名，应在一级审核完成后 5 个工作日内完成 |

### 3. 特别说明

- 所有数值字段应使用国际单位制（SI）或医学检验领域公认的单位。
- 一份校准记录表仅对应一台设备的一次校准活动。
- 如校准项目超过 6 项，可续页填写，续页应注明设备编号、校准日期和页码。
- 外部校准时，应将校准证书原件或复印件附在本表之后，一同存入设备档案。

## 附件二：记录更正规则

### 1. 更正原则

所有校准记录应做到真实、准确、完整、及时。记录填写后，不得随意更改。确需更正时，应遵循以下规则，确保记录的可追溯性和真实性。

### 2. 更正方法

1. **单划线更正法**：在错误内容上方划一条水平线（确保原内容仍可辨识），在其旁边或上方书写正确内容。
2. **签名确认**：更正人在更正处旁签署姓名缩写和更正日期。
3. **更正原因**：如更正涉及数据变更，应在"备注"栏简要注明更正原因。

### 3. 禁止行为

- **严禁**使用涂改液、修正带、橡皮擦等任何覆盖或擦除手段。
- **严禁**撕毁或重新抄写记录页。
- **严禁**在签名栏代他人签名。

### 4. 电子记录更正

对于通过 LIS/LIMS 系统记录的电子数据，更正时应保留原始记录（审计追踪），系统自动记录更正人、更正时间和更正原因。

### 5. 记录保存

- 本表纸质原件存入对应设备档案，保存期限为设备使用寿命期加 3 年，最低保存 **6 年**。
- 电子扫描件同步上传至 LIMS 系统备份。
- 校准证书原件与本表一同保存。
