## 1 审核导引

### 1.1 本次专项审核目的

对"检验过程"全部子条款进行深度符合性与有效性评价；识别系统性薄弱点；形成改进项目清单。

### 1.2 总体方针

1.2.1 逐条覆盖：每一子条款必须有独立的检查项。
1.2.2 证据优先：以客观证据为判定依据，避免凭印象判定。
1.2.3 过程追溯：重点事件（失控、EQA 不满意）逐起回溯。

### 1.3 符合性 5 级判定

- 符合：全部证据充分。
- 基本符合（观察项）：细节不完善但不影响结果。
- 一般不符合：单点失效。
- 严重不符合：系统性失效或涉及患者报告偏差。
- 不适用：有书面说明。

## 3 7.3.2 方法验证和确认

### 3.1 核心要求

3.1.1 商品化方法：引入前需验证（verify）性能声明。
3.1.2 自建或偏离方法：引入前需确认（validate）。

### 3.2 检查清单

| 编号 | 检查项 | 方法 | 判定 |
|------|--------|------|------|
| 3.2.1 | 每一定量商品化方法有验证报告（5 项必备性能指标） | 查阅 | |
| 3.2.2 | 精密度验证：2 水平，每水平 n≥20，CV 不超过厂商声明 | 重计算 | |
| 3.2.3 | 正确度验证：使用有证标准物质或 EQA 样品；偏倚 ≤TEa/2 | 重计算 | |
| 3.2.4 | 可报告范围：5 水平线性评估，r²≥0.99 | 重计算 | |
| 3.2.5 | 生物参考区间：≥20 份健康样品验证 | 查阅 | |
| 3.2.6 | 干扰：溶血 Hb≥5g/L、黄疸 TBIL≥340µmol/L、乳糜 TG≥10mmol/L | 查阅 | |
| 3.2.7 | 自建方法确认报告含 9 项性能指标 | 查阅 | |
| 3.2.8 | 确认报告经技术负责人批准后方可使用 | 查阅 | |
| 3.2.9 | 定性方法含诊断灵敏度、特异性、符合率 | 查阅 | |

## 9 追溯专项

### 9.1 IQC 数据独立重计算

9.1.1 抽 1 个月内 3 日全部 IQC 原始数据，独立计算 Mean、SD、CV，与系统报告偏差 >5% 判定一般不符合。

### 9.2 失控事件全程追溯

9.2.1 抽 2025 年度 5 起失控，检查五步闭环。

### 9.3 EQA 不满意全量回溯

9.3.1 对 2025 年度全部不满意 EQA 根因分析报告进行 5 要素核查。

### 9.4 变更控制追溯

9.4.1 对 2025 年度所有重大变更（方法、仪器、试剂品牌）进行变更控制闭环审核。

## 10 不符合开具规则

### 10.1 判定原则

10.1.1 单点失效：一般不符合。
10.1.2 系统性失效（多点、多项目）或已发报告偏差：严重不符合。
10.1.3 细节改进：观察项。

### 10.2 整改时限

| 等级 | 原因分析 | 整改完成 | 验证 |
|------|----------|----------|------|
| 严重 | 5 个工作日 | 30 个工作日 | 60 个工作日 |
| 一般 | 10 个工作日 | 60 个工作日 | 90 个工作日 |
