## 表格填写说明

### 1. 字段填写要求对照表

| 字段 | 填写要求 |
|------|---------|
| 设备名称、型号、序列号 | 与设备铭牌完全一致；不得使用俗称或缩写 |
| 设备科室编号 | 按 INS-REC-001《设备台账》的编号规则填写 |
| 所属专业组 | 在提供选项中勾选，且应与台账记录一致 |
| 校准环境条件 | 使用经检定 / 校准合格的温湿度计和气压计现场测量；记录值应为校准期间的环境监测数据 |
| 校准类型 | 在提供选项中勾选；选择"其他"时必须补充文字说明 |
| 校准单位资质 | 外部校准时填写校准机构的 CNAS 或 CMA 证书编号；内部校准时此栏填写"内部校准" |
| 校准品批号 | 与包装标识一致；自校准品开瓶之日起，其使用须在有效期内 |
| 计量溯源性声明 | 按校准品说明书中的"计量溯源"声明填写 |
| 标称值 / 实测值 / 允许偏差 | 均须带有法定计量单位；有效位数不少于制造商说明书规定 |
| 测量不确定度 | 必须按 JJF 1059.1 或等效规程评定，不得省略 |
| 证书编号 | 与校准单位出具的证书正本上的编号一致 |
| 质控物验证 | 校准完成后 2 小时内使用质控物验证；失控则进入第 I 节不合格处理流程 |

### 2. 书写规则

- 日期格式：YYYY 年 MM 月 DD 日
- 签名：使用黑色或蓝色签字笔手签，不得代签、不得盖章代替
- 涉及多页时须注明"第 X 页 / 共 X 页"并逐页由填表人签名
- 空格无内容时填写"—"或"N/A"，不得留白

### 3. 工作时限

| 工作节点 | 时限 |
|---------|------|
| 校准结束后填表 | 当日完成 |
| 一级审核（质量负责人） | 填表后 3 个工作日内 |
| 二级复核（检验科主任 / 技术负责人） | 一级审核后 5 个工作日内 |
| 完成签批的记录归档 | 二级复核后 2 个工作日内 |
| 不合格情况触发纠正措施 | 当日启动 QUA-REC-021《不符合项和纠正措施记录》 |

## 记录更正规则

依据 ISO 15189:2022 第 8.4 条记录控制要求，本表记录的更正应遵循以下规则：

1. **纸质记录更正**
   - 使用单直线划掉错误内容，确保原内容可辨认
   - 在划掉处旁书写正确内容
   - 在正确内容旁签署更正人姓名、更正日期（YYYY-MM-DD）
   - 严禁使用涂改液、修正带、橡皮擦、胶带、刮擦、挖补等手段覆盖原始记录
   - 严禁重抄表格以替代更正

2. **电子记录更正**
   - 在信息管理系统中修改时，系统应自动保留修改前原始数据、修改时间、修改人账户、修改原因，形成完整的审计追踪 (Audit Trail)
   - 审计追踪记录的保存期限不得短于原始记录

3. **重大错误处理**
   - 涉及检验结果正确度的错误更正应启动 QUA-REC-021《不符合项和纠正措施记录》
   - 需重新校准时，整张表格作废，重新填写，作废表格加盖"作废"章后归档保存

4. **表格保存与销毁**
   - 本表保存期限为 6 年
   - 到期后按 DOC-PRO-002《记录控制程序》执行销毁并保留销毁记录（记录编号、销毁日期、销毁人、销毁方式）
