# 质量手册 - 第 8.4 条 管理评审（修订版）

> 依据 ISO 15189:2022 标准修订，全面更新评审输入要素，新增风险管理和信息系统评审要求，统一使用"检验前/中/后过程"术语体系。

---

## 8.4 管理评审

### 8.4.1 目的与原则

本实验室管理层应按照 ISO 15189:2022 的要求，以策划的时间间隔对质量管理体系进行管理评审，以确保其持续的适宜性、充分性和有效性。

管理评审遵循以下原则：

- **全面性原则**：评审应覆盖管理体系所有过程领域，包括检验前过程、检验过程、检验后过程以及支持过程和管理过程
- **数据驱动原则**：评审应基于充分的数据和事实证据，避免主观判断
- **风险导向原则**：评审应关注风险和机遇的识别与管理成效
- **改进导向原则**：评审应明确改进方向，制定可执行的改进措施
- **患者安全原则**：评审应始终将患者安全和检验结果的可靠性作为核心关注点

### 8.4.2 评审策划

#### 8.4.2.1 评审周期

常规管理评审每年度至少进行 **1 次**，通常安排在每年第一季度（对上一年度进行全面评审）。

当发生以下情形之一时，实验室主任应决定启动专项管理评审：

- 重大质量安全事件或系统性不符合
- 认可标准或法规要求发生重大变更
- 实验室组织架构或关键管理岗位发生重大调整
- 外部评审结果提示管理体系存在系统性缺陷
- 实验室活动范围发生重大变更

#### 8.4.2.2 评审参与人员

| 人员 | 角色定位 |
|------|---------|
| 实验室主任 | 评审主持人，负责最终决策 |
| 质量负责人 | 评审组织者，负责材料汇编、会议协调、报告编制及后续跟踪 |
| 技术负责人 | 技术领域报告人，提供检验质量和技术运行分析 |
| 各专业组组长 | 各专业领域报告人，提供本组运行数据和问题分析 |
| 其他受邀人员 | 根据评审议题需要，可邀请相关管理、技术或安全人员参加 |

#### 8.4.2.3 评审计划编制

质量负责人应于评审前 **6 周** 编制管理评审实施计划，内容包括：

- 评审时间和地点
- 参加人员
- 评审议程和议题
- 各项输入材料的准备责任人及提交截止日期
- 上次管理评审待办事项的完成状态

评审计划经实验室主任审批后下发各相关部门。

### 8.4.3 评审输入

各责任部门应按计划时限准备并提交以下评审输入材料：

#### 一、以往管理评审决定事项的落实情况

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 上次评审决定的执行进展 | 每项决定的完成状态、完成日期、验证结论 | 质量负责人跟踪台账 |
| 未完成事项分析 | 未完成原因、新的完成计划 | 相关责任人 |

#### 二、内部审核结果

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 审核覆盖范围 | 审核计划完成率、各条款/部门覆盖情况 | 审核报告 |
| 不符合项统计 | 数量、严重程度分布、部门分布、条款分布 | 不符合项报告 |
| 整改落实率 | 纠正措施完成率及有效性验证通过率 | 整改跟踪记录 |
| 系统性趋势 | 重复出现的问题或共性不足 | 审核分析报告 |

#### 三、外部质量评价与能力验证

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| EQA/PT 参加情况 | 参加项目数、覆盖率 | 各专业组汇总 |
| 成绩汇总 | 满意/可疑/不满意结果统计 | EQA/PT 成绩单 |
| 不满意结果的原因分析 | 根本原因和纠正措施 | 原因分析报告 |
| CNAS 或其他外部评审结果 | 评审发现及整改情况 | 评审报告 |

#### 四、用户反馈与投诉

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 用户满意度调查 | 调查方式、回收率、总体满意度得分及各维度得分 | 满意度调查报告 |
| 投诉汇总 | 投诉数量、分类（检验前/检验/检验后）、处理结果 | 投诉台账 |
| 建议与需求 | 临床科室提出的改进建议和新服务需求 | 沟通记录、调查问卷 |

#### 五、检验前过程运行情况

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 标本合格率 | 各专业组标本拒收率及原因分类 | LIS 统计/拒收记录 |
| 标本运输合规性 | 温度监控、运输时限达标情况 | 运输记录 |
| 采样指南执行情况 | 更新频次、临床知晓率 | 技术负责人汇总 |
| 检验申请单合规性 | 申请单缺陷率及类型 | LIS 统计 |

#### 六、检验过程质量控制

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 室内质控（IQC） | 各项目质控合格率、失控事件统计及处理 | 质控数据统计 |
| 方法确认/验证 | 新增方法的确认/验证完成情况 | 技术负责人 |
| 仪器比对 | 同项目不同仪器间比对结果 | 比对报告 |
| 人员比对 | 操作人员间比对结果 | 比对报告 |
| 测量不确定度 | 评定和应用情况 | 技术记录 |

#### 七、检验后过程结果确认

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 报告周转时间（TAT） | 常规和急诊 TAT 达标率及趋势 | LIS 统计 |
| 危急值报告 | 及时报告率、临床确认反馈率 | 危急值记录 |
| 报告质量 | 报告差错率、修改报告统计 | LIS 统计/质量检查 |
| 标本保存 | 标本保存条件和期限的执行情况 | 各专业组记录 |

#### 八、风险管理

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 风险评估实施 | 年度风险评估的完成情况、覆盖范围 | 风险评估报告 |
| 风险登记册更新 | 已识别风险的监控状态、新增风险 | 风险登记册 |
| 风险事件 | 本周期内发生的风险事件、应对措施及效果 | 事件报告 |
| 机遇识别 | 已识别的改进机遇和利用计划 | 风险管理记录 |

#### 九、不符合项与纠正措施

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 不符合项统计 | 数量、类型分布、严重程度、趋势分析 | 不符合项台账 |
| 纠正措施有效性 | 已关闭项的有效性验证通过率 | 跟踪验证记录 |
| 重复发生项 | 多次出现的不符合项类型及根因分析 | 趋势分析报告 |

#### 十、人员管理

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 人员配置 | 总人数变化、各岗位编制与实际到岗情况 | 人员台账 |
| 培训执行 | 年度培训计划完成率、培训学时统计 | 培训记录 |
| 能力评估 | 评估完成率、评估结果分析 | 能力评估记录 |
| 人员授权 | 各类授权的发放、变更和撤销��况 | 授权记录 |

#### 十一、设备与试剂

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 设备运行 | 校准执行情况、维护完成率、故障率分析 | 设备台账 |
| 设备变更 | 新购、报废、替换的设备清单 | 采购/报废记录 |
| 试剂管理 | 试剂验收合格率、不合格品处理、供应保障情况 | 试剂台账 |
| 关键设备监控 | 关键设备性能趋势分析 | 校准/维护记录 |

#### 十二、信息管理系统有效性

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 系统可用性 | LIS 系统正常运行时间比例、停机事件统计 | IT 运维记录 |
| 数据安全 | 数据备份执行情况、恢复测试结果、安全事件 | 备份日志/安全记录 |
| 系统变更 | LIS 或相关系统的升级、接口变更 | 变更记录 |
| 网络安全 | 权限管理执行情况、安全审计结果 | 信息安全记录 |

#### 十三、法律法规与标准更新

| 评审要素 | 具体内容 | 数据来源 |
|---------|---------|---------|
| 法规变化 | 新发布/修订的相关法律法规清单 | 质量负责人 |
| 标准更新 | 相关技术标准和认可准则的变更 | 质量负责人 |
| 影响分析 | 对管理体系的影响评估及已采取的应对措施 | 影响分析报告 |

|---------|---------|---------|

| 改进项目 | 本周期内开展的持续改进项目及成果 | 改进项目报告 |

### 8.4.4 评审输出

管理评审应产生明确的决定和行动计划。评审输出应至少包含以下内容：

#### 8.4.4.1 体系总体评价

对管理体系运行状况作出整体判定：

| 评价等级 | 判定标准 |
|---------|---------|
| 适宜、充分、有效 | 各要素运行良好，质量目标基本达成，无系统性问题 |
| 基本适宜，需局部改进 | 多数要素运行正常，存在局部不足，需针对性改进 |
| 需要重大改进 | 存在系统性问题，影响检验质量或患者安全，需立即采取行动 |

#### 8.4.4.2 改进决定与行动计划

针对评审中识别的问题和改进机会，制定改进行动计划：

| 要素 | 说明 |
|------|------|
| 改进事项描述 | 需要改进的具体问题 |
| 优先级 | 高/中/低 |
| 改进措施 | 具体的改进方案 |
| 责任部门/责任人 | 负责执行改进的部门和个人 |
| 完成期限 | 预期完成日期 |
| 验证方式 | 如何确认改进有效 |

#### 8.4.4.3 资源配置决定

- 人力资源：增员需求、培训计划、外派进修安排
- 设备资源：设备采购/更换/升级计划及预算
- 试剂耗材：供应保障措施及备选方案
- 信息系统：系统升级、功能扩展或新系统引进计划
- 环境设施：实验室空间、环境条件的改善需求

#### 8.4.4.4 质量方针和目标的调整

- 质量方针是否仍然适宜，是否需要修订
- 下一年度质量目标的修订建议（提高、维持或调整目标值）
- 新增或删除的质量目标

#### 8.4.4.5 风险应对决策

- 需要升级管控的高风险事项
- 风险管理策略的调整方向
- 应急预案的修订需求

#### 8.4.4.6 对检验结果和患者安全的影响评估

- 评审期内发现的问题是否已对检验结果准确性产生实际影响
- 是否需要追溯既往检验结果
- 针对潜在患者安全风险的防范措施

### 8.4.5 评审实施流程

管理评审按以下流程实施：

```
第一阶段：评审准备（评审前 6 周至评审日）
  ├─ 质量负责人编制评审计划并报实验室主任批准
  ├─ 各责任部门按计划准备输入材料
  ├─ 质量负责人汇总编制《管理评审输入报告》
  └─ 评审前 7 日分发材料给参会人员预审

第二阶段：评审会议
  ├─ 实验室主任主持会议，确认议程
  ├─ 质量负责人报告体系运行总况
  ├─ 技术负责人报告技术质量和检验运行
  ├─ 各专业组组长报告本组运行情况
  ├─ 与会人员充分讨论，提出改进建议
  └─ 实验室主任总结评审结论，形成决议

第三阶段：评审后续（评审会议后 15 日内）
  ├─ 质量负责人编制《管理评审报告》
  ├─ 实验室主任审批评审报告
  ├─ 向相关人员通报评审决定
  ├─ 分解改进任务，明确责任人和完成期限
  └─ 定期跟踪并验证改进措施的执行效果
```

### 8.4.6 评审记录管理

#### 8.4.6.1 记录内容

管理评审应保留以下完整记录：

- 管理评审实施计划
- 管理评审输入报告及各项附件材料
- 会议签到表
- 会议记录/纪要
- 管理评审输出报告
- 评审决定事项跟踪台账
- 改进措施验证记录

#### 8.4.6.2 保存要求

| 项目 | 要求 |
|------|------|
| 保存期限 | 不少于 **5 年** |
| 保存形式 | 纸质原件归档保存，同步保存电子扫描件 |
| 存放位置 | 质量管理部门统一归档管理 |
| 保密要求 | 按本实验室保密管理规定执行，借阅需经质量负责人批准 |

所有管理评审记录按照《记录控制程序》进行标识、归档、检索和处置。
