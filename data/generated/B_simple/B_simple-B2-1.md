# ISO 15189:2022 第7.3条 检验前过程 专项内审检查表

| 审核依据 | ISO 15189:2022 第7.3条 |
|---------|----------------------|
| 审核范围 | 检验前过程 |
| 适用部门 | 样本接收、采集及处理相关岗位 |

---

| 序号 | 条款 | 检查内容 | 检查方式 | 符合 | 不符合 | 不适用 | 备注 |
|------|------|---------|---------|------|--------|--------|------|
| 1 | 7.3.1 | 确认实验室已建立并维护检验前过程的文件化程序，涵盖从申请到样本准备完毕的全部环节。 | 查阅文件 | ☐ | ☐ | ☐ | |
| 2 | 7.3.2 | 核实检验申请单包含充分的患者识别信息（至少姓名、唯一标识号、出生日期或年龄、性别），确保样本可被唯一追溯至患者。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 3 | 7.3.2 | 核实检验申请单包含申请者信息（姓名或唯一标识、联系方式），以便结果报告和临床咨询。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 4 | 7.3.2 | 确认检验申请单载明所申请的检验程序名称或检验项目代码，且与实验室检验目录一致。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 5 | 7.3.2 | 确认申请单包含必要的临床信息（如初步诊断、用药情况、采集时间要求），以支持检验程序的正确选择和结果解释。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 6 | 7.3.2 | 核实当检验申请以口头方式提出时，实验室有程序要求事后补充书面或电子申请确认。 | 查阅文件/访谈 | ☐ | ☐ | ☐ | |
| 7 | 7.3.3 | 确认实验室已向样本采集人员提供采集手册或作业指导书，包含患者准备要求、样本类型、容器及添加剂要求、采集步骤和标识要求。 | 查阅文件 | ☐ | ☐ | ☐ | |
| 8 | 7.3.3 | 核实样本采集前已按程序对患者进行正确识别（至少使用两种独立标识），并将识别过程记录在案。 | 现场观察/查阅记录 | ☐ | ☐ | ☐ | |
| 9 | 7.3.3 | 确认原始样本在采集时或采集后立即进行了标识，标识信息与检验申请单一致，且标签牢固、清晰可读。 | 现场观察 | ☐ | ☐ | ☐ | |
| 10 | 7.3.3 | 核实样本采集信息已被记录，包括采集日期和时间、采集人员标识、原始样本类型及采集部位（适用时）。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 11 | 7.3.4 | 确认实验室已建立样本运输条件要求（温度、时限、包装方式），并向所有参与运输的人员传达。 | 查阅文件/访谈 | ☐ | ☐ | ☐ | |
| 12 | 7.3.4 | 核实样本运输过程中采取了适当措施，确保样本完整性、人员安全及符合适用的法规要求。 | 查阅记录/现场观察 | ☐ | ☐ | ☐ | |
| 13 | 7.3.5 | 确认实验室已建立样本接收的验收准则，包括对样本标识、容器完整性、样本量、运输条件的检查。 | 查阅文件 | ☐ | ☐ | ☐ | |
| 14 | 7.3.5 | 核实样本接收时间已被记录，并确认接收人员按照验收准则对每份样本进行了系统检查。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 15 | 7.3.5 | 确认对不符合验收准则的样本，实验室已按程序处理（接受、拒收或有条件接受），并记录了偏离情况及处理决定。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 16 | 7.3.5 | 核实当不合格样本被接受检验时，最终报告中注明了样本偏离情况及其对结果解释的潜在影响。 | 查阅报告 | ☐ | ☐ | ☐ | |
| 17 | 7.3.6 | 确认实验室已建立检验前样本处理的作业指导书，包括离心参数、分装要求、储存条件等，并按文件执行。 | 查阅文件/现场观察 | ☐ | ☐ | ☐ | |
| 18 | 7.3.6 | 核实样本在处理和等待检验期间的储存条件（温度、光照、时限）符合检验程序的规定要求。 | 现场观察/查阅记录 | ☐ | ☐ | ☐ | |
| 19 | 7.3.7 | 确认实验室已制定检验前过程的周转时间目标，并定期监测实际周转时间，对超标情况采取措施。 | 查阅记录 | ☐ | ☐ | ☐ | |
| 20 | 7.3.7 | 核实实验室定期评审检验前过程的质量指标（如样本拒收率、标识错误率），并将评审结果用于持续改进。 | 查阅记录 | ☐ | ☐ | ☐ | |
