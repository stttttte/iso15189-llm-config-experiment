## 2 NC-2026-001 文件控制失效（严重）

### 2.1 不符合摘要

2.1.1 条款：ISO 15189:2022 第 4.2.4 条。
2.1.2 现象：生化组 SOP 版本滞后 1 版；免疫组 3 份作废文件在工作台；共 5 份过期/作废文件现场发现。

### 2.2 根本原因

2.2.1 文件分发机制无回收流程（机制缺陷）。
2.2.2 DOC-002 对纸质版本管理规定不明确。
2.2.3 6 个月未进行文件现场核查（监督失效）。
2.2.4 人员对文件版本控制意识不足。

### 2.3 纠正措施

#### 2.3.1 措施 1.1：即时清理工作岗位

- 逐岗检查、拍照取证、收回并销毁过期文件
- 完成时间：2026-02-05
- 责任人：科室主任 + 质量负责人

#### 2.3.2 措施 1.2：修订 DOC-002

- 新增：分发记录、回收流程、月度版本确认表
- 完成时间：2026-02-28
- 责任人：质量负责人

#### 2.3.3 措施 1.3：月度文件核查

- 抽样覆盖率 ≥50%
- 启动时间：2026-02-01（持续）
- 责任人：质量管理部门

#### 2.3.4 措施 1.4：全员培训

- 集中 1.5 小时 + 部门讨论 1 小时
- 考核正确率 ≥80%
- 完成时间：2026-02-28
- 责任人：质量负责人 + 科室主任

### 2.4 验证方案

#### 2.4.1 验证标准

| 验证项 | 判定条件 |
|--------|----------|
| 版本一致性 | 100% 一致；0 过期 |
| 分发回收记录 | 完整可追溯 |
| 人员理解度 | 抽查 5 人 3 问 ≥80% |
| 月度核查 | 每月有记录 |
| 核查问题数 | ≤2 个/月 |

#### 2.4.2 验证时间线

| 节点 | 日期 |
|------|------|
| 初步 | 2026-03-15 |
| 中期 | 2026-04-15 |
| 最终 | 2026-05-15 |

#### 2.4.3 证据清单

| 编号 | 证据 |
|------|------|
| DOC-REC-1001 | 修订后 DOC-002 |
| DOC-REC-1002 | 清理报告 + 照片 |
| DOC-REC-1003 | 分发记录表 |
| DOC-REC-1004 | 月度核查记录（3 个月） |
| PER-REC-1005 | 培训签到与成绩 |
| QUA-REC-1006 | 访谈记录（≥5 人） |

## 3 NC-2026-002 设备维护记录不完整（一般）

### 3.1 不符合摘要

3.1.1 条款：ISO 15189:2022 第 6.4.4 条。
3.1.2 现象：生化分析仪 AU5800 2025 年度 35 次维护中 8 次缺验证结果（22.9%）、3 次缺复核签名（8.6%）。

### 3.2 根本原因

3.2.1 EQP-004 未要求维护后验证；表单无验证栏。
3.2.2 外包维护人员对记录要求理解不足。
3.2.3 复核签名形式化。

### 3.3 纠正措施

#### 3.3.1 措施 2.1：修订 EQP-004 与表单

- 明确维护后验证要求
- 表单新增"维护后验证"栏
- 完成时间：2026-02-28
- 责任人：设备管理部门 + 质量部

#### 3.3.2 措施 2.2：二级审核制度

- 维护人员填写+技术主管或质量人员复核
- 建立维护记录复核清单
- 启动时间：2026-02-01
- 责任人：设备管理部门主任

#### 3.3.3 措施 2.3：补充 2025 年缺陷记录

- 8 条缺验证结果补充；3 条缺复核签名补签
- 完成时间：2026-03-15
- 责任人：设备管理部门 + 技术团队

#### 3.3.4 措施 2.4：外包维护管控

- 书面通知外包单位记录要求
- 外包维护验收清单建立
- 下次维护时现场培训
- 完成时间：2026-02-15
- 责任人：设备管理部门

### 3.4 验证方案

#### 3.4.1 验证标准

| 验证项 | 判定 |
|--------|------|
| 新记录完整性 | 2026-02 后新增记录 100% 合格 |
| 历史记录补充 | 8 条补充完成 |
| 维护验证执行率 | ≥95% |
| 复核签名率 | 100% |
| 外包配合 | 记录完整；验收完整 |

#### 3.4.2 验证时间线

| 节点 | 日期 |
|------|------|
| 初步 | 2026-03-20 |
| 中期 | 2026-04-20 |
| 最终 | 2026-05-20 |

#### 3.4.3 证据清单

| 编号 | 证据 |
|------|------|
| INS-REC-2001 | 修订后 EQP-004 与新表单 |
| INS-REC-2002 | 8 条补充记录 |
| INS-REC-2003 | 2026-02 后新增记录（≥5 条） |
| INS-REC-2004 | 维护复核清单（月度） |

## 4 NC-2026-003 人员能力评估缺失（一般）

### 4.1 不符合摘要

4.1.1 条款：ISO 15189:2022 第 6.2 条。
4.1.2 现象：员工 D 独立操作 2 个月无能力评估；员工 E 评估仅打勾。

### 4.2 根本原因

4.2.1 PER-002 对评估时限与质量监督不足。
4.2.2 对高学历员工放松要求。
4.2.3 缺乏"培训-评估-监控"三阶段机制。

### 4.3 纠正措施

#### 4.3.1 措施 3.1：立即评估员工 D

- 理论 + 实践全面评估
- 形成详细评估表
- 完成时间：2026-02-10
- 责任人：科室主任 + 技术负责人

#### 4.3.2 措施 3.2：修订 PER-002 与表单

- 时限：培训完成后 2 周内评估
- 标准：理论 ≥80%；实践独立完成并质量合格
- 表单：分项目评分；总体结论；不合格给出培养计划
- 监控：人事部建立监控清单
- 完成时间：2026-02-28
- 责任人：人事部 + 质量部门

#### 4.3.3 措施 3.3：补充员工 E 评估

- 深入评估形成详细记录
- 完成时间：2026-02-15
- 责任人：科室主任

#### 4.3.4 措施 3.4：新员工成长档案

- 逐人建档；记录关键节点
- 未通过评估禁止独立操作
- 完成时间：2026-02-28
- 责任人：科室主任 + 人事部

#### 4.3.5 措施 3.5：杜绝程序简化

- 程序明确所有新员工完整执行培训评估
- 管理会议强调；内审重点检查
- 启动时间：2026-02-01
- 责任人：科室主任 + 质量部门

### 4.4 验证方案

#### 4.4.1 验证标准

| 验证项 | 判定 |
|--------|------|
| 员工 D 评估 | 评估表完整、有评分与结论 |
| 员工 E 补充 | 补充记录清晰完整 |
| 新员工评估 | 整改后 100% 完整 |
| 评估时限 | 培训完成后 ≤2 周 |
| 程序合规 | 文字明确 |
| 监督机制 | 人事部每月跟踪 |

#### 4.4.2 验证时间线

| 节点 | 日期 |
|------|------|
| 初步 | 2026-03-10 |
| 中期 | 2026-04-10 |
| 最终 | 2026-05-10 |

## 5 跨项协同措施
