| 2.2 | 方法验证方案是否由技术负责人批准，包含目的、性能参数、样品要求、接受标准、时间计划 5 项内容？ | D：查阅方法验证方案 | 方案含上述全部 5 项内容；有技术负责人签字和日期 | | |
| 2.3 | 精密度验证是否使用患者样品或基质接近患者样品的物质，至少在 2 个浓度水平（覆盖临床决策点附近）上进行？ | D：查阅精密度原始数据 | 浓度水平覆盖临床决策点；数据完整可追溯 | | |
| 2.4 | 正确度验证是否使用有证参考物质（CRM）或与已知正确度的参考方法比对？ | D：查阅正确度验证数据和 CRM 证书 | 使用 CRM 或参考方法；偏倚在制造商声称范围内 | | |
| 2.5 | 可报告范围验证是否覆盖制造商声称的全部检测范围？ | D：查阅可报告范围验证数据 | 线性回归相关系数 r² 不小于 0.995；回收率 95%—105% | | |
| 2.6 | 验证报告是否包含实验数据、统计分析、与接受标准的比对结论，并有技术负责人签字批准？ | D：查阅方法验证报告 | 报告含原始数据、统计结果、明确的"通过/未通过"结论；有技术负责人签字 | | |
| 2.7 | 验证过程中发现性能不满足接受标准时，是否有文件化的处理措施（停用、联系制造商排查或更换方法）？ | D：查阅验证未通过的处理记录（若有） | 未通过项目有明确处理方案和后续跟踪记录 | | |

| 7.6 | 发生 IQC 失控时，是否立即暂停发放该项目患者结果并启动原因排查？ | D：查阅近 6 个月失控处理记录 | 失控记录每次均有"暂停发放"操作记录和排查步骤 | | |
| 7.7 | 失控后是否对失控期间已发放的患者结果进行回顾性评估，并评估是否需要召回或修正？ | D：查阅失控后的患者结果影响评估记录 | 每次失控事件均有影响评估记录 | | |

| 8.7 | 对于无 EQA 计划可用的检测项目，是否每 6 个月至少开展 1 次替代评估（实验室间比对、有证参考物质检测等）？ | D：查阅替代方案清单和执行记录 | 无 EQA 项目有替代方案；记录显示每 6 个月至少执行 1 次 | | |

## 9 不同检测程序、设备和地点间的结果可比性（ISO 15189:2022 第 7.3.7.4 条；CNAS-CL02:2023 第 7.3.7.4 条）
