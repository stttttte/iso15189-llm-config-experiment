## 1 目的

本程序对实验医学科设备在全生命周期内的管理活动作出系统规定，覆盖设备选型论证、采购、安装、验收、安装确认（IQ）、运行确认（OQ）、性能确认（PQ）、日常使用、校准、计量溯源、期间核查、预防性维护、故障处置、不良事件报告、停用、报废与退役等环节，确保所有用于检验活动的设备持续处于受控、可溯源、性能稳定的状态，其性能满足对应检验程序的要求，从而保证检验结果的正确度、精密度和可靠性，满足 ISO 15189:2022 第 6.4 条及 CNAS-CL02:2023 第 6.4 条的全部要求。

---

## 2 适用范围

2.1 本程序适用于实验医学科内所有参与检验前、检验和检验后过程的设备，分类如下：

- 分析仪器：全自动生化分析仪、化学发光免疫分析仪、电解质分析仪、血气分析仪、血细胞分析仪、凝血分析仪、尿液分析仪、粪便分析仪、全自动培养仪、微生物鉴定/药敏分析系统、质谱仪、毛细管电泳仪、实时荧光定量 PCR 仪、数字 PCR 仪、流式细胞仪、血型鉴定仪、糖化血红蛋白分析仪
- 辅助设备：离心机、水浴槽、干式恒温器、CO2 培养箱、厌氧培养箱、恒温恒湿箱、生物安全柜、通风柜、高压蒸汽灭菌器、纯水制备系统、磁力搅拌器、制冰机、显微镜（含荧光显微镜、相差显微镜）
- 计量器具：移液器、分析天平、电子天平、温度计、计时器、秒表、量筒
- 温控储存设备：2~8 ℃ 冰箱、-20 ℃ 冰柜、-80 ℃ 超低温冰箱、液氮罐
- 信息系统相关设备：LIS 服务器、自动化流水线控制终端、条码打印机、扫描仪
- POCT 设备：床旁血糖仪、床旁血气分析仪、便携式凝血分析仪、便携式免疫分析仪

2.2 本程序不覆盖：

- 厂房通用公共设施（如中央空调、电力、供水等由医院总务或设备科管理的设施）
- 属于办公信息化范畴的通用办公设备

## 4 术语与定义

| 术语 | 英文 | 定义 |
|------|------|------|
| 设备全生命周期 | Equipment lifecycle | 设备从选型论证、采购、安装、验收、使用、维护保养、故障处理直至报废退役的完整过程 |
| 安装确认 | Installation Qualification, IQ | 对设备的安装条件与安装过程符合制造商规定与设计要求的验证 |
| 运行确认 | Operational Qualification, OQ | 对设备在预定操作范围内各功能模块是否正常运行的验证 |
| 性能确认 | Performance Qualification, PQ | 使用质控物、标准品等验证设备分析性能是否满足预期要求的活动 |
| 期间核查 | Intermediate check | 在两次校准之间为验证设备性能保持可信而进行的核查 |
| 预防性维护 | Preventive maintenance, PM | 按预定计划定期执行的维护活动，旨在降低故障风险并维持性能 |
| 故障性维护 | Corrective maintenance | 设备发生故障后进行的修复性维护 |
| 设备不良事件 | Equipment adverse event | 设备使用过程中发生或可能导致人员伤害、检验结果错误的事件 |
| 计量溯源性 | Metrological traceability | 通过不间断的比较链将测量结果与参考对象关联起来的测量结果特性 |
| 设备登记册 | Equipment register | 记录本科室全部设备唯一标识与关键信息的受控清单 |
| 唯一标识 | Unique identifier, UID | 按统一编码规则赋予每一台设备的身份标识 |
| 性能验收标准 | Acceptance criteria | 设备投入使用或重新投入使用前应当满足的量化性能指标 |

## 5 职责

| 岗位 | 职责内容 |
|------|---------|
| 实验医学科主任 | 批准年度设备采购计划与重要维护预算；审批设备报废；签发重大不良事件处置结论 |
| 技术主管 | 主导新设备选型论证；审定性能验收标准；审核运行确认、性能确认方案与报告；审批设备恢复使用 |
| 质量主管 | 审核设备管理合规性；监督期间核查、校准、不良事件报告等关键流程；组织管理评审中的设备议题 |
| 设备管理员 | 维护设备登记册；组织验收与安装确认；编制预防性维护计划并组织执行；记录故障与维修；跟踪设备不良事件报告与反馈；管理设备档案 |
| 生物安全员 | 对生物安全相关设备（生物安全柜、高压灭菌器、离心机等）的验证、年检、维护进行专项监督 |
| 计量管理员 | 组织计量器具的送检与校准；维护校准证书；执行期间核查 |
| 各专业组组长 | 提出本组设备需求与故障信息；指派本组设备操作授权人员；组织本组的预防性维护日常执行 |
| 操作人员 | 按 SOP 操作设备；执行日常维护保养；记录使用日志；在发现异常时立即上报并停止使用 |

## 6 程序内容

### 6.1 设备选型与论证

#### 6.1.1 需求提出

由专业组长依据临床需求、工作量增长、现有设备老化、新检验程序开展等因素提出设备需求，填写《设备需求申请表》（LAB-EQP-REC-201），经技术主管会签、实验医学科主任批准后纳入年度采购计划。

#### 6.1.2 选型论证

技术主管组织对备选设备开展选型论证，评审维度包括：

| 评估维度 | 说明 |
|---------|------|
| 临床需求匹配 | 检验程序、样本类型、工作量、TAT 要求 |
| 技术性能 | 正确度、精密度、检测范围、分析灵敏度、特异性 |
| 方法学溯源 | 校准品溯源链、参考测量程序可获得性 |
| 兼容性 | 与 LIS、中间件、自动化流水线的接口兼容性 |
| 安装与环境要求 | 场地面积、电力、给排水、温湿度、通风 |
| 售后与备件 | 厂家服务响应时间、备件供应期限 |
| 合规性 | 注册证、生产许可、进口产品注册情况 |
| 生命周期成本 | 采购、耗材、维护、能源、人员培训 |

论证结果形成《设备选型论证报告》（LAB-EQP-REC-202）。

### 6.2 采购、到货与安装

#### 6.2.1 采购合同要素

采购合同（由医院设备科主导，实验医学科配合）应约定以下要素：性能验收标准、备件供应期（不少于设备使用年限）、培训服务内容与时长、保修期、应急服务响应时间、软件升级与接口兼容承诺。

#### 6.2.2 到货验收

到货后 3 个工作日内由设备管理员与专业组长共同开展到货验收：核对型号、序列号、配置清单、随机文件（说明书、合格证、注册证复印件、装箱单）；检查外观是否完好、封条是否完整；核对包装箱内容与装箱单一致性。到货验收结果记入《到货验收记录》（LAB-EQP-REC-203）。

#### 6.2.3 安装确认（IQ）

设备管理员组织厂家工程师开展安装确认，确认内容包括：

- 安装场地的环境条件（温度、湿度、电源、接地、排风、给排水）符合制造商规定
- 电气接线、管路连接、信号线缆符合设计要求
- 机械部件无损伤、运转正常
- 随机软件版本与合同约定一致

安装确认结果记入《安装确认记录》（LAB-EQP-REC-204）。

### 6.3 验收与启用

#### 6.3.1 运行确认（OQ）

在 IQ 合格基础上开展 OQ，验证设备在预定操作范围内各功能模块运行正常：开机自检、样本进位、试剂仓、加样系统、检测模块、数据处理、报警系统、打印或数据传输。OQ 结果记入《运行确认记录》（LAB-EQP-REC-205）。

#### 6.3.2 性能确认（PQ）

OQ 合格后由技术主管组织 PQ，内容包括：

| 项目 | 要求 |
|------|------|
| 正确度 | 使用具有溯源性的标准物质或质控物验证 |
| 精密度 | 重复性与中间精密度 |
| 线性范围 | 覆盖声明的可报告范围 |
| 分析灵敏度 | 如适用，验证最低检出限 |
| 分析特异性 | 如适用，验证干扰与交叉反应 |
| 样本携带污染 | 如适用，定量考察 |
| 与既有方法比对 | 替换或并行时的方法比对 |

PQ 结果须达到性能验收标准，方能签发《设备启用通知》（LAB-EQP-REC-206），启用通知由技术主管签发后分发各相关岗位。

### 6.4 登记与唯一标识

#### 6.4.1 设备登记册

设备管理员维护《设备登记册》（LAB-EQP-REC-207），对每一台设备记录：唯一标识、名称、型号、序列号、注册证号、制造商、供应商、到货日期、启用日期、使用部门、责任人、状态、下次校准日期、下次维护日期。

#### 6.4.2 唯一标识规则

采用"科室代码-类别代码-流水号"三段式编码，例如 LAB-ANA-0001；唯一标识以标签形式粘贴于设备显著位置。

#### 6.4.3 状态标识

设备现场状态标识分为：合格（绿）、限用（黄，注明限用条件）、停用/维修中（红）、校准中（蓝）。

### 6.5 操作、使用与授权

1. 使用人员须为经授权的岗位人员，授权依据 LAB-PER-PRO-003 执行。
2. 每台设备配备《操作 SOP》或《快速操作卡》，置于设备旁便于查阅；设备更新后 SOP 在 14 日内同步更新。
3. 每次使用后记入《设备使用日志》（LAB-EQP-REC-208），记录使用日期、开关机时间、样本量、使用人、异常情况。
4. 发现异常时立即停止使用，悬挂"停用/维修中"标识并通知设备管理员。

### 6.6 校准与计量溯源

1. 校准工作依据 LAB-EQP-PRO-002 执行；本程序仅对原则性要求做出规定。
2. 对定量分析仪器：按制造商规定和法规要求开展校准，校准品须具备明确的计量溯源性（依据 ISO 17511）。
3. 对计量器具（移液器、天平、温度计、计时器等）：按 JJF 规程送检或内部校准。
4. 校准证书（或校准记录）归入设备档案；设备现场粘贴校准状态标签（注明本次校准日期和下次校准日期）。

### 6.7 期间核查

#### 6.7.1 适用设备

对两次校准间隔较长、或性能易受使用强度影响的设备实施期间核查：温度计、温控储存设备、分析天平、离心机（转速）、移液器、生物安全柜、光源强度等。

#### 6.7.2 频次示例

| 设备 | 核查项 | 频次 |
|------|-------|------|
| 2~8 ℃ 冰箱 | 温度与报警 | 每日 |
| -20 ℃、-80 ℃ 冰柜 | 温度与报警 | 每日 |
| 水浴/恒温器 | 温度 | 每使用日 |
| 分析天平 | 砝码复核 | 每使用日 |
| 离心机 | 转速（转速表/频闪仪） | 每 6 个月 |
| 移液器 | 称重法精密度 | 每 6 个月 |
| 生物安全柜 | 气流与风速 | 每 12 个月（由资质机构） |

期间核查记录归入相应设备档案，并通过《期间核查记录表》（LAB-EQP-REC-209）追踪。

### 6.8 预防性维护

#### 6.8.1 计划编制

设备管理员依据制造商建议与使用强度，于每年 12 月编制下一年度《预防性维护计划》（LAB-EQP-REC-210），经技术主管批准后执行。

#### 6.8.2 维护分级

- 日常维护（操作人员执行）：开机自检、清洁、耗材更换、日质控
- 周/月/季维护（操作人员与专业组长）：定期清洗、过滤器更换、管路维护、内部校准
- 年度维护（厂家工程师）：全面保养、关键部件更换、系统升级

每次维护完成后记入《设备维护保养记录》（LAB-EQP-REC-211）。

### 6.9 故障处置与维修

#### 6.9.1 故障处置流程

1. 操作人员发现异常后立即停止使用，悬挂"停用/维修中"标识，填写《故障信息单》（LAB-EQP-REC-212），通知设备管理员。
2. 设备管理员组织初步判断：简单故障由内部处置；复杂故障联系厂家工程师。
3. 维修完成后记录故障原因、处置过程、更换部件、耗时。
4. 维修后须经下述"恢复使用"流程合格后方可投入检验。

#### 6.9.2 故障期间已出结果的追溯

针对可能受故障影响的先前检验结果开展追溯评估，必要时依据 LAB-QUA-PRO-007 对相关结果发起纠正措施，并与临床沟通。

#### 6.9.3 恢复使用

故障或重大维护（如更换关键检测模块、更换光路系统、更换主板）后，应按风险评估结果执行下列全部或部分确认活动：运行确认、性能确认、方法比对，确认合格后由技术主管签发《设备恢复使用通知》（LAB-EQP-REC-213）。

### 6.10 不良事件报告

1. 设备使用过程中发生或可能导致人员伤害、检验结果错误的事件，属于设备不良事件。
2. 发生后操作人员应在 24 小时内上报设备管理员与质量主管。
3. 质量主管组织评估后，须按《医疗器械不良事件监测和再评价管理办法》在法定期限内通过医院医学装备部门向国家或省级不良事件监测机构报告。
4. 不良事件记录归入设备档案并在管理评审中审议。

### 6.11 停用、报废与退役

#### 6.11.1 停用条件

出现以下情形之一时停用：经修复后性能仍不能满足验收标准；连续出现重大故障已无维修价值；制造商已停产且关键备件不可得。

#### 6.11.2 报废流程

1. 设备管理员填写《设备报废申请》（LAB-EQP-REC-214），列明报废原因。
2. 技术主管与质量主管会签。
3. 实验医学科主任批准。
4. 报医院设备科按医院报废流程处置。

#### 6.11.3 数据与去污要求

1. 报废前应按相关规定对设备硬盘或存储介质中的患者信息与校准数据进行备份和清除。
2. 涉及生物危险材料的设备按生物安全要求进行彻底去污处置，并由生物安全员复核。

### 6.12 设备档案

#### 6.12.1 档案内容

每台设备建立独立档案，内容包含：

- 身份信息：名称、型号、序列号、唯一标识、注册证、合格证、供货商
- 合同与采购文件：选型论证报告、合同副本
- 验收文件：到货验收记录、IQ/OQ/PQ 记录、启用通知
- 说明书与操作 SOP：现行版本
- 校准记录与证书
- 期间核查记录
- 预防性维护与故障维修记录
- 恢复使用确认记录
- 不良事件报告及反馈
- 报废处置记录（适用时）

#### 6.12.2 档案保管

1. 纸质档案集中存放于设备档案柜，每台设备一卷。
2. 电子档案存储于 LIS 或设备管理系统，设置分级访问权限。
3. 档案保存期：设备停用后不少于 10 年。

### 6.13 备件、耗材与配套材料管理

1. 对关键备件建立最低库存清单，由设备管理员按月核查。
2. 更换关键备件（加样针、光源、检测模块、管路等）后，按 6.9.3 执行恢复使用确认。
3. 随设备配套使用的耗材（试剂、校准品、质控品）依据 LAB-ENV-PRO-001 和 LAB-RAG-PRO-001 管理。

### 6.14 设备管理关键质量指标

每年统计并分析以下指标，纳入管理评审：

- 预防性维护计划完成率
- 校准按期完成率
- 计划外故障停机时长占比
- 故障率（故障次数/运行小时数）
- 期间核查合格率
- 不良事件数量与类型
- 平均修复时间 MTTR
