## 1. 目的

建立实验医学科设备全生命周期管理制度，涵盖设备选型论证、采购、安装验收、使用授权、校准、预防性维护、故障处理及报废退役各环节，确保所有用于检验活动的设备始终处于受控的良好运行状态，检验结果准确可靠，满足 ISO 15189:2022 第 6.4 条（6.4.1 ~ 6.4.7）的全部要求。

---

## 2. 适用范围

本程序适用于实验医学科所有用于检验活动的设备，包括但不限于：

- **分析检测仪器**：全自动/半自动生化分析仪、免疫分析仪、血细胞分析仪、凝血分析仪、尿液分析仪、血气分析仪、微生物鉴定/药敏系统、核酸扩增仪等。
- **辅助设备**：离心机、水浴箱、恒温培养箱、高压蒸汽灭菌器、生物安全柜、冰箱/冰柜、纯水系统等。
- **计量器具**：移液器、分析天平、温度计、计时器等。
- **信息系统设备**：LIS 服务器、工作站电脑、条码扫描器、条码打印机、数据接口模块等。
- **样品采集相关设备**：采血椅、真空采血装置等。
- **POCT 设备**：分布在院内各 POCT 检测点的即时检验仪器。

本程序不适用于一次性耗材和试剂的管理（参见《试剂和耗材管理程序》）。

## 4. 术语和定义

| 术语 | 定义 |
| ---- | ---- |
| 设备全生命周期 | 从设备选型论证到报废退役的全部管理过程，包括选型、采购、安装、验收、使用、校准、维护、维修、报废等阶段 |
| 安装确认（IQ） | 在安装阶段验证设备的安装环境、供电供水条件、物理安装状况均满足制造商规定要求的过程 |
| 运行确认（OQ） | 验证设备各功能模块在规定操作范围内能够正常工作的过程 |
| 性能确认（PQ） | 使用校准品、质控品或已知样品验证设备的分析性能指标达到预期要求的过程 |
| 预防性维护 | 按照预定计划对设备实施的定期保养和检查，旨在降低故障发生概率、延长设备使用寿命 |
| 设备不良事件 | 设备在正常使用过程中发生的导致或可能导致患者伤害、操作人员伤害或检验结果错误的事件 |
| 设备登记册 | 汇总记录本实验室所有设备唯一标识及关键属性信息的档案清单 |
| 报废 | 设备因技术淘汰、性能严重劣化或无法维修等原因，经评估后永久退出使用的过程 |

## 6. 程序内容

### 6.1 设备选型与采购 [对应 ISO 15189:2022 §6.4.1]

#### 6.1.1 需求论证

当出现以下情形时，应启动设备选型论证：

- 新增检验项目或现有方法学升级；
- 现有设备性能劣化或故障频率显著增加；
- 设备已达到使用年限需要更新换代；
- 工作量增长导致现有设备产能不足；
- 法规或标准变更对设备提出新要求。

由提出需求的专业组填写《设备需求论证报告》，内容包含：临床需求分析、预期检验项目及工作量估算、设备类型及技术指标要求、预算估算、与现有系统（LIS、试剂供应链）的兼容性分析。

#### 6.1.2 选型评估

技术负责人组织相关专业组组长和设备管理员成立选型评估小组，从以下维度对候选设备进行评估：

| 评估维度 | 评估内容 |
| -------- | -------- |
| 分析性能 | 精密度、正确度、线性范围、灵敏度、抗干扰能力、携带污染率 |
| 临床适用性 | 检测菜单覆盖率、样品通量、所需样品量、结果报告时间 |
| 系统集成 | 与 LIS 双向通信能力、条码识别兼容性、数据格式标准化程度 |
| 试剂模式 | 开放/封闭试剂系统、试剂消耗成本、供应稳定性 |
| 售后服务 | 工程师响应时间、备件供应周期、培训支持、质保条款 |
| 合规性 | NMPA 注册证有效性、分类管理类别 |

评估结果形成《设备选型评估报告》，经技术负责人审核、科室负责人审批后提交医院采购部门。

#### 6.1.3 采购执行

按医院采购管理制度执行采购流程（招标、议价或集中采购），签订采购合同时须明确以下条款：

- 设备配置清单及价格
- 安装调试责任和时间节点
- 操作培训内容和课时
- 保修期限及保修范围
- 售后响应时限和备件供应承诺
- 软件升级和系统维护条款
- 验收标准和验收不合格时的处置约定

### 6.2 设备标识与登记 [对应 ISO 15189:2022 §6.4.2]

#### 6.2.1 设备唯一标识编码

每台设备分配唯一的设备编号，编码规则如下：

```
编码格式：EQ-[专业组代码]-[三位序号]

专业组代码对照：
  BC — 临床化学（生化）
  IM — 临床免疫
  HM — 临床血液
  MB — 临床微生物
  UA — 临床体液
  MO — 临床分子（PCR）
  GN — 通用/公共辅助设备
  IT — 信息系统设备

示例：EQ-BC-001 表示临床化学组第 1 台设备
```

每台设备在机身显著位置粘贴设备标识卡，标识卡信息包含：设备编号、设备名称、当前管理状态（在用 / 停用 / 待报废）。管理状态变更时须同步更新标识卡。

#### 6.2.2 设备登记册

设备管理员建立并维护《设备登记册》，以电子台账和纸质备份双重形式存档。登记册应包含以下信息字段：

| 字段 | 说明 |
| ---- | ---- |
| 设备编号 | 唯一标识编码 |
| 设备名称 | 通用名称 |
| 制造商/品牌 | 生产厂家 |
| 型号/规格 | 设备型号 |
| 出厂序列号 | 制造商赋予的唯一序列号 |
| NMPA 注册证号 | 医疗器械注册或备案编号 |
| 供应商名称及联系方式 | 采购渠道和服务联络信息 |
| 购置日期 | 合同签订或到货日期 |
| 投入使用日期 | 通过验收并正式使用的日期 |
| 安装位置 | 当前放置的具体实验室区域 |
| 资产价值 | 购置原值 |
| 保修截止日期 | 保修期终止时间 |
| 当前状态 | 在用 / 停用 / 报废 |
| 责任人 | 指定的设备管理责任人 |

登记册至少每半年核查一次，确保信息准确。新增设备在通过验收后 5 个工作日内完成登记。

### 6.3 设备验收 [对应 ISO 15189:2022 §6.4.3]

#### 6.3.1 新购设备验收

新购设备到货后，按照以下步骤实施验收：

**第一阶段：到货开箱检查**

- 核对装箱清单，确认设备主机、配件、随机工具、试剂包、说明书等齐全；
- 检查设备外观，确认运输过程中无损坏、变形、锈蚀、液体泄漏；
- 如发现缺件或损坏，当场拍照记录并联系供应商处理；
- 完成《设备到货开箱检查记录》。

**第二阶段：安装确认（IQ）**

由制造商工程师完成安装，设备管理员和专业组组长现场监督并确认：

- 安装位置的空间、承重满足要求；
- 电源电压、频率、接地保护符合设备规格；
- 供水水质（纯水电阻率）和排水条件满足要求；
- 通讯线路（网络接口、串口）连接正确；
- 环境温度、湿度在设备允许范围内；
- 设备水平调节正常，无异常振动。

**第三阶段：运行确认（OQ）**

设备安装完成后，由工程师和操作人员共同执行：

- 各功能模块自检通过，无报警提示；
- 样品进样系统、检测系统、清洗系统、废液排放系统运行正常；
- 软件版本核实，操作界面功能逐项验证；
- LIS 双向通信测试：下载医嘱、上传结果、危急值标记功能验证；
- 条码识别准确性验证。

**第四阶段：性能确认（PQ）**

由技术负责人制定性能验证方案，操作人员执行，内容包括：

| 验证项目 | 方法 | 判定标准 |
| -------- | ---- | -------- |
| 精密度（批内） | 同一质控水平重复测定 ≥ 10 次 | CV% 不超过制造商声明或相关行业标准 |
| 精密度（批间） | 连续 5 天以上每天测定质控品 | CV% 不超过制造商声明的 1.5 倍 |
| 正确度 | 使用认证参考材料或与参考方法比对 | 偏倚在允许范围内 |
| 线性范围 | 系列稀释高值样品 | 回收率 95%~105%，相关系数 r ≥ 0.995 |
| 携带污染率 | 高-低-高-低交替测定 | 携带污染率 ≤ 制造商声明值 |
| 方法比对（如替换旧设备） | 至少 40 例患者样品新旧设备平行检测 | 相关系数 r ≥ 0.975，偏倚在允许范围内 |

**验收结论**

- 性能确认全部合格：技术负责人签署《设备验收测试报告》，粘贴绿色"在用"标识卡，正式投入使用。
- 存在不合格项：记录不合格内容，联系供应商/制造商处理，处理完成后重新执行相应确认阶段。

#### 6.3.2 设备重新投入使用验收

以下情况下设备需经过重新验收方可恢复使用：

| 触发场景 | 最低验收要求 |
| -------- | ------------ |
| 设备故障维修后（更换关键部件） | OQ + PQ |
| 设备故障维修后（软件修复或非关键部件） | 质控品验证 |
| 设备搬迁至新位置 | IQ + OQ + PQ |
| 长期停用（超过 30 天）后恢复启用 | OQ + 质控品验证 |
| 软件重大版本升级 | OQ + PQ（涉及计算逻辑变更时） |

重新验收方案由技术负责人根据具体情况确定，记录于《设备验收测试报告》。

### 6.4 设备使用管理 [对应 ISO 15189:2022 §6.4.4]

#### 6.4.1 授权操作制度

- 所有设备操作人员必须经过系统化培训并考核合格��方可独立操作。
- 培训内容至少包括：设备原理和结构、开/关机流程、日常操作规范、质控操作、日常维护保养、常见故障识别与初步处理、安全注意事项。
- 培训考核方式：理论考试 + 实操考核，合格标准为各项成绩均 ≥ 80 分。
- 合格人员记入《设备操作授权名单》，授权名单放置在设备操作区域，并在 LIS 中设置相应权限。
- 新员工在取得授权前须在已授权人员指导下操作，指导期不少于 2 周。
- 授权有效期为 2 年，到期需重新考核续授。

#### 6.4.2 操作文件管理

每台主要检测设备在操作区域应放置以下文件：

- **设备操作 SOP（作业指导书）**：涵盖开机自检、样品装载、常规检测、校准操作、质控操作、关机流程、常见报警代码及处置措施；
- **设备日常维护保养 SOP**：按日、周、月分别列明维护项目和操作要点；
- **快速参考卡**：列出关键参数设置、紧急关机步骤、故障报修联系人；

制造商原版使用说明书存放在设备档案中，操作人员可随时查阅。当制造商发布说明书更新版本时，设备管理员在 10 个工作日内完成替换并通知相关人员。

#### 6.4.3 设备安全保障

| 安全类别 | 措施 |
| -------- | ---- |
| 电气安全 | 所有设备使用独立接地插座，配备漏电保护器；核心设备接入 UPS 不间断电源 |
| 化学安全 | 废液桶密封收集，按规定频次更换；强腐蚀性废液单独收集处置 |
| 机械安全 | 运动部件防护罩完好；安全联锁装置定期测试确认有效 |
| 生物安全 | 处理感染性样品的设备操作区域设在生物安全柜或封闭通道内；设备维修前须进行消毒处理 |
| 信息安全 | 设备软件设置操作权限分级（操作级/管理级/维护级）；定期更改管理密码 |

### 6.5 校准管理 [对应 ISO 15189:2022 §6.4.1, 并与 §6.5 程序联动]

#### 6.5.1 分析仪器校准

分析检测仪器按照制造商说明书和《校准和计量溯源管理程序》的要求进行校准。校准时机包括：

- 新设备安装验收时（纳入 PQ 阶段）；
- 试剂批号更换时；
- 制造商规定的定期校准周期到期时；
- 关键部件更换或重大维修后；
- 质控结果显示系统偏移趋势时；
- EQA/PT 结果不满意经排查需重新校准时。

校准执行步骤：

1. 确认校准品在有效期内，按说明书要求复溶或平衡至室温；
2. 按制造商规定的校准程序在仪器上执行校准操作；
3. 审核校准曲线参数（斜率、截距、相关系数等），判断是否在可接受范围；
4. 校准通过后立即运行至少两个浓度水平的质控品进行验证；
5. 质控结果在控方可恢复临床样品检测；
6. 记录校准日期、校准品批号及有效期、校准结果参数于《仪器校准记录表》。

#### 6.5.2 计量器具检定/校准

移液器、分析天平、温度计、计时器等计量器具按国家强制检定目录和实验室需求实施外部检定或校准：

| 计量器具 | 校准/检定方式 | 周期 | 实施机构 |
| -------- | ------------- | ---- | -------- |
| 分析天平 | 法定计量检定 | 每年 1 次 | 当地计量检定机构 |
| 移液器 | 校准 | 每年 1 次（高频使用每半年） | 具备资质的校准机构 |
| 温度计（参考级） | 法定计量检定 | 每年 1 次 | 当地计量检定机构 |
| 离心机转速 | 校准 | 每年 1 次 | 当地计量检定机构 |
| 计时器 | 比对验证 | 每年 1 次 | 内部用标准时间源比对 |

校准/检定合格后在器具上粘贴合格标签（含有效期），不合格的器具立即停用并标记。

### 6.6 预防性维护 [对应 ISO 15189:2022 §6.4.5]

#### 6.6.1 维护计划编制

设备管理员于每年 12 月编制次年度《设备预防性维护计划表》，经技术负责人审批后执行。维护计划应覆盖所有在用设备，明确维护项目、频次、执行人和验收标准。

主要设备类型的典型维护内容：

| 设备类别 | 日维护 | 周维护 | 月维护 | 季度维护 | 年度维护 |
| -------- | ------ | ------ | ------ | -------- | -------- |
| 全自动生化分析仪 | 清洗比色杯/反应杯；检查试剂余量；清理废液桶 | 更换清洗液；管路冲洗；检查进样针状态 | 管路深度清洗；光源状态检查 | 光学系统清洁校准；管路完整性检查 | 制造商年度预防性维护（PM） |
| 全自动血细胞分析仪 | 运行关机清洗程序；检查试剂余量 | 浸泡清洗计数池；背景计数检测 | 管路冲洗剂深度清洗 | 激光/光学通道校准验证 | 制造商年度 PM |
| 全自动免疫分析仪 | 清洗反应杯/反应盘 | 管路冲洗 | 检查探针状态 | 光学系统维护 | 制造商年度 PM |
| 离心机 | 使用后擦拭 | 检查转子和密封件 | — | 转速和温度校准验证 | 外部计量检定 |
| 冰箱/冰柜 | 记录温度 | 检查门封密封性 | 除霜清洁（按需） | 冷凝器除尘清洁 | 温度均匀性验证 |
| 纯水系统 | 检查产水电阻率 | — | 更换预处理滤芯（按需） | 微生物检测 | 全系统维护及水质全项检测 |

#### 6.6.2 维护执行与记录

- 操作人员按维护计划执行日常维护，维护完成后在《设备维护执行记录表》上签名确认。
- 设备管理员每月核查维护记录的完整性，未按时执行的须说明原因并补做。
- 制造商工程师执行的定期维护，需提交维护服务报告，设备管理员归入设备档案。

#### 6.6.3 故障设备管理

设备在使用过程中发生故障或性能偏离规定要求时，按以下流程处置：

```
步骤 1：发现故障
  操作人员立即停止使用该设备，在设备显著位置粘贴红色"停用/故障"标识，
  标注停用日期和故障简述。
      |
步骤 2：初步处置
  暂停该设备上所有待检样品的检测，转移至备用设备（如有）。
  已完成检测但尚未发出的结果暂时不发放，标记为"待审核"。
      |
步骤 3：报告
  操作人员在 30 分钟内口头通知专业组组长和设备管理员，
  并填写《设备故障/停用记录表》。
      |
步骤 4：影响评估
  技术负责人评估故障对最后一次合格质控以来已发出检验结果的影响。
  评估内容包括：故障性质是否影响测量准确性、受影响的样品范围、
  是否需要召回结果。
  评估结论记录于《设备故障/停用记录表》。
      |
步骤 5：维修处理
  (a) 操作人员能自行排除的简单故障 → 排除后运行质控验证 → 合格后恢复使用
  (b) 需制造商或第三方维修 → 设备管理员联系维修并跟踪进度
      |
步骤 6：维修后验收
  按照 6.3.2 重新投入使用验收要求执行。验收合格后撕除"停用/故障"标识，
  更新设备登记册状态为"在用"，恢复临床检测。
      |
步骤 7：记录归档
  维修报告、验收记录、影响评估结论统一归入设备档案。
```

如故障导致需召回已发出结果，由技术负责人启动结果召回流程：通知临床科室 → 重新采样检测 → 发放更正报告 → 记录整个过程。

### 6.7 设备不良事件管理 [对应 ISO 15189:2022 §6.4.6]

#### 6.7.1 不良事件定义和分类

设备不良事件是指经营、使用合格设备过程中发生的或可能发生的、导致或可能导致人员伤害或检验结果严重失准的事件。按严重程度分为：

- **严重不良事件**：导致患者死亡或严重伤害、操作人员严重伤害；
- **一般不良事件**：导致检验结果批量错误并已发出报告、设备导致非严重人身伤害；
- **可疑不良事件**：尚未造成实际损害但有合理依据认为可能导致上述后果。

#### 6.7.2 调查与报告

```
步骤 1：发现不良事件 → 立即停用设备，保护现场
      |
步骤 2：科室内报告 → 操作人员 30 分钟内通知组长、设备管理员、质量负责人
      |
步骤 3：初步调查 → 收集事件经过、设备运行日志、相关样品和结果记录
      |
步骤 4：填写《设备不良事件报告表》→ 技术负责人和质量负责人审核
      |
步骤 5：院内报告 → 报医院设备管理部门和医务处
      |
步骤 6：监管报告（按以下时限）：
  - 导致死亡：立即报告，24 小时内通过国家医疗器械不良事件监测信息系统上报
  - 导致严重伤害：7 个工作日内上报
  - 可能导致上述后果：15 个工作日内上报
      |
步骤 7：根因调查 → 与制造商、供应商协作调查根本原因
      |
步骤 8：纠正和预防措施 → 制定并执行改进措施
      |
步骤 9：结案归档 → 报告及调查材料归入设备档案
```

#### 6.7.3 制造商召回/通知的响应

- 设备管理员负责接收和登记制造商发出的设备安全通知、召回公告、软件更新通知；
- 收到通知后 2 个工作日内评估影响范围，报技术负责人和质量负责人；
- 按照制造商指导的措施在规定时限内完成响应（停用、整改、升级或退回）；
- 响应过程和结果记录在《设备不良事件报告表》中。

### 6.8 设备报废管理

#### 6.8.1 报废条件

设备出现以下情形之一时可申请报废：

- 达到规定使用年限且性能无法满足检验要求；
- 关键部件损坏且无法获得维修备件；
- 维修费用超过设备残值的 50%，且不具有继续维修的经济合理性；
- 技术严重落后，已被新技术或新方法取代；
- 国家法规明令淘汰的设备类型。

#### 6.8.2 报废流程

```
步骤 1：使用科室/专业组提出报废申请，填写《设备报废审批表》
      |
步骤 2：设备管理员核实设备信息，附使用和维修历史记录
      |
步骤 3：技术负责人出具技术评估意见（性能状况、替代方案）
      |
步骤 4：科室负责人审批
      |
步骤 5：提交医院资产管理部门审批（按照医院固定资产管理规定）
      |
步骤 6：批准后，设备管理员执行：
  (a) 更换设备标识卡为"已报废"
  (b) 清除设备中存储的患者数据和系统参数
  (c) 对感染性样品接触过的设备进行终末消毒处理
  (d) 含放射源或危险物质的设备按环保要求处置
  (e) 物理移出实验室
      |
步骤 7：在设备登记册中注销该设备，设备档案转入归档保存
```

### 6.9 设备记录与档案管理 [对应 ISO 15189:2022 §6.4.7]

#### 6.9.1 设备档案内容

每台设备建立独立档案（一机一档），档案应至少包含以下内容：

| 序号 | 档案内容 | 来源 |
| ---- | -------- | ---- |
| 1 | 设备基本信息卡（编号、名称、型号、序列号、制造商、供应商） | 设备登记册 |
| 2 | 采购合同及发票复印件 | 采购记录 |
| 3 | 制造商使用说明书（原版） | 制造商提供 |
| 4 | NMPA 注册证/备案凭证复印件 | 供应商提供 |
| 5 | 安装记录（IQ/OQ） | 验收记录 |
| 6 | 性能确认报告（PQ） | 验收记录 |
| 7 | 操作 SOP 和维护 SOP | 实验室编制 |
| 8 | 操作人员授权名单 | 培训记录 |
| 9 | 校准记录 | 校准执行记录 |
| 10 | 预防性维护计划和执行记录 | 维护记录 |
| 11 | 故障和维修记录 | 维修报告 |
| 12 | 不良事件调查报告 | 事件记录 |
| 13 | 性能比对/验证记录（年度） | 比对报告 |
| 14 | 软件版本变更记录 | 升级记录 |
| 15 | 状态变更记录（停用、恢复、报废） | 变更记录 |

#### 6.9.2 档案管理要求

- 设备档案由设备管理员统一管理，存放在指定的设备档案柜中。
- 电子化档案存储于实验室管理系统中，定期备份。
- 在用设备档案保存至设备报废后 3 年。
- 报废设备档案至少保存至报废后 3 年。
- 档案借阅须登记，不得私自带出实验室或复制。
