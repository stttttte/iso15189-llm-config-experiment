## 第7章 过程要求

### 7.1 检验前过程

| 序号 | 检查条款 | 检查要点 | 审核方法 | 符合性判定 | 审核记录 |
|------|---------|---------|---------|-----------|---------|
| 1.1 | 7.1.1 | 实验室是否建立了与临床用户沟通的信息文件（检验手册/检验项目指南），包含采样要求、样本类型、参考区间等 | 查阅检验手册/项目目录，核实信息的完整性和可及性 | ☐符合 ☐不符合 ☐不适用 | |
| 1.2 | 7.1.1 | 检验手册是否定期评审和更新，临床用户是否能方便获取 | 查看检验手册的版本记录和发放渠道（HIS嵌入/网站/纸质） | ☐符合 ☐不符合 ☐不适用 | |
| 1.3 | 7.1.2 | 检验申请信息是否充分（含患者姓名、性别、年龄、唯一标识、申请项目、采样时间、临床信息等） | 抽查10份检验申请单，核实必填项目的完整性 | ☐符合 ☐不符合 ☐不适用 | |
| 1.4 | 7.1.3 | 样本采集是否有明确的操作规程（静脉采血、末梢血、尿液、体液等） | 查阅各类样本采集SOP，现场观察1-2次采血操作 | ☐符合 ☐不符合 ☐不适用 | |
| 1.5 | 7.1.3 | 采样人员是否经过培训并被授权，采集前是否确认患者身份 | 查阅采样人员培训及授权记录，现场观察患者身份核对流程 | ☐符合 ☐不符合 ☐不适用 | |
| 1.6 | 7.1.4 | 样本运送条件和时限是否有明确规定 | 查阅样本运送管理制度，了解温度和时限要求 | ☐符合 ☐不符合 ☐不适用 | |
| 1.7 | 7.1.4 | 样本运送过程中是否有保护措施（防溢漏、温度控制、生物安全） | 现场检查运送容器、冷链设备，抽查运送记录 | ☐符合 ☐不符合 ☐不适用 | |
| 1.8 | 7.1.5 | 样本接收是否有验收标准和拒收标准 | 查阅样本验收/拒收标准文件，查看LIS中拒收记录 | ☐符合 ☐不符合 ☐不适用 | |
| 1.9 | 7.1.5 | 不合格样本的处理是否有记录，是否通知了申请者 | 抽查近3个月不合格样本记录，核实沟通和处理结果 | ☐符合 ☐不符合 ☐不适用 | |
| 1.10 | 7.1.6 | 样本处理（离心、分注等）是否按规定执行，处理条件是否符合要求 | 现场观察样本前处理流程，查阅离心机参数设置 | ☐符合 ☐不符合 ☐不适用 | |

### 7.2 检验过程

| 序号 | 检查条款 | 检查要点 | 审核方法 | 符合性判定 | 审核记录 |
|------|---------|---------|---------|-----------|---------|
| 2.1 | 7.2.1 | 检验方法选择是否基于满足患者需求并适合预期用途 | 查阅方法选择评审记录，了解方法选择依据 | ☐符合 ☐不符合 ☐不适用 | |
| 2.2 | 7.2.2 | 检验方法是否经过验证或确认，验证/确认记录是否完整 | 抽查3个检验项目的方法验证/确认报告（含精密度、正确度、线性范围、参考区间等） | ☐符合 ☐不符合 ☐不适用 | |
| 2.3 | 7.2.3 | 方法的测量不确定度是否已评定（定量检验项目） | 抽查2个定量项目的测量不确定度评估报告 | ☐符合 ☐不符合 ☐不适用 | |
| 2.4 | 7.2.4 | 使用的参考区间/临床决定值是否适用于服务人群 | 查阅参考区间来源文件，了解是否进行了验证或转化 | ☐符合 ☐不符合 ☐不适用 | |
| 2.5 | 7.2.5 | 每个检验项目是否有标准操作规程（SOP），内容是否完整 | 抽查5份检验SOP，核实是否包含原理、标本要求、试剂、操作步骤、质控、结果报告等 | ☐符合 ☐不符合 ☐不适用 | |
| 2.6 | 7.2.5 | SOP是否为当前有效版本，操作人员是否方便获取 | 现场检查SOP文件放置情况或电子系统中SOP的可及性 | ☐符合 ☐不符合 ☐不适用 | |

### 7.3 质量保证

| 序号 | 检查条款 | 检查要点 | 审核方法 | 符合性判定 | 审核记录 |
|------|---------|---------|---------|-----------|---------|
| 3.1 | 7.3.1 | 是否建立了室内质量控制（IQC）程序，质控规则是否明确 | 查阅IQC管理程序文件，了解质控品选择、频率、规则设定 | ☐符合 ☐不符合 ☐不适用 | |
| 3.2 | 7.3.2 | IQC是否每日执行，失控时是否有处理记录 | 抽查近1个月IQC记录（3个专业组），检查失控处理记录 | ☐符合 ☐不符合 ☐不适用 | |
| 3.3 | 7.3.3 | 是否参加了外部质量评价（EQA/PT），覆盖所有已开展项目 | 查阅EQA参加计划和回报记录，核实项目覆盖率 | ☐符合 ☐不符合 ☐不适用 | |
| 3.4 | 7.3.3 | EQA不合格项目是否进行了根因分析和纠正措施 | 查阅近一年EQA不合格项目的分析和纠正记录 | ☐符合 ☐不符合 ☐不适用 | |
| 3.5 | 7.3.4 | 对于无EQA项目的检验，是否采用了替代方法（如实验室间比对、临床信息评估等） | 查阅替代评估计划和实施记录 | ☐符合 ☐不符合 ☐不适用 | |
| 3.6 | 7.3.5 | 不同设备/方法间是否进行了结果比对 | 查阅仪器间比对记录（如两台生化分析仪之间），了解比对频率和接受标准 | ☐符合 ☐不符合 ☐不适用 | |
