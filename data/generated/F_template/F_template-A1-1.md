## 1 目的

建立系统化的人员培训与能力评估管理制度，规范实验医学科培训计划的制定与实施、培训效果评估、人员能力评估方法与频次、继续教育管理及不合格处理等全过程，确保所有从事检验活动的人员持续具备胜任其岗位职责的能力，满足 ISO 15189:2022 第 6.2 条（人员）全部子条款的要求。

## 2 适用范围

2.1 本程序适用于实验医学科所有从事检验前、检验和检验后过程活动的人员，包括但不限于：

- 正式在编人员
- 合同制聘用人员
- 进修生、实习生、规培生
- 参与床旁检验（POCT）的临床科室人员
- 有权限进入检验信息系统（LIS）的管理及技术人员

2.2 本程序覆盖以下管理活动：

- 入职培训与岗前培训
- 在职培训与继续教育
- 安全培训与合规培训
- 新员工入岗能力评估
- 常规年度能力评估
- 复杂项目加密评估
- 离岗人员复岗评估
- 评估结果的统计分析与不合格处理

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求

#### 3.1.1 第 6.2.2 条 能力要求

> "实验室应确定各项职能所需的能力，包括教育、资格、培训、技术知识、技能和经验的要求。"
>
> "应按照文件化程序，在规定的时间间隔内评估每个人员执行分配任务的能力。"
>
> "评估方法应至少包括以下适用方法的组合：
> a) 直接观察常规工作过程和程序，包括所有适用的安全实践；
> b) 直接观察对设备维护和功能检查的执行；
> c) 监控检验和检验后过程的记录和结果；
> d) 评审工作记录；
> e) 评估解决问题的能力；
> f) 检查特别提供的样品，如先前检验过的样品、实验室间比对材料、分割样品。"
>
> "在新员工入职后的规定期间内（如最初六个月）应至少进行两次能力评估，之后每年进行评估。"

#### 3.1.2 第 6.2.3 条 人员培训

> "实验室应确保人员的能力通过适当的培训来获得和保持。"

#### 3.1.3 第 6.2.4 条 继续教育和职业发展

> "实验室应为所有人员提供继续教育方案。参加继续教育方案的人员应能证明其继续教育活动的效果。"
>
> "当使用电子系统或手工系统发布检验结果时，应对有权限进入检验信息系统的所有人员提供培训，以确保患者数据的保密性。"

## 4 职责

| 角色 | 培训管理职责 | 能力评估职责 |
|------|-------------|-------------|
| 科主任 | 批准年度培训计划和培训预算 | 批准年度能力评估计划，审核评估结果汇总报告 |
| 技术负责人 | 审核技术培训内容，确认培训师资 | 制定各岗位能力要求，组织年度评估计划编排，审核评估结果 |
| 质量负责人 | 组织合规培训（公正性、保密性、安全），监督培训执行 | 监督评估流程的执行合规性，协调评估与质量监督的联动 |
| 培训管理员 | 制定培训计划、组织培训实施、管理培训档案、统计继续教育学分 | 协助整理评估资料，归档评估记录 |
| 各专业组组长 | 提出本组培训需求，协助组织本组专业培训，参与带教 | 执行本组人员能力评估，记录评估结果，提出培训建议 |
| 质量监督员 | 参与培训效果的行为观察评估 | 参与直接观察评估，提供日常工作质量监控数据 |
| 全体人员 | 按计划参加培训，完成培训考核，保存继续教育证明 | 配合评估安排，按要求完成各项评估任务 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 能力评估（Competence Assessment） | 通过系统化方法评价人员执行指定任务所需知识、技能和行为的过程 |
| 直接观察（Direct Observation） | 评估人员在工作现场直接观看被评估人执行操作的评估方法 |
| 盲样测试（Blind Sample Testing） | 将已知结果的样品以常规标本形式提供给被评估人检测，评估其检验能力 |
| 入岗评估（Initial Competence Assessment） | 新员工入职后在规定期间内进行的初始能力评估 |
| 复岗评估（Return-to-duty Assessment） | 人员离岗后重新返回岗位时进行的能力评估 |
| 加密评估（Enhanced Assessment） | 对复杂或高风险检验项目增加评估频次的安排 |
| 入职培训（Orientation Training） | 新员工入职后的初始培训，包括通用培训和岗位专业培训 |
| 在职培训（On-the-job Training） | 在岗人员日常接受的持续性培训活动 |
| 继续教育（Continuing Education） | 获取和更新专业知识、技能的持续性教育活动，通常以学分计量 |
| 合规培训（Compliance Training） | 涉及法规遵从、公正性、保密性、患者权益等方面的培训 |
| POCT培训 | 对参与床旁检验（Point-of-Care Testing）人员的专项培训 |
| 培训效果评估（Training Effectiveness Evaluation） | 通过考核、实操、满意度调查等方式衡量培训达到预期目标的程度 |

## 6 程序内容

### 第一部分：培训管理

#### 6.1 年度培训计划制定与执行追踪

##### 6.1.1 培训需求分析

每年12月由培训管理员组织开展下年度培训需求分析，信息来源包括：

| 序号 | 来源 | 说明 |
|------|------|------|
| 1 | 能力评估结果 | 评估中发现的知识或技能薄弱环节（见本程序第二部分） |
| 2 | 质量监督反馈 | 日常监督中发现的操作偏差问题 |
| 3 | 不符合与纠正措施 | 内审、外审、投诉处理中发现的人员能力相关问题 |
| 4 | 新项目与新设备 | 即将开展的新检验项目或新采购的仪器设备 |
| 5 | 法规与标准更新 | 新发布或修订的法规、标准、指南 |
| 6 | 人员变动情况 | 新入职人员、调岗人员、复岗人员 |
| 7 | 管理评审建议 | 管理评审中提出的培训改进建议 |
| 8 | 人员自身需求 | 个人提出的专业发展和学习需求 |

##### 6.1.2 年度培训计划编制

1. 培训管理员汇总培训需求，编制《年度培训计划》（SYK-FM-06-001-01）。
2. 计划应包含：培训课程名称、培训类型、培训对象、培训师资、培训时间、培训学时、考核方式、预算。
3. 质量负责人审核，科主任批准后发布执行。
4. 年度培训计划应确保覆盖本程序 6.2 至 6.6 节规定的所有必修培训主题。

##### 6.1.3 计划执行追踪

1. 培训管理员每季度汇总培训执行情况，编制《培训执行追踪表》。
2. 追踪内容包括：已完成及未完成课程、参训人次、考核通过率、延期原因及补训安排。
3. 年底编制《年度培训总结报告》（SYK-FM-06-001-02），内容涵盖培训完成率、考核合格率、学分达标率等关键指标，提交管理评审。

#### 6.2 入职培训

新员工入职后2周内完成以下培训模块：

| 培训模块 | 培训内容 | 学时 | 考核方式 |
|----------|----------|------|----------|
| 科室概况 | 科室组织架构、工作流程、科室文化、岗位职责 | 2学时 | 无需考核 |
| 质量管理 | 质量方针与目标、质量管理体系概述、文件管理制度 | 2学时 | 笔试（合格线80分） |
| 生物安全 | 实验室生物安全等级、个人防护措施、职业暴露应急处理、废弃物处理 | 4学时 | 笔试+实操 |
| 信息安全与保密 | 患者数据保密要求、LIS系统使用规范、信息安全操作守则 | 2学时 | 笔试（合格线80分） |
| 公正性与伦理 | 公正性承诺、利益冲突识别、患者权益保护 | 2学时 | 签署《公正性承诺书》和《保密协议》 |
| SOP体系 | 文件查阅方法、SOP执行要求、变更管理 | 2学时 | 笔试（合格线80分） |
| 岗位专业培训 | 本岗位SOP、仪器原理与操作、质控操作规程、结果判读 | 视岗位而定 | 实操考核 |

入职培训完成后填写《入职培训清单》（SYK-FM-06-001-08），由培训管理员和新员工双方签字确认。

#### 6.3 在职培训

| 培训内容 | 频率 | 培训方式 |
|----------|------|----------|
| SOP更新培训 | SOP修订后即时开展 | 集中讲解+阅读确认签字 |
| 新检验项目培训 | 项目上线前完成 | 理论授课+实操考核 |
| 新设备操作培训 | 设备启用前完成 | 厂家培训+科内转训 |
| 质量改进培训 | 按需安排 | 案例分析+小组讨论 |
| 仪器故障排除培训 | 每半年至少1次 | 模拟故障演练 |

#### 6.4 专业培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 专业学术讲座 | 每月至少1次 | 全科人员 |
| 疑难病例讨论 | 每月1至2次 | 相关专业组 |
| 新技术与新方法学习 | 按需安排 | 相关人员 |
| 外派学术会议或培训班 | 按需安排 | 科主任批准的外派人员 |

#### 6.5 安全培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 生物安全年度培训 | 每年至少1次 | 全科人员 |
| 消防安全培训与演练 | 每年至少1次 | 全科人员 |
| 化学品安全培训 | 每年至少1次 | 接触化学危险品人员 |
| 职业暴露处置演练 | 每年至少1次 | 全科人员 |
| 应急预案演练 | 每年至少1次 | 全科人员 |

#### 6.6 合规培训

| 培训内容 | 频率 | 对应要求 |
|----------|------|----------|
| 公正性培训 | 每年至少1次 | ISO 15189:2022 4.1.1.3 |
| 保密性培训 | 每年至少1次 | ISO 15189:2022 4.1.1.4 |
| 患者权益保护培训 | 每年至少1次 | 相关法律法规 |
| 伦理培训 | 每年至少1次 | 科研伦理与知情同意 |
| 法律法规更新培训 | 法规更新后即时 | 新发布的法规与标准 |

合规培训的特殊要求：

1. 合规培训须有独立的签到记录和考核记录，即使与其他培训合并举办。
2. 公正性培训须签署《公正性承诺书》（SYK-FM-06-001-10）。
3. 保密性培训须签署《保密协议》（SYK-FM-06-001-11）。
4. 以上记录纳入个人人事档案长期保存。

#### 6.7 POCT专项培训

对参与床旁检验管理和操作的人员进行专项培训：

| 培训内容 | 适用人员 | 频率 |
|----------|----------|------|
| POCT设备操作培训 | POCT操作人员（含临床科室人员） | 上岗前+每年复训 |
| POCT质量控制培训 | POCT操作人员 | 上岗前+每年复训 |
| POCT结果判读培训 | POCT操作人员 | 上岗前 |
| POCT管理制度培训 | 实验医学科POCT管理人员 | 按需 |

POCT操作人员须经培训考核合格后方可获得POCT操作授权。

#### 6.8 培训课程与课件管理

##### 6.8.1 培训课程管理

每次培训应有正式的培训课程信息记录，包括：

- 课程名称与课程编号
- 培训类型（入职/在职/专业/安全/合规/POCT）
- 培训日期、时间、地点
- 培训师资（内部讲师或外部专家）
- 培训方式（面授、线上、实操、自学、混合）
- 培训学时
- 考核方式及合格标准

##### 6.8.2 培训课件管理

1. 培训课件由培训师编制，技术负责人或质量负责人审核后方可使用。
2. 课件统一存储于科室指定的电子文件系统或共享盘。
3. 课件应注明版本号和编制日期。
4. 涉及SOP内容的课件应与当前有效版本保持一致，SOP修订后课件应同步更新。

##### 6.8.3 培训签到管理

1. 每次培训填写《培训签到表》（SYK-FM-06-001-03），记录参加人员亲笔签名及培训起止时间。
2. 线上培训应留存系统登录记录或在线签到截图。
3. 培训签到表与培训课件一同归档保存。

#### 6.9 培训效果评估

##### 6.9.1 评估方式

| 评估方式 | 说明 | 适用场景 |
|----------|------|----------|
| 笔试或在线考核 | 理论知识测试，满分100分，合格线80分 | 理论培训 |
| 实操考核 | 现场操作演示，由培训师或组长评分 | 技能培训 |
| 满意度调查 | 培训满意度问卷，评价课程内容、师资水平和组织效果 | 所有培训 |
| 行为观察 | 培训后一段时间内观察工作行为是否改善 | 重点培训 |
| 结果分析 | 培训后检验质量指标是否改善 | 针对性培训 |

##### 6.9.2 考核不合格处理

1. 笔试或实操考核不合格者，安排2周内补考。
2. 补考仍不合格者，须重新参加培训后再行考核。
3. 关键培训（安全培训、SOP培训、岗位专业培训）未通过者，不得独立上岗操作。
4. 考核结果记录于个人培训档案。

#### 6.10 继续教育学分管理

##### 6.10.1 学分要求

根据省（市）卫生健康委员会要求，各类人员年度继续教育学分要求如下：

| 人员类别 | 年度学分要求 | 其中I类学分 | 其中II类学分 |
|----------|-------------|------------|-------------|
| 高级职称人员 | 不少于25分 | 不少于5分 | 不少于20分 |
| 中级职称人员 | 不少于25分 | 不少于5分 | 不少于20分 |
| 初级职称人员 | 不少于25分 | 不少于5分 | 不少于20分 |

##### 6.10.2 学分登记与跟踪

1. 培训管理员维护《继续教育学分登记表》（SYK-FM-06-001-04），逐人逐项登记学分获取情况。
2. 每季度汇总学分完成进度，对学分不足人员发出书面提醒。
3. 年底核查全员学分达标情况，未达标人员的分析结果纳入年度培训总结报告。

#### 6.11 外部培训与学术会议管理

1. 人员参加外部培训或学术会议须事先填写《外部培训/会议申请表》（SYK-FM-06-001-05），经专业组组长同意、科主任批准后方可参加。
2. 培训或会议结束后2周内须提交以下材料：
   - 培训或会议总结报告
   - 证书或学分证明复印件
   - 科室内转训安排（如适用）
3. 外出培训学习到的新知识或新技术，应安排向本组或全科人员进行转训分享，并做好培训记录。

### 第二部分：能力评估管理

#### 6.12 各项职能的能力要求定义

##### 6.12.1 能力要求框架

每个岗位或检验项目的能力要求应从以下六个维度进行定义：

| 维度 | 说明 | 示例 |
|------|------|------|
| 教育背景 | 学历、专业要求 | 医学检验技术专业本科及以上 |
| 专业资格 | 执业资格证书、上岗证 | 临床检验技师资格证 |
| 培训经历 | 必须完成的培训项目 | 生物安全培训、仪器操作培训 |
| 技术知识 | 应掌握的专业理论知识 | 检验原理、质控规则、参考区间 |
| 操作技能 | 应具备的实际操作能力 | 样品处理、仪器操作、结果审核 |
| 工作经验 | 相关岗位工作年限 | 本岗位工作满1年以上 |

##### 6.12.2 能力要求文件管理

1. 技术负责人组织各专业组制定《岗位/项目能力要求表》（SYK-FM-06-001-06）。
2. 能力要求应与岗位说明书中的任职条件保持一致。
3. 新项目开展或岗位职责发生重大变化时，应及时修订能力要求。
4. 能力要求文件经技术负责人审核、科主任批准后生效。

#### 6.13 年度能力评估计划编排

##### 6.13.1 年度评估计划

1. 每年1月由技术负责人组织编制《年度能力评估计划》（SYK-FM-06-001-07）。
2. 评估计划应覆盖所有在岗人员和所有已授权检验项目。
3. 常规评估间隔不超过12个月（即每年至少评估一次）。
4. 计划应明确：被评估人、评估项目、评估方法组合、评估人、计划评估时间。

##### 6.13.2 各类评估频次要求

| 评估类型 | 适用场景 | 频次要求 |
|----------|----------|----------|
| 新员工入岗评估 | 新入职人员 | 前6个月内至少2次 |
| 常规年度评估 | 在岗人员 | 每12个月至少1次 |
| 复岗评估 | 离岗超过2年的人员 | 返岗前完成评估 |
| 加密评估 | 复杂或高风险项目 | 每6个月1次（见6.17） |
| 专项评估 | 新项目或新方法上线 | 开展前完成评估 |
| 不合格后复评 | 评估不合格人员 | 完成培训后1个月内 |

#### 6.14 评估方法

##### 6.14.1 评估方法组合

每次评估应至少采用以下方法中的两种及以上组合：

| 编号 | 评估方法 | 说明 | 适用场景 |
|------|----------|------|----------|
| M1 | 直接观察 | 评估人在工作现场观察被评估人执行标准操作流程，包括安全操作和设备维护检查 | 所有岗位 |
| M2 | 盲样测试 | 将已知结果样品作为常规样品送检，评估检验准确性 | 有定量或定性结果的检验项目 |
| M3 | 理论考核 | 笔试或口试，考核专业知识、SOP理解、质量控制知识 | 所有岗位 |
| M4 | 审查工作记录 | 回顾性审查被评估人的检验记录、质控记录、异常处理记录 | 所有岗位 |
| M5 | 检查特别提供的样品 | 使用先前检验过的样品、室间质评（EQA）材料、分割样品进行评估 | 实验室间比对项目 |
| M6 | 解决问题能力评估 | 提供异常情景（如仪器故障、质控失控、危急值），评估判断和处理能力 | 中级及以上人员 |

| 岗位/项目类别 | M1 直接观察 | M2 盲样 | M3 理论 | M4 记录审查 | M5 特殊样品 | M6 问题解决 |

| 样品采集与接收 | ● | — | ● | ● | — | ○ |
| 临床化学（生化） | ● | ● | ● | ● | ○ | ○ |
| 临床免疫 | ● | ● | ● | ● | ○ | ○ |
| 临床血液与体液 | ● | ● | ● | ● | ● | ○ |
| 形态学检查 | ● | ● | ● | ● | ● | ● |
| 临床微生物 | ● | ● | ● | ● | ● | ● |
| 分子检测（PCR） | ● | ● | ● | ● | ● | ● |
| 结果审核与报告签发 | ● | — | ● | ● | — | ● |

#### 6.15 新员工入岗评估

##### 6.15.1 评估安排

1. 新员工入岗后6个月内应完成至少2次能力评估。
2. 评估时间安排：
   - 第1次评估：入岗后第2至3个月，重点评估基本操作能力和安全规范。
   - 第2次评估：入岗后第5至6个月，全面评估独立工作能力。

##### 6.15.2 评估内容

| 评估次数 | 评估重点 | 评估方法 | 合格标准 |
|----------|----------|----------|----------|
| 第1次 | 基本操作规范、安全操作、SOP理解程度 | M1 + M3 | 理论成绩不低于80分，操作无重大偏差 |
| 第2次 | 独立工作能力、结果判读、异常情况处理 | M1 + M2 + M4 + M6 | 盲样结果偏差在允许范围内，操作规范 |

##### 6.15.3 评估结果处理

- **两次评估均合格**：由技术负责人签署独立上岗确认，正式授予检验项目操作权限。
- **任一次评估不合格**：制定针对性培训计划（参照本程序第一部分），培训后1个月内安排复评。
- **经培训和复评仍不合格**：评估该人员是否适合继续在该岗位工作，必要时调整岗位或终止聘用。

#### 6.16 常规年度评估

##### 6.16.1 评估实施

1. 各专业组组长按年度计划执行本组人员能力评估。
2. 每次评估至少采用两种评估方法组合。
3. 评估过程应客观公正，评估人与被评估人不得为同一人。
4. 评估完成后及时填写《能力评估记录表》（SYK-FM-06-001-09）。

##### 6.16.2 评估记录内容

每次评估记录应包括：

- 被评估人基本信息（姓名、岗位、工号）
- 评估项目与评估方法
- 评估内容及评分标准
- 评估得分与结论（合格/不合格）
- 评估人签名及日期
- 不合格项说明及改进建议（如有）

#### 6.17 复杂项目加密评估

对以下复杂或高风险检验项目，在常规年度评估基础上增加评估频次：

| 项目类别 | 加密评估间隔 | 加密评估方法重点 | 说明 |
|----------|-------------|-----------------|------|
| 形态学检查（血涂片/骨髓/尿沉渣） | 每6个月 | M2盲样（已知形态标本）+ M5（分割样品比对） | 形态辨认高度依赖主观判断 |
| 微生物检验（培养/鉴定/药敏） | 每6个月 | M2盲样（已知菌株）+ M1直接观察（无菌操作） | 涉及培养鉴定的主观判断环节 |
| 分子检测（PCR等） | 每6个月 | M2盲样 + M1直接观察（防污染操作） | 高灵敏度项目，污染风险高 |
| 其他高风险项目 | 每6个月 | 根据项目特点选择方法组合 | 由技术负责人确定项目清单 |

#### 6.18 离岗超过2年的复岗评估

##### 6.18.1 适用条件

人员因以下原因离岗超过连续2年，返岗前须重新进行能力评估：

- 长期病假
- 产假或育儿假
- 外派或借调
- 停职
- 其他原因导致的长期脱岗

##### 6.18.2 复岗评估流程

```
复岗申请 → 岗位条件复核 → 培训安排 → 能力评估（等同新员工标准） → 评估合格 → 重新授权 → 恢复上岗
```

1. 复岗人员需重新参加岗前培训，培训内容包括生物安全、SOP更新内容、仪器操作等。
2. 能力评估标准等同于新员工入岗评估（6个月内至少2次）。
3. 所有原有授权在复岗评估合格前处于暂停状态。
4. 评估合格后由技术负责人重新签署授权文件。

#### 6.19 评估结果判定标准

| 评估方法 | 合格标准 |
|----------|----------|
| 理论考核（M3） | 得分不低于80分（满分100分） |
| 直接观察（M1） | 无关键操作偏差，总评分不低于85分 |
| 盲样测试（M2） | 所有项目结果在允许偏差范围内 |
| 工作记录审查（M4） | 无重大差错，记录完整规范 |
| 特殊样品检查（M5） | 检测结果与参考值一致或在允许范围内 |
| 问题解决评估（M6） | 正确识别问题性质并采取适当处理措施 |

#### 6.20 不合格处理流程

##### 6.20.1 处理程序

```
评估不合格 → 书面通知被评估人及其组长 → 分析不合格原因 → 制定针对性培训计划 → 执行培训 → 复评（1个月内）→ 合格/再次不合格
```

1. 评估不合格时，应以书面形式通知被评估人并详细分析原因。
2. 由组长与技术负责人共同制定针对性培训计划，明确培训内容、方式和时限。
3. 培训完成后1个月内进行复评。
4. 复评仍不合格的：
   - 限制或暂停其相关检验项目的操作授权。
   - 评估是否需要调整岗位。
   - 记录于个人档案并纳入下次管理评审议题。

##### 6.20.2 不合格评估处理记录

所有不合格评估的处理过程应填写《不合格评估处理记录》（SYK-FM-06-001-12），完整记录原因分析、培训计划、复评结果及最终处理决定。

#### 6.21 评估结果统计分析

1. 每年编制《年度能力评估汇总报告》（SYK-FM-06-001-13），内容包括：
   - 评估完成率（实际评估人次 / 计划评估人次 x 100%）
   - 一次评估合格率
   - 不合格项分布分析（按专业组、评估方法、不合格类型）
   - 复评合格率
   - 趋势分析（与上年度数据比较）
   - 培训需求建议
2. 报告经技术负责人审核后提交管理评审。

### 第三部分：培训与能力评估的联动机制

#### 6.22 能力评估与培训的联动

1. 能力评估中发现的不合格项应作为下一周期培训计划的重要输入。
2. 培训完成后应通过能力评估验证培训效果。
3. 联动关系如下：

```
能力评估发现问题 → 分析原因 → 制定培训计划 → 实施培训 → 培训效果评估 → 能力复评 → 确认改进
```

#### 6.23 能力评估与质量监督的联动

1. 日常质量监督中发现的人员能力问题应及时触发专项评估。
2. 能力评估结果应反馈给质量监督员，作为确定监督重点的依据。
3. 具体联动机制如下：

| 触发条件 | 联动动作 |
|----------|----------|
| 质量监督发现某人员多次出现同类操作偏差 | 启动该人员该项目的专项能力评估 |
| 室间质评（EQA）结果不合格 | 评估相关操作人员的检验能力 |
| 患者投诉涉及检验质量 | 评估相关操作人员能力并追溯原因 |
| 能力评估发现不合格 | 通知质量监督员加强对该人员的日常监督 |
| 新员工入岗 | 质量监督员将其纳入重点监督对象 |
| 复岗人员重新上岗 | 质量监督员将其纳入重点监督对象（至少3个月） |
