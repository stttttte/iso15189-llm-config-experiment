## 第 8 章　过程要求

### 8.1～8.2 检验前过程

| 序号 | 条款号 | 检查要点 | 审核方法 | 符合性判定 | 审核记录/证据 |
|:----:|:------:|---------|---------|:----------:|-------------|
| 20 | 8.1 | 实验室是否与临床沟通确定检验项目的临床适用性？是否提供检验项目目录和采样指南？ | 查阅检验项目目录、采样手册、临床沟通记录 | ☐C ☐NC ☐OBS ☐NA | |
| 21 | 8.2.1 | 检验申请单信息是否完整（患者标识、临床信息、样本类型、采集时间等）？ | 抽查申请单，核对必填项是否齐全 | ☐C ☐NC ☐OBS ☐NA | |
| 22 | 8.2.2 | 样本采集、运输和接收是否有明确的操作规程？是否记录样本接收状态？ | 查阅样本采集SOP、运输条件记录、样本接收登记表 | ☐C ☐NC ☐OBS ☐NA | |
| 23 | 8.2.3 | 不合格样本的识别、处理和记录流程是否建立？拒收标准是否明确？ | 查阅不合格样本处理制度、拒收记录、与临床沟通记录 | ☐C ☐NC ☐OBS ☐NA | |
| 24 | 8.2.4 | 样本是否有唯一标识？标识是否贯穿检验前、检验中和检验后全过程？ | 现场查看样本标识、追溯LIS中样本流转记录 | ☐C ☐NC ☐OBS ☐NA | |

|:----:|:------:|---------|---------|:----------:|-------------|
| 25 | 8.3.1 | 检验方法是否经过确认或验证？方法性能指标（精密度、准确度、线性范围等）是否满足临床需求？ | 查阅方法确认/验证报告、性能评价数据 | ☐C ☐NC ☐OBS ☐NA | |

| 27 | 8.3.3 | 室内质量控制是否按规定执行？质控规则是否合理？失控时是否有纠正措施？ | 查阅质控计划、质控图、失控分析和处理记录 | ☐C ☐NC ☐OBS ☐NA | |
| 28 | 8.3.4 | 实验室是否参加室间质量评价（EQA/PT）？结果是否满意？不满意结果是否有整改措施？ | 查阅EQA参加计划、EQA成绩报告、不满意结果的整改记录 | ☐C ☐NC ☐OBS ☐NA | |
| 29 | 8.3.5 | 当缺乏正式EQA计划时，实验室是否采用替代方案（如实验室间比对、临床相关性评估等）确保结果可比性？ | 查阅比对方案、比对数据分析报告 | ☐C ☐NC ☐OBS ☐NA | |
| 30 | 8.3.6 | 检验过程中的关键参数（如温度、时间、试剂批号等）是否记录？原始数据是否可追溯？ | 查阅检验原始记录、仪器运行日志、LIS数据追溯 | ☐C ☐NC ☐OBS ☐NA | |

| 36 | 8.4.6 | 样本保留和处置是否有明确规定？保留时间是否满足复检和溯源需要？ | 查阅样本保留制度、保留条件记录、样本处置记录 | ☐C ☐NC ☐OBS ☐NA | |
