# 检验过程质量控制专项内审检查表

**依据标准：** ISO 15189:2022 第 7.3 条 / CNAS-CL02:2023 第 7.3 条

**审核范围：** 检验过程质量控制（检验程序确认、质量控制、检验结果管理）

**文件编号：** E_rules_v2-B2-2

**编制日期：** 2026-04-03

---

| 序号 | 对应条款 | 检查项 | 检查方法 | 符合性判定 | 审核记录 |
|:---:|:---:|------|------|:---:|------|
| 1 | 7.3.1 | 实验室是否建立并维护了检验程序的选择、验证和确认的文件化程序？所选检验程序是否适用于预期用途并优先采用已发表的权威方法？ | 查阅检验程序选择与确认的管理文件；核查检验程序清单及其来源依据 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 2 | 7.3.2 | 实验室在引入新检验程序前，是否按规定进行了性能验证或性能确认？验证/确认内容是否涵盖正确度、精密度、测量区间、生物参考区间、测量不确定度等关键性能指标？ | 查阅新项目性能验证/确认报告；核对验证参数是否完整 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 3 | 7.3.2 | 实验室使用的检验程序如有修改（包括更换试剂批号、更换设备等），是否重新评价其对检验结果的影响并保留记录？ | 查阅检验程序变更记录；抽查试剂批号更换时的比对记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 4 | 7.3.3 | 检验程序的性能确认是否由具备资质的人员实施？确认方案是否经过审批？确认结论是否经授权人员批准？ | 查阅确认方案审批记录；核查实施人员的资质与授权文件 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 5 | 7.3.3 | 当检验程序来源为实验室自建方法（LDT）时，是否按 ISO 15189:2022 要求进行了全面的方法确认，并形成完整的确认报告？ | 查阅自建方法确认报告；核查确认项目是否覆盖分析灵敏度、分析特异性、测量区间、干扰因素等 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 6 | 7.3.4 | 实验室是否建立了测量不确定度的评定程序？是否对每个定量检验程序评定了测量不确定度？评定结果是否定期复审？ | 查阅测量不确定度评定程序文件及各检验程序的评定报告；核查复审周期记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 7 | 7.3.4 | 测量不确定度的评定是否考虑了检验前和检验后过程中影响测量结果的各分量？评定结果是否用于临床决策和结果解释？ | 查阅测量不确定度报告中的分量来源分析；询问相关人员如何将不确定度信息应用于临床沟通 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 8 | 7.3.5 | 实验室是否建立了生物参考区间的确定和验证程序？引用的生物参考区间是否适用于本实验室服务的人群？ | 查阅生物参考区间管理文件；抽查具体检验程序的生物参考区间来源与验证记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 9 | 7.3.5 | 当实验室的服务人群特征发生变化或检验程序发生变更时，是否重新评估了生物参考区间的适用性？ | 查阅变更后的生物参考区间复评记录；核查是否存在应评估而未评估的情况 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 10 | 7.3.6 | 实验室是否建立了内部质量控制程序？质控计划是否覆盖所有已开展的检验程序？质控频次是否合理并形成文件？ | 查阅室内质量控制管理文件及质控计划；核查质控频次设置的依据 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 11 | 7.3.6 | 实验室是否使用适当的质控物（含独立第三方质控物）监控检验过程的正确度和精密度？质控物的选择是否尽可能模拟患者样本基质？ | 查阅质控物采购与使用记录；核查是否包含独立第三方质控物 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 12 | 7.3.6 | 质控数据是否按规定进行统计分析和评价（如 Westgard 规则或等效方法）？失控情况是否有明确的判断标准和处理流程？ | 查阅质控数据分析记录（如 Levey-Jennings 图）；抽查失控记录及纠正措施 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 13 | 7.3.6 | 失控后实验室是否评估了对已报告患者结果的影响？是否根据评估结果采取了适当的纠正措施（包括必要时的结果召回）？ | 查阅失控影响评估记录及纠正措施记录；询问相关人员失控后处理流程 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 14 | 7.3.7 | 实验室是否参加了 CNAS 认可的能力验证计划？能力验证计划是否覆盖所有已认可的检验程序？ | 查阅能力验证参加计划及报名记录；核查覆盖范围是否完整 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 15 | 7.3.7 | 能力验证样本是否按照常规检验流程进行检测？是否存在特殊处理或优待能力验证样本的情况？ | 询问检测人员能力验证样本处理流程；查阅操作记录核实是否与常规样本一致 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 16 | 7.3.7 | 能力验证结果不满意时，实验室是否及时进行了根本原因分析并采取了纠正措施？纠正措施的有效性是否得到验证？ | 查阅能力验证不满意结果的原因分析报告及纠正措施记录；核查有效性验证记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 17 | 7.3.7 | 对于无法获得能力验证计划的检验程序，实验室是否建立了替代评估方案（如实验室间比对）？替代方案是否形成文件？ | 查阅替代评估方案文件及实施记录；核查覆盖了哪些无能力验证的检验程序 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 18 | 7.3.7 | 实验室是否定期对能力验证和实验室间比对的结果进行趋势分析？分析结果是否纳入质量改进？ | 查阅能力验证年度总结报告；核查趋势分析记录及改进措施 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 19 | 7.3.8 | 实验室是否建立了检验结果可比性的评价程序？使用多个设备或多种检验程序检测同一项目时，是否进行了结果比对？ | 查阅设备间/方法间比对程序文件及比对记录；核查比对频次是否符合要求 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 20 | 7.3.8 | 比对结果不可接受时，实验室是否采取了纠正措施？是否评估了对已报告结果的潜在影响？ | 查阅比对不可接受时的处理记录；核查影响评估和纠正措施的闭环情况 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 21 | 7.3.1-7.3.8 | 检验过程质量控制相关的所有记录是否完整、可追溯？记录的保存期限是否符合实验室文件管理要求？ | 抽查各类质控记录的完整性和可追溯性；核查记录保存期限 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 22 | 7.3.1-7.3.8 | 实验室是否对检验过程质量控制相关人员进行了充分的培训？相关人员是否理解质控规则、失控处理和能力验证要求？ | 查阅培训记录；现场询问操作人员质控规则和失控处理流程 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 23 | 7.3.6 | 实验室是否对定性检验程序和半定量检验程序建立了适当的质量控制措施？控制频次和判定标准是否明确？ | 查阅定性/半定量项目的质控计划；核查质控物选择和控制频次是否合理 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 24 | 7.3.4 | 实验室在向临床提供检验结果时，是否在必要时提供测量不确定度信息？临床医生是否能方便地获取相关信息？ | 询问临床科室是否了解如何获取测量不确定度信息；查阅结果报告格式 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |
| 25 | 7.3.2 | 实验室是否建立了检验程序定期复审机制？是否根据最新的循证医学证据、技术发展和临床需求对检验程序进行评价和更新？ | 查阅检验程序复审计划及复审记录；核查近一年内是否有检验程序更新 | ☐ 符合 ☐ 不符合 ☐ 不适用 | |

**备注：**
1. 本检查表依据 ISO 15189:2022 第 7.3 条及 CNAS-CL02:2023 相应条款编制，涵盖检验程序的选择与确认（7.3.1-7.3.3）、测量不确定度（7.3.4）、生物参考区间（7.3.5）、内部质量控制（7.3.6）、能力验证与实验室间比对（7.3.7）、结果可比性（7.3.8）等全部子条款。
2. 审核时应结合实验室实际开展的检验程序，对每条检查项进行逐一核实，必要时扩展至具体科室和岗位。
3. 不符合项应在内审报告中详细描述客观事实、不符合条款及严重程度，并要求受审核部门制定纠正措施。
