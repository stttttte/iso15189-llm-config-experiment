## 1. 整改总体策略

本整改方案采用 **PDCA**（Plan-Do-Check-Act）循环驱动模式，将 3 项不符合项视为体系性缺陷的体现，从根源上消除问题。整改分为 4 阶段、4 个月完成：

```
Plan         Do          Check       Act
策划阶段 → 执行阶段 → 验证阶段 → 闭环阶段
2/1—2/14   2/15—4/30  5/1—5/20  5/21—5/31
```

**整改原则**：
1. 严重不符合优先于一般不符合处理。
2. 立即措施（防扩散、保护患者）优先于体系措施（防再发）。
3. 程序修订须与人员培训和工具更新同步。
4. 验证必须有客观证据；不接受主观陈述。

## 2. 不符合项 1（NC-2026-001）— 文件控制失效

### 2.1 P 阶段（Plan）— 策划

#### 2.1.1 不符合现象再现

| 维度 | 数据 |
|------|------|
| 现象 | 工作岗位发现 5 份过期/作废 SOP；2 个组按旧版操作超 2 个月 |
| 涉及条款 | ISO 15189:2022 第 4.2.4 条 |
| 等级 | 严重不符合（NC-S） |
| 涉及范围 | 生化组、免疫组（直接）；质控、报告（间接） |
| 评估患者影响 | 已对 2 个月内全部相关报告进行抽查；未发现重大偏差 |

#### 2.1.2 立即应对

- 立即评估对患者结果的影响（已完成）。
- 紧急清理工作岗位（2026-02-01 至 02-05）。
- 临时使用文件库标准版。

#### 2.1.3 策划纠正措施

| 措施 | 描述 |
|------|------|
| 1.1 立即清理 | 14 天内 |
| 1.2 修订 DOC-002 程序 | 28 天内 |
| 1.3 月度版本核查 | 持续执行 |
| 1.4 全员培训与考核 | 28 天内 |

### 2.2 D 阶段（Do）— 执行

#### 行动 D1.1 立即清理（科室主任 + 质量管理组）

- 时间：2026-02-01 至 2026-02-05
- 步骤：
  1. 逐个工作岗位检查。
  2. 收回过期/作废文件。
  3. 拍照取证（每个岗位至少 1 张）。
  4. 销毁旧版本（粉碎机或专用销毁通道）。
  5. 替换为现行版本。
  6. 形成清理报告（含照片附件）。
- 输出：清理报告（QUA-REC-001-A）；销毁记录（QUA-REC-001-B）。

#### 行动 D1.2 修订 DOC-002（质量负责人）

- 时间：2026-02-06 至 2026-02-28
- 修订重点：
  1. 新增"纸质版分发-回收-销毁流程"章节（含分发记录、回收记录、销毁记录 3 份新表单）。
  2. 新增"月度文件版本核查"章节。
  3. 新增"月度版本确认表"章节。
  4. 明确各角色职责（质量负责人、部门主任、检验员）。
- 输出：DOC-002 v1.6（含主任批准签字）。

#### 行动 D1.3 月度版本核查（质量管理组，持续）

- 启动：2026-02-01
- 频率：每月 1 次
- 抽样：覆盖率不低于 50% 的工位
- 输出：月度核查报告（每月 1 份）

#### 行动 D1.4 全员培训（质量负责人 + 主任）

- 时间：2026-02-15 至 2026-02-28
- 形式：集中培训 1.5 小时 + 部门讨论 1 小时
- 内容：文件版本重要性、识别方法、风险后果、新流程
- 考核：笔试或口试（3 题，答对率不低于 80%）
- 输出：培训记录、考试成绩单（不少于 25 人）

### 2.3 C 阶段（Check）— 验证

| 验证项 | 标准 | 验证时间 | 验证人 |
|-------|------|---------|-------|
| 工位文件版本 | 100% 现行版本 | 3/15、4/15、5/15 | 内审员 |
| 分发-回收-销毁记录 | 完整且可追溯 | 3/15、4/15、5/15 | 内审员 |
| 培训效果 | 抽查 5 人，答对率不低于 80% | 3/15、5/15 | 内审员 |
| 月度核查执行 | 每月有报告，问题数下降 | 4/15、5/15 | 质量负责人 |

### 2.4 A 阶段（Act）— 闭环

- 整改完成后向 CNAS 提交《NC-2026-001 整改闭环报告》。
- 经验总结：纸质版控制是文件控制的薄弱环节；需建立持续监督机制。
- 防止再发：将月度版本核查纳入年度内审检查项。

## 3. 不符合项 2（NC-2026-002）— 设备维护记录不完整

### 3.1 P 阶段 — 策划

#### 3.1.1 不符合现象

- 2025 年 35 次维护中：8 次缺验证结果（22.9%）；3 次缺复核签名（8.6%）。
- 涉及条款：ISO 15189:2022 第 6.4.4 条。
- 等级：一般不符合（NC-G）。

#### 3.1.2 立即应对

- 暂时要求所有维护活动按完整流程执行。
- 加强当下维护记录的复核。

#### 3.1.3 策划纠正措施

| 措施 | 描述 |
|------|------|
| 2.1 修订 EQP-004 程序 | 28 天内 |
| 2.2 重新设计维护记录表单 | 14 天内 |
| 2.3 二级审核制度 | 立即执行 |
| 2.4 补充 2025 年缺陷记录 | 45 天内 |
| 2.5 强化外包维护管理 | 14 天内 |

### 3.2 D 阶段 — 执行

#### 行动 D2.1 修订 EQP-004（设备管理员 + 质量负责人）

- 时间：2026-02-01 至 2026-02-28
- 修订重点：
  1. 明确"维护后验证"要求（按维护类型差异化）。
  2. 维护类型分类（故障维修、预防性维护、校准后维护、重大维修）。
  3. 各类型验证内容标准化。
  4. 二级审核制度（维护人员一级签字 + 技术主管二级复核签字）。
- 输出：EQP-004 v1.4

#### 行动 D2.2 重新设计维护记录表单

- 时间：2026-02-01 至 2026-02-15
- 表单包含：
  - 设备信息（设备编号、名称、型号）
  - 维护信息（日期、维护类型、维护内容、维护人）
  - **维护后验证**：验证方法、验证结果（合格/不合格）、IQC 在控状态、验证人签名
  - 复核：技术主管复核签名、复核日期
- 输出：新表单模板 INS-REC-XXX

#### 行动 D2.3 二级审核制度（设备管理员）

- 启动：2026-02-01
- 流程：
  1. 维护人员完成维护并填写记录第一部分（维护内容、验证结果、签名）。
  2. 技术主管在 24 小时内复核：维护内容完整、验证充分、记录准确。
  3. 复核签名和日期。
  4. 复核未通过的，须补充维护或验证。
  5. 月度统计复核情况。
- 输出：维护记录复核清单（月度）

#### 行动 D2.4 补充 2025 年度缺陷记录

- 时间：2026-02-15 至 2026-03-15
- 步骤：
  1. 8 条缺验证记录：审查原维护单据、设备运行日志、IQC 记录，由当时维护人员或技术主管补充签名（注明补记日期"2026-XX-XX 补记"）。
  2. 3 条缺复核签名：由技术主管补签。
- 输出：补充记录归档。

#### 行动 D2.5 强化外包维护管理

- 时间：2026-02-01 至 2026-02-15
- 措施：
  1. 与外包维护单位明确记录要求（合同补充协议）。
  2. 建立"外包维护验收清单"，接收方检查记录完整性后才签署接收。
  3. 下次维护时现场培训外包人员。
- 输出：合同补充协议、外包验收清单模板、培训记录。

### 3.3 C 阶段 — 验证

| 验证项 | 标准 | 验证时间 | 验证人 |
|-------|------|---------|-------|
| 新维护记录完整性 | 100% 含验证结果和复核签名 | 3/20、4/20、5/20 | 内审员 |
| 历史缺陷补充 | 8 条 + 3 条全部补齐 | 3/20 | 内审员 |
| 程序执行率 | 验证执行率不低于 95% | 4/20、5/20 | 设备管理员 |
| 外包配合度 | 外包记录符合要求 | 4/20、5/20 | 设备管理员 |

### 3.4 A 阶段 — 闭环

- 经验总结：维护"后续验证"是设备管理的重要环节；二级审核制度是关键保障。
- 防止再发：将设备维护记录完整性纳入月度内审。

## 4. 不符合项 3（NC-2026-003）— 人员能力评估不到位

### 4.1 P 阶段 — 策划

#### 4.1.1 不符合现象

- 员工 D（硕士）独立操作 2 个月无能力评估记录。
- 员工 E 评估过于简单（仅打勾）。
- 涉及条款：ISO 15189:2022 第 6.2 条。
- 等级：一般不符合（NC-G）。

#### 4.1.2 立即应对

- 立即停止员工 D 独立操作（在评估完成前）。
- 由资深技术人员陪同操作。

#### 4.1.3 策划纠正措施

| 措施 | 描述 |
|------|------|
| 3.1 立即评估员工 D | 10 天内 |
| 3.2 补充员工 E 详细评估 | 14 天内 |
| 3.3 修订 PER-002 程序 | 28 天内 |
| 3.4 设计详细评估表单 | 28 天内 |
| 3.5 建立成长档案制度 | 28 天内 |
| 3.6 全员宣导"无差别"原则 | 立即执行 |

### 4.2 D 阶段 — 执行

#### 行动 D3.1 立即评估员工 D（主任 + 技术负责人）

- 时间：2026-02-01 至 2026-02-10
- 评估内容：
  - 知识：检测原理、仪器原理、质量控制、异常处理（4 项）
  - 实践：样品准备、仪器操作、质控判读、异常处理、结果审核、报告填写（6 项）
- 评估方法：笔试（30 题）+ 实际操作 + 案例分析
- 评估结论：合格 / 需进一步观察 / 不合格
- 输出：详细评估表（PER-REC-XXX）

#### 行动 D3.2 补充员工 E 详细评估

- 时间：2026-02-01 至 2026-02-15
- 步骤同 D3.1。
- 输出：补充评估表。

#### 行动 D3.3 修订 PER-002（人力 + 质量负责人）

- 时间：2026-02-01 至 2026-02-28
- 修订重点：
  1. 明确评估时限（培训完成后 1—2 周内）。
  2. 明确评估标准（笔试得分≥80%；实践合格）。
  3. 详细评估表内容要求。
  4. 评估方法（笔试 + 实践 + 案例分析至少 3 种）。
  5. 评估周期（在岗人员每 12 个月至少 1 次）。
  6. 不合格处理（培养计划、限制独立操作）。
  7. 监控清单制度。
- 输出：PER-002 v1.3。

#### 行动 D3.4 设计详细评估表单

- 时间：2026-02-15 至 2026-02-28
- 表单包含：
  - 被评估人员信息
  - 评估方法（笔试、实践、案例）
  - 各评估项（按维度展开，含具体评分项）
  - 评估人评分
  - 总体结论（合格/需进一步观察/不合格）
  - 培养计划（如不合格）
  - 评估人和被评估人签名
- 输出：新表单 PER-REC-XXX。

#### 行动 D3.5 建立成长档案制度（主任 + 人力）

- 时间：2026-02-15 至 2026-02-28
- 制度内容：
  1. 每个新员工建立成长档案。
  2. 记录培训内容、日期、学习进度、能力评估结果、独立操作授权日期。
  3. 主任和人力共同监督。
  4. 评估不合格或未评估的不允许独立操作。
- 输出：成长档案制度文件、模板。

#### 行动 D3.6 全员宣导"无差别"原则（主任）

- 时间：2026-02-01（立即执行）
- 形式：
  - 管理会议宣导。
  - 团队沟通会再次强调。
  - 内审中重点检查。
- 输出：会议纪要。

#### 行动 D3.7 月度评估监控清单（人力资源组）

- 启动：2026-03-01
- 频率：每月 1 次
- 内容：
  - 应进行评估人员名单
  - 实际进行评估情况

  - 整改建议
- 输出：月度监控清单。

## 6. 闭环报告

### 6.1 闭环报告内容

整改完成后，向 CNAS 提交《NC-2026 整改闭环报告》（QUA-REC-063-001），包含：

1. **整改概述**：3 项不符合的整改情况。
2. **每项不符合的纠正措施执行**：详细描述各措施完成情况。
3. **验证证据汇总**：含所有附件清单。
4. **整改有效性自评**：基于验证标准的自我评价。
5. **改进效果说明**：新程序对问题解决的作用。
6. **持续改进与防止再发**：将整改成果固化为常规管理活动的安排。
