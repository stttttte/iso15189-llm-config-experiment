## 使用说明

本检查表依据 ISO 15189:2022 第 7.3 条"检验结果质量保证"全部子条款（7.3.1～7.3.7）逐条编制，供内审员对检验过程质量控制模块开展专项审核时使用。审核员应综合运用文件审查、记录检查、现场观察和人员访谈等方法对每条检查项进行审核，在"符合性判定"栏填写结果，并在"客观证据"和"备注"栏记录具体发现。

## 检查表

| 条款号 | 审核要点 | 审核方法 | 符合性判定 | 客观证据 | 备注 |
|:------:|----------|----------|:----------:|----------|------|
| 7.3.1 | 实验室是否建立并实施了质量控制程序，以监控检验过程的有效性并确保向患者提供检验结果的质量？该程序是否覆盖所有已开展的检验程序？ | 文件审查：查阅质量控制管理程序（QUA-005 或等效文件）；记录检查：核查质控项目清单与实际开展检验程序的对应关系 | C / NC / PC / NA | | |
| 7.3.1 | 质量控制程序是否充分考虑了检验前过程和检验后过程对检验结果质量的影响，并纳入了相应的监控措施？ | 文件审查：核查质控程序中是否包含检验前过程（样本质量评估）和检验后过程（结果审核）的风险识别与监控内容；人员访谈：询问相关人员对检验前后过程质量风险的认知 | C / NC / PC / NA | | |
| 7.3.2 | 实验室是否针对每个检验程序设计了质量控制方案，包括选用适当的质控物、设定合理的质控频次和控制规则？方案设计是否基于对检验程序风险的评估？ | 文件审查：查阅各检验程序对应的质控方案及其风险评估依据；记录检查：核查质控物选择记录和质控频次设定文件 | C / NC / PC / NA | | |
| 7.3.3 a) | 实验室是否使用了内部质量控制（IQC）来评价检验结果的可靠性？IQC 是否涵盖所有检验系统和检验程序？质控物的基质类型和浓度水平选择是否适当（含医学决定水平附近的浓度）？ | 记录检查：核查 IQC 实施记录和质控物信息（来源、基质、浓度水平覆盖情况）；现场观察：查看各检验系统当日 IQC 执行情况 | C / NC / PC / NA | | |
| 7.3.3 a) | 实验室是否建立了 IQC 评价规则（如 Westgard 规则或等效规则）？失控判定标准是否明确？当 IQC 结果超出可接受范围时，是否有明确的处置措施和结果放行判断准则？ | 文件审查：查阅 IQC 评价规则和失控处理程序；记录检查：抽查近 3 个月内失控事件的处置记录，核查是否按规定流程执行 | C / NC / PC / NA | | |
| 7.3.3 a) | 失控后实验室是否评估了对已报告患者检验结果的影响？是否根据评估结果采取了纠正措施（包括必要时的结果召回和临床通知）？ | 记录检查：查阅失控影响评估记录和纠正措施记录；人员访谈：询问检验人员失控后处理流程和结果召回的触发条件 | C / NC / PC / NA | | |
| 7.3.3 b) | 实验室是否参加了能力验证（PT）/ 实验室间比对（ILC）计划？参加的计划是否覆盖所有可获得 PT/ILC 计划的检验程序？覆盖率是否达到实验室质量目标要求？ | 文件审查：查阅年度 PT/ILC 参加计划和报名记录；记录检查：核查 PT/ILC 参加覆盖率，比对已认可检验程序清单 | C / NC / PC / NA | | |
| 7.3.3 b) | 能力验证样本是否按照常规检验流程进行检测，由日常执行检验的人员操作？是否存在对能力验证样本的特殊处理或优待情况？ | 人员访谈：询问检测人员能力验证样本的处理流程；记录检查：核查 PT 样本的操作记录是否与常规样本处理流程一致 | C / NC / PC / NA | | |
| 7.3.3 b) | 对于无法获得能力验证计划的检验程序，实验室是否建立了替代评估方案（如实验室间比对、使用认定参考物质等）？替代方案是否形成文件并定期实施？ | 文件审查：查阅替代评估方案文件；记录检查：核查替代方案覆盖了哪些无 PT 计划的检验程序及其实施记录 | C / NC / PC / NA | | |
| 7.3.3 c) | 当同一检验程序使用不同的检验系统、设备、试剂或在不同地点开展时，实验室是否建立了检验结果可比性的监控机制？是否定期进行系统间比对？ | 文件审查：查阅结果可比性评估方案和比对计划；记录检查：核查不同检验系统间的比对记录和偏倚评估结果 | C / NC / PC / NA | | |
| 7.3.3 c) | 可比性监控是否覆盖了试剂批号变更、校准物批号变更等关键变更时点的结果一致性评价？比对结果不可接受时是否采取了纠正措施并评估了对已报告结果的影响？ | 记录检查：核查试剂批号更换时的比对验证数据和变更评价记录；查阅比对不可接受时的纠正措施和影响评估报告 | C / NC / PC / NA | | |
| 7.3.4 | 实验室是否对质量控制数据进行了定期评审和趋势分析？评审是否能识别检验性能的系统性变化（如均值漂移、精密度变化）？ | 记录检查：查阅质控数据月度/季度评审记录和趋势分析报告（如 Levey-Jennings 图）；人员访谈：询问质量负责人评审频次和方法 | C / NC / PC / NA | | |
| 7.3.4 | 当质控数据评审发现趋势性偏移或潜在问题时，实验室是否采取了预防措施？是否对已发出的可能受影响的患者检验结果进行了评估和追溯？ | 记录检查：查阅趋势偏移处置记录和影响评估报告；人员访谈：询问发现趋势偏移后的具体处理流程和追溯机制 | C / NC / PC / NA | | |
| 7.3.5 | 实验室是否建立了 PT/ILC 结果不满意时的处置程序？是否包括根本原因分析和纠正措施？不满意结果的处置是否评估了对已发出患者检验结果的潜在影响？ | 文件审查：查阅 PT/ILC 不满意结果处置程序；记录检查：核查近 2 年内 PT 不满意结果的根本原因分析报告、纠正措施记录和影响评估记录 | C / NC / PC / NA | | |
| 7.3.5 | 实验室是否定期对能力验证和实验室间比对的参加情况和结果进行总结分析？分析结果是否纳入质量改进和管理评审？ | 记录检查：查阅 PT/ILC 年度总结报告和趋势分析记录；文件审查：核查管理评审输入材料中是否包含 PT/ILC 结果分析 | C / NC / PC / NA | | |
| 7.3.6 | 对于定性检验程序和无法使用常规质控物进行 IQC 的检验程序（如形态学检查、分子检验等），实验室是否建立了替代质量控制策略？替代策略的有效性是否经过评价？ | 文件审查：查阅替代质控策略的书面方案和有效性验证报告；记录检查：核查替代质控措施的实施记录和评价结果 | C / NC / PC / NA | | |
| 7.3.7 | 实验室是否将质量指标（quality indicators）纳入了检验全过程的质量监控？质量指标是否覆盖检验前过程、检验过程和检验后过程？指标目标值和阈值是否基于适当的依据设定？ | 文件审查：查阅质量指标清单和监控计划；记录检查：核查质量指标的数据收集、统计分析报告和目标值设定依据 | C / NC / PC / NA | | |
| 7.3.7 | 当质量指标超出阈值时，实验室是否有触发评审和改进的机制？改进措施是否得到落实和有效性验证？ | 记录检查：核查超阈值时的评审记录和改进措施跟踪表；人员访谈：询问质量负责人质量指标超标后的处理流程 | C / NC / PC / NA | | |
| 7.3.1～7.3.7 | 检验过程质量控制相关的所有记录（IQC 记录、PT/ILC 记录、比对记录、质控评审记录等）是否完整、可追溯？记录的保存期限是否符合 DOC-002 记录控制程序的要求？ | 记录检查：抽查各类质控记录的完整性和可追溯性；核查记录保存期限是否满足要求（检验记录至少保存 2 年） | C / NC / PC / NA | | |
| 7.3.1～7.3.7 | 实验室是否对从事质量控制工作的人员进行了充分的培训和能力评估？相关人员是否理解质控规则、失控处理流程、能力验证要求和结果可比性评价方法？ | 记录检查：查阅相关人员的培训记录和能力评估记录（PER-REC 系列）；现场访谈：随机抽查操作人员对 IQC 评价规则和失控处理流程的掌握情况 | C / NC / PC / NA | | |
