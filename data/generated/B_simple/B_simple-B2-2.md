## 使用说明

本检查表针对 ISO 15189:2022 第 7.3 条"检验结果质量保证"的所有子条款逐条编制，供内审员在专项审核时使用。审核员应根据现场观察、文件审查、记录检查和人员访谈等方式对每条进行审核，在"符合性判定"栏填写判定结果，并在"审核证据/备注"栏记录具体发现。

**符合性判定标准**：C = 符合（Conformity）；NC = 不符合（Nonconformity）；OFI = 改进机会（Opportunity for Improvement）；NA = 不适用（Not Applicable）

## 检查表

| 序号 | 对应条款 | 检查要点 | 审核方法 | 符合性判定 | 审核证据/备注 |
|:----:|:--------:|----------|----------|:----------:|---------------|
| 1 | 7.3.1 | 实验室是否建立了质量控制程序，用于监控检验过程的有效性并确保向患者提供结果的质量？该程序是否覆盖所有已开展的检验项目？ | 文件审查：查阅质量控制程序文件；记录检查：核查质控项目清单与实际开展项目的对应关系 | | |
| 2 | 7.3.1 | 质量控制程序是否考虑了检验前过程和检验后过程对结果质量的影响，并对这些影响进行了适当的监控？ | 文件审查：审核质控程序中对检验前/检验后过程风险的识别；人员访谈：询问相关人员对检验前后过程质量监控的认知 | | |
| 3 | 7.3.2 | 实验室是否针对每个检验程序设计了质量控制方案（quality control scheme），包括选用适当的质控物和频次，以验证检验结果的预期质量？ | 文件审查：查阅各检验项目的质控方案；记录检查：核查质控物的选择依据和质控频次设定 | | |
| 4 | 7.3.2 | 质量控制方案的设计是否基于对检验程序风险的评估？是否考虑了对患者护理结果的潜在影响？ | 文件审查：查阅风险评估记录和质控方案设计依据；人员访谈：询问技术负责人质控方案制定的理由 | | |
| 5 | 7.3.3 a) | 实验室是否使用了内部质量控制（IQC）来评价检验结果的可靠性？IQC 是否涵盖所有检验系统和检验项目？ | 文件审查：查阅 IQC 程序和 IQC 项目清单；记录检查：核查各检验系统的 IQC 实施记录 | | |
| 6 | 7.3.3 a) | IQC 物质的选择是否适当（如基质类型、浓度水平）？IQC 的频次和时机是否满足检验结果质量监控的需要？ | 记录检查：核查 IQC 物质的来源、基质效应评价、浓度水平覆盖情况；现场观察：观察 IQC 的实际执行频次 | | |
| 7 | 7.3.3 a) | 实验室是否建立了 IQC 的评价规则（如 Westgard 规则或等效规则）？当 IQC 结果超出可接受范围时，是否有明确的处置措施和结果放行判断准则？ | 文件审查：查阅 IQC 评价规则和失控处理程序；记录检查：核查近期失控事件的处置记录 | | |
| 8 | 7.3.3 b) | 实验室是否参加了能力验证（PT）/实验室间比对（ILC）计划？参加的计划是否覆盖所有可获得 PT/ILC 计划的检验项目？ | 文件审查：查阅 PT/ILC 参加清单和年度计划；记录检查：核查 PT/ILC 参加覆盖率和相关证明 | | |
| 9 | 7.3.3 b) | 当某些检验项目无法获得 PT/ILC 计划时，实验室是否采取了替代方法来评估检验结果的质量（如与其他实验室的样本交换比对、使用认定参考物质等）？ | 文件审查：查阅替代评估方案；记录检查：核查替代方法的实施记录和结果评价 | | |
| 10 | 7.3.3 c) | 实验室是否监控了检验结果的可比性？当同一检验项目使用不同的检验程序、设备或地点时，是否有确保结果一致性的机制？ | 文件审查：查阅结果可比性评估方案；记录检查：核查不同系统间的比对记录和偏倚评估结果 | | |
| 11 | 7.3.3 c) | 可比性监控是否涵盖了在不同时间段（如试剂批号变更、校准品批号变更）的结果一致性评价？ | 记录检查：核查试剂批号更换记录及相应的比对验证数据；人员访谈：询问技术人员试剂批号更换时的操作流程 | | |
| 12 | 7.3.4 | 实验室是否对质量控制数据进行了定期评审？评审是否包括趋势分析以识别检验性能的系统性变化？ | 记录检查：查阅质控数据月度/季度评审报告；文件审查：核查趋势分析方法和评审周期 | | |
| 13 | 7.3.4 | 当质量控制数据评审发现潜在问题或趋势性偏移时，实验室是否采取了预防措施？是否对已发出的可能受影响的患者结果进行了评估？ | 记录检查：查阅质控趋势偏移的处置记录和影响评估报告；人员访谈：询问发现趋势偏移后的具体处理流程 | | |
| 14 | 7.3.5 | 实验室是否建立了 PT/ILC 结果不满意时的处置程序？处置措施是否包括根本原因分析和纠正措施？ | 文件审查：查阅 PT/ILC 不满意结果处置程序；记录检查：核查近期 PT/ILC 不满意结果的调查和纠正措施记录 | | |
| 15 | 7.3.5 | PT/ILC 不满意结果的处置是否评估了对已发出患者检验结果的潜在影响？是否在必要时采取了适当措施（如通知临床、结果召回等）？ | 记录检查：查阅 PT 不合格后的影响评估和患者结果追溯记录；人员访谈：询问相关处置流程的执行情况 | | |
| 16 | 7.3.6 | 对于无法使用常规质控物进行 IQC 的检验项目（如某些分子检验、形态学检查等），实验室是否采用了替代质量控制策略？ | 文件审查：查阅替代质控策略的书面方案和依据；记录检查：核查替代质控措施的实施记录 | | |
| 17 | 7.3.6 | 替代质量控制策略的有效性是否经过了评价？是否有证据表明替代策略能够有效监控检验过程的质量？ | 记录检查：查阅替代策略的有效性验证报告；人员访谈：询问技术负责人对替代策略有效性的评价 | | |
| 18 | 7.3.7 | 实验室是否将质量指标（quality indicators）纳入了检验过程质量监控？质量指标是否覆盖了检验前、检验中和检验后过程？ | 文件审查：查阅质量指标清单和监控计划；记录检查：核查质量指标的数据收集和统计分析报告 | | |
| 19 | 7.3.7 | 质量指标的目标值和阈值是否基于适当的依据设定？当指标超出阈值时是否有触发评审和改进的机制？ | 文件审查：查阅质量指标目标值设定依据和触发机制；记录检查：核查超阈值时的评审和改进记录 | | |
| 20 | 7.3.3/7.3.4 | 实验室对质控规则的选择（如 IQC 规则类型、PT 评价标准）是否有文件化的依据？是否定期评价这些规则的适宜性？ | 文件审查：查阅质控规则选择依据的文件；人员访谈：询问质量负责人质控规则定期评审的情况 | | |
