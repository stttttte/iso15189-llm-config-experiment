# 设备校准记录表（修订版）

## 修订概览

本次修订针对原版 FRM-EQP-002《设备校准记录表》V1.0 中的 4 类缺失字段进行补充完善，使之符合 ISO 15189:2022 第 6.4 条（设备）、第 6.5 条（设备校准和计量溯源）、第 6.3 条（设施和环境条件）及 CNAS-CL02:2023 相应条款要求。

| 修订项 | 原版状态 | 修订后状态 |
|--------|---------|-----------|
| 测量不确定度记录栏 | 缺失 | 新增第五部分"测量不确定度评定" |
| 校准环境条件记录栏（温度 / 湿度 / 气压） | 缺失 | 新增第二部分"校准环境条件" |
| 审核 / 复核签名栏 | 仅有填表人签名 | 新增"一级审核（质量负责人）"与"二级复核（科室主任 / 技术负责人）"，形成三级签审机制 |
| 填写说明与更正规则 | 缺失 | 增补"表格填写说明"与"记录更正规则" |
| 校准后质控物验证 | 缺失 | 新增第七部分"校准后验证" |
| 不合格校准处理机制 | 缺失 | 新增第八部分"不合格校准处理" |
| 表单编号规范 | FRM-EQP-002 | INS-REC-002（按新编号体系） |
| 保存期限 | 未明确 | 明确为 6 年 |

## 附录 A ：表格填写说明（新增）

### A.1 字段定义与填写要求

| 字段 | 定义 | 填写要求 |
|------|------|---------|
| 设备名称、型号、序列号 | 来源于设备铭牌 | 与设备铭牌完全一致，严禁缩写 |
| 设备科室唯一编号 | 科室内部管理编号 | 按 INS-REC-001《设备台账》的编号规则填写 |
| 所属专业组 | 设备日常使用的专业组 | 在提供选项中勾选 |
| 校准环境条件 | 校准时的实际温度、湿度、气压 | 使用经检定 / 校准合格的测量仪器现场测量 |
| 校准类型 | 本次校准的触发原因 | 在提供选项中勾选；选"其他"必须文字补充 |
| 校准周期 | 两次校准间隔 | 按制造商建议或计量法规要求填写；单位为月 |
| 校准单位资质编号 | 外部校准机构的 CNAS / CMA 编号 | 外部校准必填；内部校准填写"内部校准" |
| 校准品 / 标准物质名称 | 校准时使用的参考标准品 | 填写完整名称，禁止使用商品简称 |
| 计量溯源性声明 | 校准品量值溯源至的更高级参考标准 | 按校准品说明书中的溯源声明填写 |
| 标称值 / 实测值 / 允许偏差 | 三者配合用于判定 | 含法定计量单位；有效位数不低于制造商规定 |
| 扩展不确定度 U | 测量结果的分散性表征 | 按 JJF 1059.1 评定；注明单位和包含因子 k |
| 质控物验证 | 校准有效性验证 | 至少使用 2 个浓度水平的质控物 |

## 附录 B ：记录更正规则（新增）

依据 ISO 15189:2022 第 8.4 条"记录控制"要求，本表记录更正必须遵循以下规则。

### B.1 纸质记录更正

1. 使用单直线划掉错误内容，确保原内容清晰可辨
2. 在划掉处旁书写正确内容
3. 在正确内容旁由更正人签名并注明更正日期（YYYY-MM-DD）
4. **严禁行为**：
   - 使用涂改液、修正带、橡皮擦、胶带覆盖原内容
   - 刮擦、挖补、剪贴以掩盖原内容
   - 重新抄写整张表格以替代更正

### B.2 电子记录更正

1. 在信息管理系统中修改时，系统应自动保留：
   - 修改前的原始数据
   - 修改时间（精确到秒）
   - 修改人账户
   - 修改原因
2. 审计追踪 (Audit Trail) 记录不得人为删除或修改
3. 审计追踪保存期限不得短于原始记录

### B.3 重大错误的升级处理

1. 涉及检验结果正确度的错误更正，应同步启动 QUA-REC-021《不符合项和纠正措施记录》
2. 需重新校准时，本表整张作废，重新填写一份；作废表加盖"作废"章后归档保存

### B.4 表格保存与销毁

| 项目 | 要求 |
|------|------|
| 保存期限 | 6 年（自记录生成之日起计算） |
| 保存形式 | 纸质原件 + 电子扫描件 |
| 保存位置 | 检验科设备档案柜 + 科室电子档案系统 |
| 销毁流程 | 到期后按 DOC-PRO-002《记录控制程序》执行；销毁前由质量负责人审核；销毁后留存《记录销毁记录》 |
