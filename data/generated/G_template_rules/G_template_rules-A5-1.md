## 表格填写说明

### 1. 各字段定义与填写要求

| 字段 | 定义 | 填写要求 |
|------|------|----------|
| 设备名称 | 设备的完整中文名称 | 与设备铭牌及设备登记册一致，不得使用缩写 |
| 设备型号 | 制造商标注的产品型号 | 与设备铭牌一致 |
| 设备序列号 | 制造商出厂编号 | 与设备铭牌一致 |
| 设备唯一编号 | 科室内部管理编号 | 按照科室设备编号规则填写 |
| 所属专业组 | 该设备日常使用的专业组 | 在提供的选项中圈选 |
| 校准环境条件 | 校准时实际环境温度、湿度、气压 | 从经过校准的温湿度计读取，精密天平校准须同时记录气压 |
| 校准类型 | 本次校准的触发原因 | 在提供的选项中勾选，选择"其他"时必须在说明栏补充具体原因 |
| 校准周期 | 两次校准之间的间隔时间 | 按照制造商建议或计量法规要求填写，单位为月 |
| 校准品/标准物质 | 校准时使用的参考标准品名称 | 填写完整名称，禁止仅写商品简称 |
| 校准品批号 | 校准品的生产批号 | 与校准品包装标识一致 |
| 校准品计量溯源性声明 | 校准品量值可追溯至的更高级参考标准 | 根据校准品说明书中的溯源声明填写 |
| 校准项目 | 本次校准所覆盖的具体测量参数 | 逐项填写，每行一个参数 |
| 标准值/标称值 | 校准品或标准物质的标称赋值 | 含单位 |
| 实测值 | 仪器测量的实际读数 | 含单位，保留有效位数与标准值一致 |
| 允许偏差 | 判定合格与否的偏差限值 | 根据制造商说明书或相关计量检定规程填写 |
| 测量不确定度 | 校准结果的合理赋予的量值分散度 | 填写扩展不确定度 U 及其包含因子 k |
| 质控物验证 | 校准完成后使用质控物对校准有效性的确认 | 至少使用 2 个浓度水平的质控物进行验证，记录实测值与靶值范围 |

### 2. 填写一般规则

- 所有日期统一使用"YYYY年MM月DD日"格式
- 数值须注明单位，小数位数与制造商说明书或检定规程要求一致
- 所有签名必须使用手签（黑色或蓝色签字笔），不得代签
- 校准不合格时，第八部分"不合格校准处理"为必填项
- 填写完成后，由填表人当日提交质量负责人审核；质量负责人在 3 个工作日内完成一级审核；科室主任/技术负责人在 5 个工作日内完成二级复核

## 记录更正规则

依据 ISO 15189:2022 第 7.7 条及 CNAS-CL02:2023 第 7.7 条的记录管理要求：

1. **更正方式**：在错误内容上画单条横线（确保原始内容仍可辨认），在旁边写上正确内容
2. **更正标注**：每处更正须注明更正日期，并由更正人签名确认
3. **禁止行为**：
   - 禁止使用涂改液、修正带、橡皮擦或胶带覆盖
   - 禁止刮擦、挖补或重新抄写整张表格以替代更正
   - 禁止以任何方式使原始记录内容无法辨认
4. **电子记录更正**：在信息系统中修改时，系统须自动保留修改前的原始数据、修改时间和修改人信息，形成完整的审计追踪 (Audit Trail)
5. **表格保存**：本表保存期限为 6 年，保存期满后按照科室文件销毁程序执行销毁并保留销毁记录
