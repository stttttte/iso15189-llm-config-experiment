## 1 目的

建立实验医学科人员培训与能力评估的系统化管理制度，规范培训计划制定、培训实施、效果评估、能力评估方法与结果处理、继续教育学分管理等全过程活动，确保所有从事检验活动的人员持续具备胜任其岗位的知识、技能和能力，满足 ISO 15189:2022 第 6.2 条（人员）的全部要求。

## 2 适用范围

2.1 本程序适用于实验医学科所有从事检验前、检验和检验后过程活动的人员，包括正式在编人员、合同制聘用人员、进修人员、实习人员及参与 POCT 操作的临床科室人员。

2.2 覆盖以下管理活动：
- 入职培训、在职培训、专项培训、安全培训、合规培训
- 新员工入岗能力评估、年度能力评估、复岗评估、专项评估
- 继续教育学分管理与职业发展
- 培训与能力评估结果的统计分析与持续改进

## 3 标准依据

### 3.1 ISO 15189:2022 原文要求摘录

> **6.2.2 能力要求**
>
> "实验室应确定各项职能所需的能力，包括教育、资格、培训、技术知识、技能和经验的要求。"
>
> "应按照文件化程序，在规定的时间间隔内评估每个人员执行分配任务的能力。"
>
> "评估方法应至少包括以下适用方法的组合：
> a) 直接观察常规工作过程和程序，包括所有适用的安全实践；
> b) 直接观察对设备维护和功能检查的执行；
> c) 监控检验和检验后过程的记录和结果；
> d) 评审工作记录；
> e) 评估解决问题的能力；
> f) 检查特别提供的样品，如先前检验过的样品、实验室间比对材料、分割样品。"
>
> "在新员工入职后的规定期间内（如最初六个月）应至少进行两次能力评估，之后每年进行评估。"

> **6.2.4 继续教育和职业发展**
>
> "实验室应为所有人员提供继续教育方案。参加继续教育方案的人员应能证明其继续教育活动的效果。"
>
> "当使用电子系统或手工系统发布检验结果时，应对有权限进入检验信息系统的所有人员提供培训，以确保患者数据的保密性。"

## 4 职责

| 角色 | 培训管理职责 | 能力评估职责 |
|------|-------------|-------------|
| 科主任 | 批准年度培训计划和培训预算 | 批准年度能力评估计划，审核评估汇总报告 |
| 技术负责人 | 审核技术培训内容，确认培训师资 | 制定各岗位能力要求，组织年度评估计划编排，审核评估结果 |
| 质量负责人 | 组织合规培训（公正性、保密性、安全），监督培训执行 | 监督评估流程的执行合规性，协调评估与质量监督的联动 |
| 培训管理员 | 制定培训计划、组织培训实施、管理培训档案、统计继续教育学分 | 协助编制评估计划，汇总评估结果数据 |
| 各专业组组长 | 提出本组培训需求，协助组织本组专业培训，参与带教 | 执行本组人员评估，记录评估结果，提出培训建议 |
| 质量监督员 | 将日常监督中发现的能力问题纳入培训需求 | 参与直接观察评估，提供日常工作质量监控数据 |
| 全体人员 | 按计划参加培训，完成培训考核，保存继续教育证明 | 配合评估安排，按要求完成各项评估任务 |

## 5 术语与定义

| 术语 | 定义 |
|------|------|
| 能力评估 (Competency Assessment) | 通过系统化方法评价人员执行指定任务所需知识、技能和行为的过程 |
| 直接观察 (Direct Observation) | 评估人员在工作现场直接观看被评估人执行操作的评估方法 |
| 盲样测试 (Blind Sample Testing) | 将已知结果的样品以常规标本形式提供给被评估人检测，评估其检验能力 |
| 入岗评估 | 新员工入职后在规定期间内进行的初始能力评估 |
| 复岗评估 | 人员长期离岗后重新返回岗位时进行的能力评估 |
| 入职培训 | 新员工入职后的初始培训，包括通用培训和岗位专业培训 |
| 在职培训 | 在岗人员日常接受的持续性培训活动 |
| 继续教育 (Continuing Education) | 获取和更新专业知识、技能的持续性教育活动，通常以学分计量 |
| 合规培训 | 涉及法规遵从、公正性、保密性、患者权益等方面的培训 |
| POCT 培训 | 对参与床旁检验（Point-of-Care Testing）人员的专项培训 |
| 培训效果评估 | 通过考核、实操、满意度调查等方式衡量培训达到预期目标的程度 |
| 加密评估 | 对复杂或高风险检验项目增加评估频次的安排 |

## 6 程序内容

### 第一部分：培训管理

#### 6.1 年度培训计划制定与执行追踪

##### 6.1.1 培训需求分析

每年12月进行下年度培训需求分析，信息来源包括：

| 来源 | 说明 |
|------|------|
| 能力评估结果 | 年度能力评估中发现的知识或技能薄弱环节 |
| 质量监督反馈 | 日常监督中发现的操作问题和能力偏差 |
| 不符合与纠正措施 | 内审、外审、投诉中发现的人员能力相关问题 |
| 新项目与新设备 | 即将开展的新检验项目或新采购的检验设备 |
| 法规与标准更新 | 新发布或修订的法律法规、行业标准 |
| 人员变动情况 | 新入职人员、调岗人员、复岗人员 |
| 管理评审建议 | 管理评审中提出的培训改进建议 |
| 人员自身需求 | 人员提出的专业发展和学习需求 |

##### 6.1.2 年度培训计划编制

1. 培训管理员汇总各渠道培训需求，编制《年度培训计划》（SYS-FM-06-001-01）。
2. 计划应包含：培训课程名称、培训类型、培训对象、培训师资、计划时间、培训学时、考核方式、预算估算。
3. 计划应确保覆盖所有必修培训主题（见6.2各类培训要求）。
4. 质量负责人审核计划的合规性和完整性，科主任批准。
5. 批准后的计划发放至各专业组，并在科室公告栏或信息系统中公布。

##### 6.1.3 计划执行追踪

1. 培训管理员每季度汇总培训执行情况，编制培训执行追踪记录。
2. 追踪内容包括：已完成课程数/计划课程数、参训人次、考核通过率、未完成课程及延期原因。
3. 对于未按计划完成的培训，分析原因并安排补训。
4. 年底编制《年度培训总结报告》（SYS-FM-06-001-02），内容包括培训计划完成率、考核通过率、继续教育学分达标率、改进建议等，提交管理评审。

#### 6.2 培训类型与内容

##### 6.2.1 入职培训

新员工入职后2周内完成以下培训：

| 培训模块 | 培训内容 | 学时 | 考核方式 |
|----------|----------|------|----------|
| 科室概况 | 科室组织架构、工作流程、科室制度 | 2h | 无需考核 |
| 质量管理 | 质量方针与目标、质量管理体系概述、质量手册要点 | 2h | 笔试 |
| 生物安全 | 实验室生物安全等级与防护措施、职业暴露处置、应急处理流程 | 4h | 笔试 + 实操演练 |
| 信息安全与保密 | 患者数据保密要求、LIS/HIS使用规范、信息系统权限管理 | 2h | 笔试 |
| 公正性与伦理 | 公正性承诺、利益冲突识别与处理、患者权益保护 | 2h | 签署公正性承诺书 |
| SOP体系 | 文件管理制度、SOP查阅和执行要求、文件变更流程 | 2h | 笔试 |
| 岗位专业培训 | 本岗位SOP操作规程、仪器操作、质控操作、结果判读 | 视岗位而定 | 实操考核 |

入职培训完成后，由培训管理员填写《入职培训完成清单》（SYS-FM-06-001-08），经组长和技术负责人签字确认。

##### 6.2.2 在职培训

| 培训内容 | 频率 | 培训方式 |
|----------|------|----------|
| SOP更新培训 | SOP修订后即时开展 | 集中讲解 + 阅读确认签字 |
| 新检验项目开展培训 | 项目上线前 | 理论学习 + 实操练习 |
| 新设备操作培训 | 设备启用前 | 厂家工程师培训 + 科内转训 |
| 质量改进培训 | 按需 | 案例分析与讨论 |
| 仪器故障排除培训 | 每半年至少1次 | 模拟演练 |
| 室间质评培训 | 每次室间质评前 | 规范操作流程讲解 |

##### 6.2.3 专业培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 专业学术讲座 | 每月至少1次 | 全科人员 |
| 疑难病例讨论 | 每月1-2次 | 相关专业组人员 |
| 新技术新方法学习 | 按需 | 相关人员 |
| 学术会议与培训班 | 按需 | 外派人员 |
| 科内学术交流 | 每季度至少1次 | 全科人员 |

##### 6.2.4 安全培训

| 培训内容 | 频率 | 适用人员 |
|----------|------|----------|
| 生物安全年度培训 | 每年至少1次 | 全科人员 |
| 消防安全培训与演练 | 每年至少1次 | 全科人员 |
| 化学品安全培训 | 每年至少1次 | 接触化学危险品人员 |
| 职业暴露处置演练 | 每年至少1次 | 全科人员 |
| 应急预案演练 | 每年至少1次 | 全科人员 |
| 放射安全培训 | 每年至少1次 | 涉及放射性操作人员（如适用） |

##### 6.2.5 合规培训

| 培训内容 | 频率 | 对应标准/法规 |
|----------|------|--------------|
| 公正性培训 | 每年至少1次 | ISO 15189:2022 4.1.1.3 |
| 保密性培训 | 每年至少1次 | ISO 15189:2022 4.1.1.4 |
| 患者权益与知情同意培训 | 每年至少1次 | 相关法律法规 |
| 伦理培训 | 每年至少1次 | 科研伦理、知情同意规范 |
| 法律法规更新培训 | 法规更新后即时 | 新发布或修订的法规 |

合规培训须签署相应承诺书或协议：公正性培训签署《公正性承诺书》，保密性培训签署《保密协议》。

##### 6.2.6 POCT 培训

对参与 POCT（床旁检验）管理和操作的人员进行专项培训：

| 培训内容 | 适用人员 | 频率 |
|----------|----------|------|
| POCT设备操作培训 | POCT操作人员（含临床科室护理人员） | 上岗前 + 每年复训 |
| POCT质量控制培训 | POCT操作人员 | 上岗前 + 每年复训 |
| POCT结果判读与临床意义 | POCT操作人员 | 上岗前 |
| POCT管理制度培训 | 检验科POCT管理人员 | 按需 |

POCT操作人员须经培训考核合格后方可独立操作，由实验医学科发放POCT操作资格确认。

#### 6.3 培训课程、课件与签到管理

##### 6.3.1 培训课程管理

每次培训应形成正式的培训记录，包括：
- 课程名称与课程编号
- 培训类型（入职/在职/专业/安全/合规/POCT）
- 培训日期、时间、地点
- 培训师资（内部讲师姓名/外部机构名称）
- 培训方式（面授、线上、实操、自学、混合式）
- 培训学时
- 考核方式及合格标准

##### 6.3.2 培训课件管理

1. 培训课件由培训师编制，涉及技术内容的由技术负责人审核，涉及质量管理内容的由质量负责人审核。
2. 课件统一存储于科室共享文件夹或LIMS系统的培训模块中。
3. 课件应注明版本号和编制日期，涉及SOP内容的课件应与当前有效版本保持一致。
4. SOP修订后，相关培训课件应同步更新。

##### 6.3.3 培训签到管理

1. 每次面授培训应使用《培训签到表》（SYS-FM-06-001-03），记录参加人员姓名、签名、培训时间。
2. 线上培训应保留系统登录记录或在线签到截图作为参训证明。
3. 培训签到表与培训课件一同归档保存。

#### 6.4 培训效果评估

##### 6.4.1 评估方式

| 评估方式 | 说明 | 适用场景 |
|----------|------|----------|
| 笔试/在线考核 | 理论知识测试，满分100分，合格标准80分 | 理论培训、合规培训 |
| 实操考核 | 现场操作演示，由培训师或组长按评分表评分 | 仪器操作、检验技能培训 |
| 满意度调查 | 培训满意度问卷，评价课程内容、师资水平、组织安排 | 所有培训（抽样或全员） |
| 行为观察 | 培训后一段时间内观察工作行为是否发生改变 | 重点培训项目 |
| 结果分析 | 培训后检验质量指标（如室内质控在控率、室间质评成绩）是否改善 | 针对性技能培训 |

##### 6.4.2 考核不合格处理

1. 笔试或实操考核不合格者，安排2周内补考。
2. 补考仍不合格者，须重新参加该培训课程。
3. 关键培训（生物安全培训、岗位SOP培训）未通过者，不得独立上岗操作。
4. 所有考核结果记录于个人培训档案。

#### 6.5 继续教育学分管理

##### 6.5.1 学分要求

根据省/市卫生健康委员会相关规定，各类专业技术人员年度继续教育学分要求如下：

| 人员类别 | 年度学分总要求 | I类学分最低要求 | II类学分最低要求 |
|----------|---------------|----------------|-----------------|
| 高级职称 | 不少于25分 | 不少于5分 | 不少于20分 |
| 中级职称 | 不少于25分 | 不少于5分 | 不少于20分 |
| 初级职称 | 不少于25分 | 不少于5分 | 不少于20分 |

I类学分来源：国家级或省级继续教育项目、学术会议等。II类学分来源：科内学术活动、自学、在线学习等。

##### 6.5.2 学分登记与跟踪

1. 培训管理员维护《继续教育学分登记表》（SYS-FM-06-001-04），逐人逐项登记学分获取情况。
2. 每季度汇总学分完成进度，对学分不足人员发出书面提醒。
3. 年底核查全员学分达标情况，纳入年度培训总结报告。
4. 学分未达标人员的情况报科主任，作为年度考核参考。

#### 6.6 外部培训与会议管理

1. 人员参加外部培训或学术会议须事先填写《外部培训/会议申请表》（SYS-FM-06-001-05），经所在组组长同意、科主任批准后方可参加。
2. 培训或会议结束后2周内须提交以下材料：
   - 培训/会议总结报告（SYS-FM-06-001-09）
   - 培训证书或学分证明复印件
   - 科室内转训安排（如适用）
3. 外出培训学习到的新知识、新技术、新方法，应安排向本专业组或全科进行转训分享，转训活动纳入科室培训记录。

### 第二部分：能力评估管理

#### 6.7 各项职能的能力要求定义

##### 6.7.1 能力要求框架

每个岗位或检验项目的能力要求应从以下六个维度进行定义：

| 维度 | 说明 | 示例 |
|------|------|------|
| 教育背景 | 学历和专业要求 | 医学检验技术本科及以上 |
| 专业资格 | 执业资格和职称要求 | 持有检验技师及以上资格 |
| 培训经历 | 完成的必要培训 | 已完成生物安全培训和岗位SOP培训 |
| 技术知识 | 岗位相关的理论知识 | 掌握临床化学分析原理 |
| 操作技能 | 实际操作能力 | 能独立完成全自动生化分析仪的校准和质控 |
| 工作经验 | 相关工作经验年限 | 具有1年以上检验工作经验 |

##### 6.7.2 能力要求文件

1. 技术负责人组织各专业组制定《岗位/项目能力要求表》（SYS-FM-06-001-06），每个岗位或授权项目均应有对应的能力要求描述。
2. 能力要求应与岗位说明书中的任职条件保持一致。
3. 新检验项目开展或岗位职责发生重大变化时，应及时修订对应的能力要求。
4. 能力要求表经技术负责人审核、科主任批准后生效。

#### 6.8 评估计划编排

##### 6.8.1 年度评估计划

1. 每年1月由技术负责人组织编制《年度能力评估计划》（SYS-FM-06-001-07）。
2. 评估计划应覆盖所有在岗人员和所有授权的检验项目。
3. 常规评估间隔不超过12个月（即每年至少评估一次）。
4. 计划应明确：被评估人姓名、评估项目、评估方法组合、评估人、计划评估时间。
5. 计划经科主任批准后执行。

##### 6.8.2 评估频次要求

| 评估类型 | 适用场景 | 频次要求 |
|----------|----------|----------|
| 新员工入岗评估 | 新入职人员 | 前6个月内至少2次 |
| 常规年度评估 | 在岗人员 | 每12个月至少1次 |
| 复岗评估 | 离岗超过2年 | 返岗前完成评估 |
| 加密评估 | 复杂/高风险项目 | 每6个月1次（见6.11） |
| 专项评估 | 新项目或新方法上线 | 开展前完成评估 |
| 不合格后复评 | 评估不合格人员 | 完成培训后1个月内 |

#### 6.9 评估方法

##### 6.9.1 评估方法体系

每次能力评估应至少采用以下方法中的两种及以上组合：

| 编号 | 评估方法 | 说明 | 适用场景 |
|------|----------|------|----------|
| M1 | 直接观察 | 评估人在工作现场观察被评估人执行标准操作流程，包括安全操作和设备维护操作 | 所有岗位，所有评估类型 |
| M2 | 盲样测试 | 将已知结果的样品作为常规样品送检，评估检验结果的准确性和可靠性 | 有定量或定性结果的检验项目 |
| M3 | 理论考核 | 笔试或口试，考核专业知识、SOP理解程度、质量控制知识 | 所有岗位 |
| M4 | 工作记录审查 | 回顾性审查被评估人的检验记录、质控记录、仪器维护记录、异常处理记录 | 所有岗位 |
| M5 | 检查特别提供的样品 | 使用先前检验过的样品、室间质评（EQA）材料、分割样品进行评估 | 参与室间质评的检验项目 |
| M6 | 解决问题能力评估 | 提供异常情景（如仪器故障、质控失控、危急值处理），评估判断和处置能力 | 中级及以上人员，承担值班和审核职责的人员 |

#### 6.10 新员工入岗评估

##### 6.10.1 评估安排

1. 新员工入岗后6个月内应完成至少2次能力评估。
2. 具体安排如下：
   - 第1次评估：入岗后第2-3个月，重点评估基本操作能力和安全规范的掌握程度。
   - 第2次评估：入岗后第5-6个月，全面评估独立工作能力、结果判读和异常处理能力。

##### 6.10.2 评估内容与标准

| 评估次数 | 评估重点 | 评估方法组合 | 合格标准 |
|----------|----------|-------------|----------|
| 第1次 | 基本操作规范、安全操作、SOP理解 | M1（直接观察）+ M3（理论考核） | 理论成绩不低于80分，操作无关键步骤偏差 |
| 第2次 | 独立工作能力、结果判读、异常处理 | M1 + M2（盲样测试）+ M4（记录审查）+ M6（问题解决） | 盲样结果在允许偏差范围内，操作规范，能正确处理异常情况 |

##### 6.10.3 评估结果处理

- 两次评估均合格：由技术负责人签署独立上岗确认，正式授权开展相关检验项目。
- 任一次评估不合格：由组长与技术负责人共同制定针对性培训计划（见本程序6.1-6.4），培训完成后1个月内进行复评。
- 经培训和复评仍不合格：评估该人员是否适合继续在该岗位工作，必要时调整岗位或终止试用，由科主任做出最终决定。

#### 6.11 复杂项目加密评估

对于以下复杂或高风险检验项目，在常规年度评估基础上增加评估频次至每6个月1次：

| 项目类别 | 加密评估间隔 | 评估方法重点 | 依据说明 |
|----------|-------------|-------------|----------|
| 形态学检查（血涂片分类、骨髓细胞学、尿沉渣） | 每6个月 | M2盲样（已知形态标本）+ M5（分割样品比对） | 形态辨认高度依赖主观判断，个体差异大 |
| 微生物检验（培养、鉴定、药敏） | 每6个月 | M2盲样（已知菌株）+ M1直接观察（无菌操作） | 涉及培养鉴定的主观判断和无菌技术 |
| 分子检测（PCR等） | 每6个月 | M2盲样 + M1直接观察（防污染操作） | 高灵敏度项目，污染风险高 |
| 流式细胞术（如开展） | 每6个月 | M2盲样 + M1直接观察（设门策略） | 设门分析高度依赖操作者经验 |

#### 6.12 离岗超过2年的复岗评估

##### 6.12.1 适用条件

人员因以下原因连续离岗超过2年，返岗前须重新进行能力评估：
- 长期病假
- 产假或育儿假
- 外派或借调至非检验岗位
- 停职
- 其他原因导致的长期脱离检验工作

##### 6.12.2 复岗评估流程

```
复岗申请 → 岗位条件复核 → 复岗培训安排 → 能力评估（等同新员工标准）→ 评估合格 → 重新授权 → 恢复上岗
```

具体要求：

1. 复岗人员须重新参加岗前培训，内容包括离岗期间更新的生物安全要求、修订的SOP、新增或更换的仪器设备操作等。
2. 能力评估标准等同于新员工入岗评估（6个月内至少2次），但可根据离岗时间长短和岗位变化程度适当调整。
3. 所有原有授权在复岗评估合格前处于暂停状态。
4. 评估合格后由技术负责人重新签署授权文件。

#### 6.13 评估结果记录与不合格处理

##### 6.13.1 评估结果记录

每次评估应填写《能力评估记录表》（SYS-FM-06-001-10），内容包括：
- 被评估人信息（姓名、工号、岗位、专业组）
- 评估类型（入岗/年度/复岗/加密/专项/复评）
- 评估项目与评估方法
- 评估内容详细描述及评分标准
- 评估得分或评估结论（合格/不合格）
- 评估人签名及评估日期
- 不合格项具体说明及改进建议

##### 6.13.2 评估结果判定标准

| 评估方法 | 合格标准 |
|----------|----------|
| 理论考核（M3） | 得分不低于80分（满分100分） |
| 直接观察（M1） | 无关键操作偏差，总评分不低于85分 |
| 盲样测试（M2） | 所有检测项目结果在允许偏差范围内 |
| 工作记录审查（M4） | 无重大差错，记录完整且规范 |
| 特殊样品检查（M5） | 结果与参考值或其他实验室结果一致性达到要求 |
| 问题解决评估（M6） | 能正确识别问题并采取适当处置措施 |

所有评估方法均为合格时，该次能力评估总体判定为合格；任一方法评估为不合格，则该次能力评估总体判定为不合格。

##### 6.13.3 不合格处理流程

```
评估不合格 → 书面通知被评估人及其组长 → 分析不合格原因 → 制定针对性培训计划 → 执行培训 → 复评（1个月内）→ 合格/再次不合格
```

1. 评估不合格时，评估人应书面通知被评估人，并与组长共同分析不合格原因。
2. 由组长与技术负责人共同制定针对性培训计划，明确培训内容、方式、时限和预期目标。
3. 培训完成后1个月内进行复评，复评方法和标准与原评估一致。
4. 复评合格的，恢复或确认其授权。
5. 复评仍不合格的，采取以下措施：
   - 限制或暂停其相关检验项目的操作授权。
   - 由技术负责人评估是否需要调整其工作岗位。
   - 处理结果记录于人员档案，并纳入下次管理评审议题。

##### 6.13.4 评估结果统计分析

1. 每年编制《年度能力评估汇总报告》（SYS-FM-06-001-11），内容包括：
   - 评估计划完成率（实际评估人次/计划评估人次 x 100%）
   - 一次评估合格率
   - 不合格项按评估方法和专业组的分布分析
   - 复评合格率
   - 与上年度的趋势对比分析
   - 改进建议
2. 报告提交管理评审，作为持续改进和下年度培训计划制定的依据。

#### 6.14 能力评估与质量监督的联动

能力评估体系与日常质量监督体系应建立双向联动机制：

1. 日常质量监督中发现的人员能力问题应及时触发专项能力评估。
2. 能力评估结果应反馈给质量监督员，作为确定日常监督重点的依据。
3. 以下情况应启动联动机制：

| 触发条件 | 联动动作 |
|----------|----------|
| 质量监督发现某人员多次出现同类操作偏差 | 启动该人员该项目的专项能力评估 |
| 室间质评结果不合格 | 评估相关操作人员的能力，分析是否为人员因素 |
| 患者投诉涉及检验质量问题 | 评估相关操作人员的能力 |
| 能力评估结果为不合格 | 通知质量监督员加强对该人员的日常工作监督 |
| 新员工入岗 | 质量监督员将其纳入重点监督对象 |
| 培训后行为观察异常 | 启动能力评估确认培训效果 |

### 第三部分：培训与能力评估的整合管理

#### 6.15 培训与评估的闭环管理

培训与能力评估构成相互驱动的闭环：

```
培训需求分析 → 制定培训计划 → 实施培训 → 培训效果评估
       ↑                                        ↓
  改进措施 ← 不合格原因分析 ← 能力评估 ← 评估计划制定
```

1. 能力评估中发现的不足是下一轮培训需求的核心来源。
2. 培训效果通过后续能力评估进行验证。
3. 培训管理员和技术负责人每半年联合审查培训与评估数据，识别系统性问题和改进机会。

#### 6.16 合规培训与能力评估的整合

##### 6.16.1 整合安排

为提高管理效率，合规培训可与能力评估活动适当整合：

| 合规培训主题 | 可整合的评估/培训场景 | 整合方式 |
|-------------|---------------------|----------|
| 公正性培训 | 入职培训、年度质量管理培训 | 设置独立考核模块 |
| 保密性培训 | 入职培训、信息系统操作培训 | 设置独立考核模块 |
| 患者权益培训 | 入职培训、伦理培训 | 合并课程统一考核 |
| 安全培训 | 入职培训、年度安全培训 | 可合并或分开安排 |

##### 6.16.2 整合时的记录要求

即使合规培训与其他培训整合开展，仍须保证：
1. 合规培训有独立的签到记录和考核记录。
2. 公正性培训完成后签署《公正性承诺书》。
3. 保密性培训完成后签署《保密协议》。
4. 以上记录均纳入个人人员档案。

#### 6.17 培训档案与评估档案管理

1. 每位人员应建立完整的培训与能力评估个人档案，包括：
   - 入职培训完成清单及考核记录
   - 历次在职培训签到表和考核记录
   - 历次能力评估记录表及评估结论
   - 继续教育学分登记与证明材料
   - 外部培训/会议证书及总结报告
   - 公正性承诺书、保密协议
   - 授权文件
2. 档案由培训管理员统一管理，存放于锁定档案柜或LIMS系统中。
3. 档案保存期限：在职期间全部保存，离职后保存不少于5年。
4. 档案信息的查阅须经质量负责人或科主任授权。
