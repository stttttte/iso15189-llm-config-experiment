## 判定标准说明

| 缩写 | 含义 |
|------|------|
| C | 符合（Conformity）——要求已完全实施且有效运行 |

| NA | 不适用（Not Applicable）——该条款不适用于受审区域 |

## 第 6 章 资源要求

### 6.2 人员

| 条款号 | 审核要点 | 审核方法 | 符合性判定（C/NC/PC/NA） | 客观证据 | 备注 |
|--------|----------|----------|--------------------------|----------|------|
| 6.2.1 | 实验室是否对所有影响实验室活动的人员规定了能力要求？人员岗位说明书是否涵盖学历、资质、技能和经验要求？ | 查文件 | | | |
| 6.2.2 | 实验室是否制定了年度培训计划并按计划实施？培训记录是否完整（含培训日期、内容、讲师、参加人员、考核结果）？ | 查记录 | | | |
| 6.2.3 | 实验室是否对所有检验人员进行了能力评估？评估是否覆盖理论知识和实际操作两个维度？评估频率是否至少每年一次？ | 查记录、人员访谈 | | | |
| 6.2.3 | 新员工是否经过培训和能力评估后方可独立工作？是否有明确的授权文件记录授权范围和生效日期？ | 查记录 | | | |
| 6.2.4 | 实验室是否保留了所有人员的资质档案（学历证书、执业证书、职称证书、培训记录、能力评估记录、授权记录）？档案是否及时更新？ | 查记录 | | | |
