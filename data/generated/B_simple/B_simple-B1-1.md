# 2026年度内部审核检查表

| 文件编号 | QR-IA-2026-001 |
|---|---|
| 版本号 | A/0 |
| 编制依据 | ISO 15189:2022《医学实验室——质量和能力要求》；CNAS-CL02:2023《医学实验室质量和能力认可准则》 |
| 审核周期 | 2026年1月—2026年12月 |
| 适用范围 | 本实验室质量管理体系全过程 |

---

## 使用说明

1. 审核员依据本检查表逐条实施审核，在"符合性判定"栏填写判定结果。
2. 符合性判定标记：**C**（符合）、**NC**（不符合）、**OB**（观察项）、**NA**（不适用）。
3. 发现不符合项时，在"审核记录/证据"栏记录客观证据并开具不符合报告。
4. 本检查表覆盖 ISO 15189:2022 第5章至第8章的关键要求。

## 一、通用要求（ISO 15189:2022 第5章）

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 1 | 5.1 | **法律实体**：确认实验室具有明确的法律实体地位或由具有法律实体地位的组织明确授权运营。 | 查阅法人证书、授权文件、营业执照或事业单位登记证。 | | |
| 2 | 5.2 | **公正性**：验证实验室已识别影响公正性的风险，并采取措施消除或最小化此类风险。确认实验室人员不受任何不当商业、财务或其他压力的影响。 | 查阅公正性风险评估记录；访谈关键岗位人员。 | | |
| 3 | 5.3 | **保密性**：确认实验室建立并实施患者信息保密制度，包括电子和纸质记录的访问控制。验证信息共享取得合法授权。 | 查阅保密制度文件；检查信息系统权限设置；抽查知情同意记录。 | | |

## 二、结构要求（ISO 15189:2022 第6章）

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 4 | 6.1 | **组织与管理职责**：确认实验室组织架构明确，规定实验室主任（负责人）的职责和权限。验证实验室主任具备 ISO 15189:2022 要求的资质和能力。 | 查阅组织架构图、岗位职责说明书、实验室主任任命文件及资质证明。 | | |
| 5 | 6.2 | **人员**：确认实验室建立人员能力评估程序，包括培训、考核、授权和持续能力评价。验证所有执行检验的人员经授权上岗。 | 查阅人员能力评估程序、培训计划及记录、授权清单；抽查3-5名人员的能力评估档案。 | | |
| 6 | 6.2 | **人员继续教育**：确认实验室制定并实施年度继续教育和培训计划，覆盖新方法、新设备引入及安全培训。 | 查阅2025-2026年度培训计划及执行记录；核实培训效果评价。 | | |
| 7 | 6.3 | **设施和环境条件**：确认实验室设施满足检验活动要求，环境条件（温度、湿度、照明、洁净度等）得到监控和记录。验证生物安全等级符合所开展检验项目的要求。 | 现场观察实验室布局和环境；查阅环境监测记录；核实生物安全设施配置。 | | |
| 8 | 6.3 | **区域隔离与标识**：确认不相容活动区域有效隔离，危险区域有明确标识和准入控制。 | 现场检查区域划分、警示标识、门禁或准入记录。 | | |
| 9 | 6.4 | **设备**：确认实验室建立设备管理程序，包括选型验证、校准/核查计划、维护保养和停用/报废流程。验证关键设备有唯一性标识并纳入管理台账。 | 查阅设备管理程序文件、设备台账；抽查3-5台关键设备的校准证书和维护记录。 | | |
| 10 | 6.4 | **设备校准溯源**：确认计量设备的校准溯源至国际单位制（SI）或有证参考物质（CRM），校准状态标识清晰。 | 查阅校准证书、溯源链文件；现场检查设备校准状态标识。 | | |
| 11 | 6.5 | **试剂和耗材**：确认实验室建立试剂与耗材的接收、验收、储存、使用和库存管理程序。验证关键试剂在有效期内使用，并记录批号信息。 | 查阅试剂管理程序、验收记录、库存台账；现场抽查试剂储存条件和有效期。 | | |
| 12 | 6.5 | **试剂验证/确认**：确认新批号试剂或供应商变更时执行验证或确认程序，记录完整。 | 查阅试剂批间验证记录；核实供应商评价与变更管理。 | | |
| 13 | 6.6 | **信息管理系统（LIS）**：确认实验室信息系统经验证后投入使用，定期进行系统维护和数据备份。验证系统变更经过授权和验证。 | 查阅LIS验证报告、系统变更记录、数据备份日志；核实系统权限分级管理。 | | |
| 14 | 6.6 | **数据完整性**：确认采取措施保障数据的完整性、安全性和保密性，防止未授权的数据篡改。 | 查阅数据完整性策略文件；检查审计追踪功能；测试数据修改权限。 | | |

## 三、过程要求（ISO 15189:2022 第7章）

### 3.1 检验前过程

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 15 | 7.2 | **检验申请信息**：确认检验申请单包含充分的患者识别信息、临床信息、申请的检验项目及样品信息。验证申请信息的完整性和准确性。 | 抽查10-15份检验申请单，核对信息完整性；查阅申请单审核规则。 | | |
| 16 | 7.3 | **样品采集与处理**：确认实验室编制并向采集人员提供样品采集手册或指南，规定采集要求、患者准备和样品标识规则。 | 查阅样品采集手册；访谈采集人员；现场观察采集过程。 | | |
| 17 | 7.3 | **样品运输与接收**：确认建立样品运输条件要求（温度、时限等），接收时核查样品符合接收标准，拒收不合格样品并记录。 | 查阅样品运输规程和接收标准；抽查样品接收记录和拒收记录。 | | |
| 18 | 7.3 | **样品标识与可追溯性**：确认每个样品有唯一标识，可追溯至对应患者和检验申请。验证标识在整个检验过程中保持完整。 | 现场检查样品标识方法；随机追踪3份样品从采集到报告的标识链。 | | |

### 3.2 检验过程

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 19 | 7.4 | **检验方法验证/确认**：确认实验室对采用的检验程序进行验证（标准方法）或确认（非标准方法/实验室自建方法），记录方法性能特征（精密度、正确度、测量不确定度、线性范围、干扰等）。 | 查阅方法验证/确认报告；核对性能指标是否满足预期用途。 | | |
| 20 | 7.4 | **测量不确定度**：确认实验室对定量检验项目评定测量不确定度，并定期评审其适用性。 | 查阅测量不确定度评定报告；核实评定方法和结果的合理性。 | | |
| 21 | 7.4 | **生物参考区间**：确认实验室建立或验证适用于本实验室服务人群的生物参考区间，并定期评审。 | 查阅参考区间建立/验证记录；核实信息来源和评审频次。 | | |
| 22 | 7.4 | **标准操作规程（SOP）**：确认每项检验项目配备现行有效版本的SOP，内容涵盖样品要求、检验步骤、质控规则、结果判读和报告标准。 | 抽查5-8个检验项目的SOP，核对版本受控状态和内容完整性。 | | |
| 23 | 7.5 | **质量控制（IQC）**：确认实验室制定并执行室内质量控制计划，使用适当的质控物和质控规则，及时识别和处理失控情况。 | 查阅IQC计划和记录；抽查失控处理记录；核实质控物的来源和稳定性。 | | |
| 24 | 7.5 | **室间质量评价（EQA/PT）**：确认实验室参加覆盖所有检验项目的室间质量评价计划或替代方案，分析和处理不满意结果。 | 查阅EQA/PT参加记录和结果分析报告；核实不满意结果的纠正措施。 | | |
| 25 | 7.5 | **可比性**：确认使用多台设备或多种方法检测同一检验项目时，建立结果比对方案，定期实施比对并评价一致性。 | 查阅仪器比对方案和比对结果记录；核实超出允许偏差时的处理措施。 | | |

### 3.3 检验后过程

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 26 | 7.6 | **结果审核与报告**：确认检验结果经授权人员审核后发出。报告内容符合 ISO 15189:2022 规定要求，包含必要的识别信息、结果、参考区间和解释性注释。 | 抽查10-15份检验报告，核对报告格式和内容完整性；核实结果审核签名/电子审核。 | | |
| 27 | 7.6 | **危急值通报**：确认实验室建立危急值清单和通报程序，及时向临床通报危急值，并记录通报过程和接收确认。 | 查阅危急值清单和通报程序；抽查5-10条危急值通报记录。 | | |
| 28 | 7.6 | **结果修改与撤回**：确认已发出报告的修改/更正有明确程序，修改后的报告清楚标识为修改版本并通知相关方。 | 查阅报告修改程序；抽查报告修改记录和通知记录。 | | |
| 29 | 7.7 | **样品储存与处置**：确认实验室规定检验后样品的储存条件、保留时间和处置方法，满足复检和法规要求。 | 查阅样品保留制度；现场检查样品储存条件和标识；核实生物安全废物处置。 | | |

## 四、管理体系要求（ISO 15189:2022 第8章）

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |
|:---:|---|---|---|---|:---:|
| 30 | 8.1 | **管理体系选项**：确认实验室建立、实施和维护与其活动范围相适应的管理体系（选项A或选项B），并形成文件。 | 查阅质量手册或管理体系文件；确认体系覆盖范围的充分性。 | | |
| 31 | 8.2 | **文件控制**：确认实验室建立文件控制程序，管理体系文件的编制、审核、批准、发放、修订和作废均受控。确认使用场所获得现行有效版本文件。 | 查阅文件控制程序；抽查3-5份受控文件的版本状态；现场检查是否有作废文件留存。 | | |
| 32 | 8.3 | **记录控制**：确认实验室建立记录标识、收集、索引、存取、存储、维护和安全处置的管理程序。验证记录保存期限符合法规和认可要求。 | 查阅记录管理程序；抽查记录的完整性、可读性和可追溯性。 | | |
| 33 | 8.4 | **应对风险和机遇的措施**：确认实验室识别与管理体系相关的风险和机遇，制定并实施应对措施，评价措施的有效性。 | 查阅风险评估记录和应对措施计划；核实风险矩阵或登记册。 | | |
| 34 | 8.5 | **改进**：确认实验室策划并实施持续改进活动，利用质量指标、审核结果、不符合项管理、EQA结果等信息驱动改进。 | 查阅年度质量改进计划和执行记录；核实改进成效评价。 | | |
| 35 | 8.6 | **纠正措施**：确认实验室对检出的不符合项进行根本原因分析，制定并实施纠正措施，跟踪验证措施有效性。 | 抽查3-5份不符合项报告，核实根因分析方法和纠正措施闭环。 | | |
| 36 | 8.6 | **不符合工作管理**：确认实验室建立不符合检验工作的识别和处理程序，包括已发出不符合结果的召回和通知。 | 查阅不符合工作管理程序和处理记录；核实已发出结果的召回记录。 | | |
| 37 | 8.7 | **内部审核**：确认实验室按策划的时间间隔实施内部审核，覆盖管理体系全部要求。审核员独立于受审活动，审核发现得到跟踪处理。 | 查阅年度内审计划、内审报告和不符合项跟踪记录；核实审核员资格和独立性。 | | |
| 38 | 8.8 | **管理评审**：确认实验室管理层按策划的时间间隔对管理体系进行评审，评审输入和输出符合 ISO 15189:2022 要求。 | 查阅管理评审计划、评审记录和输出决议；核实评审输入的完整性。 | | |
| 39 | 8.5 | **投诉处理**：确认实验室建立投诉接收、评价和处理程序，记录投诉及处理结果，必要时采取纠正措施。 | 查阅投诉处理程序和记录；核实投诉趋势分析。 | | |
| 40 | 8.9 | **外部提供的产品和服务（外包）**：确认实验室对外部提供的检验和校准服务进行选择评价和监控，确保外包结果的质量。 | 查阅外包管理程序、供应商评价记录和外包实验室资质。 | | |

## 五、补充专项检查

| 序号 | 检查条款 | 检查要点 | 审核方法 | 审核记录/证据 | 符合性判定 |

| 41 | 6.5 / 7.4 | **关键试剂和校准品溯源**：确认实验室使用的校准品和质控品溯源至高等级参考物质或参考方法，记录溯源链。 | 查阅校准品/质控品证书和溯源声明文件。 | | |
| 42 | 7.4 | **患者样品的检验结果可比性与可转移性**：确认当实验室变更检验系统时，对结果影响进行评估并通知用户。 | 查阅系统变更评估记录和用户通知文件。 | | |

| 44 | 6.2 | **人员健康与安全**：确认实验室建立职业健康安全程序，包括个人防护装备配置、职业暴露处理和健康监测。 | 查阅安全手册和职业暴露记录；现场检查防护装备配置和使用情况。 | | |
| 45 | 7.1 | **咨询服务**：确认实验室为用户提供检验项目选择、结果解释和临床咨询的支持，并记录重要的咨询活动。 | 查阅咨询服务记录；访谈临床医师了解实验室咨询服务满意度。 | | |
