# 表格填写说明

## 一、字段定义与填写规范

| 字段名称 | 定义 | 填写要求 | 数据格式 |
|----------|------|----------|----------|
| 设备名称 | 被校准设备的完整中文名称 | 与设备铭牌及设备登记册完全一致 | 文本 |
| 设备型号 | 制造商标注的型号标识 | 与设备铭牌一致，含字母和数字 | 文本 |
| 出厂序列号 | 设备出厂时制造商分配的唯一编号 | 与设备铭牌一致 | 文本 |
| 科室设备管理编号 | 科室内部赋予设备的唯一管理编号 | 按科室设备编号规则（如 EQ-BC-001）填写 | 文本 |
| 校准环境条件 | 校准操作时实验室内部的温度、湿度和气压 | 使用经计量检定/校准且在有效期内的温湿度计读取，同时记录温湿度计的编号和校准有效期 | 数值+单位 |
| 校准触发原因 | 启动本次校准的具体原因 | 在预设选项中勾选，选择"其他"时在说明栏写明具体原因 | 选择项 |
| 校准品批号 | 校准品外包装上标注的生产批号 | 直接抄录包装标签 | 文本 |
| 计量溯源信息 | 校准品量值与上级参考标准之间的追溯关系 | 依据校准品附带的溯源声明或产品说明书填写，写明溯源至哪一级参考物质/参考方法 | 文本 |
| 校准参数 | 本次校准针对的具体测量量 | 每个参数填写一行，如"波长正确度""吸光度线性""温度示值" | 文本 |
| 标准值 | 校准品或标准物质的赋值 | 含单位和有效数字位数 | 数值+单位 |
| 实测值 | 被校设备对校准品/标准的实际测量读数 | 有效数字位数与标准值保持一致 | 数值+单位 |
| 偏差 | 实测值与标准值之间的差异 | 偏差 = 实测值 - 标准值，或偏差百分比 = (实测值 - 标准值) / 标准值 x 100% | 数值 |
| 允许偏差限 | 判定校准合格与否的最大允许偏差 | 依据制造商说明书、计量检定规程或行业标准填写 | 数值 |
| 扩展不确定度 (U) | 以一定置信概率限定被测量值的区间半宽度 | U = k x uc，其中 uc 为合成标准不确定度，k 为包含因子 | 数值+单位 |
| 质控物验证 | 校准后使用质控物确认校准有效性的测试 | 至少使用 2 个浓度水平的质控物，校准完成后 2 小时内执行 | 数值 |

## 二、填写通用规定

1. 所有日期填写格式：YYYY年MM月DD日
2. 所有数值必须注明单位
3. 签名须使用黑色或蓝黑色签字笔手写，禁止铅笔、红色笔或印章代替手签
4. 校准不合格时，第十部分为强制填写部分，不得空白
5. 填表人在校准完成当日提交一级审核人；一级审核人在收到记录后 3 个工作日内完成审核；二级复核人在收到记录后 5 个工作日内完成复核
6. 外部校准时，校准证书原件须在收到后 5 个工作日内归入设备档案

# 记录更正规则

依据 ISO 15189:2022 第 7.7 条、CNAS-CL02:2023 第 7.7 条的要求，对记录的更正须遵循以下规则：

1. **单线划改**：在需要更正的内容上画一条水平线，使原始内容仍然清晰可辨
2. **旁注正确内容**：在原始内容旁边（上方或右侧）书写正确内容
3. **签注信息**：每处更正必须在旁边标注更正日期（格式：YYYY.MM.DD）和更正人签名
4. **严格禁止的更正方式**：
   - 禁止使用涂改液（修正液）、修正带
   - 禁止使用橡皮擦擦除
   - 禁止用胶带覆盖粘贴
   - 禁止刮擦纸面
   - 禁止以任何手段使原始记录内容不可辨认
   - 禁止撕毁或重新抄写整页表格
5. **电子系统记录更正**：在 LIS 或其他电子系统中修改记录时，系统须自动生成审计追踪 (Audit Trail)，完整记录修改前数据、修改后数据、修改时间、修改人身份
6. **保存要求**：本表纸质原件保存 6 年，保存于设备档案或科室记录存档室，保持干燥、防潮、防虫蛀；电子版本在 LIS/LIMS 中同步保存，每日自动备份
