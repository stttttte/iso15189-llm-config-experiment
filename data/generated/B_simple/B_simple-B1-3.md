**评定标准说明：**
- **C（符合）**：完全符合标准要求，有充分的客观证据支持
- **NC（不符合）**：未满足标准要求，需开具不符合项报告
- **OBS（观察项）**：虽符合但存在改进空间，或潜在不符合趋势
- **NA（不适用）**：该条款不适用于受审核区域

## 第5章 结构和治理要求

| 序号 | 条款号 | 审核内容 | 审核方法 | 评定(C/NC/OBS/NA) | 审核记录/客观证据 |
|------|--------|---------|---------|-------------------|-----------------|
| 1 | 5.1 | 实验室是否为能够承担法律责任的实体，或是法律实体中明确界定的一部分？是否有文件证明实验室的法律地位？ | 查阅法人证书、授权文件、组织机构代码等 | | |
| 2 | 5.2 | 实验室管理层是否确保实验室活动的公正性？是否识别了可能影响公正性的利益冲突风险并采取措施消除或最小化？ | 查阅公正性声明、利益冲突管理程序；访谈管理层 | | |
| 3 | 5.3 | 实验室是否定义了组织架构和管理结构？是否明确了管理层、技术运作和质量管理体系之间的关系？ | 查阅组织机构图、岗位职责描述文件 | | |
| 4 | 5.4 | 实验室是否任命了实验室主任（laboratory director），并明确其职责和权限？实验室主任是否具备必要的能力来承担其职责？ | 查阅任命文件、资质证明、职责描述 | | |
| 5 | 5.5 | 实验室是否任命了质量管理者（quality manager），确保质量管理体系的实施和维护？质量管理者是否有权直接向最高管理层报告？ | 查阅任命书、职责权限说明；访谈质量管理者 | | |
| 6 | 5.6 | 实验室是否与使用者（users）就检验服务的需求进行了有效沟通？是否建立了实验室咨询服务机制（advisory services）？ | 查阅沟通记录、用户满意度调查、咨询服务记录 | | |

## 第6章 资源要求

| 序号 | 条款号 | 审核内容 | 审核方法 | 评定(C/NC/OBS/NA) | 审核记录/客观证据 |
|------|--------|---------|---------|-------------------|-----------------|
| 7 | 6.2.1 | 实验室是否对所有影响实验室活动的人员规定了能力要求（competence requirements）？人员是否具备相应的教育、资质、培训、技术知识、技能和经验？ | 查阅岗位能力要求、人员档案、资质证书 | | |
| 8 | 6.2.2 | 实验室是否制定了人员培训计划并实施？是否对培训效果进行了评估（evaluation of effectiveness of training）？ | 查阅年度培训计划、培训记录、考核评估记录 | | |
| 9 | 6.2.3 | 实验室是否对人员进行了授权（authorization），明确其可执行的具体活动？是否定期对已授权人员的能力进行再评估？ | 查阅授权记录、能力评估/再评估记录、评估周期 | | |
| 10 | 6.3.1 | 实验室设施和环境条件是否适合所开展的实验室活动？是否对影响检验结果的环境条件进行了监控和记录？ | 现场观察实验室布局、温湿度记录；查阅环境监控程序 | | |
| 11 | 6.3.2 | 实验室是否对不同活动区域进行了有效分隔（separation），以防止交叉污染？工作流程是否合理？ | 现场观察区域划分、人流物流路径 | | |
| 12 | 6.3.3 | 实验室存储区域是否能确保样品、试剂、耗材、记录等的完整性？是否监控存储条件？ | 现场查看冰箱/冷库温度记录、存储区域管理 | | |
| 13 | 6.4.1 | 实验室是否制定了设备管理程序，包括设备的选择、验收、使用、校准、维护和报废？是否保留了设备的唯一性标识记录？ | 查阅设备管理SOP、设备台账、设备档案 | | |
| 14 | 6.4.2 | 设备是否在投入使用前经过验证/确认（verification/validation），确认其能达到必要的性能？校准计划是否按时执行？ | 查阅设备验收记录、校准证书、校准计划执行情况 | | |
| 15 | 6.4.3 | 设备出现故障或偏离规定状态时，实验室是否采取了适当措施？是否评估了设备故障对已发出结果的影响？ | 查阅设备故障记录、纠正措施记录、结果回顾评估记录 | | |
| 16 | 6.5.1 | 实验室是否建立了试剂和耗材的验收、储存和库存管理程序？是否对关键试剂和耗材在使用前进行了验证？ | 查阅试剂验收记录、库存台账、入库检验记录 | | |
| 17 | 6.5.2 | 实验室使用的信息系统（包括LIS）是否经过确认（validated），确保其功能满足要求？数据完整性是否得到保护？ | 查阅信息系统确认记录、数据备份策略、访问权限设置 | | |

## 第7章 过程要求

| 序号 | 条款号 | 审核内容 | 审核方法 | 评定(C/NC/OBS/NA) | 审核记录/客观证据 |
|------|--------|---------|---------|-------------------|-----------------|
| 18 | 7.1 | 实验室是否策划并实施了检验前过程（pre-examination processes）的程序？申请单信息是否充分、准确？ | 查阅检验申请单、样品采集手册、信息要素核查 | | |
| 19 | 7.2.1 | 实验室是否制定了原始样品（primary sample）的采集指导书？是否涵盖了患者准备、采集技术、标识和运输要求？ | 查阅样品采集指导书/手册内容的完整性 | | |
| 20 | 7.2.2 | 样品运输条件是否符合要求？是否评估了运输时间和温度对样品完整性的影响？ | 查阅样品运输程序、温度监控记录、运输时限规定 | | |
| 21 | 7.2.3 | 实验室是否建立了样品接收和处理的准则？对于不符合接收标准的样品是否有明确的处理流程和记录？ | 查阅样品接收标准、不合格样品处理记录 | | |
| 22 | 7.3.1 | 实验室是否对所使用的检验程序（examination procedures）进行了验证（verification）或确认（validation），以证明其适用于预期用途？ | 查阅方法验证/确认方案和报告（精密度、正确度、线性范围、检出限等） | | |
| 23 | 7.3.2 | 实验室是否采用文件化的程序来执行检验？检验人员是否能方便获取当前有效版本的SOP？ | 现场查看SOP可及性；查阅SOP受控清单与文件发放记录 | | |
| 24 | 7.3.3 | 实验室是否确保了检验结果的计量溯源性（metrological traceability）？是否使用了有证参考物质（certified reference materials）或适当的校准品？ | 查阅校准品溯源性证书、参考物质证书及使用记录 | | |
| 25 | 7.3.4 | 实验室是否建立了室内质量控制（internal quality control, IQC）程序？QC规则、频次和判定标准是否明确？失控处理是否有记录？ | 查阅IQC程序、Levey-Jennings图/Westgard规则应用记录、失控处理记录 | | |
| 26 | 7.3.5 | 实验室是否参加了室间质量评价（external quality assessment, EQA）/能力验证（proficiency testing, PT）计划？对于不满意结果是否进行了根本原因分析？ | 查阅EQA/PT参加证明、成绩回报单、不满意结果分析改进记录 | | |
| 27 | 7.3.6 | 当没有可用的EQA/PT计划时，实验室是否采取了替代措施来评估检验质量（如实验室间比对、临床样品复检等）？ | 查阅替代方案文件及实施记录 | | |
| 28 | 7.4.1 | 检验后过程（post-examination processes）是否包括了结果的审核、释放和报告？是否有明确的结果授权发布流程？ | 查阅结果审核流程、授权签发记录 | | |
| 29 | 7.4.2 | 检验报告是否清晰、准确、无歧义？是否包含了所有必要的信息（患者信息、检验项目、结果、参考区间、必要的解释性注释等）？ | 抽查检验报告的完整性和规范性 | | |
| 30 | 7.4.3 | 实验室是否建立了危急值（critical/alert values）报告程序？危急值是否及时通知了负责临床医师？通知记录是否完整？ | 查阅危急值管理程序、危急值报告及通知记录、时效性统计 | | |
| 31 | 7.4.4 | 实验室是否制定了已发出报告的修改和更正程序？修改后的报告是否明确标注，并保留了原始结果的可追溯性？ | 查阅报告修改记录、修改前后对比、通知临床记录 | | |
| 32 | 7.5 | 实验室是否建立了样品保留和处置的程序？样品保留期限是否满足临床和法规要求？ | 查阅样品保留政策、处置记录 | | |
| 33 | 7.6 | 实验室是否建立了确保检验结果有效性的过程？是否对检验过程的风险进行了评估并采取了相应的控制措施？ | 查阅风险评估文件、质量指标监控数据 | | |

## 第8章 管理体系要求

| 序号 | 条款号 | 审核内容 | 审核方法 | 评定(C/NC/OBS/NA) | 审核记录/客观证据 |
|------|--------|---------|---------|-------------------|-----------------|
| 34 | 8.1 | 实验室是否建立、文件化、实施和维护了质量管理体系（quality management system, QMS）？是否持续改进其有效性？ | 查阅质量手册、体系文件框架 | | |
| 35 | 8.2.1 | 实验室是否建立了文件控制程序？所有文件（内部和外部）是否经过审批、标识和受控分发？ | 查阅文件控制程序、文件受控清单、文件审批记录 | | |
| 36 | 8.2.2 | 实验室是否建立了记录控制程序？记录是否清晰可辨、易于检索？电子记录是否有适当的保护措施？ | 查阅记录管理程序、记录保存条件和期限、电子记录备份 | | |
| 37 | 8.3 | 实验室是否对不符合工作（nonconforming work）进行了识别、记录和管理？是否评估了不符合对已发出结果的影响并采取了适当措施？ | 查阅不符合工作管理程序、不符合项记录、影响评估记录 | | |
| 38 | 8.4 | 实验室是否建立了纠正措施（corrective action）程序？是否对不符合的根本原因进行了分析？纠正措施的有效性是否得到验证？ | 查阅纠正措施程序、根本原因分析记录、有效性验证记录 | | |
| 39 | 8.5.1 | 实验室是否制定了应对风险和机遇（risks and opportunities）的措施？是否考虑了对实验室活动和QMS有效性的影响？ | 查阅风险评估程序、风险登记册/矩阵、应对措施 | | |
| 40 | 8.5.2 | 实验室是否识别了改进机会并实施了持续改进活动？是否通过质量指标、审核发现、EQA结果等驱动改进？ | 查阅持续改进记录、质量指标趋势分析报告 | | |
| 41 | 8.6 | 实验室是否定期评价使用者的反馈（包括投诉和建议）？投诉处理流程是否文件化？是否对投诉进行了调查、跟踪和关闭？ | 查阅投诉管理程序、投诉台账、处理和关闭记录、满意度调查 | | |
| 42 | 8.7 | 实验室是否按照策划的时间间隔进行了内部审核（internal audit）？内审是否覆盖了管理体系的所有要素？审核员是否独立于被审核活动？ | 查阅年度内审计划、内审检查表、内审报告、审核员资质 | | |
| 43 | 8.8 | 实验室管理层是否按照策划的时间间隔进行了管理评审（management review）？管理评审输入是否涵盖了标准要求的所有方面？评审输出是否包含了改进的决定和措施？ | 查阅管理评审计划、评审报告（输入/输出）、措施跟踪记录 | | |
| 44 | 8.9.1 | 当实验室将检验或其他活动外包给外部供应商时，是否对外部供应商的能力进行了评价和监控？ | 查阅外部服务方评价程序、合格供应商清单、评价记录 | | |
| 45 | 8.9.2 | 实验室是否定期对外部服务和供应品的质量进行了评价？是否保留了评价记录？ | 查阅供应商定期评价记录、采购验收记录 | | |
