## 1 目的

本程序对本实验医学科所有用于检验活动的设备的全生命周期管理作出规定，使其从选型论证、采购、安装与验收、确认、使用、校准、期间核查、预防性维护、故障处置、维修后的恢复使用、不良事件报告，直至停用、报废与退役的每一环节均处于受控并可追溯的状态，确保设备性能持续满足对应检验程序的要求，进而保障检验结果的正确度、精密度、可追溯性和患者安全，满足 ISO 15189:2022 第 6.4 条的全部要求。

---

## 2 适用范围

本程序适用于实验医学科以下设备：

- 分析类：生化分析仪、免疫分析仪、血气分析仪、血细胞分析仪、凝血分析仪、尿液分析仪、微生物培养与鉴定系统、药敏仪、质谱分析系统、PCR 仪、数字 PCR 仪、流式细胞仪、毛细管电泳仪、血型鉴定仪
- 辅助类：离心机、水浴槽、干式加热器、恒温摇床、CO2 培养箱、厌氧工作站、生物安全柜、通风柜、高压蒸汽灭菌器、纯水机、磁力搅拌器、制冰机、超声波清洗器、各类显微镜
- 计量器具：移液器、分析天平、台式天平、温度计、湿度计、计时器
- 温控储存：药品冰箱、标本冷冻柜、超低温冰箱、液氮罐
- 信息系统硬件：LIS 服务器、工作站、前处理流水线控制柜、条码打印机、扫描枪
- POCT 设备：血糖仪、血气仪、心肌标志物便携仪、便携式凝血仪

不适用范围：

- 由医院物业或总务部门管理的公共设施
- 非用于检验活动的办公设备

## 5 职责

| 岗位 | 职责 |
|------|------|
| 实验医学科主任 | 批准设备采购计划、重要维护与报废申请；批准设备启用与恢复使用；主持涉及设备的管理评审议题 |
| 技术负责人 | 组织选型论证、运行与性能确认方案评审；对重大故障、维修后恢复使用作技术判定 |
| 质量负责人 | 监督设备管理程序合规执行；协调不良事件报告与纠正措施 |
| 设备工程师（设备管理员） | 维护设备登记册；组织验收、确认活动；编制并跟踪预防性维护与校准计划；处理故障信息；管理设备档案 |
| 生物安全员 | 对生物安全相关设备（安全柜、高压灭菌器、离心机等）的验证与年检进行监督 |
| 计量管理员 | 组织计量器具送检或内部校准；维护校准证书 |
| 各专业组组长 | 提出本组设备需求与故障信息；组织本组预防性维护与使用培训 |
| 操作人员 | 按 SOP 操作；执行日常维护；填写使用日志；发现异常立即停用并上报 |

## 6 程序内容

### 6.1 选型与采购阶段

#### 6.1.1 需求申请

各专业组提出需求，填写《设备需求申请表》（QMS-EQ-04-R01），内容包括用途、预计工作量、现有设备状况、技术指标、预算、拟采购时点。需求表经技术负责人会签、科主任批准，纳入年度采购计划。

#### 6.1.2 选型论证

技术负责人组织选型论证小组开展评审，输出《设备选型论证报告》（QMS-EQ-04-R02），评审维度至少包含：

| 维度 | 评审要点 |
|------|---------|
| 临床需求匹配 | 检验程序、样本类型、TAT、工作量弹性 |
| 技术性能 | 正确度、精密度、检测范围、灵敏度、特异性 |
| 计量溯源 | 校准品溯源链、参考物质、参考方法 |
| 信息兼容 | 接口协议、与 LIS/中间件/流水线的兼容 |
| 安装环境 | 场地、电力、给排水、温湿度、通风 |
| 售后服务 | 响应时限、备件供应、软件升级 |
| 合规性 | 注册证、许可、进口报关材料 |
| 生命周期成本 | 设备、耗材、维护、能源、人员培训 |

#### 6.1.3 合同要求

合同应明确性能验收标准、备件供应期（不少于设备使用年限）、培训时长、保修期、故障响应时间和软件升级承诺。

### 6.2 到货、安装与确认

#### 6.2.1 到货验收

到货后 3 个工作日内由设备工程师与专业组长共同开展到货验收。对照合同与装箱单核对型号、序列号、配置、随机文件与包装完好性。验收结果记入《到货验收记录》（QMS-EQ-04-R03）。

#### 6.2.2 安装确认（IQ）

内容包括环境条件符合性、电气与管路连接、机械部件完好、软件版本核对。IQ 由厂家工程师执行，设备工程师监督，结果记入《安装确认记录》（QMS-EQ-04-R04）。

#### 6.2.3 运行确认（OQ）

开机自检、样本进位、试剂仓、加样系统、检测模块、数据处理、报警、输出。OQ 结果记入《运行确认记录》（QMS-EQ-04-R05）。

#### 6.2.4 性能确认（PQ）

OQ 合格后由技术负责人组织 PQ：

| 项目 | 要点 |
|------|------|
| 正确度 | 使用溯源性参考物质或质控物验证 |
| 精密度 | 重复性、中间精密度 |
| 线性范围 | 覆盖可报告范围 |
| 分析灵敏度 | 最低检出限（如适用） |
| 分析特异性 | 干扰与交叉反应（如适用） |
| 携带污染 | 定量考察（如适用） |
| 方法比对 | 与原方法或参考方法比对（替换或并行时） |

PQ 合格后由技术负责人签发《设备启用通知》（QMS-EQ-04-R06）。

### 6.3 登记与标识

#### 6.3.1 设备登记册

设备工程师维护《设备登记册》（QMS-EQ-04-R07），每台设备至少记录：唯一标识、名称、型号、序列号、注册证、制造商、供应商、到货日期、启用日期、使用部门、责任人、当前状态、校准状态、维护状态。

#### 6.3.2 唯一标识编码规则

采用"科室-类别-流水号"形式（例：LAB-IMM-0012），标签牢固粘贴于设备显著位置。

#### 6.3.3 现场状态标识

现场设备必须保持一种状态标识在显著位置：

| 颜色 | 含义 |
|------|------|
| 绿 | 合格在用 |
| 黄 | 限用（注明限用条件） |
| 红 | 停用或维修中 |
| 蓝 | 校准中 |

### 6.4 使用与授权

1. 使用人员须获得授权，授权依据 QMS-HR-03。未授权人员不得操作。
2. 每台设备配备现行 SOP 或快速操作卡置于设备旁。
3. 每次使用填写《设备使用日志》（QMS-EQ-04-R08），记录开关机时间、样本数量、异常情况、使用人签字。
4. 设备参数修改与配置变更须经技术负责人授权，系统保留操作日志。

### 6.5 校准与计量溯源

1. 详见 QMS-EQ-05；本程序做原则性规定。
2. 对定量分析仪器按制造商和法规要求进行校准，使用具有溯源性的校准品（ISO 17511）。
3. 对计量器具按 JJF 规程送检或内部校准。
4. 校准证书与记录归档；设备粘贴校准状态标签，注明本次和下次校准日期。

### 6.6 期间核查

#### 6.6.1 执行原则

对校准间隔较长、性能波动风险较高的设备实施期间核查。核查方法包括使用参考物质、使用内控样本长期稳定性监控、使用可溯源标准器具等。

#### 6.6.2 频次

| 设备 | 核查项 | 频次 |
|------|-------|------|
| 2~8 ℃ 冰箱 | 温度、报警功能 | 每日 |
| -20 ℃ 冰柜 | 温度、报警 | 每日 |
| -80 ℃ 冰柜 | 温度、报警 | 每日（自动记录+人工复核） |
| 液氮罐 | 液位 | 每日 |
| 水浴/干式加热器 | 温度 | 每使用日 |
| 分析天平 | 砝码复核 | 每使用日 |
| 离心机 | 转速 | 每 6 个月 |
| 移液器 | 称重法精密度 | 每 6 个月 |
| 生物安全柜 | 气流与风速 | 每 12 个月（具备资质的机构） |

核查结果记入《期间核查记录表》（QMS-EQ-04-R09）。

### 6.7 预防性维护

#### 6.7.1 计划

设备工程师每年 12 月依据制造商建议与使用强度编制《预防性维护计划》（QMS-EQ-04-R10），经技术负责人批准。

#### 6.7.2 维护分级

| 级别 | 执行人 | 典型内容 |
|------|-------|---------|
| 日常 | 操作人员 | 开机自检、清洁、日质控、耗材更换 |
| 周/月/季 | 操作人员/专业组长 | 深度清洁、过滤器更换、管路维护 |
| 年度 | 厂家工程师 | 全面保养、关键部件更换、系统升级 |

每次维护完成后记入《设备维护保养记录》（QMS-EQ-04-R11）。

### 6.8 故障处置与维修

#### 6.8.1 故障信息

操作人员发现异常后应：立即停止使用，悬挂"停用/维修中"红色标识，填写《故障信息单》（QMS-EQ-04-R12），并通知设备工程师。

#### 6.8.2 维修

设备工程师组织初步判断，简单故障内部处置，复杂故障联系厂家。维修完成后记录故障原因、处置过程、更换部件、耗时。

#### 6.8.3 故障期间结果追溯

对可能受故障影响的已发结果开展追溯评估，必要时依据 QMS-QA-07 启动纠正措施并与临床沟通召回或更正报告。

#### 6.8.4 恢复使用确认

维修或重大维护（更换关键检测模块、光路、主板、样本通道等）后，按风险评估结果执行下列全部或部分确认：运行确认、性能确认、方法比对。合格后由技术负责人签发《设备恢复使用通知》（QMS-EQ-04-R13）。

### 6.9 不良事件报告

1. 设备使用过程中发生或可能导致人员伤害、检验结果错误的事件，属于设备不良事件。
2. 发生后操作人员 24 小时内上报设备工程师与质量负责人。
3. 质量负责人评估后，按《医疗器械不良事件监测和再评价管理办法》通过医院医学装备部门按法定期限向监测机构报告。
4. 不良事件记录归入设备档案并纳入管理评审。

### 6.10 停用、报废与退役

#### 6.10.1 停用情形

修复后性能仍不达标；连续重大故障无修复价值；备件停供无法维持运行；技术严重落后影响质量。

#### 6.10.2 报废流程

1. 设备工程师填写《设备报废申请》（QMS-EQ-04-R14）。
2. 技术负责人、质量负责人会签。
3. 科主任批准。
4. 报医院设备科按医院报废流程处置。

#### 6.10.3 数据清除与去污

1. 报废前对设备存储介质中的患者信息与校准数据按规定备份和清除。
2. 涉及生物危险材料的设备按生物安全要求进行彻底去污，生物安全员复核。

### 6.11 设备档案

#### 6.11.1 档案构成

每台设备一份完整档案：

- 身份信息：名称、型号、序列号、唯一标识、注册证、合格证
- 采购与合同：需求申请、论证报告、合同副本
- 验收：到货验收、IQ/OQ/PQ、启用通知
- 说明书与 SOP：现行版本
- 校准证书与记录
- 期间核查记录
- 维护计划与记录
- 故障与维修记录
- 恢复使用确认记录
- 不良事件报告与反馈
- 报废处置记录（适用时）

#### 6.11.2 保管

1. 纸质档案设备档案柜一柜一台，按唯一标识排列。
2. 电子档案存储于设备管理系统，分级访问权限。
3. 停用后档案保存不少于 10 年。

### 6.12 备件与耗材

1. 关键备件建立最低库存清单，每月复查。
2. 更换关键备件后按 6.8.4 执行恢复使用确认。
3. 配套试剂、校准品、质控品依 QMS-EN-01 及相关程序管理。

### 6.13 信息系统与设备互联

1. 所有与 LIS、中间件、流水线联网的设备，其接口配置、数据映射均须经信息系统管理员确认。
2. 软件或固件版本升级前应经风险评估；升级后须验证接口通信、数据完整性与历史数据兼容性，结果记入《接口验证记录》（QMS-EQ-04-R15）。
3. 网络设备的安全策略（访问控制、日志审计、防病毒）按 QMS-IT-01 执行。

### 6.14 关键质量指标

每年统计并纳入管理评审：预防性维护完成率；校准按期完成率；计划外停机时间占比；故障率（故障次数/运行小时数）；期间核查合格率；设备不良事件数量与类型；平均修复时间 MTTR；接口故障次数。
