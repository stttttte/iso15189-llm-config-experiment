|:---:|:---:|------|------|:---:|------|
| 1 | 7.3.1 | 实验室是否制定了检验程序的选择、验证和确认的书面规程？ | 查阅程序文件清单，确认是否覆盖检验方法的选择依据、验证/确认流程及接受准则。 | ☐符合 ☐不符合 ☐不适用 | |
| 2 | 7.3.1 | 实验室选用的检验程序是否适合其预期用途，并优先采用已发布的公认方法？ | 抽查 3-5 个检验项目，核实方法来源（国家标准、行业标准、试剂厂商说明书等），确认选择理由有文件记录。 | ☐符合 ☐不符合 ☐不适用 | |
| 3 | 7.3.2 | 实验室在引入新检验程序前，是否对其进行了方法验证或方法确认？ | 查阅近 12 个月内新开展项目的验证/确认报告，确认包含准确度、精密度、测量不确定度、线性范围、检出限等性能参数的评估。 | ☐符合 ☐不符合 ☐不适用 | |
| 4 | 7.3.2 | 方法验证/确认的结果是否经授权人员审核批准后方可用于临床检验？ | 查看验证报告的审批签字，确认审批人具备相应授权资质。 | ☐符合 ☐不符合 ☐不适用 | |
| 5 | 7.3.2 | 当已验证的方法发生变更（如试剂批号更换、仪器维修后）时，是否重新评估了方法性能？ | 抽查设备维修记录及试剂批号变更记录，核实是否有对应的再验证记录。 | ☐符合 ☐不符合 ☐不适用 | |
| 6 | 7.3.3 | 实验室是否建立了检验前过程（样本采集、运送、接收、处理）的书面程序？ | 查阅样本采集手册及样本管理 SOP，确认涵盖患者准备、采集要求、标识、运送条件及时限、接收判定标准。 | ☐符合 ☐不符合 ☐不适用 | |
| 7 | 7.3.3 | 是否对不合格样本（溶血、脂血、量不足、标识不清等）有明确的拒收/处理准则？ | 查阅不合格样本处理记录，确认有判定标准及处理流程，并通知临床的机制。 | ☐符合 ☐不符合 ☐不适用 | |
| 8 | 7.3.4 | 实验室是否对每个检验项目建立并实施了内部质量控制（IQC）程序？ | 抽查 3-5 个检验项目的质控记录，确认质控品浓度水平、频次、判定规则（如 Westgard 规则）均有文件规定且被执行。 | ☐符合 ☐不符合 ☐不适用 | |
| 9 | 7.3.4 | 质控数据是否定期进行统计分析和趋势评估？ | 查看质控图（Levey-Jennings 图）及月度/季度质控数据分析报告，确认有偏倚和趋势的监控记录。 | ☐符合 ☐不符合 ☐不适用 | |
| 10 | 7.3.4 | 当质控结果超出可接受范围时，是否有明确的纠正措施并在放行患者结果前解决？ | 抽查质控失控记录，核实失控原因分析、纠正措施及患者结果是否暂停报告直至质控恢复。 | ☐符合 ☐不符合 ☐不适用 | |
| 11 | 7.3.4 | 实验室是否参加室间质量评价（EQA）/能力验证（PT）计划，并覆盖所有可获得 EQA 方案的检验项目？ | 查阅 EQA/PT 参加登记表及覆盖率统计，确认已开展项目均已参加可获得的外部质评方案。 | ☐符合 ☐不符合 ☐不适用 | |
| 12 | 7.3.4 | EQA/PT 结果是否被及时评价？不满意结果是否进行了根因分析和纠正？ | 查阅近 4 个周期的 EQA 回报结果及不满意结果的调查处理记录。 | ☐符合 ☐不符合 ☐不适用 | |
| 13 | 7.3.4 | 对于无法获得 EQA 方案的检验项目，实验室是否采用了替代方法（如实验室间比对、回收试验、临床相关性评估）监控结果质量？ | 查阅替代质量监控方案文件及实施记录，确认监控频次和接受准则已定义。 | ☐符合 ☐不符合 ☐不适用 | |
| 14 | 7.3.5 | 实验室是否制定了检验结果审核与放行的书面规程？ | 查阅结果审核 SOP，确认规定了自动审核规则及人工审核的触发条件和审核权限。 | ☐符合 ☐不符合 ☐不适用 | |
| 15 | 7.3.5 | 检验结果在报告前是否经过授权人员的审核？ | 抽查 10 份报告，核实审核签名/电子签章及审核时间，确认由被授权人员执行。 | ☐符合 ☐不符合 ☐不适用 | |
| 16 | 7.3.5 | 是否建立了危急值报告制度，并确保在规定时间内通知临床？ | 查阅危急值清单、报告时限要求及通知记录（含接收人、通知时间、回读确认）。 | ☐符合 ☐不符合 ☐不适用 | |
| 17 | 7.3.6 | 实验室是否对检验结果进行了临床可报告范围的确定和文件化？ | 抽查 3-5 个定量检验项目，确认其可报告范围（含稀释后扩展范围）有验证数据支撑。 | ☐符合 ☐不符合 ☐不适用 | |
| 18 | 7.3.6 | 当检验结果超出分析测量范围时，是否有明确的样本复检/稀释/转介程序？ | 查阅超线性结果处理 SOP 及实际处理记录，确认操作符合规定。 | ☐符合 ☐不符合 ☐不适用 | |
| 19 | 7.3.7 | 实验室是否确保检验报告的内容完整、准确，并符合标准要求？ | 抽查 10 份检验报告，确认包含：患者信息、样本信息、检验项目、结果、参考区间、检验方法（必要时）、报告日期、审核人等要素。 | ☐符合 ☐不符合 ☐不适用 | |
| 20 | 7.3.7 | 是否对修改后的报告有明确的标识，保留原始结果并注明修改原因、修改人和修改时间？ | 查阅报告修改记录（LIS 审计追踪），确认修改可追溯。 | ☐符合 ☐不符合 ☐不适用 | |
| 21 | 7.3.7 | 参考区间（生物参考区间）是否定期审查并适用于服务人群？ | 查阅参考区间来源文件及定期审查记录，确认审查周期和依据。 | ☐符合 ☐不符合 ☐不适用 | |
| 22 | 7.3.8 | 实验室是否建立了检验后样本保留、储存和处置的书面规程？ | 查阅样本保留 SOP，确认不同样本类型的保存条件、保留时间及废弃处理流程。 | ☐符合 ☐不符合 ☐不适用 | |
| 23 | 7.3.8 | 样本的保留条件是否得到监控（如冰箱温度记录）以确保复检可靠性？ | 查看样本存储区域温度监控记录及超温报警处理记录。 | ☐符合 ☐不符合 ☐不适用 | |
