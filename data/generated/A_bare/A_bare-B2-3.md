| 1 | 实验室是否建立了检验前过程的程序文件，涵盖从临床医生下达检验申请到样品准备就绪进行检验的全过程？ | 7.3.1 | 查阅文件 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |

| 3 | 检验申请单是否包含临床相关信息，如申请医师信息、检验项目、临床诊断/用药信息、采样日期时间、样品类型等？ | 7.3.2 | 查阅申请单 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |

| 8 | 样品标识是否在采集时即完成标记，标识信息与检验申请单一致，且采用不易脱落、不易模糊的方式？ | 7.3.4 | 现场观察 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |

| 10 | 样品运输过程是否有明确的时间、温度和保存条件要求，并对运输过程进行监控和记录？ | 7.3.5 | 查阅运输规程/记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 11 | 运输包装是否符合生物安全要求，确保样品在运输过程中不会泄漏、损坏或降解？ | 7.3.5 | 现场检查 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 12 | 实验室是否建立了样品接收的验收标准，包括对样品完整性、标识正确性、容器适宜性、运输条件等的检查？ | 7.3.6 | 查阅接收程序 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 13 | 对于不符合接收标准的样品，实验室是否有明确的拒收或有条件接收的处理程序，并记录不符合情况及处置决定？ | 7.3.6 | 查阅不合格样品记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 14 | 当实验室决定对不合格样品进行检测时，是否在报告中注明样品偏差及对结果解释的潜在影响？ | 7.3.6 | 查阅报告实例 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 15 | 实验室是否对检验前过程的周转时间进行监控（如从样品采集到实验室接收的时间）？ | 7.3.7 | 查阅监控数据 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 16 | 样品在实验室内部的分发、分装和处理是否有文件化程序，确保样品完整性和可追溯性？ | 7.3.7 | 查阅程序文件/现场观察 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 17 | 实验室是否制定了样品储存条件的要求（如温度、时间限制），并对储存条件进行监控和记录？ | 7.3.7 | 查阅储存记录/温度监控记录 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
| 18 | 实验室是否有样品检测后保留的政策，规定保留时间和条件，以满足复检或追加检验的需要？ | 7.3.7 | 查阅样品保留政策 | ☐ 符合 ☐ 不符合 ☐ 不适用 | | |
