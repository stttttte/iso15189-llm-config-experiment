## 审核说明

本检查表逐条覆盖 ISO 15189:2022 第 7.3 条"检验过程"所有子条款（7.3.1 至 7.3.7），共计 20 条检查项。审核员应逐项核实，在"符合情况"栏填写"符合 / 不符合 / 部分符合 / 不适用"，并在"客观证据"栏记录所查阅的文件编号、记录编号、观察事实或访谈内容。

## 检查表

| 序号 | 对应条款 | 检查项目 | 检查方法 | 符合情况 | 客观证据 |
|:---:|:---:|---|---|:---:|---|
| 1 | 7.3.1 | 实验室是否针对每项检验程序建立并维护了经确认的标准操作规程（SOP），且 SOP 中明确规定了样本类型、检验原理、设备型号、试剂批号要求、校准物与质控物规格、操作步骤、结果判读规则及生物参考区间？ | 抽查不少于 3 项检验程序的 SOP 文件，核对上述要素是否齐全 | ☐ | |
| 2 | 7.3.1 | 实验室是否确保所有检验程序在投入常规使用前，经过方法验证或方法确认，并保留完整的验证/确认记录（含正确度、精密度、测量区间、测量不确定度、干扰因素评估等数据）？ | 查阅最近 12 个月内新开展或变更检验程序的验证/确认报告，核实数据完整性 | ☐ | |
| 3 | 7.3.1 | 当检验程序发生变更（包括试剂批号更换、设备更换、软件升级等）时，实验室是否按照文件化程序重新进行性能验证，并经授权人员审批后方可恢复使用？ | 抽查近 6 个月内试剂批号更换或设备维修后的再验证记录及审批签字 | ☐ | |
| 4 | 7.3.2 | 实验室是否建立了文件化的检验程序性能规范，包括但不限于：测量不确定度、精密度（重复性与中间精密度）、正确度（偏倚评估）、分析灵敏度、分析特异性、测量区间及临床可报告范围？ | 查阅检验程序性能规范汇总表，核对是否覆盖所有在用检验程序 | ☐ | |
| 5 | 7.3.2 | 实验室是否定期（至少每年一次）评审检验程序的性能规范，并在性能未达到预设要求时采取纠正措施？ | 查阅最近一次年度性能评审报告及相关纠正措施记录 | ☐ | |
| 6 | 7.3.3 | 实验室是否建立并实施了覆盖所有检验程序的内部质量控制（IQC）计划，包括质控物的选择依据、质控频次、质控规则（如 Westgard 规则）及失控处理流程？ | 查阅 IQC 方案文件，核对质控规则是否明确具体，失控判定标准是否量化 | ☐ | |
| 7 | 7.3.3 | 质控物的种类和浓度水平是否覆盖临床决定水平，且质控物与患者样本在同一分析批次中同条件检测？ | 抽查不少于 3 个检验程序的质控数据，核对质控物浓度水平设置及检测时序 | ☐ | |
| 8 | 7.3.3 | 实验室是否在每个工作日/每个分析批次开始前运行质控物，并在质控结果满足接受标准后方可报告患者结果？ | 现场观察当日质控运行流程，查阅质控数据审核签字记录 | ☐ | |
| 9 | 7.3.3 | 当质控结果超出预设限值（失控）时，实验室是否立即停止报告患者结果，执行失控调查，采取纠正措施，并对失控期间已发出的患者结果进行回顾性评估？ | 查阅近 6 个月内失控事件记录，核对失控处理流程是否完整执行 | ☐ | |
| 10 | 7.3.3 | 实验室是否定期（至少每月一次）对 IQC 数据进行统计分析（如绘制 Levey-Jennings 图、计算变异系数），识别趋势和偏移，并保留分析记录？ | 抽查 3 个检验程序近 3 个月的质控图及月度统计分析报告 | ☐ | |
| 11 | 7.3.4 | 实验室是否参加覆盖所有检验程序的能力验证（PT）/实验室间比对计划，且参加频次满足 CNAS-CL02:2023 及相关专业领域要求？ | 查阅能力验证参加计划表，核对覆盖率及参加频次 | ☐ | |
| 12 | 7.3.4 | 对于无法获得适宜能力验证计划的检验程序，实验室是否通过其他方式（如与其他实验室间比对、使用有证参考物质等）确定检验程序的性能可接受性？ | 查阅替代方案文件及实施记录，核对替代方式的科学合理性 | ☐ | |
| 13 | 7.3.4 | 能力验证结果不满意时，实验室是否在规定时限内完成根本原因分析、制定纠正措施并验证有效性，且相关记录完整可追溯？ | 查阅近 2 年内能力验证不满意项的整改报告及效果验证记录 | ☐ | |
| 14 | 7.3.5 | 实验室是否建立并维护了检验设备（含自动化分析系统）的校准计划，包括校准物的溯源性说明、校准周期、校准方法及校准接受标准？ | 查阅设备校准计划及校准物溯源证书，核对校准周期是否合理设定 | ☐ | |
| 15 | 7.3.5 | 校准活动是否按计划执行，校准结果是否在接受标准范围内，校准记录是否完整保留（含校准日期、操作人员、校准物批号及有效期、校准结果数据）？ | 抽查不少于 3 台设备的校准记录，逐项核对记录要素完整性 | ☐ | |
| 16 | 7.3.6 | 实验室是否针对每项定量检验程序评估了测量不确定度，并将评估方法、数据来源、计算过程及结果形成文件？ | 查阅测量不确定度评估报告，核对评估方法是否符合 GUM 或相关指南要求 | ☐ | |
| 17 | 7.3.6 | 测量不确定度的评估结果是否在需要时（如临床决策点附近结果、与其他实验室结果比较时）提供给临床使用者？ | 访谈临床医师或查阅检验报告模板，确认测量不确定度信息的传递机制 | ☐ | |
| 18 | 7.3.6 | 实验室是否定期（至少每年一次或在检验程序发生重大变更时）重新评审测量不确定度，并在必要时更新评估结果？ | 查阅测量不确定度年度评审记录及变更后的再评估记录 | ☐ | |
| 19 | 7.3.7 | 实验室是否对检验过程中可能影响结果质量的风险进行了识别和评估（如样本溶血、脂血、干扰物质等），并制定了相应的控制措施？ | 查阅检验过程风险评估记录，核对风险识别是否涵盖分析前/中/后各环节 | ☐ | |
| 20 | 7.3.7 | 实验室是否建立了检验结果可比性监控机制，对使用不同检验程序、不同设备或不同检测地点产生的同一检验程序结果进行定期比对，并在比对结果超出允许偏差时采取纠正措施？ | 查阅仪器间/方法间比对计划及实施记录，核对比对频次和判定标准 | ☐ | |
