## 摘要

本次自查采用"ISO 15189:2022 全条款映射 → 文件符合性审查 → 风险分级 → 整改计划"四步法，对实验医学科现行 35 份文件与 ISO 15189:2022 第 4—8 章全部条款进行逐条对标。识别出 3 项文件缺失和 5 项弱化覆盖（Part Ⅲ 详述），按风险级别和业务影响排序制定整改计划，确保 3 个月内（2026 年 7 月）完成体系闭环，达到 CNAS 监督评审要求。

## Part Ⅲ 风险分级与影响评估

### 3.1 风险评估矩阵

风险等级 = 发生概率 × 影响严重程度

| | 影响轻微 | 影响中等 | 影响严重 | 影响重大 |
|---|---------|---------|---------|---------|
| 概率极少 | 低 | 低 | 中 | 中—高 |
| 概率偶尔 | 低 | 中 | 高 | 极高 |
| 概率经常 | 中 | 高 | 极高 | 极高 |
| 概率频繁 | 中—高 | 极高 | 极高 | 极高 |

### 3.2 各缺口风险评估

#### UG-01：设施与环境条件程序缺失（极高风险）

- **发生概率**：经常（评审员每次必查）
- **影响程度**：重大（涉及患者安全、员工安全、检验质量）
- **综合等级**：极高
- **可能的评审结论**：严重不符合项（NC-S）
- **可能的连锁影响**：评审中断；追溯 12 个月内报告质量；严重时暂停认可资格

#### UG-02：检验后过程程序缺失（高风险）

- **发生概率**：经常（评审员重点关注）
- **影响程度**：严重（影响报告质量、样品追溯）
- **综合等级**：极高
- **可能的评审结论**：严重不符合或一般不符合项（NC-S 或 NC-G）

#### UG-03：不符合工作纠正措施程序缺失（中—高风险）

- **发生概率**：偶尔至经常（取决于评审员关注点）
- **影响程度**：中等（管理体系完整性）
- **综合等级**：高
- **可能的评审结论**：一般不符合或观察项（NC-G 或 OBS）

#### PG-01：风险管理体系化不足（中风险）

- **发生概率**：偶尔
- **影响程度**：中等
- **综合等级**：中
- **可能的评审结论**：观察项（OBS）

#### PG-02：应急准备体系化不足（中风险）

- **发生概率**：偶尔
- **影响程度**：中等（应急响应能力）
- **综合等级**：中
- **可能的评审结论**：观察项（OBS）

### 3.3 风险优先级排序

1. UG-01（极高）→ 紧急整改
2. UG-02（极高）→ 紧急整改
3. UG-03（高）→ 限期整改
4. PG-01（中）→ 规划整改
5. PG-02（中）→ 规划整改

## Part Ⅳ 整改行动方案

### 4.1 紧急整改行动（2 周内启动，8 周内完成）

#### 行动 1：编写《设施与环境条件管理程序》（SAF-001）

| 项目 | 内容 |
|------|------|
| 负责人 | 质量负责人（牵头）；生物安全员（技术支持） |
| 启动 | 2026-04-15 |
| 初稿完成 | 2026-05-05 |
| 评审与修订 | 2026-05-06 至 2026-05-20 |
| 批准发布 | 2026-05-31 |
| 试运行 | 2026-06-01 至 2026-06-30 |
| 验证闭环 | 2026-07-01 至 2026-07-10 |

**关键要求**：
- 温度控制 20—25℃，每日 8:00 和 14:00 两次记录；偏差超 ±2℃ 须启动应急处理。
- 湿度控制 30%—70%，记录同上。
- 生物安全分级清晰；BSL-2 实验室防护措施完整。
- 生物安全柜每 12 个月检定 1 次。
- 医疗废物分类存放；暂存不超过 48 小时。

#### 行动 2：编写《检验后过程管理程序》（ANA-010）

| 项目 | 内容 |
|------|------|
| 负责人 | 技术负责人（牵头）；各专业组长 |
| 启动 | 2026-04-15 |
| 初稿完成 | 2026-05-05 |
| 评审与修订 | 2026-05-06 至 2026-05-20 |
| 批准发布 | 2026-05-31 |
| 试运行 | 2026-06-01 至 2026-06-30 |
| 验证闭环 | 2026-07-01 至 2026-07-10 |

**关键要求**：
- 结果审核规则：自动审核（LIS 规则库）+ 人工复核（异常触发）。
- 报告前质量控制：IQC 数据关联、结果合理性核查、历史对比。
- 样品保留：血清/血浆 2—8℃ 不少于 7 天；全血不少于 48 小时；分子检测样品按具体项目要求。
- 样品处置：符合生物安全要求，送医疗废物处置系统。

### 4.2 限期整改行动（4 周内启动，10 周内完成）

#### 行动 3：编写《不符合项和纠正措施管理程序》（QUA-007）

| 项目 | 内容 |
|------|------|
| 负责人 | 质量负责人 |
| 启动 | 2026-04-20 |
| 初稿完成 | 2026-05-15 |
| 评审与修订 | 2026-05-16 至 2026-06-05 |
| 批准发布 | 2026-06-15 |
| 试运行 | 2026-06-16 至 2026-07-10 |
| 验证闭环 | 2026-07-11 至 2026-07-15 |

**关键要求**：
- 不符合项分类：一般（NC-G）/ 严重（NC-S）/ 关键（NC-C）。
- 即时处理：24 小时内启动；评估对患者结果的影响。
- 根因分析：5-Why、鱼骨图、帕累托图。
- 纠正措施：30 天内完成；有效性验证。
- 趋势分析：每季度 1 次；纳入管理评审。

## Part Ⅵ 监控与验证

### 6.1 周度监控

每周一组织 15 分钟整改进度会，由质量负责人主持，汇报：
- 上周完成任务
- 本周计划任务
- 遇到的问题与解决措施
- 风险预警

### 6.2 月度验证

每月最后一个工作日组织质量委员会会议，评估整改进度：
- 文件完成度
- 试运行记录完整性
- 人员培训参与率
- 整体风险下降评估
