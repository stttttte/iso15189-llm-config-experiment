# 1 目的

建立实验医学科人员培训、能力要求确定、能力评估、继续教育、授权前能力确认及相关记录管理的文件化程序，确保所有参与检验前过程、检验过程和检验后过程的人员具备与岗位相适应的教育背景、资格、培训、技术知识、技能和经验，并持续保持胜任能力。

本程序用于满足以下要求：

- ISO 15189:2022 第 6.2.1 条
- ISO 15189:2022 第 6.2.2 条
- ISO 15189:2022 第 6.2.4 条
- ISO 15189:2022 第 6.2.5 条
- 与授权相关的 ISO 15189:2022 第 6.2.3 条前置能力要求
- CNAS-CL02:2023 相应条款要求

# 4 术语和定义

## 4.1 能力 competence

人员运用知识和技能实现预期结果的本领。

## 4.2 能力评估 competence assessment

采用规定方法，对人员执行已分配任务的实际能力进行评价的活动。

## 4.3 培训 training

为使人员获得或维持履行岗位职责所需知识、技能和行为规范而实施的系统性活动。

## 4.4 再培训 retraining

当人员因岗位变更、程序变更、评估不合格、长期离岗后返岗或发生质量事件时，针对特定不足开展的补充培训。

## 4.5 授权 authorization

实验室管理层批准人员实施特定活动的正式行为，包括方法选择、方法确认、结果审核、报告发布、信息系统使用等。

## 4.6 直接观察 direct observation

评估人员在现场观察受评估人员执行常规工作、设备维护或功能检查的评估方式。

## 4.7 特别提供的样品 specially provided samples

用于评估人员能力的样品，包括既往检验样品、室间质量评价材料、分样样品、盲样等。

## 4.8 继续教育 continuing education

人员为维持和提升专业能力，参加的内部或外部教育活动。

# 5 职责

## 5.1 科主任

1. 应批准年度培训计划和年度能力评估计划。  
2. 应批准关键岗位人员上岗和相关授权前能力确认结论。  
3. 应对培训和能力评估资源配置负责。  
4. 应审阅年度培训与能力评估总结。

## 5.2 技术负责人

1. 应组织制定各岗位能力要求。  
2. 应审核培训内容的技术适宜性。  
3. 应组织实施能力评估。  
4. 应确认人员是否具备独立上岗条件。  
5. 应对新项目、新方法、新设备启用前的专项培训和能力确认负责。

## 5.3 质量负责人

1. 应监督本程序符合 ISO 15189:2022 第 6.2 条和 CNAS-CL02:2023 要求。  
2. 应审核培训与能力评估记录的完整性和可追溯性。  
3. 应将能力不足、不符合项、投诉、内部审核发现与培训需求联动。  
4. 应监督保密、公正性、生物安全相关培训的实施。

## 5.4 各专业组组长

1. 应提出本组培训需求。  
2. 应编制岗位培训实施安排。  
3. 应担任或指定培训教师、带教老师、评估人。  
4. 应对本组人员的日常工作表现进行监测并提出再培训建议。  
5. 应审核本组人员独立上岗申请。

## 5.5 培训管理员/人事档案管理员

1. 应汇总培训需求并编制培训计划。  
2. 应组织培训签到、课件归档、考核安排和记录保存。  
3. 应维护人员培训档案、能力评估档案和继续教育记录。  
4. 应按时发出培训、评估、证书到期提醒。  

## 5.6 评估人

1. 应具备相应专业能力和评估资格。  
2. 应按照批准的评估方法实施评估。  
3. 应客观记录评估结果。  
4. 应回避可能影响公正性的评估情形。

## 5.7 全体人员

1. 应按要求参加培训、考核和能力评估。  
2. 应在获得授权前不得独立从事相应活动。  
3. 应及时提交继续教育、资质证书和岗位变更相关资料。  
4. 应对个人培训和能力保持负责。

# 6 程序内容

## 6.1 总则

1. 实验医学科应建立文件化程序，管理所有对检验前过程、检验过程和检验后过程有影响的人员。  
2. 每一岗位均应明确能力要求，并形成受控文件。  
3. 每一名人员均应接受与岗位相关的培训和能力评估。  
4. 未完成规定培训、未通过能力评估或未获得授权的人员，不得独立执行相应工作。  
5. 所有培训、评估、授权前确认和继续教育活动均应形成记录并可追溯。

## 6.2 岗位能力要求的确定

## 6.2.1 能力要求内容

各岗位能力要求至少应包括以下内容：

1. 教育背景  
2. 专业资格  
3. 执业资格或岗位资质  
4. 培训要求  
5. 技术知识  
6. 操作技能  
7. 工作经验  
8. 信息系统使用能力  
9. 保密和公正性要求  
10. 生物安全要求

## 6.2.2 岗位能力要求文件

实验医学科应按岗位建立《岗位能力要求表》，至少覆盖以下岗位：

- 科主任
- 技术负责人
- 质量负责人
- 授权签字人
- 各专业组检验人员
- 采样人员
- 样品接收人员
- 报告审核人员
- LIS用户
- POCT管理与操作人员

## 6.2.3 岗位能力要求制定与审批

1. 各专业组组长应提出岗位能力要求建议。  
2. 技术负责人应审核技术内容。  
3. 质量负责人应审核合规性。  
4. 科主任应批准后实施。  
5. 岗位职责或检测项目发生变化时，应在 **10个工作日内** 修订。

## 6.3 培训分类与培训对象

## 6.3.1 入职培训

新入科人员应在上岗前完成入职培训。入职培训至少包括：

1. 实验医学科组织架构与岗位职责  
2. 质量管理体系基础  
3. 文件与记录管理  
4. 生物安全与职业暴露处置  
5. 消防与应急  
6. 公正性和保密性要求  
7. 患者信息保护  
8. LIS/HIS相关权限和信息安全要求  
9. 岗位基础SOP  
10. 仪器设备基础操作

## 6.3.2 岗位培训

岗位培训应依据人员所在专业组和承担任务实施，包括：

- 检验前过程培训
- 检验过程培训
- 检验后过程培训
- 设备操作与维护培训
- 质量控制与室间质量评价培训
- 危急值处理培训
- 异常结果处理培训

## 6.3.3 变更培训

当发生以下情况之一时，必须实施变更培训：

1. 新项目开展  
2. 新设备投入使用  
3. 新方法建立或旧方法重大修改  
4. SOP修订  
5. 信息系统升级  
6. 法律法规或认可准则更新  
7. 岗位调整  
8. 发生重大质量事件后

培训应在变更生效前完成。

## 6.3.4 再培训

出现以下情况之一，应实施再培训：

1. 能力评估不合格  
2. 内部质量控制异常与个人操作相关  
3. 室间质量评价不满意且与人员能力有关  
4. 发生报告差错、样品差错、识别错误  
5. 投诉调查显示人员执行不到位  
6. 长期离岗后返岗

## 6.3.5 继续教育

实验医学科应为所有人员提供继续教育方案。继续教育至少包括：

1. 科内业务学习  
2. 专题讲座  
3. 病例讨论  
4. 外部培训班  
5. 学术会议  
6. 线上课程  
7. 新技术新进展学习

## 6.4 培训计划的制定与实施

## 6.4.1 年度培训计划

1. 培训管理员应于每年 **12月20日前** 完成下一年度培训需求收集。  
2. 技术负责人和质量负责人应于每年 **12月31日前** 审核年度培训计划。  
3. 科主任应于每年 **1月10日前** 批准年度培训计划。  

年度培训计划至少包括：

- 培训主题
- 培训对象
- 培训时间
- 培训方式
- 培训教师
- 培训学时
- 考核方式
- 记录编号

## 6.4.2 临时培训计划

当发生 6.3.3 规定事项时，应在变更实施前制定临时培训安排，并形成记录。

## 6.4.3 培训实施要求

1. 培训应采用授课、示教、现场操作、案例分析、线上学习等方式。  
2. 培训课件、教材、签到记录、考核结果应统一归档。  
3. 涉及患者数据处理的培训必须包含保密措施和权限控制要求。  
4. 涉及生物样品操作的培训必须包含生物安全要求。

## 6.5 培训效果评价

## 6.5.1 评价方式

培训效果应采用以下一种或多种方式评价：

1. 理论考试  
2. 口头提问  
3. 现场操作考核  
4. 直接观察  
5. 工作记录审查  
6. 盲样考核  
7. 质量指标改善情况分析

## 6.5.2 合格判定

1. 理论考试合格标准应为 **80分及以上**。  
2. 操作考核应无关键步骤错误，且总评分应为 **85分及以上**。  
3. 盲样考核结果应符合该项目预设接受标准。  
4. 不符合上述任一标准者，应判定为培训效果未达要求。

## 6.5.3 不合格处理

1. 培训考核不合格者，应在 **10个工作日内** 安排补训。  
2. 补训完成后，应在 **5个工作日内** 进行补考或复评。  
3. 连续两次不合格者，专业组组长、技术负责人和质量负责人应共同分析原因，并决定是否限制上岗范围。

## 6.6 能力评估管理

## 6.6.1 评估对象

所有从事以下活动的人员均应纳入能力评估：

- 样品采集
- 样品接收与处理
- 检测项目操作
- 结果审核
- 报告发布
- 设备维护
- LIS操作
- POCT相关活动

## 6.6.2 评估时机与频次

1. 新入职人员应在入职后最初 **6个月内至少进行2次能力评估**。  
2. 在岗人员应每 **12个月** 至少进行1次能力评估。  
3. 岗位变更人员应在独立承担新岗位工作前完成能力评估。  
4. 长期离岗超过 **24个月** 的返岗人员，应按新上岗人员要求重新评估。  
5. 新项目、新方法、新设备启用前，相关人员应完成专项能力评估。  
6. 评估不合格人员完成再培训后，应在 **30日内** 复评。

## 6.6.3 评估方法

能力评估方法应至少采用以下适用方法中的两种组合：

1. 直接观察常规工作过程和程序，包括所有适用的安全实践  
2. 直接观察设备维护和功能检查的执行  
3. 监控检验和检验后过程的记录和结果  
4. 评审工作记录  
5. 评估解决问题的能力  
6. 检查特别提供的样品，如既往检验样品、室间质量评价材料、分样样品

## 6.6.4 不同岗位评估方法示例

| 岗位 | 评估方法 |
|---|---|
| 采样人员 | 直接观察、理论考核、记录审查 |
| 常规检验人员 | 直接观察、盲样、记录审查 |
| 微生物岗位 | 直接观察、特别提供的样品、问题解决能力评估 |
| 分子检测岗位 | 直接观察、盲样、污染防控专项评估 |
| 报告审核人员 | 结果审核记录审查、问题解决能力评估 |
| LIS使用人员 | 系统操作考核、保密要求考核 |

## 6.6.5 评估实施流程

1. 制定年度能力评估计划。  
2. 确定评估对象、评估项目、评估方法和评估人。  
3. 按计划实施评估。  
4. 填写《能力评估记录表》。  
5. 由技术负责人审核评估结论。  
6. 对不合格项制定纠正和培训措施。  
7. 跟踪复评结果。  
8. 评估结果归入个人档案。

## 6.6.6 新员工能力评估要求

1. 第1次评估应在入职后 **3个月内** 完成。  
2. 第2次评估应在入职后 **6个月内** 完成。  
3. 两次评估均合格后，方可申请独立上岗授权。  
4. 任一次评估不合格者，不得独立上岗。

## 6.7 特殊岗位和关键信息系统培训

## 6.7.1 信息系统使用培训

对有权限进入检验信息系统的所有人员，必须开展以下培训：

1. 账号和密码管理  
2. 权限分级使用  
3. 数据保密要求  
4. 报告查询与发布规则  
5. 电子签名规范  
6. 异常登录和数据泄露报告流程

未完成培训和考核者，不得分配LIS相应权限。

## 6.7.2 特殊技术岗位培训

涉及以下岗位的人员，应增加专项培训与评估：

- 分子检测
- 微生物鉴定与药敏
- 形态学检查
- 输血相关检测
- HIV初筛
- 产前筛查
- POCT管理

如相关法规要求持证上岗，人员必须取得有效资质后方可授权。

## 6.8 继续教育与职业发展

1. 实验医学科应制定年度继续教育安排。  
2. 所有人员每年应参加继续教育活动，具体学时或学分按【待补充：当地卫生行政部门要求】执行。  
3. 外出学习、学术会议、线上课程、院内培训均可计入继续教育，但应提供可验证证据。  
4. 参加继续教育的人员应证明继续教育效果，方式包括：
   - 学习报告
   - 转训授课
   - 理论测试
   - 技术改进应用记录

## 6.9 能力评估结果与授权联动

1. 人员获得独立上岗资格前，必须完成规定培训并通过能力评估。  
2. 授权签字人、方法确认人员、LIS高权限用户、设备关键操作人员，应在授权前完成专项能力确认。  
3. 能力评估不合格者，应暂停相应授权申请或暂停既有授权范围。  
4. 复评合格后，方可恢复申请或恢复授权。  
5. 本程序与《人员授权控制程序》联动执行。

## 6.10 能力不足和改进措施

1. 发生下列情形时，应判定存在能力风险：
   - 连续培训考核不合格
   - 能力评估不合格
   - 同类差错重复发生2次及以上
   - 室间质量评价结果不满意且与个人操作有关
   - 违反生物安全或保密规定

2. 发现能力风险后，应实施以下措施之一或组合：
   - 再培训
   - 增加监督
   - 限制工作范围
   - 暂停独立上岗
   - 暂停授权
   - 岗位调整

3. 所有措施应形成记录，并评价其有效性。

## 6.11 人员档案和记录管理

## 6.11.1 人员记录内容

每位人员的人事和能力记录至少应包括：

1. 岗位能力要求  
2. 教育背景和专业资格证明  
3. 既往工作经历或推荐材料  
4. 岗位说明书  
5. 培训和再培训记录  
6. 能力评估记录  
7. 继续教育和成就记录  
8. 不良事件/事故报告  
9. 工作表现评价  
10. 授权或许可文件  
11. 与健康状况有关的信息（如适用）

## 6.11.2 保存要求

1. 纸质记录应存放于带锁档案柜。  
2. 电子记录应设置访问权限和备份措施。  
3. 涉及个人隐私和患者信息的记录应受控保存。  
4. 人员培训与能力记录保存期限应为在职期间加离职后 **10年**。如法规或医院制度要求更长时间，应从其规定。

## 6.12 年度总结与管理评审输入

1. 培训管理员应于每年 **12月31日前** 形成年度培训总结。  
2. 技术负责人应于每年 **12月31日前** 形成年度能力评估总结。  
3. 总结内容至少应包括：
   - 计划完成率
   - 参训率
   - 考核合格率
   - 评估完成率
   - 初评合格率
   - 复评合格率
   - 主要薄弱环节
   - 改进建议

4. 上述总结应作为管理评审输入。
