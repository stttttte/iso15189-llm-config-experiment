## 一、审查总体评价

本次审查聚焦于程序文件的术语使用规范性、各章节间的逻辑一致性，以及程序在实际执行中的可行性。审查发现文件中存在与 ISO 15189:2022 术语体系不一致的用词问题，质量目标设定的合理性值得商榷，附录表单完全缺失，且质控数据保存期限与现行法规要求不完全匹配。

**总体评级**：需针对性修订，涉及术语统一替换、目标值校准和附录补充。

---

## 二、审查发现

### 发现 1：术语使用与 ISO 15189:2022 不一致——"分析过程"应为"检验过程"

| 项目 | 内容 |
|------|------|
| 严重程度 | 一般不符合 |
| 涉及条款 | ISO 15189:2022 §3（术语和定义）、§7.3 |
| 位置 | 第 6.2 条标题及正文 |

**事实描述**：

第 6.2 条标题为"分析过程中的异常处理"，正文中使用"当分析过程中发现质控不合格时"。ISO 15189:2022 已将检验全流程统一为"检验前过程（pre-examination process）""检验过程（examination process）""检验后过程（post-examination process）"三阶段术语体系，取代了旧版（ISO 15189:2012）中的"分析前""分析""分析后"表述。

在 CNAS 评审中，术语使用是否与现行标准版本一致是评审员关注的要点之一。使用旧版术语可能被认为文件未按新标准完成术语更新。

**不符合判据**：

文件术语与 ISO 15189:2022 的术语体系不一致，不满足标准对统一使用规范术语的隐含要求。

**纠正措施建议**：

1. 在全文中执行以下术语替换：

   | 当前用词 | 应替换为 | 理由 |
   |---------|---------|------|
   | 分析过程 | 检验过程 | ISO 15189:2022 统一术语 |
   | 分析前 | 检验前 | ISO 15189:2022 统一术语 |
   | 分析中 | 检验中 | ISO 15189:2022 统一术语 |
   | 开始分析前 | 开始检验前 | 保持术语一致 |
   | 后续检验（原文无误） | 保留 | 不需修改 |

2. 第 6.2 条标题修改为"检验过程中的异常处理"。
3. 第 3.4.2 条"每日质控（分析前）"修改为"每日质控（检验前）"，"分析中"修改为"检验中"。

### 发现 2：术语使用与 ISO 15189:2022 不一致——"标本"应为"样本"

| 项目 | 内容 |
|------|------|
| 严重程度 | 一般不符合 |
| 涉及条款 | ISO 15189:2022 §3（术语和定义） |
| 位置 | 第 6.2 条"标本重新检验" |

**事实描述**：

第 6.2 条第 2 步操作写为"标本重新检验：对之前分析的标本进行重新检验"。ISO 15189:2022 中使用的规范术语为"样本（sample）"，而"标本（specimen）"虽然在临床实践中仍被广泛使用，但在标准文件层面应统一使用 ISO 15189:2022 的官方术语。

需要注意的是，ISO 15189:2022 中"sample"对应的中文翻译在 CNAS 官方文件中为"样品"或"样本"，各实验室可根据 CNAS-CL02 应用说明的推荐选择统一用词，但不应使用"标本"。

**不符合判据**：

文件中的术语选用与 ISO 15189:2022 术语体系不符。

**纠正措施建议**：

1. 在全文搜索"标本"一词，逐处替换为"样本"或"样品"（建议选择"样本"，与国内检验医学界主流用法一致）。
2. 同步核查第 3.4.2 条中是否存在"标本"用词并替换。
3. 在新增的"术语和定义"章节中，明确注释："本程序中的'样本'即临床实践中的'标本'，与 ISO 15189:2022 中 sample 含义一致。"

### 发现 3：质量目标设定的合理性与可行性存疑

| 项目 | 内容 |
|------|------|
| 严重程度 | 观察项 |
| 涉及条款 | ISO 15189:2022 §8.8.2（质量指标） |
| 位置 | 第 9 条"质量目标" |

**事实描述**：

第 9 条设定了三项质量目标：

| 目标 | 设定值 | 问题分析 |
|------|-------|---------|
| 质控合格率 | 大于等于 99%（月度评估） | 过高。以每日 3 个浓度水平、每月约 60-90 次质控计算，99% 意味着每月最多允许 1 次失控。实际运行中，单纯统计概率下 Westgard 规则的假拒绝率约 3-5%，即使操作完全正常也可能达不到 99%。建议参考实验室实际基线数据设定。 |
| 异常处理及时率 | 100%（当日处理完成） | 部分复杂异常（如需要联系厂商、等待新批号试剂等）可能无法在当日完成全部调查和纠正。100% 作为目标值过于绝对。 |
| 样本重测率 | 小于等于 2% | 需明确计算口径。"由于质控不合格而重测"与"由于样本本身原因重测"应区分，否则指标失去监控意义。 |

**建议**：

1. 根据实验室近 12 个月的实际质控数据建立基线，参考基线设定切实可行的目标值。例如：
   - 质控合格率：建议设定为大于等于 95%（月度），如持续达标可逐年上调。
   - 异常处理及时率：建议改为"质控失控事件的初始响应率 100%（当日启动调查），纠正措施完成率大于等于 90%（5 个工作日内）"。
   - 样本重测率：明确计算公式中"重测"的定义范围。

2. 增加目标值的评审机制：每年在管理评审中对质量目标的适宜性进行评审，必要时调整。

3. 补充达标/未达标时的处理规定：当某月目标未达标时，应触发什么层级的分析和报告。

### 发现 4：附录表单完全缺失

| 项目 | 内容 |
|------|------|
| 严重程度 | 一般不符合 |
| 涉及条款 | ISO 15189:2022 §7.3.2、§8.3.2 |
| 位置 | 附录部分 |

**事实描述**：

文件末尾的"附录：质控表单"处标注"此处应该包括 Levey-Jennings 图的绘制模板、质控原始记录表等，但本版本未提供"。

程序文件应当配套完整的记录表单，这是"程序可操作性"的关键组成部分。ISO 15189:2022 §7.3.2 要求记录质控数据，§8.3.2 要求受控文件完整可用。缺少配套表单意味着执行人员没有标准化的记录工具，可能导致各专业组自行设计格式不一的记录方式，影响数据可比性和可追溯性。

**不符合判据**：

程序文件配套的记录表单缺失，不满足标准对文件完整性和可操作性的要求。

**纠正措施建议**：

1. 至少应补充以下配套表单：

   | 表单编号（建议） | 表单名称 | 用途说明 |
   |---------------|---------|---------|
   | ANA-FR-003-01 | 内部质量控制原始记录表 | 记录每日质控检测的原始数据（项目、批号、浓度水平、测定值、判定结果） |
   | ANA-FR-003-02 | Levey-Jennings 控制图模板 | 绘制质控物逐日检测值的趋势图，标注均值线和 1s/2s/3s 控制限 |
   | ANA-FR-003-03 | 质控失控处理记录表 | 记录失控事件的发现时间、触发规则、原因调查、纠正措施、重新验证结果 |
   | ANA-FR-003-04 | 质控物使用登记表 | 记录质控物的启用/停用日期、批号、有效期、储存条件 |
   | ANA-FR-003-05 | 月度质控数据汇总与评价表 | 汇总当月各项目质控数据，计算 CV%、偏倚、合格率等统计量 |
   | ANA-FR-003-06 | 测量不确定度评估记录表 | 记录各检验项目的不确定度评估过程和结果（配合新增的不确定度章节） |

2. 每个表单应附带填写说明和示例。
3. 表单编号应纳入实验室文件编号体系，并在文件控制清单中登记。

### 发现 5：质控数据保存期限需要核实

| 项目 | 内容 |
|------|------|
| 严重程度 | 观察项 |
| 涉及条款 | ISO 15189:2022 §8.4（记录控制）；相关法规 |
| 位置 | 第 8.2 条"数据保存" |

**事实描述**：

第 8.2 条规定"质控数据应保存大于等于 6 年"。该要求需要与以下规定进行校核：

- CNAS-CL02:2023 应用说明中关于记录保存期限的要求
- 《医疗机构临床实验室管理办法》中对检验记录保存期限的规定
- 实验室自身文件管理程序中关于记录保存期限的统一规定

如果实验室文件管理程序规定记录保存期为"不少于 3 年"而本程序规定"不少于 6 年"，则存在内部文件间的不一致。

**建议**：

1. 核查实验室文件管理程序（如记录控制程序）中关于记录保存期限的统一规定，确保本程序的保存期限与之一致或取较长者。
2. 如果 6 年确为实验室统一要求，在本程序中注明依据（如"依据本实验室记录控制程序 SOP-XX-XX 第 X 条规定"），增强可追溯性。
3. 考虑区分电子记录和纸质记录的保存方式和期限要求。

### 发现 6：Levey-Jennings 拼写错误

| 项目 | 内容 |
|------|------|
| 严重程度 | 观察项 |
| 涉及条款 | 文件准确性 |
| 位置 | 第 3.4.1 条标题 |

**事实描述**：

第 3.4.1 条正文中将"Levey-Jennings"误写为"Levy-Jennings"（遗漏字母 e）。虽然这是一个拼写错误，但在正式的质量管理体系文件中，专有名词的准确性直接影响文件的专业性和可信度。

**建议**：

1. 将全文中的"Levy-Jennings"统一修正为"Levey-Jennings"。
2. 在文件修订审核流程中增加术语和专有名词校对环节。

## 三、综合改进建议

除上述逐项发现外，从整体可执行性角度提出以下综合建议：

### 建议 1：增加"职责"章节

当前文件缺少明确的职责分工章节。建议在"适用范围"之后增加"职责"章节，明确：

| 角色 | 职责 |
|------|------|
| 科室负责人 | 审批质控方案、审查月度质控汇总报告 |
| 质量负责人 | 监督质控制度执行情况，组织质控数据评审 |
| 各专业组组长 | 负责本组质控方案制定和日常质控执行的监督 |
| 检验技术人员 | 执行日常质控操作、记录质控数据、发现失控及时报告 |

### 建议 2：增加定性和半定量项目的质控说明

第 2 条明确排除了定性和半定量项目，但 ISO 15189:2022 §7.3.2 要求对所有检验项目实施质量控制。建议在本程序中增加说明，指出定性和半定量项目的质控要求在何处另行规定（如引用其他 SOP 编号），避免评审员质疑条款覆盖缺口。

### 建议 3：增加风险管理与质控策略的关联说明

ISO 15189:2022 强调基于风险的思维。建议在程序中增加一段说明，阐述质控方案的制定如何考虑风险因素（如高风险项目增加质控频率、关键检验方法采用更严格的质控规则等），将质控活动与实验室风险管理体系建立联系。

| 统计项目 | 数量 |
|---------|------|
| 严重不符合 | 0 |
| 一般不符合 | 3 |
| 观察项 | 3 |

**结论**：本文件在术语规范性和执行可行性方面存在 3 项一般不符合和 3 项观察项。术语使用不一致的问题虽然修改成本低，但在 CNAS 评审中容易被指出，且反映文件未按新版标准完成全面更新。附录表单的缺失直接影响程序的可操作性。质量目标的合理性问题则需结合实验室实际数据进行论证和调整。建议在修订时将术语统一、表单补充、目标值校准三项工作同步完成，经科室内部评审后发布新版本。
