# 2026年度内部审核检查表

**依据标准：** ISO 15189:2022 医学实验室——质量和能力的要求  
**审核周期：** 2026年1月—2026年12月  
**编制日期：** 2026-04-04  
**适用范围：** 第5章—第8章（通用要求、资源要求、过程要求、管理体系要求）

---

## 使用说明

- **审核判定：** C＝符合（Conformity）；NC＝不符合（Nonconformity）；OFI＝改进机会（Opportunity for Improvement）；NA＝不适用
- 审核员应记录客观证据，对NC项开具不符合报告

---

## 第5章 通用要求

| 序号 | 条款 | 检查内容 | 判定 | 客观证据/备注 |
|------|------|----------|------|---------------|
| 1 | 5.1 | 确认实验室具备独立法律实体资格或在母体组织中有明确界定的法律地位，并提供相应的法律文件。 | | |
| 2 | 5.2 | 验证实验室已识别可能影响公正性的利益冲突，并记录消除或最小化此类风险的措施。 | | |
| 3 | 5.3 | 核实实验室对所有在其固定设施、远程场所或相关临时/移动设施中开展的活动承担责任。 | | |
| 4 | 5.4 | 确认实验室的活动符合适用的法律法规要求，并保留合规性评审的证据。 | | |
| 5 | 5.5 | 审查实验室与用户之间的协议或合同，确认其明确规定了检验要求、样品接收条件及报告方式。 | | |
| 6 | 5.6 | 核实实验室在委托外部实验室（转介实验室）时，已评价其能力并保留评价记录。 | | |
| 7 | 5.7 | 确认实验室已建立外部服务和供应品的采购程序，包括对试剂、耗材和服务的验收准则及评价记录。 | | |

## 第6章 资源要求

| 序号 | 条款 | 检查内容 | 判定 | 客观证据/备注 |
|------|------|----------|------|---------------|
| 8 | 6.1 | 确认实验室配备了充足的人员，并对其能力（包括教育、培训、技能和经验）进行了评估和记录。 | | |
| 9 | 6.2.a | 核实人员能力评估包括初始培训后的授权以及定期再评估，并保留授权记录。 | | |
| 10 | 6.2.b | 审查人员继续教育和专业发展计划，确认其与实验室工作范围相关并已实施。 | | |
| 11 | 6.3 | 确认实验室环境条件（温度、湿度、照明、洁净度、电磁干扰等）已被识别、监控并记录，且符合检验要求。 | | |
| 12 | 6.4.a | 核实设备（包括仪器、软件、测量系统和辅助设备）在投入使用前已进行确认或验证。 | | |
| 13 | 6.4.b | 审查设备维护计划，确认预防性维护按计划执行，并保留维护记录。 | | |
| 14 | 6.4.c | 确认设备校准和计量溯源性符合要求，校准结果可溯源至SI单位或有证参考物质，并保留校准证书。 | | |
| 15 | 6.4.d | 核实设备故障或偏离规定条件时的处置程序，包括对先前检验结果的影响评估。 | | |
| 16 | 6.5 | 确认试剂和耗材在使用前已按规定进行了验收检验或验证，并保留批次验收记录。 | | |

## 第7章 过程要求

| 序号 | 条款 | 检查内容 | 判定 | 客观证据/备注 |
|------|------|----------|------|---------------|
| 17 | 7.1 | 审查检验前过程，确认医嘱/申请单信息充分（包括患者识别、临床信息、检验项目和样品要求）。 | | |
| 18 | 7.2 | 核实样品采集、运送和接收程序，确认样品标识唯一且可追溯，并规定了样品拒收准则。 | | |
| 19 | 7.3.a | 确认检验方法已经过确认（validation）或验证（verification），并保留确认/验证记录，包括性能特征评估。 | | |
| 20 | 7.3.b | 核实测量不确定度已针对每个检验程序进行了评估，并在需要时向用户提供。 | | |
| 21 | 7.3.c | 审查生物参考区间或临床决定值的建立或验证过程，确认其适用于服务人群。 | | |
| 22 | 7.4 | 确认检验过程中实施了质量控制程序，包括内部质量控制（IQC）策略，并记录了质控数据的评审和措施。 | | |
| 23 | 7.5 | 核实实验室参加了外部质量评价（EQA）/能力验证（PT）计划，并对结果进行了评审和跟踪纠正。 | | |
| 24 | 7.6 | 确认结果在报告前经过授权人员的审核和批准，包括自动化审核规则的确认。 | | |
| 25 | 7.7.a | 审查检验报告格式和内容，确认包含患者识别、检验项目、结果、参考区间、实验室识别及报告日期等必要信息。 | | |
| 26 | 7.7.b | 核实危急值（critical values/alert values）通知程序，确认已及时通知临床并记录通知时间和接收人。 | | |
| 27 | 7.7.c | 确认报告修改/更正程序已建立，修改后的报告清晰标识原报告和修改内容，并保留审批记录。 | | |
| 28 | 7.8 | 核实检验后样品的保留、储存和处置程序符合规定的时限和条件要求。 | | |
| 29 | 7.9 | 确认实验室已建立结果发布和信息管理系统（LIS）的数据完整性控制，包括数据备份和访问权限管理。 | | |
| 30 | 7.10 | 审查实验室信息系统的确认和变更管理程序，确认系统变更前后均进行了验证。 | | |

## 第8章 管理体系要求

| 序号 | 条款 | 检查内容 | 判定 | 客观证据/备注 |
|------|------|----------|------|---------------|
| 31 | 8.1 | 确认实验室已建立、文件化、实施并维护了管理体系，且管理体系覆盖ISO 15189:2022的全部要求。 | | |
| 32 | 8.2 | 核实质量方针和质量目标已由管理层制定并传达至全体人员，且质量目标可测量并定期评审。 | | |
| 33 | 8.3 | 审查文件控制程序，确认文件的批准、发布、分发、变更和废止均受控，且人员可获取现行有效版本。 | | |
| 34 | 8.4 | 核实记录管理程序，确认记录的标识、存储、保护、检索、保留期限和处置符合规定要求。 | | |
| 35 | 8.5 | 确认实验室已对风险和机遇进行了系统性识别和评估，制定了应对措施并评审其有效性（基于风险的思维）。 | | |
| 36 | 8.6 | 审查纠正措施程序，确认不符合工作发生时已进行根本原因分析，并实施了纠正措施且验证了有效性。 | | |
| 37 | 8.7 | 核实持续改进活动的策划和实施，包括利用质量指标、审核结果、管理评审输出和用户反馈推动改进。 | | |
| 38 | 8.8 | 确认实验室已建立并实施投诉处理程序，对投诉进行调查、评价和处置，并向投诉人反馈结果。 | | |
| 39 | 8.9.a | 审查内部审核程序和年度计划，确认内审覆盖管理体系的所有要素，且审核员独立于被审核活动。 | | |
| 40 | 8.9.b | 核实内部审核发现的不符合项已制定纠正措施计划，并在规定时间内完成整改和验证。 | | |
| 41 | 8.10 | 确认管理评审按策划的时间间隔进行，评审输入包括：审核结果、质量指标、投诉、纠正措施状态、外部评价结果、风险评估等。 | | |
| 42 | 8.10 | 核实管理评审输出包括改进决定、资源需求和体系变更，且已分配责任和完成时限。 | | |
