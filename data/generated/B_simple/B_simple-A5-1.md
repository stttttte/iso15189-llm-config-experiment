# 设备校准记录表（FRM-EQP-002）

## 1 目的与范围

依据 ISO 15189:2022 第 6.5.1 条"设备校准与计量溯源性"要求，记录实验室检测设备每次校准的完整信息，确保测量结果可溯源至国际单位制（SI）或有证参考物质，并为校准周期管理和质量审核提供客观证据。本表适用于所有纳入校准计划的检测设备与测量系统。

## 2 记录内容

### 2.1 基本信息

| 字段 | 填写说明 |
|------|----------|
| 设备名称 | 填写设备全称 |
| 设备编号 | 填写实验室内部唯一编号 |
| 规格型号 | 填写制造商标注的型号 |
| 制造商 | 填写设备制造商名称 |
| 安装位置 | 填写设备所在科室或房间编号 |
| 校准日期 | 填写本次校准执行日期（YYYY-MM-DD） |
| 校准机构/人员 | 填写执行校准的机构名称或内部校准人员姓名及资质编号 |
| 环境条件 | 记录校准时的温度、湿度等环境参数 |

### 2.2 校准依据/方法

依据 ISO 15189:2022 第 6.5.2 条，记录本次校准所采用的标准方法或规程。填写内容包括：

- **校准规程编号及名称**：引用国家计量检定规程（如 JJG/JJF 编号）、国际标准方法或制造商推荐的校准程序。
- **校准标准物质/标准器**：记录使用的有证标准物质（CRM）编号、标准器名称及其证书编号和有效期。
- **溯源链描述**：简要说明测量结果如何溯源至 SI 单位或国际公认的参考标准，确保符合 ISO 15189:2022 第 6.5.1 条计量溯源性要求。

### 2.3 校准结果与测量不确定度

依据 ISO 15189:2022 第 6.5.3 条，记录校准数据及其不确定度评定结果。

| 校准点 | 标准值 | 测量值 | 偏差 | 扩展不确定度（U） | 包含因子（k） | 判定结果 |
|--------|--------|--------|------|---------------------|----------------|----------|
| | | | | | | 合格/不合格 |

填写要求：

- 在每个校准点记录标准值与实际测量值，计算偏差。
- 按照 GUM（测量不确定度表示指南）或 CNAS-GL006 评定测量不确定度，填写扩展不确定度（U）及对应的包含因子（k），通常取 k=2，置信水平约 95%。
- 将校准偏差与扩展不确定度之和与允许误差限进行比较，判定该校准点是否合格。

### 2.4 下次校准日期

依据 ISO 15189:2022 第 6.5.1 条关于校准间隔管理的要求，明确记录下次校准到期日。

- **下次校准日期**：根据校准计划中规定的校准周期，填写下次校准截止日期（YYYY-MM-DD）。
- **校准周期确定依据**：注明周期设定依据（如制造商建议、历史校准数据趋势分析、风险评估结果或计量法规要求）。
- 将下次校准日期录入设备管理系统，设置到期前提醒，防止超期使用未经校准的设备。

### 2.5 审核人/批准人

依据 ISO 15189:2022 第 6.5.4 条及第 8.2 条对记录审核与授权的要求，设置双重签署确认。

| 角色 | 姓名 | 签名 | 日期 |
|------|------|------|------|
| 校准执行人 | | | |
| 审核人 | | | |
| 批准人 | | | |

填写要求：

- **审核人**：由具备计量专业能力的质量管理人员对校准数据、不确定度评定及判定结论进行技术审核，确认记录完整性与准确性，签署姓名及日期。
- **批准人**：由实验室技术负责人或其授权代表批准校准结果，确认设备可继续投入使用或需采取纠正措施，签署姓名及日期。
- 审核人与批准人不得为同一人，且不得由校准执行人兼任。

## 3 记录管理

按照 ISO 15189:2022 第 8.2 条要求管理本记录。保留校准记录原件及校准证书副本，保存期限不少于设备使用寿命结束后两个校准周期。确保记录可追溯、防篡改，并在内部审核或外部评审时可及时检索。
